### Task 1.2: 카드 풀 설계 (Card Pool) ⏰ 2026-03-05

#### 📌 목표

**50-100장의 플레이 가능한 카드 설계**

#### 🎴 카드 분류 체계

```
Card Pool (50-100 cards)
├─ 공격 카드 (Attack)        [10-15장]
│  ├─ 기본 공격 (1-2 cost)
│  ├─ 강력한 공격 (2-3 cost)
│  └─ 극대 공격 (3-4 cost)
│
├─ 방어 카드 (Skill/Defend)   [10-15장]
│  ├─ 기본 방어 (1-2 cost)
│  ├─ 강력한 방어 (2-3 cost)
│  └─ 버프 (1-3 cost)
│
├─ 특수 카드 (Powers)         [5-10장]
│  ├─ 지속 효과
│  ├─ 캐릭터 특수 능력
│  └─ 극대 연계 기술
│
└─ 상황 카드 (Situational)    [10-20장]
   ├─ 회복 (Rare)
   ├─ 드로우 (Uncommon)
   └─ 코스트 감소 (Uncommon)
```

#### 📋 카드 설계 템플릿

```yaml
# 공격 카드 예시
Card:
  Name: "꿈의 검격"                    # 타로 + 꿈 테마
  Cost: 2                              # 에너지 비용
  Type: Attack                         # 카드 유형
  Rarity: Uncommon                     # 등급
  Effect: "적에게 피해 8 입히기"
  Upgraded: "적에게 피해 10 입히기"
  SynergyWith: ["힘의 기억", "전투열"]  # 상성 좋은 카드
  UpgradeCount: 0                      # 강화 횟수
  Obtained: ["Shop", "Reward"]         # 획득 방식

# 방어 카드 예시
Card:
  Name: "꿈의 방패"
  Cost: 1
  Type: Skill
  Rarity: Common
  Effect: "방어도 5 획득, 취약 1 해제"
  Upgraded: "방어도 7 획득, 취약 1 해제"
  SynergyWith: ["보호의 기억"]
  UpgradeCount: 0
  Obtained: ["StartingDeck", "Shop"]

# 특수 카드 예시
Card:
  Name: "기억의 소용돌이"
  Cost: 3
  Type: Power
  Rarity: Rare
  Effect: "매턴 카드 1장 드로우. 턴 시작 시 에너지 1 소비"
  Upgraded: "매턴 카드 2장 드로우. 턴 시작 시 에너지 1 소비"
  SynergyWith: ["드로우 효과 전체"]
  Duration: "영구"
  Obtained: ["Reward", "Rare Shop"]
```

#### 🎯 진행 순서

```
1️⃣ 전투 시스템 결정 받음 (3/3)
   ├─ ATB인 경우: 비용 0-2 중심
   └─ Turn인 경우: 비용 1-3 중심

2️⃣ 카드 비용 기준표 작성 (3/4)
   ├─ 공격 카드: 비용별 피해량 기준
   ├─ 방어 카드: 비용별 방어도 기준
   └─ 특수 카드: 비용별 효과 기준

3️⃣ 기본 카드 목록 작성 (3/4)
   ├─ 각 카테고리별 10-15장
   ├─ 중복 회피
   └─ 밸런스 검증

4️⃣ 카드별 상세 설계 (3/5)
   ├─ 효과 정확히 명시
   ├─ 강화(Upgrade) 정의
   ├─ 시너지 맵핑
   └─ 획득 방식 결정

5️⃣ 최종 밸런싱 (3/5)
   ├─ 파워 레벨 검증
   ├─ 초보자/고급 용이성
   └─ 메타게임 분석
```

#### 📊 카드 분배 예시 (총 60장)

```
공격 카드 (14장)
├─ 비용 1 (3장): 기본공격, 후려치기, 찌르기
├─ 비용 2 (6장): 열화 공격, 회전 공격, 연속 공격 등
├─ 비용 3 (4장): 강력한 공격, 극대 공격, 시너지 공격
└─ 비용 4 (1장): 궁극 기술

방어 카드 (14장)
├─ 비용 1 (5장): 방어, 회피, 준비
├─ 비용 2 (6장): 강화 방어, 버프 추가
├─ 비용 3 (2장): 극대 방어, 복합 버프
└─ 비용 4 (1장): 궁극 방어

특수 카드 (12장)
├─ 파워 (3장): 지속 효과 (에너지, 드로우, 회복)
├─ 드로우 (4장): 카드 추가 (드로우, 서치)
├─ 코스트 감소 (3장): 비용 절감 효과
└─ 기타 (2장): 정화, 타겟팅 등

상황 카드 (20장)
├─ 회복 (3장): 체력 회복 (Rare)
├─ 버프 (8장): 힘, 민첩, 보호 증가
├─ 디버프 제거 (5장): 취약, 약화 해제
├─ 조건부 카드 (4장): 적 상태에 따른 효과
```

### Task 1.3: 몬스터 & 보스 설계 ⏰ 2026-03-07

#### 📌 목표

**50-70종 몬스터 + 3명 보스 설계**

#### 🗂️ 몬스터 분류

```
몬스터 풀 (총 50-70종)
├─ Act 1 (튜토리얼)           [15-20종]
│  ├─ 일반 몬스터 (8-10)
│  ├─ 엘리트 (3-4)
│  └─ 보스 (1)
│
├─ Act 2 (주 본편)             [20-25종]
│  ├─ 일반 몬스터 (12-15)
│  ├─ 엘리트 (5-7)
│  └─ 보스 (1)
│
└─ Act 3 (최종)                [15-20종]
   ├─ 일반 몬스터 (8-10)
   ├─ 엘리트 (4-6)
   ├─ 보스 (1)
   └─ 특수 보스 변형 (1)
```

#### 🎯 진행 순서

```
1️⃣ 카드 풀 설계 받음 (3/5)
   └─ 적의 드롭 테이블 결정

2️⃣ 몬스터 이름 & 테마 결정 (3/5)
   ├─ 각 Act별 테마 선택
   │  ├─ Act 1: 꿈의 시작 (청순한 이미지)
   │  ├─ Act 2: 꿈의 왜곡 (이질적 이미지)
   │  └─ Act 3: 공백의 식식 (어두운 이미지)
   └─ 몬스터 이름 목록 (50-70개)

3️⃣ 몬스터 기본 스펙 정의 (3/6)
   ├─ 각 단계별 체력 곡선
   ├─ 공격력 증가 추이
   ├─ 엘리트 배수 (× 1.5 ~ 2.0)
   └─ 보스 배수 (× 3.0 ~ 5.0)

4️⃣ 몬스터 행동 패턴 설계 (3/6)
   ├─ 일반 몬스터: 2-3 턴 패턴
   ├─ 엘리트: 3-4 턴 패턴
   └─ 보스: 멀티페이즈 + 특수 능력

5️⃣ 보스 스토리 연결 (3/7)
   ├─ 각 보스의 배경
   ├─ 전투 전 대사
   ├─ 전투 중 반응
   └─ 격파 후 스토리 진행

6️⃣ 아트 레퍼런스 작성 (3/7)
   ├─ 각 몬스터 시각적 설명
   ├─ 컨셉 아트 방향
   └─ 아트팀과 협력
```
