# 🔧 Scripts - 유틸리티 및 자동화 스크립트

이 폴더는 **프로젝트 자동화 및 유틸리티 스크립트**를 포함합니다.

## 📂 구조

### 실행 스크립트 (.sh)
- **run_atlas.sh** - Atlas PM 에이전트 실행
- **run_post.sh** - 블로그 포스팅 자동화 실행
- (향후) run_game.sh - 게임 빌드 및 배포

### Python 스크립트
- (향후) build_game.py - 게임 빌드 자동화
- (향후) sync_content.py - 콘텐츠 동기화

## 사용법

```bash
# Atlas 실행
./scripts/run_atlas.sh

# 블로그 포스팅 실행
./scripts/run_post.sh
```
