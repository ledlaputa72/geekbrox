# 운영/사업 현황 (OPS_STATE)

> 이 파일은 운영및사업팀장과 모든 서브에이전트가 **작업 시작 전 반드시 읽어야** 하는 핵심 상태 파일이다.
> 작업 후 변경사항을 반드시 업데이트한다.

## 현재 우선순위 업무
| 우선순위 | 업무 | 상태 |
|---------|------|------|
| 1 | 게임 시장 리서치 | ✅ 완료 |
| 2 | 블로그 SEO/트렌드 리서치 | 🔄 진행 중 |
| 3 | 게임 유료화 모델 설계 | ⏳ 컨셉 확정 후 |
| 4 | QA 프로세스 수립 | ⏳ 개발 착수 후 |

## 리서치 현황

### 게임 시장 분석 (완료)
- **장르:** 로그라이크 + 방치형 + 덱빌딩 하이브리드
- **결론:** 🟡 High Risk, High Reward 블루오션
- **경쟁작:** Slay the Spire, Loop Hero 분석 완료
- **상세:** `research/RESEARCH_LOG.md` 참고

### 블로그 트렌드 리서치 (진행 중)
- Tistory 블로그 성장 전략 수립 중
- 애니메이션 시즌별 트렌드 분석
- Google Trends 키워드 최적화

## 유료화 현황
- 기본 방향: 모바일 무료 + 광고/IAP
- 구체적 설계: 게임 컨셉 확정 후
- 상세: `monetization/MODEL.md` 참고

## QA 현황
- 현재 단계: 개발 전 (QA 해당 없음)
- 프로세스 준비만 완료
- 상세: `qa/QA_LOG.md` 참고
