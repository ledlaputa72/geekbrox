## 1️⃣ 카드 획득 곡선 분석

### 게임 진행 구간별 분석

#### 초기 구간 (Lv 1~30, 0~10시간)

**무료 경로:**
```
스토리 보상:      25장 (5장/시간)
일일 뽑기:        5장 (0.5장/시간)
총 획득:         30장 (3장/시간)

기대값: 초보자 10시간 내 30장 획득 → 게임 시작 족
```

**유료 경로:**
```
초기 전환율: 15% (초기 구매욕)
평균 구매: 5,000원 (약 $4)
예상 유료 카드: 10장 추가

총 획득: 40장
```

**결론:** 초보자가 빠르게 강해지는 느낌 제공 ✅

#### 중기 구간 (Lv 30~60, 10~30시간)

**무료 경로:**
```
일일 퀘스트:      0.5장/시간 (5시간/일 기준)
주간 보상:        2장/일
월 이벤트:        5장
총 월 획득:       70~80장

기대값: 20시간에 약 40~50장 (2.5장/시간)
```

**유료 경로:**
```
전환율: 25% (게임 몰입도 증가)
평균 월 지출: $15 (약 18,000원)
예상 유료 카드: 30장

총 획득: 80~100장
```

**재화 수급:**
```
골드 월 획득: 100,000
골드 월 소비: 80,000 (카드 강화)
수급 비율: 1.25 (125% - 약간 잉여) ✅
```

**결론:** 무료 플레이어도 따라갈 수 있는 속도 ✅

#### 엔드게임 (Lv 60+, 30시간+)

**무료 경로:**
```
일일 뽑기: 0.2장/시간 (유지 정도)
월 획득: 20~30장 (수집용)
성능 향상: 미미 (이미 강함)
```

**유료 경로:**
```
전환율: 35% (수집 욕구)
평균 월 지출: $30 (약 36,000원)
예상 유료 카드: 50장

월 획득: 70~80장
```

**장기 플레이 유도:**
```
시즌 카드 (한정): 분기별 10~15장
이벤트 카드: 월별 5~10장
수집 컨텐츠: 200/200 → 300/300 로드맵
```

**결론:** 장기 플레이 유도하면서 수익화 ✅
