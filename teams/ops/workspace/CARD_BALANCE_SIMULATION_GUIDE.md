## 📌 개요

Game팀이 생성한 **200개 카드 데이터**를 기반으로 전투 밸런스와 재화 밸런스를 시뮬레이션합니다.

### 목표
- ✅ 4가지 카드 타입(공격/방어/강화/약화)의 전투 균형성 검증
- ✅ 초보자 → 중급자 → 고급자 진행 곡선 분석
- ✅ 유료/무료 카드 수익성 검증
- ✅ 월 수익화 목표(약 $1M) 달성 가능성 평가
- ✅ 조정 필요 카드 목록 도출

### 기한
- 데이터 인수: 2026-03-09 (Game팀 Step 4 완료)
- 시뮬레이션: 2026-03-09 ~ 2026-03-12 (3-4일)
- 최종 보고: 2026-03-12

## 🎮 Part 1: 전투 밸런스 시뮬레이션

### 1.1 분석 대상

**4가지 카드 타입:**
- ATTACK (42장): 직접 데미지
- SKILL (60장): 방어/리액션
- POWER (50장): 지속 버프
- CURSE (48장): 적 약화

**3가지 리액션 태그 (SKILL 내부):**
- GUARD: 즉시 블록 (오토 가능)
- PARRY: 패링 윈도우 (수동만)
- DODGE: 회피 윈도우 (오토 45~65%)

### 1.2 핵심 분석 항목

#### A. 비용 효율성 분석

**목표:** 각 카드의 비용 대비 효과가 균형적인가?

**메트릭:**
```
비용 효율도 = 카드 효과값 / 비용

예시:
  ATK_001 (검의 에이스): 6데미지 / 1비용 = 6.0 효율도
  ATK_005 (세계): 20데미지 / 4비용 = 5.0 효율도
  
목표: 모든 카드 효율도 4.5~6.5 범위 내
```

**분석 방법:**
1. 각 타입별 비용대별 효율도 평균 계산
2. 극단값(4.0 이하 또는 7.0 이상) 추출
3. 조정 필요 카드 목록화

**산출물:**
```
비용 효율도 분석표
├─ ATTACK 타입: 비용별 평균 효율도
├─ SKILL 타입: 비용별 평균 효율도
├─ POWER 타입: 비용별 평균 효율도
├─ CURSE 타입: 비용별 평균 효율도
└─ 극단값 카드 목록 (조정 대상)
```

#### B. 타입 간 밸런스 분석

**목표:** 공격형이 방어형을 이기는가? (의도한 관계 검증)

**기본 게임 루프:**
```
공격형(ATTACK) → 방어형(SKILL)이 블록 → 강화형(POWER)로 버프 → 약화형(CURSE)로 약화
```

**메트릭:**
```
승률 분석:
  공격형 vs 방어형 = ?:? (목표: 50:50)
  공격형 vs 약화형 = ?:? (목표: 60:40 공격 유리)
  방어형 vs 강화형 = ?:? (목표: 40:60 강화 유리)

시뮬레이션 방법:
  100회 × (공격 3장 조합 vs 방어 3장 조합) = 총 10,000회 전투
```

**분석 범위:**
- 저비용 조합 (1~2비용): 초보자 플레이
- 중비용 조합 (2~3비용): 중급자 플레이
- 고비용 조합 (3~5비용): 고급자 플레이

**산출물:**
```
타입 간 밸런스 분석표
├─ 공격형 vs 방어형 승률 (저/중/고 비용대별)
├─ 공격형 vs 강화형 승률
├─ 공격형 vs 약화형 승률
├─ 타입별 최강 조합 (상위 5개)
└─ 타입별 최약 조합 (하위 5개)
```

#### C. 리액션 시스템 검증

**목표:** PARRY(패링)와 DODGE(회피)의 난이도 대비 보상이 균형적인가?

**메트릭:**
```
리액션 가치도 = 리액션 성공 시 보상 / 난이도

GUARD (난이도: 매우 낮, 즉시 발동)
  보상: 즉시 블록
  기대값: 블록량 × 100% (무조건 발동)

PARRY (난이도: 매우 높, 0.4초 윈도우)
  보상: 에너지 +1~3, 부가효과
  기대값: 보상 × 80% (플레이어 패링 성공률 가정)

DODGE (난이도: 높음, 1.4초 윈도우)
  보상: 에너지 +1, 부가효과
  기대값: 보상 × 55% (오토 성공률 평균)
```

**분석 방법:**
1. 각 리액션 카드의 기대값 계산
2. 난이도 대비 보상 비교
3. 플레이어가 선택하고 싶은 카드인지 검증

**산출물:**
```
리액션 시스템 검증 보고서
├─ GUARD 카드별 기대값 분석
├─ PARRY 카드별 기대값 분석
├─ DODGE 카드별 기대값 분석
├─ 패링/회피 선택 유인도 분석
└─ 조정 필요 카드 (너무 약한/강한 리액션)
```

#### D. 리스크 카드 검증

**목표:** 고위험 카드(자해/부작용)의 위험도가 보상과 균형적인가?

**분석 대상:**
- ATK-RCK (5장): 고데미지 + 자해
- CRS-RSK (12장): 저비용 약화 + 플레이어 부작용

**메트릭:**
```
리스크 비율 = 플레이어 부작용 / 적 약화

예시:
  CUR_취약: 적 피해 +100%, 플레이어도 +25% (1턴)
  → 리스크 비율 = 25% / 100% = 0.25 (25% 리스크)

목표: 0.2 ~ 0.4 범위 (20~40% 리스크)
```

**산출물:**
```
리스크 카드 검증 보고서
├─ ATK-RCK 카드별 위험도 분석
├─ CRS-RSK 카드별 위험도 분석
├─ 플레이어가 사용하고 싶은 강도 검증
└─ 너무 안전/위험한 카드 목록
```

## 💰 Part 2: 재화 밸런스 시뮬레이션

### 2.1 분석 대상

**게임 진행 구간:**
- 초기: Lv 1~30 (첫 10시간)
- 중기: Lv 30~60 (다음 20시간)
- 엔드게임: Lv 60+ (장기 플레이)

**재화 종류:**
- 골드: 일반 인게임 화폐
- 다이아: 프리미엄 재화
- 카드: 덱 강화 자원

### 2.2 핵심 분석 항목

#### A. 카드 획득 곡선

**목표:** 각 게임 구간에서 플레이어가 얼마나 빨리 카드를 얻는가?

**메트릭:**
```
카드 획득 속도 (장/시간):
  초기 (무료): 예상 1.5장/시간
  초기 (유료): 예상 3.0장/시간
  중기 (무료): 예상 0.8장/시간
  중기 (유료): 예상 2.0장/시간
  엔드게임: 예상 0.3장/시간 (수집용)

목표: 초보자가 충분히 강해지되, 너무 빨리 끝나지 않기
```

**분석 방법:**
1. 스토리 보상 카드 수
2. 일일 무료 뽑기 기대값
3. 스토리 보상 타이밍
4. 유료 카드 가격대별 기대 전환율

**산출물:**
```
카드 획득 곡선 분석
├─ 초기 구간 (0~10시간)
│  ├─ 무료 경로 카드 획득 그래프
│  └─ 유료 경로 카드 획득 그래프
├─ 중기 구간 (10~30시간)
│  ├─ 재화 부족/과잉 분석
│  └─ 유료 유도 지점 검증
├─ 엔드게임 (30시간+)
│  ├─ 수집 컨텐츠 만족도
│  └─ 월 카드 획득 기대값
└─ 플레이어 만족도 예상 (초보/중급/고급)
```

#### B. 재화 흐름 분석

**목표:** 각 구간에서 골드와 다이아의 수급 밸런스는?

**메트릭:**
```
재화 수급 비율 = 월 획득 재화 / 월 필요 재화

예시 (초기 구간):
  초보자 월 골드 획득: 10,000
  초보자 월 골드 소비: 8,000 (카드 강화)
  수급 비율 = 10,000 / 8,000 = 1.25 (25% 잉여)

목표: 0.9 ~ 1.2 범위 (약간 부족 ~ 약간 잉여)
```

**분석 방법:**
1. 일일 골드 드롭 (던전, 전투, �est)
2. 일일 골드 소비 (강화, 카드 변환 등)
3. 일일 다이아 드롭 (미니게임, 보상)
4. 일일 다이아 소비 (프리미엄 뽑기)

**산출물:**
```
재화 흐름 분석 보고서
├─ 초기 구간 재화 수급 그래프
├─ 중기 구간 재화 수급 그래프
├─ 엔드게임 재화 수급 그래프
├─ 구간별 재화 부족/과잉 판정
└─ 조정 필요 재화 배분
```

#### C. 유료 vs 무료 카드 성능 분석

**목표:** 유료 카드가 무료 카드보다 얼마나 더 강한가?

**메트릭:**
```
성능 지수 = 유료 카드 평균 효과값 / 무료 카드 평균 효과값

목표: 1.2 ~ 1.5배 (20~50% 성능 우위)

과도한 경우:
  - 1.5배 이상: "게임이 너무 P2W" → 조정 필요
  - 1.0배 미만: "유료 구매 유인 없음" → 조정 필요
```

**분석 방법:**
1. 유료 카드(SPECIAL/LEGENDARY) 평균 효과값
2. 무료 카드(COMMON) 평균 효과값
3. 비용 대비 효율도 비교
4. 실제 전투에서의 승률 차이

**산출물:**
```
유료 vs 무료 성능 분석
├─ 카드 등급별 평균 효과값
├─ 비용 효율도 비교
├─ 전투 승률 차이 (무료 덱 vs 유료 덱)
├─ 유료 카드 매력도 평가
└─ P2W 우려 여부 판정
```

#### D. 수익화 시뮬레이션

**목표:** 200개 카드로 월 $1M 수익 달성 가능한가?

**메트릭:**
```
월 수익 = DAU × 전환율 × 평균 ARPU

가정값:
  DAU: 1,000 (초기)
  전환율: 30% (게임 유료 사용 비율)
  평균 ARPU: $33 (월 평균 지출)
  → 월 수익 = 1,000 × 0.3 × $33 = $10,000

목표: $1,000,000 → 필요 DAU 약 100,000
```

**분석 방법:**
1. 카드 티어별 판매가 책정
2. 초기/중기/엔드게임 구간별 구매 예상
3. 시즌 카드(한정) 판매 피크 예상
4. 월별 수익 곡선 시뮬레이션

**산출물:**
```
수익화 시뮬레이션 보고서
├─ Tier 1 (프리미엄) 월 수익 기대값
├─ Tier 2 (선택) 월 수익 기대값
├─ Tier 3 (기본) 월 수익 기대값
├─ DAU별 월 수익 추정 ($1k ~ $1M)
├─ 초기 6개월 수익 곡선
└─ $1M 달성 난이도 평가
```
