> 세계적으로 수천만 판매고를 올리며 현대 액션 RPG 전투 설계의 표준이 됨.

## 1. 소울류 전투 시스템 개요

### 1.1 공통 철학

소울류 전투를 관통하는 핵심 설계 철학:

```
1. 모든 공격에는 예고(tell)가 있다
2. 플레이어의 실수는 반드시 결과를 낳는다
3. 성공은 학습의 결과이지, 스펙의 결과가 아니다
4. 위험과 보상은 항상 비례한다
```

### 1.2 시리즈별 전투 스타일 진화

| 작품 | 핵심 메카닉 | 방어 중심 | 속도감 |
|------|----------|---------|--------|
| Dark Souls 1~3 | 스태미나 + 롤 회피 | 방패 > 패링 | 느림~중간 |
| Bloodborne | 스태미나 폐지 + 반격 | 회피 > 방패 | 빠름 |
| Sekiro | 자세 게이지 + 튕겨내기 | 패링 = 핵심 | 매우 빠름 |
| Elden Ring | 위 3작의 혼합 | 모두 선택 가능 | 중간~빠름 |

## 2. 핵심 메카닉 1: 스태미나 시스템 (Dark Souls)

### 2.1 스태미나의 역할

```
[스태미나 바]
████████████░░ 85%

소모 행동: 공격, 방패 막기, 롤 회피, 달리기, 패링
회복: 가만히 서 있거나 천천히 걸을 때 (방패 내리면 더 빠름)
```

### 2.2 스태미나 기반 전투의 재미

스태미나는 단순한 자원이 아닌 **전술적 의사결정의 연속**을 만들어냄:

```
전투 중 판단:
├── 공격 한 번 더 할 스태미나가 있는가?
├── 지금 회피할 여유가 있는가?
├── 방패로 막으면 스태미나가 부족해진다 → 언제 내릴 것인가?
└── 스태미나 관리 실패 → 방패 깨짐(Guard Break) or 무방비 노출
```

**핵심 긴장감 요소**: 스태미나 0에서 적이 공격하면 → 완전한 위기 상황.

### 2.3 Guard Break (방패 관통)

적이 연속 공격 시 플레이어 스태미나 소진 → Guard Break:
```
스태미나 0에서 방패 막기 → Guard Break 연출 → 무방비 피격
→ 극한의 긴장감 (스태미나 관리 중요성 체감)
```

---

## 3. 핵심 메카닉 2: 패링 & 반격 (Parry & Riposte)

### 3.1 다크소울 패링의 구조

다크소울 패링은 고위험/고보상의 완벽한 예시:

```
[패링 성공 순서]
1. 적 공격 직전 (파라이 시작)에 패링 버튼
2. 성공: 적이 비틀거림(Staggered)
3. 즉시 반격(Riposte) 가능: 2~4배 데미지 + 처형 모션
4. 실패: 아무 효과 없거나 패링 중 피격 (역대 리스크)
```

| 타이밍 | 결과 |
|--------|------|
| 너무 빠름 | 패링 없이 공격 맞음 |
| 완벽한 타이밍 | Parry 성공 → Riposte 기회 |
| 너무 늦음 | 방패로 막거나 그냥 맞음 |

### 3.2 패링의 게임 디자인적 의미

패링은 **선택적** 시스템. 안 써도 클리어 가능하지만:
- 패링 마스터 = 어려운 적도 쉽게 처리
- 패링 전혀 안 함 = 어렵지만 회피로 가능

→ 숙련도에 비례하는 보상 시스템. 게임이 스스로 학습을 장려함.

## 4. 핵심 메카닉 3: 자세 게이지 (Sekiro Posture System)

Sekiro는 소울류 전투의 가장 큰 혁신 중 하나.

### 4.1 자세 게이지 개념

```
[Sekiro 전투 상태 표시]
HP:    ████████░░ 80%
자세:  ████████░░ 80%  ← 두 번째 HP바처럼 동작
```

- **자세 게이지**: 공격 받거나 튕겨내기(Deflect) 당할 때 충전
- **자세 파쇄(Posture Break)**: 게이지 100% → 비틀거림 → **처형(Deathblow)** 가능

### 4.2 Sekiro의 패러다임 전환

기존 소울류: **HP를 0으로 만들어라** → Sekiro: **자세를 파쇄하라**

```
두 가지 승리 조건 (보스에 따라 다름):
방법 1: HP를 0으로 만들기 (일반 데미지)
방법 2: 자세를 파쇄 후 처형 반복 (처형 = HP 1회분 제거)

→ 보스는 대부분 방법 2가 핵심 공략
```

### 4.3 튕겨내기(Deflect) — Sekiro의 패링

다크소울 패링보다 **훨씬 넓은 윈도우**, 하지만 다른 보상:

```
튕겨내기 성공 시:
└── 데미지 0 (완벽 방어)
└── 적 자세 게이지 +대량
└── 내 자세 게이지 +소량 (완전 무료가 아님!)

연속 튕겨내기 리듬 유지:
└── 적 자세 급속 충전 → 빠른 처형 기회
```

**Sekiro의 핵심 재미**: 튕겨내기 리듬을 유지하며 자세를 깎는 두뇌와 손의 협력.

### 4.4 스태미나 폐지의 효과

Sekiro는 스태미나 바를 완전히 제거:
- 이동/공격/방어 모두 무제한 → 더 빠르고 공격적인 전투
- 대신 자세 게이지로 무분별한 방어를 처벌
- 결과: "공격이 최선의 방어" 라는 전투 리듬 형성

## 5. 핵심 메카닉 4: 위험 알림 (危 — Peril System)

Sekiro의 유명한 시스템. 특정 공격 직전 화면에 빨간 한자 **"危"(위험)** 표시:

```
일반 공격: 튕겨내기 가능
危 표시 공격: 튕겨내기 불가 → 점프 or 회피만 유효
```

```
危가 표시되는 공격 유형:
├── 하단 쓸기 공격 → 점프로 회피
├── 잡기 공격 → 점프 or 뒤로 회피
└── 꿰뚫기 공격 → 옆으로 회피
```

**게임 디자인 의의**: 예고 없이 패링 불가 공격이 오면 불공평하게 느낌. 危 표시로 "이건 다르게 대처하라"는 정보를 명확히 전달. Expedition 33의 "관통 공격" 에 직접 영향을 준 시스템.

## 6. 핵심 메카닉 5: 혈귀 시스템 (Bloodborne — Rally)

### 6.1 Rally (반격 회복)

Bloodborne은 패링 대신 **Rally** 시스템:

```
적에게 피해를 받음 → HP 일부가 주황색(회복 가능 상태)
    ↓
일정 시간 내 공격하면 → 주황색 HP 회복
    ↓
피해 받아도 공격으로 즉시 만회 가능!
```

**효과**: 방어적 플레이가 아닌 **공격적 플레이를 장려**. "맞으면 더 공격해서 회복"이라는 역발상.

## 7. 공격 예고(Tell) 시스템 — 소울류 보편 원칙

소울류의 가장 중요한 설계 원칙 중 하나.

### 7.1 예고의 3단계

```
1단계 — 자세 변화 (1~2초 전)
    몬스터가 공격 자세로 이동/전환
    플레이어: "공격 온다"

2단계 — 모션 시작 (0.5~1초 전)
    실제 공격 애니메이션 시작
    플레이어: "어떤 공격인지 파악"

3단계 — 임팩트 직전 (0~0.3초 전)
    가장 위험한 순간
    플레이어: "패링/회피 타이밍"
```

### 7.2 예고 설계의 원칙

- **모든 공격은 예고 가능해야 함** (숨겨진 공격 = 불공평)
- **예고 → 반응 → 결과** 의 인과관계가 명확해야 함
- **학습 후에는 예측 가능** → 숙련도의 보상
- **예고가 너무 뻔하면 재미 없음** → 페이크(위장) 공격 삽입

### 7.3 페이크 공격 설계

보스가 실제로 공격하지 않고 예비 동작만 하는 "페이크":
```
보스: 공격 자세 진입 → [페이크] → 실제 다른 방향 공격
플레이어: 일찍 회피 → 허공에 회피 → 반격 당함
→ 한 번 속으면 → 다음엔 더 신중하게 예고를 읽음
```

## 9. 소울류 전투의 재미 심리학

### 9.1 "죽음 = 학습" 사이클

```
도전 → 실패 → 분석 → 재도전 → 성공 → 성취감
   ↑_____________________________________________↓
```

이 사이클이 소울류를 중독성 있게 만드는 핵심. 성공이 외부 도움(아이템, 레벨업)이 아닌 **플레이어 본인의 성장**에서 나옴.

### 9.2 정보의 비대칭성

적은 플레이어보다 강함 → 단, **패턴을 익히면 쉽게 이길 수 있음**
이 비대칭성이 학습 동기를 만듦.

## 10. Dream Collector 적용 가능성 평가

| 소울류 메카닉 | 적용 가능성 | 모바일 변환 방식 |
|-------------|-----------|--------------|
| 자세 게이지 (Sekiro) | ★★★★★ 최고 | 그대로 도입, UI만 모바일화 |
| 攻격 예고 시스템 | ★★★★★ 최고 | 시각 큐 강화 (3단계 글로우) |
| 패링/튕겨내기 | ★★★★☆ 높음 | 탭 타이밍으로 변환, 윈도우 확장 |
| 危 표시 (관통 공격) | ★★★★★ 최고 | "방어 불가!" 텍스트로 직접 적용 |
| Rally (반격 회복) | ★★★☆☆ 중간 | 패링 성공 시 소량 HP 회복으로 변형 |
| 스태미나 | ★★☆☆☆ 낮음 | 에너지 시스템이 이미 대체 역할 |
| 페이크 공격 | ★★★★☆ 높음 | 보스 전용 페이크 패턴 설계 |

**종합 평가**: 소울류의 **"자세 게이지 + 위험 예고 + 패링"** 삼위일체가 Dream Collector에 가장 직접적으로 적용 가능한 시스템. 특히 모바일에서 시각적 명확성이 더 중요하므로, 소울류의 명확한 예고 시스템이 더 빛날 수 있음.
