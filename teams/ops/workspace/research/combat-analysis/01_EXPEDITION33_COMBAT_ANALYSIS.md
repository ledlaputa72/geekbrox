## 2. 핵심 전투 시스템: Reactive Turn-Based

### 2.1 개념 정의

Expedition 33의 전투는 기존 턴제와 실시간 액션을 결합한 **"반응형 턴제(Reactive Turn-Based)"** 라고 공식 명명됨. 핵심 혁신은 다음 한 문장으로 요약됨:

> **"적의 턴에도 플레이어는 쉬지 않는다."**

기존 턴제 RPG에서 적의 공격은 그냥 맞거나, 기껏해야 방어 선택을 했음. Expedition 33에서는 모든 적 공격에 대해 플레이어가 **실시간으로 패링/회피/점프**를 실행하며, 이것이 전투의 핵심 재미가 됨.

### 2.2 플레이어 턴 구조

```
[플레이어 턴]
├── 근접 공격: AP 획득 (Ability Point)
├── 원거리 공격: 자유 에임 (3인칭 슈터 방식)
├── 스킬: AP 소모 + QTE (퀵 타임 이벤트)로 효과 강화
└── 아이템 사용
```

**AP 시스템**: 에너지처럼 쌓이고 소모하는 자원. 근접 공격으로 획득, 스킬로 소모.
QTE 성공 시 스킬 효과 추가 강화 — 조준 정밀도가 보상으로 연결됨.

### 2.3 적의 턴 구조 (핵심 혁신)

```
[적의 공격 시퀀스]

예고 애니메이션 (시각/청각 큐)
    ↓
플레이어 실시간 선택:
├── ★ 패링 (Parry): 좁은 윈도우, 최고 보상
├── ★ 완벽한 회피 (Perfect Dodge): 패링과 같은 타이밍, 연습용
├── ★ 회피 (Dodge): 넓은 윈도우, 낮은 보상
├── ★ 점프 (Jump): 특정 공격 전용
└── ❌ 아무것도 안 함: 풀 데미지
```

## 3. 패링 시스템 (Parry) — 핵심 분석

### 3.1 패링의 보상 구조

패링은 단순히 "안 맞기"가 아님. **적극적 공격 도구**로 설계됨:

| 패링 결과 | 보상 |
|---------|------|
| 성공 (일반) | 피해 완전 무효 + AP 획득 |
| 체인 패링 완성 | 마지막 공격에 강력한 카운터 |
| 완벽한 회피 (동일 타이밍) | AP 획득 (Dodger Picto 장착 시) |
| 실패 | 풀 데미지 |

**핵심 철학**: 패링 = 리스크(좁은 윈도우) × 리워드(반격) 교환. 잘하는 플레이어는 더 빠르게 이길 수 있음.

### 3.2 타이밍 학습 방법 — 시청각 큐 설계

Expedition 33이 특히 주목받은 이유: 패링 타이밍을 **무작위 반응이 아닌 학습 가능한 패턴**으로 설계.

```
공격 예고 신호:
1. 독특한 애니메이션 피크 (공격 직전 자세 고정)
2. 무기/사지의 발광 (시각)
3. 특정 사운드 이펙트 (청각)
→ 이 신호가 임팩트 직전 수백 ms에 발생 → 반복 가능한 기준점
```

**Story 모드**: 패링 윈도우 40% 확장 → 초보자도 경험 가능
**Expert 모드**: 원래 윈도우 → 고난이도 집중 게임

### 3.3 체인 패링 (연속 공격 방어)

일부 적은 **2~5회 연속 공격 체인**을 사용. 체인의 모든 공격을 패링하면:
```
체인 완성 → 마지막 공격에 카운터 발동 → 대미지 + 추가 보상
```
실패 시: 체인 끊김, 남은 공격 풀 데미지.

## 4. 브레이크 시스템 (Break Mechanics)

### 4.1 브레이크 게이지 개념

```
[적 상태 표시]
HP: ████████░░ 80%
BR: ████░░░░░░ 40%  ← 브레이크 게이지
```

- 패링 성공, 특정 스킬 사용 시 브레이크 게이지 누적
- 게이지 100% → **브레이크(Break)** 상태 진입

### 4.2 브레이크 상태 효과

```
브레이크 진입:
├── 적이 스턴/비틀거림
├── 방어력 감소
├── 한정 시간 내 대형 버스트 데미지 기회
└── 高AP 스킬 집중 사용의 창
```

브레이크는 전투의 **"페이즈 구조"** 를 만듦: 브레이크 셋업 → 브레이크 폭발 → 재충전 → 반복

## 5. 특수 공격 유형

### 5.1 점프(Jump) 전용 공격

일부 공격은 패링/회피 불가 → 점프만 유효. 점프 타이밍 성공 시 카운터 가능.
```
점프 카운터:
특정 적의 공격 → 점프 성공 → 공중에서 카운터 공격 발동
```

### 5.2 원정대 카운터 (Expedition Counter)

광역 공격에 대응하는 **파티 전체의 공동 패링**:
```
보스 광역 공격 → 전 파티 동시 패링 입력 → 성공 시 강력한 집단 카운터
```

### 5.3 그라디언트 공격 (Gradient Attack)

보스 전용 특수 기술. 화면이 **흑백(모노크롬)** 으로 전환되며 특수 카운터만 유효:
```
화면 흑백 전환
    ↓
"그라디언트 카운터" 입력 (별도 키)
    ↓
성공: 화면 컬러 복구 + 보스에게 막대한 반격
실패: 파티 전체 대형 피해
```
게임에서 가장 **스펙타클한 전투 순간**을 연출하는 메카닉.

## 6. 의도 예고 시스템

Expedition 33은 소울류처럼 **공격 직전 명확한 시각/청각 예고**를 제공:

| 예고 레이어 | 타이밍 | 표현 방식 |
|----------|--------|---------|
| 애니메이션 준비 | 공격 1~2초 전 | 적 자세 변화 |
| 발광/글로우 | 공격 0.5초 전 | 무기/사지 빛남 |
| 사운드 큐 | 공격 직전 | 특징적 효과음 |

→ 패링을 "반응 게임"이 아닌 "학습 게임"으로 만드는 핵심 설계.
