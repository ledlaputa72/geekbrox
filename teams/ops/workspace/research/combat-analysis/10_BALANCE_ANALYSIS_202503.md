## 1️⃣ ATB 전투 게임성 분석

### 1.1 기본 공격 + 카드 이중 구조

#### 메커니즘
```
플레이어 ATB (spd 70 기준):
├─ 자동 기본 공격: 약 8초~12초마다 자동 발동 (spd 70 = ATB 100 도달 시간)
│  └─ 데미지: ATK 10 기본값 (효율 1.0/초)
│
└─ 카드 시스템: 에너지 3~5 사용해 추가 행동
   ├─ ATK_001 (1 에너지, 6 데미지): 효율 6/에너지
   └─ ATK_010 (3 에너지, 12 데미지): 효율 4/에너지
```

#### 게임성 평가
| 측면 | 평가 | 상세 |
|------|------|------|
| **플레이어 주도성** | ⭐⭐⭐⭐ | 기본 공격(자동) + 카드(선택) 조합이 운과 전략의 균형 |
| **템포** | ⭐⭐⭐⭐ | ATB 게이지 충전으로 긴장감 유지, 방치 느낌 적음 |
| **전략 깊이** | ⭐⭐⭐ | 선택의 폭이 넓지는 않음 (대부분 공격 카드 사용) |
| **직관성** | ⭐⭐⭐⭐⭐ | 신규 유저도 쉽게 이해 가능 |

**결론**: **실시간 액션성** + **턴 기반 구조**의 좋은 균형. 다만 전략성은 약함.

### 1.2 플레이어 SPD 70 vs 적 SPD 밸런스

#### 현재 설정
| 엔티티 | SPD | ATB 충전 시간 | 기본 공격 주기 |
|--------|-----|-------------|-------------|
| **플레이어** | 70 | 5.7초 | ~11.4초 (기본 공격 2회) |
| **슬라임** | 50 | 8초 | ~16초 |
| **고블린** | 65 | 6.1초 | ~12.2초 |
| **고속 몬스터** | 100 | 4초 | ~8초 |

#### 문제점
```
고속 몬스터(spd 100) 만남:
  ├─ 플레이어 ATB: 5.7초마다 충전
  ├─ 몬스터 ATB: 4초마다 충전 (1.4배 빠름)
  ├─ 결과: 리액션 윈도우 연속 3회 이상 → 피로도 ↑
  └─ 신규 유저 이탈 위험 ⚠️
```

#### 권고 조정
```
[옵션 1] 플레이어 SPD 상향 (70 → 75)
  └─ 효과: 플레이어 기본 공격 주기 단축, 카드 사용 빈도 ↑
  └─ 리스크: 오토 플레이 난이도 하향

[옵션 2] 적 SPD 정규화 (100 → 85)
  └─ 효과: 고속 몬스터도 플레이어 속도와 비슷해짐
  └─ 리스크: 나머지 몬스터 대비 상대적으로 느려짐

[옵션 3] 리액션 윈도우 시간 단축 (X초 → 0.1초 감소)
  └─ 효과: 빈도 줄지 않지만 각 윈도우 부담 경감
  └─ 리스크: 어려워짐 (신규 유저용이 아님)
```

**추천**: **옵션 2** (적 SPD 상향값 조정 85~90 범위)

### 1.3 에너지 시스템 고찰

#### 메커니즘
```
ATB 에너지 충전:
├─ 기본: 5초마다 +1 (0 → 3 구간)
├─ 방어 보상:
│  ├─ 패링 성공: +2 (즉시)
│  ├─ 회피 성공: +1
│  └─ 가드 성공: +0.5
└─ 오버플로우: 최대 5까지 (3초 유지 후 감소)
```

#### 현재 이슈

| 시간 배율 | 문제 | 영향 |
|---------|------|------|
| **1.0×** | 5초 회복이 길게 느껴짐 | 카드 사용 빈도 ↓ |
| **2.5×** | 에너지가 너무 빠르게 쌓임 | 카드 선택의 긴급성 ↓ |

**예시**: 
```
1.0× 속도:
  └─ 에너지 3/3 → 1장 사용 → 2/3 → 2초 대기 → 3/3
  └─ 차음: 에너지 부족으로 패스 고려

2.5× 속도:
  └─ 에너지 3/3 → 1장 사용 → 2/3 → 0.8초 후 3/3 (자동 회복)
  └─ 차음: 계속 카드 사용 가능, 선택 압박 없음
```

#### 권고 조정
```
[안내]: scaled_delta 기반 회복 유지하되,
        속도별 기본 회복 시간 재정의

현재: const ENERGY_AUTO_INTERVAL = 5.0  (모든 속도 동일)

변경안:
  1.0×: 5.0초 (현재 유지) → 카드 선택 압박 O
  2.5×: 2.0초로 단축 (속도에 비례) → 빠른 진행과 균형

공식: base_interval = 5.0
      actual_interval = base_interval / (speed_multiplier * 0.75)
      
결과:
  1.0×: 5.0 / 0.75 = 6.67초 (더 길게 → 신규 유저 배려)
  2.5×: 5.0 / 1.875 = 2.67초 (더 짧게 → 빠른 진행 지원)
```

## 2️⃣ 턴베이스 전투 게임성 분석

### 2.1 의도(Intent) 시스템

#### 메커니즘
```
보스 턴 시작:
├─ 다음 3행동 예고 (아이콘 + 수치)
├─ 표시 방식:
│  ├─ 1번째: 강조 (큰 아이콘, 빨강)
│  ├─ 2번째: 중간 (중간 크기, 노랑)
│  └─ 3번째: 약함 (작은 크기, 회색)
└─ 효과: 플레이어가 2~3턴 앞서 계획 가능
```

#### 게임성 평가
| 측면 | 점수 | 상세 |
|------|------|------|
| **전략 깊이** | ⭐⭐⭐⭐⭐ | 의도 예고로 최고의 전략성 제공 |
| **긴장감** | ⭐⭐⭐⭐ | "다음 공격이 강할 것 같다" 느낌 |
| **신규 유저 학습곡선** | ⭐⭐ | 3개 기호 기억 필요, 약간 복잡 |

**결론**: **턴베이스 = 전략 게임**, ATB = **액션 게임** 역할 분담 성공

### 2.2 타로 에너지 시스템

#### 메커니즘
```
조건: 메이저 아르카나 카드 3장 연속 사용
보상: "태양의 폭발" (30 데미지 전체 적용)

카드 분류:
├─ 메이저 아르카나 (11장/30장):
│  ├─ ATK: 검의 에이스, 이중 베기, 마법사, ..., 황제
│  └─ DEF: 방패의 왕, 여황제, 교황, 달, 정의, 은둔자, 절제
└─ 일반 (19장): 나머지

스타터 덱 내 아르카나: 4장/12장 (33%)
  ├─ ATK_001 × 2 (약함)
  ├─ DEF_002 × 1
  └─ SKL_001 × 1
```

#### 현재 이슈
```
✅ 메커니즘 자체는 우수
❌ 스타터 덱에서 달성 어려움

실제 게임:
  └─ 3장 연속 조건 → 스타터 덱 10장 중 4장만 아르카나
  └─ 손패 5장 × 2턴 = 실제 사용 10장에서 아르카나 연속 3장 확률: ~15%
  └─ 대부분 플레이어가 타로 에너지 체험 못하고 보스전 끝남 ⚠️
```

#### 권고 조정
**스타터 덱 아르카나 비중 상향**:
```
현재 (12장):
  ATK_001 × 2 (아르카나)
  ATK_006 × 1 (일반)
  DEF_002 × 2 (일반)
  PAR_001 × 3 (일반)
  DOD_001 × 2 (일반)
  SKL_001 × 2 (일반)
  → 아르카나: 2장 (17%)

변경안 (12장):
  ATK_001 × 2 (아르카나) ✓
  ATK_010 × 1 (아르카나) ← 추가
  DEF_003 × 1 (아르카나) ← 교체
  DEF_006 × 1 (일반)
  PAR_001 × 2 (일반) ← 감소
  DOD_001 × 2 (일반)
  SKL_001 × 2 (일반)
  → 아르카나: 4장 (33%)
  
결과: 손패 5장에서 아르카나 2장 포함 → 타로 체험 확률 ↑
```

## 3️⃣ 카드 밸런스 상세 분석

### 3.1 공격 카드 (ATK) 메타

#### 데미지 효율 정렬
| 순위 | 카드 | 비용 | 데미지 | 효율 | 평가 |
|------|------|------|--------|------|------|
| 1 | ATK_001 | 1 | 6 | 6.0 | ⭐⭐⭐⭐⭐ 최고 효율 |
| 2 | ATK_006 | 1 | 8 | 8.0 | ⭐⭐⭐⭐⭐ |
| 3 | ATK_009 | 2 | 7 | 3.5 | ⭐⭐⭐ (드로우 포함) |
| 4 | ATK_010 | 3 | 12 | 4.0 | ⭐⭐⭐⭐ |
| 5 | ATK_008 | 3 | 18 | 6.0 | ⭐⭐⭐⭐⭐ (레어) |

**문제**: 1비용 카드(ATK_001, ATK_006)의 효율이 3비용 카드보다 높음
```
현재:
  최고 효율: 1비용 6~8 데미지
  3비용: 4~6 데미지
  → 결론: 항상 저비용 카드만 사용하는 게 수학적으로 이득

예상 플레이:
  1. ATK_001 사용 (1비용, 6 데미지)
  2. ATK_006 사용 (1비용, 8 데미지)
  3. Pass (손패 교체)
  → ATK_008, ATK_010 같은 강력한 카드는 사용 기회 거의 없음
```

#### 권고 조정
```
[안내 1] 저비용 카드는 기본 공격 보조 용도
[안내 2] 고비용 카드에 특수 효과 추가

변경안:
  ATK_001: 6 데미지 → 5 데미지 (효율 5.0)
  ATK_006: 8 데미지 → 7 데미지 (효율 7.0, 여전히 좋음)
  ATK_008: 18 데미지 → 18 데미지 (효율 6.0) + "힘 +1" 추가 효과
  ATK_010: 12 데미지 → 12 데미지 (효율 4.0) + "힘 +2" 추가 효과

결과:
  저비용: 보조 공격
  고비용: 버프 + 강력한 데미지
  → 플레이어가 고비용 카드도 사용할 동기 생김
```

### 3.3 리액션 카드 (패링·회피·가드) 메타

#### 현재 구성
| 유형 | 장수 | 효과 | 평가 |
|------|------|------|------|
| **패링** | 3~5장 | 에너지 +2 + 드로우 1 | ⭐⭐⭐⭐⭐ 최고의 보상 |
| **회피** | 2~3장 | 에너지 +1 + 드로우 0 | ⭐⭐⭐⭐ |
| **가드** | 5~8장 | 블록 0~2 | ⭐⭐⭐ 기본값 |

**현재 이슈**:
```
리액션 윈도우 중심 게임:
  ├─ 패링 성공 = 가장 큰 보상 (에너지 +2 + 드로우)
  ├─ 회피 성공 = 중간 보상 (에너지 +1)
  └─ 가드 = 기본값 (데미지 경감만)

→ 메타: 패링을 열심히 연습 → 회피로 대체 → 가드는 부스러기
→ 신규 유저는 가드를 연습할 동기가 적음

개선안:
  패링: 에너지 +2 + 드로우 1 (유지)
  회피: 에너지 +2 (상향) + 드로우 0
  가드: 에너지 +1 (신규 추가) + 블록 경감
  
  결과: 세 옵션 모두 가치있음 → 선택의 폭 넓어짐
```

## 4️⃣ 리액션 시스템 종합 평가

### 4.1 패링·회피·가드의 학습곡선

#### 난이도 순서
```
배우기 쉬운 순:
  1순. 가드 (에너지 버튼 누르기, 타이밍 관계없음)
  2순. 회피 (노랑+빨강 구간, 약 1.4초)
  3순. 패링 (빨강 구간만, 약 0.4초)
```

#### 신규 유저 경험
```
현재 세팅:
  └─ 리액션 윈도우 자동 열림
  └─ 3가지 중 뭘 눌러도 성공 가능한 시점이 있음
  └─ 실패 → 윈도우 유지 → 다시 시도 가능 ← ✅ 좋은 설계

문제점:
  └─ 옵션이 3개인데, 보상이 패링>>회피>>가드 (불균형)
  └─ 신규 유저가 "패링만 하면 된다"고 착각 가능
  └─ 실패할 때마다 +50% 데미지 페널티 → 좌절감 ↑
```

#### 권고 개선
```
1️⃣ 보상 재균형:
   └─ 패링/회피/가드 모두 "같은 윈도우에서 한 번 성공" 시 혜택 O

2️⃣ 튜토리얼 추가:
   └─ 첫 전투: "가드는 항상 가능 → 회피를 배워보세요 → 패링 도전"

3️⃣ 피드백 명확화:
   └─ 실패 시: "패링 실패! 타이밍을 다시 맞춰보세요" (설명 추가)
```

## 5️⃣ 오토 AI 게임성
