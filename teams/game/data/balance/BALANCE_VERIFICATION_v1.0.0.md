## ⚠️ 4. 발견된 밸런스 이슈

### 4.1 심각 (즉시 조정 필요)

1. **일반 뽑기 손실률 과다 (-62.5%)**
   - 현상: 플레이어가 뽑기를 꺼려함
   - 원인: Common 카드 가치 너무 낮음
   - 해결: Rare 확률 30% → 35%, Epic 4% → 6%

2. **난이도 7+ 몬스터 데미지 과다**
   - 현상: HP 80으로 생존 불가
   - 원인: 몬스터 DPS 증가율 과다
   - 해결: 난이도 7+ 몬스터 데미지 10-15% 감소

### 4.2 중간 (모니터링 필요)

3. **0코스트 카드 4장 존재**
   - 현상: 무한 콤보 가능성
   - 원인: 코스트 0 제한 없음
   - 해결: 카드 효과 약화 또는 "턴당 1회" 제한 추가

4. **Legendary 카드 파워 격차 (20-30)**
   - 현상: 컬렉션 완성 시 과도한 파워 증가
   - 원인: 전설 레어 보정 과다
   - 해결: 프레임워크 기준 (×1.5) 준수 → 일부 하향 (25-30 → 18-24)

5. **프레스티지 보너스 부족**
   - 현상: 엔드게임 난이도 과다
   - 원인: 프레스티지 보너스 25% 부족
   - 해결: 보너스 강화 (25% → 30%) 또는 유물 시스템 추가

### 4.3 경미 (장기 검토)

6. **Elite 몬스터 보상 vs 난이도**
   - 현상: Elite 전투 보상이 Boss보다 효율적
   - 원인: Boss 전투 시간 과다
   - 해결: Boss 보상 20% 증가 또는 Elite 난이도 증가

7. **Skill 카드 사용률 예측 낮음**
   - 현상: 공격/방어 우선 메타 예상
   - 원인: Skill 카드 즉시 효과 부족
   - 해결: Skill 카드 시너지 강화 (태그 시스템)

   - 조정 기준: <1% 버프, >20% 너프

2. **난이도별 승률** (Win Rate)
   - 목표: 난이도 1-3 (75-85%), 4-6 (60-70%), 7-10 (45-55%)
   - 조정 기준: ±10% 벗어날 시 몬스터 조정

3. **재화 획득/소비 비율**
   - 목표: 하루 평균 뽑기 3-5회
   - 조정 기준: <2회 시 보상 증가, >8회 시 인플레이션 주의

4. **플레이 타임별 이탈률**
   - 목표: 5시간 이탈률 <30%, 20시간 이탈률 <50%
   - 조정 기준: 이탈률 +10% 시 해당 구간 보상 강화
