## 🔧 밸런스 조정 고려사항

### 몬스터 강화 요소
- **HP 증가:** 전투 길이 조절 (+10~20%)
- **공격력 증가:** 위협도 증가 (+15~30%)
- **새로운 행동 추가:** 복잡도 증가
- **소환 빈도 증가:** 난이도 급상승

### 몬스터 약화 요소
- **HP 감소:** 전투 속도 증가 (-10~20%)
- **공격력 감소:** 생존성 증가 (-15~25%)
- **행동 간격 증가:** 대응 시간 확보
- **디버프 효과 감소:** 스트레스 완화

### 조정 기준
- **Normal:** Win Rate 70-80%
- **Elite:** Win Rate 50-65%
- **Boss:** Win Rate 40-55%
