## 🃏 CardThumbnail (카드 썸네일)

가장 핵심적인 공통 컴포넌트. 두 게임의 카드 시스템을 동일한 구조로 표현.

### 크기 규격

| 용도 | 너비 | 높이 | 비고 |
|------|------|------|------|
| 덱 빌더 그리드 | 100px | 140px | 3열 기준 |
| 카드 라이브러리 | 110px | 154px | 약간 크게 |
| 패 (인게임, 던전) | 72px | 101px | 5장 가로 배치 |
| 상세 보기 팝업 | 280px | 392px | 화면 중앙 |
| 덱 슬롯 (가로 스크롤) | 64px | 90px | 현재 덱 미리보기 |

### 카드 레이어 구조 (위에서 아래)

```
┌─────────────────────┐
│  [희귀도 색상 테두리] │  ← RarityFrame (2px border)
│  ┌───────────────┐  │
│  │               │  │
│  │  [카드 일러]  │  │  ← 일러스트 이미지 (상단 60%)
│  │               │  │
│  ├───────────────┤  │
│  │ [카드 이름]   │  │  ← 텍스트 (12px Bold)
│  │ [타입 아이콘] │  │  ← 타입 배지
│  │ [효과 설명]   │  │  ← 텍스트 (10px Regular, 2줄 제한)
│  └───────────────┘  │
│  [코스트: 에너지 숫자] │  ← 좌상단 오버레이 (원형 배지)
└─────────────────────┘
```

### 희귀도 색상 (두 게임 공통)

| 희귀도 | Dream 테마 테두리 | Dark 테마 테두리 | 배경 틴트 |
|--------|-----------------|-----------------|----------|
| 일반 (Common) | `#888888` | `#666666` | 없음 |
| 언커먼 (Uncommon) | `#4CAF50` | `#228B22` | 10% 초록 |
| 레어 (Rare) | `#7B68EE` | `#6A0DAD` | 10% 보라 |
| 전설 (Legendary) | `#FFD700` | `#B8860B` | 10% 금색 |

### 상태 (States)

- **기본:** 정상 표시
- **선택됨:** 2px 흰 테두리 추가 + scale(1.05)
- **덱에 추가됨:** 우하단 초록 체크 배지
- **최대 보유 수:** 좌하단 회색 "MAX" 배지
- **잠김:** 전체 50% 어둡게 + 자물쇠 아이콘 오버레이
- **신규:** 좌상단 "NEW" 붉은 배지

## 💰 CurrencyChip (재화 칩)

### 규격

```
높이: 32px
패딩: 8px 12px
라운드: 16px (완전 캡슐형)
배경: rgba(0,0,0,0.4) + blur(4px)
테두리: 1px solid --color-currency-x (15% 투명도)

[아이콘 20×20px] [숫자 텍스트 14px Medium]
   간격: 6px
```

### 숫자 포맷

| 범위 | 표시 형식 | 예시 |
|------|---------|------|
| 0 ~ 9,999 | 원본 | `1,234` |
| 10,000 ~ 999,999 | K | `12.3K` |
| 1,000,000+ | M | `1.2M` |
| 1,000,000,000+ | B | `1.5B` |

### 증가 애니메이션
1. 숫자 카운트업 (500ms, ease-out)
2. 칩 전체 briefly 빛남 (300ms, --color-currency 색상 glow)
3. +XX 플로팅 텍스트 위로 사라짐 (600ms, fade+float)

## ⬆️ UpgradeNode (업그레이드 노드)

### 규격

```
너비: 화면 너비 / 3 (약 120px)
높이: 120px
배경: --color-bg-panel
라운드: 12px
패딩: 12px

구조:
┌────────────────────┐
│  [아이콘 40×40px]  │  ← 중앙 상단
│  [업그레이드 이름]  │  ← 12px Bold, 1줄
│  [현재효과/다음효과] │  ← 10px, 회색→흰색
│  [레벨: X / MAX]   │  ← 10px, 우하단
│  [▲ 업그레이드]    │  ← 버튼 (전체 너비, 높이 32px)
└────────────────────┘
```

### 상태

- **업그레이드 가능:** 버튼 활성 (--color-primary)
- **최대 레벨:** 버튼 "MAX" 표시, 금색
- **잠김:** 전체 어둡게, 자물쇠 + "필요 조건" 텍스트
- **방금 업그레이드:** 0.6초 파티클 애니메이션

### 트리 연결선

```
두 노드 간 연결: SVG 곡선 (bezier)
색상: 활성화된 경로 → --color-primary
      미활성 경로 → #444
두께: 2px
```

## 🔎 FilterBar (카드 필터 바)

카드 라이브러리와 덱 빌더에서 공통 사용.

```
높이: 44px
가로 스크롤 (오버플로우 허용)

[전체] [공격] [방어] [유틸] [특수]   ← 타입 필터 (게임별 레이블 교체)
   +
[일반▼] [희귀도▼] [정렬: 이름▼]    ← 드롭다운 필터
```

### 필터 칩 스타일

```
높이: 32px
패딩: 8px 14px
라운드: 16px
비활성: 배경 #333, 텍스트 #888
활성: 배경 --color-primary, 텍스트 흰색
폰트: 13px Medium
```

## 🔔 팝업 시스템 (CommonPopup)

### 팝업 유형

| 유형 | 크기 | 사용처 |
|------|------|--------|
| Alert | 300×200px | 에러, 알림 1버튼 |
| Confirm | 300×220px | 확인/취소 2버튼 |
| Info | 340×480px | 카드 상세, 긴 정보 |
| Reward | 320×400px | 카드 획득, 업그레이드 완료 |
| Loading | 100×100px | 로딩 스피너 |

### 공통 팝업 레이아웃

```
배경 딤: rgba(0,0,0,0.6) + blur(4px)
팝업 컨테이너:
  배경: --color-bg-panel
  라운드: 20px
  패딩: 24px
  최소 너비: 280px

┌──────────────────────────┐
│  [× 닫기 버튼]  (선택)   │  ← 우상단 (Info/Reward만)
│                          │
│  [아이콘/이미지]          │  ← 중앙 (유형별 다름)
│  [제목 18px Bold]        │
│  [내용 14px Regular]     │
│                          │
│  [보조버튼] [주버튼]      │  ← 하단 (2버튼) 또는 [확인] (1버튼)
└──────────────────────────┘
```

### 카드 획득 팝업 (Reward) 특별 사양

```
팝업 등장: scale(0.5)→scale(1.05)→scale(1.0), 400ms spring
배경 파티클: 희귀도 색상 별 파티클 방사 (100개, 2초)
카드 이미지: 360도 회전 후 정면 착지 (600ms)
텍스트: "새 카드 획득!" fade-in (700ms 딜레이)
사운드: 희귀도별 효과음 (별도 사운드 사양서 참조)
```

## 🔤 타이포그래피 시스템

### 폰트 계층 (두 테마 공통 구조)

| 역할 | Dream 폰트 | Dark 폰트 | 크기 | Weight |
|------|-----------|----------|------|--------|
| 헤드라인 H1 | Nunito ExtraBold | Crimson Text Bold | 28px | 800 |
| 헤드라인 H2 | Nunito Bold | Crimson Text SemiBold | 22px | 700 |
| 서브타이틀 | Nunito SemiBold | Crimson Text Regular | 18px | 600 |
| 본문 강조 | Noto Sans KR Medium | Noto Sans KR Medium | 14px | 500 |
| 본문 기본 | Noto Sans KR Regular | Noto Sans KR Regular | 13px | 400 |
| 캡션 | Noto Sans KR Light | Noto Sans KR Light | 11px | 300 |
| 카드 코스트 | Nunito ExtraBold | Crimson Text Bold | 16px | 800 |
| 숫자/재화 | Nunito Bold | Nunito Bold | 14px | 700 |

### 텍스트 색상 (라이트/다크 공통 사용)
- 기본 텍스트: `#FFFFFF` (흰색)
- 보조 텍스트: `#AAAAAA` (회색)
- 비활성 텍스트: `#555555`
- 경고 텍스트: `#FF6B6B` (빨간계열)
- 성공 텍스트: `#66BB6A` (초록계열)
