## 2. 덱 빌더 화면

### 2.1 레이아웃 (세로 390×844px)

```
┌────────────────────────────────┐
│ [← 뒤로]  덱 빌더  [💾 저장]   │ ← 헤더 (60px)
├────────────────────────────────┤
│ 현재 덱 (8/12):  DPS: 45       │ ← 덱 요약 (40px)
├────────────────────────────────┤
│ ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐ ... │ ← 가로 스크롤
│ │C1│ │C2│ │C3│ │C4│ │C5│     │   현재 덱 (120px)
│ └──┘ └──┘ └──┘ └──┘ └──┘     │   카드: 64×90px
├────────────────────────────────┤
│ [필터▼] [정렬▼] [🔍검색]      │ ← 필터 바 (50px)
├────────────────────────────────┤
│ ┌──┐ ┌──┐ ┌──┐                │
│ │A1│ │A2│ │A3│  사용 가능     │ ← 스크롤 가능 그리드
│ └──┘ └──┘ └──┘  카드          │   (3열)
│ ┌──┐ ┌──┐ ┌──┐                │   카드: 100×140px
│ │A4│ │A5│ │A6│                │
│ └──┘ └──┘ └──┘                │
│ ...                            │
└────────────────────────────────┘
```

### 2.2 UI 컴포넌트

#### 2.2.1 헤더 (60px)
- **뒤로 버튼**
  - 위치: 좌측 (20px 마진)
  - 크기: 40×40px
  - 아이콘: ← (Feather Icons)
  - 인터랙션: 탭 → 변경사항 확인 모달 (수정 있을 경우)

- **제목**
  - 텍스트: "덱 빌더" (Nunito Bold 20px)
  - 위치: 중앙

- **저장 버튼**
  - 위치: 우측 (20px 마진)
  - 크기: 40×40px
  - 아이콘: 💾 (활성) / 🔒 (비활성)
  - 상태:
    - 활성: 변경사항 있음 + 유효한 덱
    - 비활성: 변경 없음 또는 무효한 덱

#### 2.2.2 덱 요약 바 (40px)
- **레이아웃**
  - 배경: `#2C2C3E`
  - 패딩: 10px 20px

- **정보 표시**
  - 덱 크기: "현재 덱 (8/12)" (Nunito Bold 14px)
  - DPS: "DPS: 45" (Nunito Regular 14px, 색상: `#FFD700`)
  - 평균 비용: "평균: 2.8" (Nunito Regular 14px)

#### 2.2.3 현재 덱 영역 (120px)
- **가로 스크롤**
  - 카드 크기: 64×90px
  - 간격: 8px
  - 패딩: 좌우 20px

- **카드 표시**
  - 배경: 희귀도별 그라데이션
  - 텍스트: 카드명 (Nunito Bold 10px)
  - 비용: 우상단 배지 (16×16px)
  - 애니메이션: 탭 시 확대 (150ms)

- **인터랙션**
  - 탭: 덱에서 제거 (페이드 아웃 200ms)
  - 롱 프레스: 카드 상세 모달

- **빈 슬롯**
  - 표시: 점선 테두리 (2px dashed `#555555`)
  - 텍스트: "+" (Nunito Bold 24px, 색상: `#888888`)

#### 2.2.4 필터 바 (50px)
- **레이아웃**
  - 배경: `#1A1A2E`
  - 패딩: 10px 20px

- **필터 버튼 (3개)**
  - **필터▼**: 유형, 희귀도, 태그, 소유
  - **정렬▼**: 이름, 비용, 데미지, 희귀도
  - **🔍 검색**: 텍스트 입력 (모달 오버레이)

- **드롭다운 스타일**
  - 배경: `#2C2C3E`
  - 테두리 반경: 8px
  - 그림자: `0 4px 8px rgba(0,0,0,0.3)`
  - 항목 높이: 44px (터치 최적화)

#### 2.2.5 사용 가능 카드 그리드
- **그리드 설정**
  - 열: 3개
  - 간격: 12px
  - 패딩: 좌우 20px

- **카드 크기: 100×140px**
  - 배경: 희귀도별 그라데이션
  - 테두리 반경: 12px
  - 그림자: `0 2px 4px rgba(0,0,0,0.2)`

- **카드 정보**
  - 카드명: Nunito Bold 12px
  - 비용: 우상단 배지 (20×20px)
  - 유형 아이콘: 좌하단 (16×16px)
  - 소유 표시: 체크마크 (우하단, 녹색)

- **인터랙션**
  - 탭: 덱에 추가 (슬라이드 업 200ms)
  - 롱 프레스: 카드 상세 모달
  - 드래그: 덱 영역으로 드래그 (선택적)

- **상태**
  - 덱에 포함됨: 불투명도 0.5 + "덱에 포함" 레이블
  - 미소유: 흑백 + 잠금 아이콘
  - 최대치: 희미함 + "최대치" 레이블
