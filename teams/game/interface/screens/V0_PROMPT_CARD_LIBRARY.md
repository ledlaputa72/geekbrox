## 📋 시스템 컨텍스트 프롬프트 (먼저 입력)

```
시스템 컨텍스트:
모바일 방치형 덱빌딩 게임 "꿈 수집가 (Dream Collector)" UI 제작 중입니다.

기술 스택:
- React + TypeScript + Tailwind CSS + shadcn/ui
- 기준 해상도: 390×844px (세로 모드 전용, 모바일)
- 렌더링: 웹 미리보기이지만 실제 모바일 게임 UI처럼 보여야 함

공통 테마 변수 (CSS Custom Properties로 구현):
:root {
  /* Dream 테마 (꿈 수집가) */
  --color-primary: #7B9EF0;
  --color-secondary: #C4A8E8;
  --color-accent: #F5F0FF;
  --color-bg-main: #0D1B3E;
  --color-bg-panel: rgba(255, 255, 255, 0.10);
  --color-currency-1: #FFE066;
  --color-currency-2: #E8D5FF;
  --radius-card: 16px;
  --radius-button: 20px;
}

레이아웃 원칙:
- 상단: 헤더 64px + Safe Area 44px
- 중앙: 메인 콘텐츠 영역 (가변, 스크롤)
- 하단: 5탭 내비게이션 80px
- 모든 버튼 최소 44×44px (접근성)
- 글래스모피즘 패널: backdrop-blur + 반투명 배경

이 컨텍스트를 기억하고 이후 요청하는 화면을 제작해주세요.
```
