## 🎯 화면 목적

플레이어가 보유한 모든 카드를 탐색하고 관리할 수 있는 컬렉션 화면. 필터링, 정렬, 검색 기능을 제공하며, 카드 상세 정보 확인 및 덱 빌더로의 빠른 이동을 지원.

**스타일:**
- 배경: rgba(255,255,255,0.08) + backdrop-blur(8px)

### ③ 필터 바 (44px)

```
┌────────────────────────────────────────────────┐
│ [전체] [수집▼] [액션] [시너지] [이벤트] → 스크롤 │
└────────────────────────────────────────────────┘
```

**카드 타입 필터 칩:**

| 칩 | 아이콘 | 설명 |
|----|--------|------|
| 전체 | 🃏 | 모든 카드 표시 (기본) |
| 수집 | 💰 | 패시브 소득 카드 |
| 액션 | ⚡ | 액티브 이벤트 카드 |
| 시너지 | 🔗 | 콤보 효과 카드 |
| 이벤트 | 🎲 | 로그라이크 랜덤 카드 |

**하위 필터 (드롭다운):**
- **희귀도:** [전체], [일반], [언커먼], [레어], [전설]
- **소장 상태:** [전체], [보유], [미보유], [최대 보유]
- **새 카드:** [전체], [최근 획득]

**스타일:**
- 칩 높이: 32px
- 비활성: 배경 #333, 텍스트 #888
- 활성: 배경 --color-primary, 텍스트 흰색
- 드롭다운 칩: 우측 ▼ 화살표 아이콘

### ⑤ 카드 그리드 (가변 높이, 스크롤)

```
패딩: 16px
간격: 12px (가로/세로)
열: 3개 (고정)
카드 너비: (390 - 32 - 24) / 3 = 약 111px
카드 높이: 156px (비율 1:1.4)
```

#### 카드 컴포넌트 구조

```
┌──────────────────┐
│ [NEW] [⭐]       │  ← 좌상단 배지 영역
│                  │
│   [카드 일러]    │  ← 일러스트 (상단 60%)
│                  │
├──────────────────┤
│ [기억 파편] ⚡2  │  ← 이름 + 코스트
│ 수집 | 일반      │  ← 타입 | 희귀도
│ +5 레버리/시간   │  ← 효과 (2줄 제한)
└──────────────────┘
```

**카드 요소 상세:**

1. **배지 오버레이 (좌상단):**
   - **NEW:** 빨간 배지 (신규 획득 3일 이내)
   - **즐겨찾기 별:** ⭐ (유저가 설정)
   - **보유 수량:** "×3" 회색 칩 (우하단)

2. **일러스트 영역:**
   - 비율: 60% (상단)
   - 배경: 희귀도 색상 틴트 10%
   - 미보유 카드: 흑백 + 50% 어둡게 + 🔒 오버레이

3. **정보 영역 (하단 40%):**
   - **라인 1:** 카드 이름 (12px Bold) + 코스트 (에너지 아이콘 + 숫자)
   - **라인 2:** 타입 | 희귀도 (10px, 회색)
   - **라인 3:** 효과 설명 (10px Regular, 2줄 ellipsis)

4. **희귀도 테두리:**
   - 일반: #888888
   - 언커먼: #4CAF50
   - 레어: #7B68EE
   - 전설: #FFD700 + glow 효과

5. **인터랙션:**
   - **탭:** 카드 상세 팝업 표시
   - **롱 프레스:** 빠른 메뉴 (즐겨찾기 추가/제거, 덱에 추가)

## 💬 카드 상세 팝업 (인터랙션)

카드 탭 시 표시되는 팝업:

```
┌──────────────────────────────────┐
│              [× 닫기]             │
├──────────────────────────────────┤
│                                  │
│      [카드 대형 일러스트]        │
│         280×392px                │
│                                  │
├──────────────────────────────────┤
│ [기억 파편]       ⚡ 비용: 2     │
│ 수집 카드 | 일반                  │
│                                  │
│ 효과:                            │
│ 시간당 레버리 +5 획득             │
│                                  │
│ 다음 레벨 (Lv.4 → Lv.5):         │
│ 시간당 레버리 +7 (💎 200 필요)   │
│                                  │
│ 보유: 3장 / 최대: 10장           │
│                                  │
│ "작은 꿈 조각도 모이면            │
│  큰 기억이 됩니다..."             │
│  — 평온의 몽상가                 │
├──────────────────────────────────┤
│ [업그레이드] [덱에 추가] [⭐즐겨찾기] │
└──────────────────────────────────┘
```

**팝업 스타일:**
- 크기: 340×560px
- 배경 딤: rgba(0,0,0,0.7) + backdrop-blur(4px)
- 팝업 배경: --color-bg-panel + 희귀도 색상 테두리 2px
- 등장 애니메이션: scale(0.8) → scale(1.0), 300ms ease-out

**버튼:**
- **업그레이드:** Primary 버튼 (재화 충분 시 활성)
- **덱에 추가:** Secondary 버튼 → 덱 빌더로 이동
- **즐겨찾기:** 아이콘 버튼 (토글)

## 🔄 상태 및 애니메이션

### 로딩 상태
```
카드 그리드 로딩 중:
  - 스켈레톤 카드 9개 표시 (맥박 애니메이션)
  - 배경: rgba(255,255,255,0.05)
```

### 필터 변경 애니메이션
```
1. 현재 카드들 fade-out (200ms)
2. 그리드 재배치
3. 새 카드들 fade-in (300ms, stagger 50ms)
```

### 카드 획득 알림
```
새 카드 획득 시:
  - 해당 카드에 "NEW" 배지 3일간 표시
  - 카드 라이브러리 진입 시 자동으로 새 카드로 스크롤
  - 반짝임 효과 (2초)
```

### 스크롤 상태
```
- 상단 도달: 헤더에 그림자 제거
- 스크롤 중: 헤더에 그림자 추가 (depth 효과)
- 하단 도달: "모든 카드를 확인했습니다" 메시지 (1초 표시)
```

## 🎮 사용자 플로우

### 메인 플로우
```
1. 메인 로비에서 "카드" 탭 선택
   ↓
2. 카드 라이브러리 진입
   ↓
3. (선택) 필터/검색으로 원하는 카드 찾기
   ↓
4. 카드 탭 → 상세 팝업
   ↓
5. (선택) 업그레이드 또는 덱에 추가
```

### 서브 플로우
```
덱 빌더 진입:
  - 상단 더보기 → [덱 빌더로 이동]
  - 또는 카드 상세 팝업 → [덱에 추가] → 자동 이동

새 카드 획득:
  - 팝업 알림 → [확인하러 가기] → 카드 라이브러리 (해당 카드로 스크롤)

컬렉션 보상:
  - 통계 바 탭 → 마일스톤 팝업 → [보상 받기]
```

## 🛠️ 기술 사양

### 컴포넌트 구조 (React)

```typescript
<CardLibraryScreen>
  <Header>
    <BackButton />
    <Title />
    <MoreMenu />
  </Header>
  
  <StatsBar 
    collected={34} 
    total={200} 
    nextMilestone={50} 
  />
  
  <FilterBar 
    types={cardTypes}
    activeFilters={filters}
    onChange={handleFilterChange}
  />
  
  <SearchBar 
    value={searchQuery}
    onSearch={handleSearch}
    sortBy={sortOption}
    onSortChange={handleSortChange}
  />
  
  <CardGrid>
    {filteredCards.map(card => (
      <CardThumbnail
        key={card.id}
        card={card}
        onTap={handleCardTap}
        onLongPress={handleCardLongPress}
      />
    ))}
  </CardGrid>
  
  <BottomNav activeTab="cards" />
  
  {selectedCard && (
    <CardDetailPopup
      card={selectedCard}
      onClose={handleClosePopup}
      onUpgrade={handleUpgrade}
      onAddToDeck={handleAddToDeck}
    />
  )}
</CardLibraryScreen>
```

### 데이터 모델

```typescript
interface Card {
  id: string;
  name: string;
  type: 'collection' | 'action' | 'synergy' | 'event';
  rarity: 'common' | 'uncommon' | 'rare' | 'legendary';
  cost: number; // 에너지 비용
  effect: string;
  illustration: string; // 이미지 URL
  level: number;
  maxLevel: number;
  owned: number; // 보유 수량
  maxOwned: number; // 최대 보유
  isNew: boolean;
  isFavorite: boolean;
  acquiredAt?: Date;
}

interface FilterState {
  type: string[];
  rarity: string[];
  owned: 'all' | 'owned' | 'unowned' | 'maxed';
  showNew: boolean;
}

interface SortOption {
  by: 'name' | 'rarity' | 'cost' | 'recent' | 'favorite';
  order: 'asc' | 'desc';
}
```

### 성능 최적화

- **가상 스크롤:** react-window 사용 (200장 이상 카드 대응)
- **이미지 레이지 로딩:** Intersection Observer API
- **필터링:** 디바운스 300ms (검색어 입력)
- **애니메이션:** CSS transform/opacity만 사용 (reflow 방지)
