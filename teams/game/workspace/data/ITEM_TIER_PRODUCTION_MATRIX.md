## 개요

### 목적

등급별 아이템 생산 메트릭스는 **일관성 있고 균형잡힌 아이템 생성**을 위한 표준입니다.

```
아이템 생성 프로세스:
1. 아이템 타입 결정 (무기/방어구/반지/목걸이)
   ↓
2. 등급 결정 (Common → Legendary)
   ↓
3. 이 메트릭스에 따라 능력치 범위 결정
   ↓
4. 스킬 및 옵션 결정
   ↓
5. 최종 검증 (메트릭스 충족 여부)
```

## 등급별 능력치 범위

### 1️⃣ Common (일반) — 초반 진입 용

**레벨:** 1-2  
**특징:** 가장 기본적인 능력치, 초보자용

**능력치 범위 (무기 기준):**

| 스탯 | 최소 | 최대 | 예시 |
|------|------|------|------|
| ATK | 15 | 20 | 18 |
| Crit Rate | 0 | 3% | 2% |
| Mana | 0 | 5 | 0 |
| Special | 0 | 5% | 0% |

**스킬 효율:** 100% (기본값)  
**옵션 개수:** 2개 (1개 선택)  
**시간:** 초반 1-5레벨 (약 30분)

---

## 등급별 스킬 시스템

### 스킬 기본 데미지 범위

| 등급 | 최소 | 최대 | 예시 | 재사용 대기 |
|------|------|------|------|----------|
| Common | 80% | 120% | 100% | 0-1턴 |
| Uncommon | 120% | 160% | 140% | 1-2턴 |
| Rare | 160% | 220% | 190% | 1-3턴 |
| Epic | 220% | 280% | 250% | 2-4턴 |
| Legendary | 280% | 360% | 320% | 3-5턴 |

### 스킬 특성

```
Common:
├─ 기본 공격 또는 단순 효과
├─ 추가 효과: 없음 또는 매우 약함
└─ 예: "기본 베기" (100% 데미지)

Uncommon:
├─ 기본 공격 + 약한 추가 효과
├─ 추가 효과: 버프/약화 15% 정도
└─ 예: "강화된 공격" (140% 데미지 + ATK +10%)

Rare:
├─ 공격 + 확실한 추가 효과
├─ 추가 효과: 버프/약화 20-30%
└─ 예: "폭염 돌격" (190% 데미지 + 꿈 수집 +50)

Epic:
├─ 강력한 공격 + 복합 효과
├─ 추가 효과: 다중 효과 또는 높은 수치 (30-40%)
└─ 예: "신성한 심판" (250% 데미지 + 상태이상 제거 + 2배 피해)

Legendary:
├─ 극강 공격 + 극강 효과
├─ 추가 효과: 극도의 효과 (40%+) 또는 매우 특수한 효과
└─ 예: "공허의 쓸기" (320% 데미지 + 흡수 30% + 상태이상 무시)
```

## 등급별 특징

### Common (일반)

```
목적: 게임 초입 경험
├─ 획득: 게임 시작 또는 초반 몬스터
├─ 비용: 저가
├─ 성능: 기본 성능
├─ 특수성: 거의 없음
└─ 보관 기간: 5-10레벨 정도 후 버림

특징:
├─ 스킬이 가장 단순
├─ 옵션이 적음 (2개만)
├─ 능력치가 낮음
└─ 빠르게 교체됨
```

### Uncommon (고급)

```
목적: 초중반 진행 보조
├─ 획득: 초중반 던전 또는 상점
├─ 비용: 중가
├─ 성능: 기본보다 나음
├─ 특수성: 약간의 특수 능력

특징:
├─ 스킬에 추가 효과 포함
├─ 옵션 선택의 자유도 시작
├─ 초중반 약 10-15레벨 사용
└─ 자주 교체되지만 보관 가치 있음
```

### Rare (레어) — 주력 장비

```
목적: 중반의 안정적인 플레이
├─ 획득: 중반 던전 또는 보스 드롭
├─ 비용: 중고가
├─ 성능: 확실한 성능
├─ 특수성: 다양한 효과

특징:
├─ 중반의 주력 장비
├─ 플레이 스타일에 따른 선택 가능
├─ 약 15-35레벨 사용
├─ 강화 가치가 높음
└─ 모든 플레이어가 보유할 수 있음
```

### Epic (에픽) — 도전 장비

```
목적: 후반의 강력한 성능
├─ 획득: 후반 보스 또는 고가 상점
├─ 비용: 고가
├─ 성능: 매우 강함
├─ 특수성: 강력한 특수 능력

특징:
├─ 후반 콘텐츠 필수
├─ 강화 가치가 매우 높음
├─ 약 35-70레벨 사용
├─ 제한적 획득
└─ 수집가치 높음
```

### Legendary (전설) — 최고급

```
목적: 최고 난이도 콘텐츠 및 수집
├─ 획득: 극후반 또는 매우 제한적
├─ 비용: 극고가
├─ 성능: 최강 성능
├─ 특수성: 매우 특별한 효과

특징:
├─ 게임의 궁극 목표
├─ 모든 능력이 극대화됨
├─ 약 70+레벨 사용
├─ 매우 제한적 획득
└─ 수집 가치 극대화
```

## 검증 규칙

### 생산 시 검증 체크리스트

```
각 아이템 생성 시:

1️⃣ 등급 확인
   [ ] 해당 등급 개수 충족 중인가?
   [ ] 타입별 분배가 균형잡혀 있는가?

2️⃣ 능력치 검증
   [ ] ATK가 등급 범위 내인가?
   [ ] 추가 스탯이 등급에 맞는가?

3️⃣ 스킬 검증
   [ ] 데미지 범위가 등급 맞나?
   [ ] 재사용 대기가 적절한가?

4️⃣ 옵션 검증
   [ ] 옵션 개수가 맞나? (Common/Uncommon: 2개, 나머지: 2-3개)
   [ ] 옵션 효율이 등급에 맞나?

5️⃣ 특징 검증
   [ ] 등급의 특징이 반영되었나?
   [ ] 게임 진행도에 맞는 난이도인가?
```

## 생산 가이드

### Step 1: 타입 결정
```
무기/방어구/반지/목걸이 중 선택
```

### Step 2: 등급 결정
```
생산 메트릭스에서 남은 개수 확인
예: "반지 Rare는 10개 중 7개 완료, 3개 남음"
```

### Step 3: 능력치 결정
```
해당 등급의 능력치 범위에서 무작위 선택
예: Rare 무기 → ATK 35-45 범위에서 선택
```

### Step 4: 스킬 결정
```
해당 등급의 스킬 데미지 범위 적용
예: Rare 무기 → 160-220% 범위에서 스킬 설정
```

### Step 5: 옵션 결정
```
등급에 맞는 옵션 개수 및 효율 적용
예: Rare → 2-3개 옵션, 120% 효율
```

### Step 6: 검증
```
위의 검증 체크리스트 실행
```
