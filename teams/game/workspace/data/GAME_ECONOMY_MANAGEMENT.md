## 개요

### 목적

Dream Collector의 게임내 경제 시스템을 관리하여:
- **플레이어 만족도**: 공정한 보상과 진행도 유지
- **게임 밸런스**: 파워 크리프 방지, 콘텐츠 수명 연장
- **운영 수익성**: 프리미엄 재화 판매 최적화
- **건강한 경제**: 인플레이션 방지, 물가 안정성

### 경제 구조도

```

└────────────┬────────────────────────────────┘
             │
    ┌────────┼────────┬──────────┬────────────┐
    │        │        │          │            │
    ▼        ▼        ▼          ▼            ▼
┌─────┐ ┌──────┐ ┌────────┐ ┌─────────┐ ┌──────────┐
│ 골드  │ │ 보석  │ │ 아이템  │ │  카드   │ │기타 요소│
│(무료)│ │(유료)│ │(등급)│ │(능력)  │ │(경험치) │
└──┬──┘ └──┬───┘ └────┬───┘ └────┬────┘ └────┬─────┘
```

## 재화 시스템

### 1️⃣ 골드 (Gold) — 기본 화폐

**특징:**
- 모든 플레이어가 획득 가능
- 게임 진행에 필수
- 인플레이션 위험

**획득 경로:**
```
몬스터 드롭:         기본 × 1-5 (변동)
퀘스트:             50-1,000
로그인:             100-500
상인 판매:          아이템당 변동
시즌 패스:          1,000-3,000
이벤트:             500-2,000
```

**소비처:**
```
상인 구매:          Common~Epic 아이템
뽑기 (골드 가우차):  500골드/회
길드 숍:            200-1,000골드
기타:               튜토리얼, 특수 기능
```

**월간 예상 순환량:**
```
신입 플레이어 (Lv1-10):    약 5,000 골드 획득
초중급 (Lv10-30):         약 50,000 골드 획득
중급 (Lv30-50):          약 200,000 골드 획득
고급 (Lv50+):            약 500,000+ 골드 획득
```

### 2️⃣ 보석 (Gems) — 프리미엄 화폐

**특징:**
- 유료 판매의 주요 수입원
- 제한적 무료 획득
- 희소성 유지

**획득 경로:**
```
유료 구매:          최소 100원 (1,200 보석)
로그인:             7일 10개, 30일 200개
시즌 패스 (프리):   50-100개
마일스톤:           50-500개
이벤트:             10-100개
선물 (운영):        50-500개 (특수)
```

**소비처:**
```
뽑기 (보석 가우차):   50보석/회
시즌 패스 프리미엄:   4,900보석/월
특수 상품:           1,000-10,000보석
```

**월간 수익 목표:**
```
월간 활동 플레이어:   30,000명
평균 과금율:         3-5%
평균 ARPPU:         3,000-5,000원
월간 예상 수익:      90,000,000 ~ 150,000,000원
```

### 3️⃣ 아이템 (Items)

**특징:**
- 게임 진행도에 필수
- 사고팔 수 없음 (매매 금지)
- 강화/진화 시스템 (미래)

**등급별 획득 난이도:**
```
Common:     초반 쉬움 (30분 내)
Uncommon:   초중반 (1-2시간)
Rare:       중반 (5-10시간)
Epic:       후반 (20-50시간)
Legendary:  극후반 (100+시간 또는 유료)
```

### 4️⃣ 카드 (Cards)

**특징:**
- 게임 플레이의 핵심 메커니즘
- 조합/시너지 중요
- 수집 요소

**획득:**
```
게임 진행:     초반 자동 획득
뽑기:         골드/보석 뽑기로 획득
퀘스트:       특정 카드 선물
마일스톤:     선택 획득
이벤트:       한정 카드
```

## 보상 밸런싱

### 플레이 타임별 기대값

```
┌────────────────────────────────────────────────┐
│      월간 플레이 시간별 예상 획득량              │
└────────────────────────────────────────────────┘

매우 가벼운 플레이어 (5시간/월):
├─ 골드: 10,000~20,000
├─ 보석: 100~150 (로그인만)
├─ 아이템: Common 20-30개
└─ 진행도: Lv10-15

가벼운 플레이어 (20시간/월):
├─ 골드: 50,000~100,000
├─ 보석: 200~300
├─ 아이템: Common 50 + Uncommon 10
└─ 진행도: Lv30-40

적극적 플레이어 (60시간/월):
├─ 골드: 200,000~300,000
├─ 보석: 300~500 (로그인 + 퀘스트)
├─ 아이템: Uncommon 50 + Rare 10-15
└─ 진행도: Lv60-70

매우 적극적 플레이어 (100+시간/월):
├─ 골드: 500,000+
├─ 보석: 500~1,000
├─ 아이템: Rare 30 + Epic 5-10
└─ 진행도: Lv70+

유료 플레이어 (월 50,000원):
├─ 골드: 무료와 동일
├─ 보석: +2,500 (50,000원 환율)
├─ 아이템: Epic 10-20 추가
└─ 진행도: 2-3주 가속
```

### 등급별 획득 시간 기준

```
일반 플레이어가 각 등급 아이템을 획득하는 데 걸리는 시간:

Common:      2시간 (3-5개)
             └─ 초반 몬스터 드롭, 퀘스트

Uncommon:    5시간 (1-2개)
             └─ 중반 몬스터, 스테이지 클리어

Rare:        15시간 (1개)
             └─ 후반 보스, 주간 퀘스트

Epic:        50시간 (1개)
             └─ 극후반, 특수 이벤트, 유료 고려

Legendary:   150+시간 또는 유료
             └─ 마일스톤, 콜라보, 보석 뽑기
```

## 게임 플레이 밸런싱

### 파워 상승 곡선

```
          플레이어 능력
          ^
          │         ╱╱
          │      ╱╱
          │   ╱╱  (Rare 획득 후 가속)
          │╱╱
          ├─────────────────────────► 플레이 시간
    초반  중반    후반      극후반

목표:
├─ 초반 (0-10h): 빠른 진행 (재미 확보)
├─ 중반 (10-30h): 안정적 진행 (몰입 유지)
├─ 후반 (30-60h): 약간의 둔화 (장기 플레이 유도)
└─ 극후반 (60h+): 매우 느린 상승 (자동 진행 유도)
```

### 콘텐츠별 난이도 조정

```
Stage:              Difficulty ← Equipment Level
─────────────────────────────────────────────
Stage 1-10:         ★☆☆☆☆  (Common 권장)
Stage 11-20:        ★★☆☆☆  (Uncommon 필요)
Stage 21-30:        ★★★☆☆  (Rare 권장)
Stage 31-40:        ★★★★☆  (Epic 권장)
Stage 41-50:        ★★★★★  (Legendary 필수)

조정:
- Stage 클리어 불가 → 아이템 획득 기대도 감소 (악순환 방지)
- 충분한 보상 → 다음 스테이지 진행 가능 (선순환)
```

## 운영 정책

### 월별 운영 계획

```
[1월]
├─ 신규 플레이어 유입 집중
├─ 온보딩 보상 강화 (신입 환영 패키지)
├─ 신입 이벤트 (쉬운 콘텐츠, 많은 보상)
└─ 목표: 신규 10,000명

[2월]
├─ 초기 리텐션 관리 (7일, 30일 로그인)
├─ 일일/주간 퀘스트 보상 정상화
├─ 시즌 패스 시작 (1시즌: 60일)
└─ 목표: 30일 리텐션 40%

[3월]
├─ 시즌 이벤트 (한정 아이템 추가)
├─ 콜라보 이벤트 (새로운 수익 기회)
├─ 길드 시스템 활성화
└─ 목표: 일일 활성 플레이어 20,000명

[반기별]
├─ 게임 밸런스 패치
├─ 보상 조정 (인플레이션 확인 후)
├─ 새로운 콘텐츠 추가
└─ 목표: 플레이어 만족도 유지
```

### 인플레이션 관리

```
위험 신호 (Inflation Warning):
├─ 평균 골드 보유량 월 50% 이상 증가
├─ 상인 물품 판매 가격 상승 필요
├─ Rare 아이템 획득 시간 단축
└─ → 조정 필요!

대응 방안:
├─ 1) 보상 감소 (플레이어 불만)
├─ 2) 지출 처 증가 (강화/진화 시스템 추가)
├─ 3) 점진적 조정 (매달 5-10%)
└─ 추천: 2+3 조합
```

## 모니터링 KPI

### 핵심 지표 (Core KPIs)

```
1. 수익 지표
   ├─ ARPU (평균 플레이어 수익): 목표 4,000원/월
   ├─ ARPPU (과금자 평균 수익): 목표 100,000원/월
   ├─ 과금율: 목표 3-5%
   └─ LTV (플레이어 생애 가치): 목표 100,000원

2. 플레이어 지표
   ├─ DAU (일일 활성): 목표 15,000명
   ├─ MAU (월간 활성): 목표 30,000명
   ├─ 신규: 목표 3,000명/월
   ├─ 리텐션 (7일): 목표 50%
   └─ 리텐션 (30일): 목표 30%

3. 게임 플레이 지표
   ├─ 평균 플레이 시간: 목표 40시간/월
   ├─ 평균 레벨: 목표 Lv35
   ├─ 스테이지 진행도: 목표 Stage 25
   ├─ 완료율 (메인): 목표 60%
   └─ 참여율 (이벤트): 목표 70%

4. 경제 지표
   ├─ 평균 골드 보유: 목표 100,000~500,000
   ├─ 평균 아이템 개수: 목표 20-50개
   ├─ Rare 아이템 평균: 목표 3-5개
   └─ 인플레이션율: 목표 5-10%/월
```

```json
{
  "rewardManagementSystem": {
    "gameEconomy": {
      "currency": {
        "gold": { /* 골드 관리 */ },
        "gems": { /* 보석 관리 */ }

      }
    },
    "rewardSources": {
      "monsterDrops": { /* 몬스터 드롭 테이블 */ },
      "questRewards": { /* 퀘스트 보상 */ },
      "gachaDrops": { /* 뽑기 확률 */ },

    "balanceMetrics": {
      "inflationRate": { /* 인플레이션 감시 */ },
      "playerProgression": { /* 플레이어 진행도 */ },
      "rewardDifficulty": { /* 획득 난이도 */ }
    }
  }
}
```
