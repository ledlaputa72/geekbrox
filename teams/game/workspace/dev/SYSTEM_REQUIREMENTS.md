### 1.2 CARD_POOL.md

**상태**: 🔲 미작성  
**우선순위**: 🔴 CRITICAL  
**마감**: 2026-03-05

**목적**: 
게임에 등장할 모든 카드를 명시하는 마스터 스프레드시트 (타로 제외)

**필수 내용**:
```
| 카드 ID | 이름 | 타입 | 비용 | 효과 | 등급 | 해금 조건 |
|---------|------|------|------|------|------|----------|
| C001 | 분노의 기억 | 공격 | 1 | 체력 6 피해 | Common | 게임 시작 |
| ... | ... | ... | ... | ... | ... | ... |
```

**구체적 요구**:
- [ ] 총 카드 개수 (50-100 사이)
- [ ] 각 카드별 이름, 타입(공격/방어/스킬), 비용
- [ ] 카드 효과 명확한 설명
- [ ] 등급 시스템 정의 (Common/Uncommon/Rare/Epic)
- [ ] 카드 획득 로드맵 (어느 막에서 해금?)
- [ ] 카드별 시각적 컨셉 (아트팀용)

**예시**:
```
| C087 | 악몽의 휩쓸림 | 스킬 | 3 | 모든 적에게 2 피해, 1 취약 부여 | Epic | 2막 이후 |
```

**담당자**: 기획팀  
**영향도**: 카드 획득, 카드 강화, 덱 구성 모두 영향

### 1.3 ENEMY_DESIGN.md

**상태**: 🔲 미작성  
**우선순위**: 🔴 CRITICAL  
**마감**: 2026-03-07

**목적**: 
게임에 등장할 모든 적(일반/엘리트/보스)의 설계 명시

**필수 내용**:
```
## 1막 - "길 잃은 아이의 침실"

### 일반 적 (x5)
| ID | 이름 | 체력 | 의도 1 | 의도 2 | 특수 기능 |
|----|------|------|---------|---------|----------|
| E001 | 어린 그림자 | 8 | 공격 3 | 약화 | - |

### 엘리트 (x3)
| E1E001 | 악몽의 파편 | 20 | 공격 7 | 저주 | 턴마다 취약 +1 |

### 보스
| E1B001 | 기억의 괴물 | 50 | 공격 10 (1페이즈) | 공격 15 (2페이즈) | 페이즈 전환 시 HP 회복 |
```

**구체적 요구**:
- [ ] 각 막별 10-15종의 일반 적
- [ ] 각 막별 3-5종의 엘리트 적
- [ ] 각 막별 1-2종의 보스 적
- [ ] 모든 적의 체력, 피해량, 특수 능력 명시
- [ ] 적 이미지 컨셉 (아트팀용)
- [ ] 의도(Intent) 명확한 설명

**담당자**: 기획팀 + 밸런스팀  
**영향도**: 게임 난이도 곡선 전체

### 2.1 BOSS_NARRATIVE.md

**상태**: 🔲 미작성  
**우선순위**: 🟠 HIGH  
**마감**: 2026-03-07

**목적**: 
각 보스가 게임의 서사와 어떻게 연결되는지 명시

**필수 내용**:
```
## 1막 보스: 기억의 괴물
- **서사 역할**: 녹스의 첫 번째 기억 조각을 지키는 존재
- **스토리 대사**: "너... 넌 누구지? 왜 내 기억을 빼앗아?"
- **패배 시**: 녹스가 한 가지 기억 조각(핵심 카드)을 얻음
- **전투 연출**: 초반 약하지만, 중반부터 공격력 급증
- **음악**: "기억의 울림" (보스 테마)
```

**담당자**: 시나리오팀  
**연계**: ENEMY_DESIGN.md + 스토리팀

### 2.3 ECONOMY_BALANCE.md

**상태**: 🔲 미작성  
**우선순위**: 🟠 HIGH  
**마감**: 2026-03-12

**목적**: 
게임의 모든 경제 수치 명시 (골드, 상점, 강화 비용 등)

**필수 내용**:
```
## 보상

| 상황 | 기본 골드 | 보너스 조건 |
|------|---------|-----------|
| 일반 전투 승리 | 50 | 엘리트 처치 시 +50 |
| 상점 방문 | - | 카드 구매 시 내림가 가능 |

## 상점 가격

| 항목 | 기본 가격 | 비용 |
|------|---------|------|
| 카드 (Common) | 75 | 골드 |
| 카드 (Rare) | 200 | 골드 |
| 유물 | 300+ | 골드 |

## 강화 비용

| 강화 대상 | 기본 비용 | 누적 비용 |
|----------|---------|--------|
| 카드 (1회) | 50 | 50 × N |
| 유물 (1회) | 100 | 100 × N |
| 카드 제거 | 75 | - |
```

**담당자**: 밸런스팀  
**연계**: Card Pool + Relic System

### 2.4 CHARACTER_DESIGN.md

**상태**: 🔲 미작성  
**우선순위**: 🟠 HIGH  
**마감**: 2026-03-14

**목적**: 
플레이어가 선택 가능한 캐릭터 설계 (녹스 제외, 2-4명 추가)

**필수 내용**:
```
## 캐릭터 1: 꿈의 현자

- **시작 체력**: 25 (기본 20)
- **시작 에너지**: 3 (기본)
- **시작 카드**: [기억의 파편, 지혜의 목소리, 명상]
- **고유 능력**: 카드 강화 효율 +15%
- **스토리**: 녹스를 돕는 신비로운 존재
```

**구체적 요구**:
- [ ] 초기 2-4개의 캐릭터
- [ ] 각 캐릭터의 시작 수치 (체력, 에너지, 속도)
- [ ] 각 캐릭터의 시작 카드 풀 (3-5장)
- [ ] 고유 능력 (패시브)
- [ ] 캐릭터별 스토리 라인
- [ ] 아트 컨셉 (3~4등신 SD 스타일)

**담당자**: 기획팀 + 아트팀
