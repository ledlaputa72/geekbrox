## 엔진 비교 (결정 전 참고)

### Unity
- 언어: C#
- 장점: 에셋 스토어, 커뮤니티, iOS/Android 지원 강력
- 단점: 라이선스 비용 (수익 발생 시), 무거움
- 적합: 모바일 게임 출시 경험 없는 팀

### Godot
- 언어: GDScript (Python 유사) / C#
- 장점: 완전 무료, 가벼움, 오픈소스
- 단점: 커뮤니티/에셋 상대적으로 작음
- 적합: 소규모 인디, 빠른 프로토타이핑
