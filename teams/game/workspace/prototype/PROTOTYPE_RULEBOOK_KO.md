# 꿈 수집가 - 종이 프로토타입 룰북
**버전:** 1.0  
**작성일:** 2026년 2월 23일  
**프로토타입 유형:** 카드 기반 테이블탑 시뮬레이션

---

## 🎯 목적

이 종이 프로토타입은 **꿈 수집가**의 **핵심 게임플레이 루프**를 시뮬레이션합니다:
- **덱빌딩** (카드 선택)
- **로그라이크 런** (랜덤 인카운터)
- **방치형 진행** (종이 프로토타입용으로 단순화)

**플레이 시간:** 1회 런당 15-30분  
**플레이어:** 1명 (솔로) 또는 2명 (대전)

---

## 📦 게임 준비

### 필요한 재료

1. **카드 덱:**
   - 플레이어 덱 (카드 풀에서 선택한 8-12장)
   - 이벤트 덱 (셔플한 인카운터 카드)
   - 보상 덱 (런 중 획득하는 카드)

2. **토큰:**
   - 레버리 토큰 (동전/구슬 사용) - 50개 이상
   - 에너지 토큰 (5-10개)
   - 체력 토큰 (10개)

3. **게임 보드:**
   - 종이 격자에 8-12개 노드로 간단한 경로 그리기
   - 노드 표시: 기억 (○), 이벤트 (?), 전투 (⚔), 보스 (★)

4. **참조 시트:**
   - 카드 타입 & 효과
   - 액션 비용
   - 승리/패배 조건

### 준비 단계

1. **드리머 선택** (난이도 레벨):
   - 세레니티 (쉬움): 10 HP, 턴당 3 에너지로 시작
   - 앙자이어티 (보통): 8 HP, 턴당 4 에너지로 시작
   - 피어 (어려움): 6 HP, 턴당 5 에너지로 시작

2. **시작 덱 구성:**
   - 기본 카드 풀에서 8장 선택
   - 덱을 셔플하고 4장 드로우 (시작 핸드)

3. **게임 보드 설정:**
   - 이벤트 덱 셔플 (20장)
   - 레버리 토큰을 근처에 배치 ("은행")
   - 체력 및 에너지 트래커 설정

4. **노드 1에서 시작**

## 🎮 핵심 규칙

### 턴 구조

경로의 각 노드는 하나의 턴을 나타냅니다:

```
┌─────────────────────────────────────┐
│ 1. 드로우 페이즈                     │
│    - 카드 1장 드로우 (핸드 제한: 5) │
├─────────────────────────────────────┤
│ 2. 노드 공개                         │
│    - 이벤트 카드 드로우 (? 또는 ⚔)  │
│    - 또는 레버리 수집 (○일 경우)    │
├─────────────────────────────────────┤
│ 3. 액션 페이즈                       │
│    - 에너지를 소모해 카드 플레이     │
│    - 카드 효과 해결                  │
│    - 선택하기 (이벤트 카드)          │
├─────────────────────────────────────┤
│ 4. 종료 페이즈                       │
│    - 초과 카드 버리기 (>5)          │
│    - 에너지를 최대로 리셋            │
│    - 다음 노드로 이동                │
└─────────────────────────────────────┘
```

### 에너지 시스템

- **시작 에너지:** 턴당 3-5 (드리머에 따라)
- **카드 비용:** 0-3 에너지 범위
- **에너지 리셋:** 각 턴 종료 시
- **미사용 에너지:** 이월되지 않음

### 체력 시스템

- **시작 HP:** 6-10 (드리머에 따라)
- **피해:** 전투 인카운터에서 받음
- **힐링:** 희귀 (일부 이벤트 카드, 방어 카드)
- **사망:** HP 0 도달 = 런 실패 (실패 규칙 참조)

## 🗺️ 게임 보드 & 노드 타입

### 경로 레이아웃 (8-12 노드)

10노드 런 예시:

```
시작 → ○ → ? → ○ → ⚔ → ○ → ? → ⚔ → ○ → ★ 종료
```

### 노드 타입

#### ○ 기억 노드 (60% 빈도)
**안전한 수집 지점**

**효과:**
- 기본 레버리 수집 (10 토큰)
- "기억 파편" 카드가 핸드에 있으면 +2 보너스
- 비용 없음, 자동

#### ? 이벤트 노드 (20% 빈도)
**선택 기반 인카운터**

**절차:**
1. 이벤트 덱에서 이벤트 카드 1장 드로우
2. 이벤트 설명 읽기
3. 옵션 A 또는 옵션 B 선택
4. 결과 적용

**이벤트 카드 예시:**
```
[갈림길]
선택 A: 20 레버리 획득 (안전)
선택 B: 동전 던지기 - 앞면: 50 레버리, 뒷면: 10 HP 손실
```

#### ⚔ 전투 노드 (15% 빈도)
**악몽과의 전투**

**절차:**
1. 이벤트 덱에서 전투 카드 1장 드로우
2. 적 HP 및 공격 패턴 확인
3. 액션 카드로 피해 입히기
4. 모든 적 공격 생존

**전투 예시:**
```
[그림자 악마]
HP: 15
공격: 턴 종료 시 3 피해
보상: 30 레버리 + 랜덤 카드 1장
```

#### ★ 보스 노드 (5% 빈도 - 최종 노드)
**최종 인카운터**

**절차:**
1. 보스 카드 드로우
2. 보스는 30-50 HP와 특수 능력 보유
3. 격파하여 런 완료 및 주요 보상 획득

## 🃏 카드 규칙

### 카드 타입

#### 1. 수집 카드 (패시브)
**한 번 플레이하면 이번 런 동안 영구 적용**

**메커니즘:**
- 에너지 비용 지불
- 앞에 배치 (활성 상태 유지)
- 매 관련 턴마다 효과 자동 적용

**예시:**
```
[기억 파편]
비용: 1 에너지
타입: 수집
효과: 각 기억 노드(○)에서 +2 레버리 획득
```

#### 2. 액션 카드 (일회용)
**강력한 일회성 효과**

**메커니즘:**
- 에너지 비용 지불
- 효과를 즉시 해결
- 사용 후 버리기

**예시:**
```
[루시드 스트라이크]
비용: 2 에너지
타입: 액션
효과: 현재 적에게 10 피해
```

#### 3. 시너지 카드 (콤보)
**조합 시 보너스 효과**

**메커니즘:**
- 단독으로 플레이 가능 또는 콤보 파트너와 함께
- 조건 충족 시 콤보 효과 자동 활성화

**예시:**
```
[악몽의 수호]
비용: 2 에너지
타입: 시너지
효과: 5 피해 막기
콤보: [공포 정수]가 플레이 중이면 대신 10 막기
```

#### 4. 방어 카드 (리액티브)
**피해에 대응해 플레이**

**메커니즘:**
- 자신의 턴 외에도 플레이 가능 (즉시)
- 피해 적용 전에 선언해야 함

**예시:**
```
[꿈의 방패]
비용: 1 에너지
타입: 방어
효과: 8 피해 막기 (반응으로 플레이 가능)
```

### 카드 플레이 규칙

**핸드 제한:** 최대 5장  
**드로우:** 턴당 1장 (시작 시)  
**에너지:** 카드는 0-3 에너지 소모  
**버리기:** 핸드 제한 초과 시 턴 종료 때 발생

## 🎲 전투 해결

### 전투 페이즈 단계

⚔ 전투 노드 만났을 때:

```
1. 적 공개
   - 이벤트 덱에서 전투 카드 드로우
   - 확인: 적 HP, 공격력
   
2. 플레이어 턴
   - 액션/공격 카드 플레이
   - 에너지를 소모해 피해 입히기
   - 총 피해 계산
   - 적 HP 감소
   
3. 적 턴
   - 적이 자동으로 공격
   - 피해 = 적 공격력
   - 플레이어는 방어 카드 플레이 가능
   - 플레이어 HP를 (공격 - 방어)만큼 감소
   
4. 반복
   - 적 HP = 0 또는 플레이어 HP = 0이 될 때까지
   
5. 승리/패배
   - 승리: 보상 수집 (레버리 + 새 카드)
   - 패배: 아래 실패 규칙 참조
```

### 피해 계산

**플레이어가 적에게:**
```
총 피해 = 이번 턴에 플레이한 모든 공격 카드의 합
```

**적이 플레이어에게:**
```
받은 피해 = 적 공격력 - 당신의 방어 카드
(최소 0, 음수 불가)
```

### 전투 예시

**설정:**
- 플레이어 HP: 10
- 적: 그림자 악마 (HP: 15, 공격: 3)
- 플레이어 핸드: [루시드 스트라이크] (10 피해, 2 에너지), [꿈의 방패] (8 막기, 1 에너지)
- 플레이어 에너지: 3

**턴 1:**
1. 플레이어가 [루시드 스트라이크] 플레이 (2 에너지 소모)
   - 적 HP: 15 → 5
2. 적 공격 (3 피해)
3. 플레이어가 [꿈의 방패] 플레이 (1 에너지 소모, 8 막기)
   - 피해 완전히 막음
4. 플레이어 HP: 10 (변화 없음)

**턴 2:**
1. 플레이어가 새 카드 드로우, 공격 플레이 (5 피해)
   - 적 HP: 5 → 0 (격파!)
2. 플레이어가 보상 수집: 30 레버리 + 랜덤 카드 1장

**승리 방법:**
- 최종 노드(★ 보스 노드) 도달

**보상:**
- 기본: 100 레버리

| 등급 | 조건 | 보상 배수 |
|------|------|-----------|
| 완벽 | 100% HP 남음 | 2.0x 레버리 |
| 강력 | 50%+ HP 남음 | 1.5x 레버리 |
| 표준 | 보스 격파 | 1.0x 레버리 |

## 💀 패배 조건

### 실패 상태

**런 실패 조건:**
- 플레이어 HP가 0 도달
- 필요할 때 카드를 드로우할 수 없음 (덱 고갈)

### 실패 시 부분 보상

**여전히 획득:**
- 런 중 수집한 레버리의 50%
- 새 카드 없음
- 메타 언락 진행도 (정식 게임에서)

**예시:**
- 노드 6에서 80 레버리를 모은 상태로 사망
- 보유: 40 레버리 (50%)
- 손실: 이후 노드의 잠재적 카드

## 🎴 덱 빌딩 규칙

### 시작 덱 (8장)

**필수 시작 카드:**
- 3x 기억 파편 (○ 노드에서 +2 레버리 수집)
- 2x 기본 공격 (5 피해, 비용 1)
- 1x 꿈의 방패 (8 피해 막기, 비용 1)
- 2x 레버리 폭발 (즉시 10 레버리 획득, 비용 1)

**합계:** 8장

### 덱 확장 (런 중)

**적을 격파하거나 이벤트 완료 시:**
- 보상 덱에서 3장 드로우
- 1장을 선택해 덱에 추가
- 새 카드를 즉시 덱에 셔플

**덱 크기:**
- 최소: 8장
- 최대: 12장 (이 프로토타입 기준)
- 전략: 작은 덱 = 더 일관된 드로우

### 덱빌딩 전략

**시너지 중심:**
- 테마 중심으로 구성 (모두 공격, 모두 수집 등)
- 콤보 카드는 서로를 강화

**시너지 덱 예시:**
```
[공포 정수] + [악몽의 수호] = 2배 방어
[기억 파편] x3 = ○ 노드당 +6 레버리
[루시드 드림] + [드림 위버] = 연쇄 공격
```

## 🔄 방치형 시뮬레이션 (단순화)

### 종이 프로토타입 방치형 규칙

정식 게임에서는 오프라인 중에도 레버리를 획득합니다. 종이 프로토타입의 경우:

**"방치형 런" 모드:**
1. 덱 구성 (8장)
2. 덱의 "방치형 파워" 계산:
   - 각 수집 카드 = +5 포인트
   - 각 시너지 카드 = +3 포인트
3. 1시간의 방치형 시간 시뮬레이션:
   - 획득 레버리 = 방치형 파워 × 10

**예시:**
- 덱에 수집 카드 4장, 시너지 카드 2장
- 방치형 파워 = (4 × 5) + (2 × 3) = 26
- 1시간 = 260 레버리 획득

**사용 예:**
- 플레이테스트 런 사이에 획득했을 양 계산
- 진행 균형 테스트에 도움

## 📊 점수 & 진행

### 레버리 점수

**레버리 획득 방법:**
- 기억 노드 (○): 노드당 10-15
- 전투 승리: 적당 30-50
- 이벤트: 10-100 (위험/보상)
- 보스 격파: 100 기본 + 보너스
- 방치형 (시뮬레이션): 시간당 덱 파워 × 10

### 메타 진행 (런 사이)

**레버리 소비 (정식 게임에서):**
- 카드 업그레이드 (레벨당 20 레버리)
- 새 드리머 언락 (100 레버리)
- 프레스티지 보너스 (다양)

**종이 프로토타입의 경우:**
- 여러 런에 걸쳐 누적 레버리 추적
- 5번의 런 후, 플레이어가 업그레이드 카드 1장 "구매" 허용

### 테스트 대상

1. **밸런스:**
   - 피해가 균형 잡혔나? (너무 쉽지/어렵지 않나?)
   - 레버리 보상이 공정한가?
   - 에너지 비용이 적절하게 느껴지나?

2. **페이싱:**
   - 10 노드가 너무 길거나/짧나?
   - 전투 인카운터가 만족스러운가?

3. **전략:**
   - 다양한 덱 빌드가 가능한가?
   - 시너지 콤보가 강력하게 느껴지나?

4. **재미:**
   - 선택이 의미 있나?
   - 다른 덱으로 다시 플레이하고 싶나?

### 일반적인 문제 & 해결책

| 문제 | 해결책 |
|------|--------|
| 너무 쉬움 | 적 HP 증가, 시작 HP 감소 |
| 너무 어려움 | 방어 카드 추가, 적 공격력 감소 |
| 지루함 | 흥미로운 선택이 있는 이벤트 카드 추가 |
| 너무 김 | 경로를 8 노드로 축소 |
| 카드가 비슷함 | 시너지 보너스 증가 |
