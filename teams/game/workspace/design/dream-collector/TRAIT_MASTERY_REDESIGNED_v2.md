# 🎯 Dream Collector — 특성 시스템 재설계 v2.0 (플레이어 선택 중심)

**작성:** Atlas (PM) → Game팀  
**기준:** 모든 성장이 플레이어의 명시적 선택에서 나오도록 재설계  
**버전:** v2.0 (완전히 다른 접근)  
**상태:** 📋 최종 설계

---

## 📌 설계 철학 변경

### 이전 설계의 문제점

```
❌ 강화된 무기 착용 → 자동으로 마스터리 레벨 증가
   → 플레이어가 선택한 게 아님
   → 숨겨진 메커니즘
   → 왜 이 보너스를 주는가? 이유 불명확

❌ 무기 +10 강화 + 자동 특성 = 인플레이션에 추가 보너스
   → 게임 밸런스 불안정
   → 플레이어가 의도한 성장이 아님
```

### 새로운 설계 원칙

```
✅ 모든 성장 = 플레이어의 명시적 선택
   (레벨업과 기본 스탯은 예외)

✅ "강화"와 "특성 강화"는 별개
   - 강화: 장비의 스탯만 증가 (무기 ATK +20%)
   - 특성: 플레이어가 수동으로 투자 (특성 포인트/특성석 사용)

✅ 모든 선택에 명확한 비용과 보상
   - 비용이 명확 (특성 포인트, 특성석, 골드)
   - 보상이 명확 (%로 표시)
   - 트레이드오프가 있음 (이걸 쓰면 저건 못 씀)
```

## 🎮 재설계된 특성 시스템

### 핵심 변경: 마스터리는 "특성 포인트"로 투자

```
기존:
  강화된 무기 착용 → 자동 특성 레벨 증가
  
새로운 방식:
  강화된 무기 착용 → 장비의 스탯만 증가 (기본 공격력 ↑)
  + 플레이어가 명시적으로 "특성 포인트" 투자
    → "무기 마스터리에 3 포인트 투자"
    → 보너스 +15% 획득
```

## 1️⃣ 무기 마스터리 (Weapon Mastery) - 재설계

### 이전

```
무기 +5 강화 → 마스터리 Lv 5 (자동) → +30% 보너스
```

### 새로운 방식

```
Step 1: 무기 강화
  무기 +5 강화 (골드 투자)
  → 무기 ATK 20 → 26 (스탯만 증가)
  
Step 2: 플레이어 선택 - 특성 포인트 투자
  "무기 마스터리에 포인트를 줄까?"
  → 특성 포인트 5개 소비
  → 무기 마스터리 Lv 5 획득
  
Step 3: 결과
  기본 공격: 26 × (1 + 25%) = 32.5 데미지
  
비용:
  강화: 2,000 G
  특성 포인트: 5pt (다른 곳에 쓸 수 없음)
  
선택:
  "이 5포인트를 무기에 줄까? 방어에 줄까? 속도에 줄까?"
```

### 특성 포인트 분배 방식

```
기본 특성 포인트 (매 레벨업):
  +5pt (특성 포인트)
  
분배 대상:
  1. 무기 마스터리 (1pt = +5% 공격력)
  2. 방어 강화 (1pt = +5% 방어력)
  3. 악세서리 동조 (1pt = +5% 악세서리 효과)
  4. 듀얼 웰드 (1pt = +5% 추가 데미지)
  5. 기본 스탯 (ATK/DEF/HP/SPD)
  
트레이드오프:
  "5pt를 무기에만 줄까? 아니면 골고루 줄까?"
  → 플레이어의 명확한 전략 선택
```

### 효과 공식 (같음, 하지만 명시적 투자)

```
무기 마스터리 보너스 = 투자한 포인트 × 5%

예:
  5pt 투자 → +25%
  10pt 투자 → +50%
  15pt 투자 → +75% (최대)
```

## 2️⃣ 방어 강화 (Defense Mastery) - 재설계

### 새로운 방식

```
Step 1: 방어구 강화 (골드 투자)
  방어구 +5 강화
  → 방어력만 증가
  
Step 2: 플레이어 선택 - 특성 포인트 투자
  "방어 마스터리에 포인트를 줄까?"
  → 특성 포인트 5개 소비
  → 방어력 +25%, HP +25% 추가
  
선택지:
  "공격을 강화할까? 방어를 강화할까?"
```

## 3️⃣ 악세서리 동조 (Accessory Resonance) - 재설계

### 새로운 방식

```
Step 1: 악세서리 획득 (가챭 필요)
  공격 반지 획득 (+3 ATK)
  → 착용하면 기본 +3 ATK
  → 추가 포인트 없이도 효과 있음
  
Step 2: 추가 효과 원하면 포인트 투자 (선택)
  "악세서리의 효과를 강화할까?"
  → 특성 포인트 3개 소비
  → 악세서리 효과 +15% 추가 증폭
  
예:
  반지: 기본 +3 ATK
  + 포인트 3개 투자 (추가 동조 강화)
  → 반지 효과: +3 × 1.15 = +3.45 ATK

선택지:
  "반지 여러 개는 기본 효과만으로도 좋으니
   포인트를 공격/방어에 줄까?"
```

## 4️⃣ 듀얼 웰드 (Dual Wield) - 재설계

### 새로운 방식

```
Step 1: 보조 무기 획득 (가챭 필요)
  단검 획득 (ATK 8)
  → 착용하면 기본 추가 데미지 발생
  
Step 2: 강화 원하면 포인트 투자 (선택)
  "보조 무기의 효과를 강화할까?"
  → 특성 포인트 4개 소비
  → 추가 데미지 +20% 증가
  
예:
  단검: 기본 8 ATK → 8 × 1.20 = 9.6 ATK

선택지:
  "보조 무기는 기본이 좋으니
   포인트를 다른 곳에 줄까?"
```

## 5️⃣ 장비 보너스 (Equipment Bonus) - 재설계

### 새로운 방식

```
Step 1: 여러 장비 강화 (골드 투자)
  무기 +3, 방어구 +2, 악세서리 +1 강화
  → 각각의 스탯만 증가
  → 별도 보너스는 없음
  
Step 2: 추가 보너스 원하면 포인트 투자 (선택)
  "여러 장비를 갖춘 보너스를 받을까?"
  → 특성 포인트 8개 소비
  → 모든 스탯 +8% 추가
  
조건:
  이 보너스를 얻으려면 최소 4개 장비 강화 필수
  (기본 조건)
  
선택지:
  "4개 장비를 강화했으니 보너스를 받을까?
   아니면 포인트를 다른 특성에 줄까?"
```

## 📊 새로운 특성 시스템 정리

### 포인트 투자 방식

```
레벨업 보상: +5 특성 포인트

분배 대상 (모두 선택):
  1. 무기 마스터리: 1pt = +5% 공격력
  2. 방어 강화: 1pt = +5% 방어력
  3. 악세서리 동조: 1pt = +5% 악세서리 효과
  4. 듀얼 웰드: 1pt = +5% 추가 데미지
  5. 기본 스탯: ATK/DEF/HP/SPD
  
최대 투자:
  각 마스터리별 최대 15pt (이상 무의미)
  
선택의 자유:
  "이번 레벨업에는 공격에 집중할까?
   아니면 방어를 강화할까?"
```

### 명확한 비용-보상

```
예시:

무기 마스터리에 5pt 투자
  비용: 5pt (다른 곳에 못 씀)
  보상: +25% 공격력
  → 명확함

방어 강화에 8pt 투자
  비용: 8pt
  보상: +40% 방어력, +40% HP
  → 명확함

트레이드오프:
  무기 5pt + 방어 8pt = 13pt 사용
  남은: 2pt
  "이 2pt를 어디에 줄까?"
  → 플레이어의 명확한 전략 선택
```

## 🎮 플레이어 경험 (새로운 방식)

### Day 1 신규 플레이어

```
기본 무기로 시작
→ 첫 전투: 15 데미지 (특성 포인트 없음)

레벨업
→ 특성 포인트 +5 획득

선택 순간:
  "포인트를 어디에 줄까?"
  
  옵션 1: 무기 마스터리에 5pt
    → 무기 공격력 +25%
    → 더 강해 보임
    
  옵션 2: 기본 스탯 ATK에 5pt
    → 모든 공격력 +5
    → 보너스가 더 강함
    
선택 시 느낌:
  "내가 선택했더니 강해졌다!"
  (자동이 아니라 명시적 선택)
```

### Week 2 일반 플레이어

```
누적 특성 포인트: 50pt

분배 고민:
  "무기 마스터리 10pt, 방어 15pt, 기본 스탯 25pt"
  또는
  "균형: 12pt, 12pt, 12pt, 14pt"
  
플레이 스타일별 선택:
  Archer (공격형): 무기 마스터리에 집중
  Defender (방어형): 방어 강화에 집중
  Balanced: 고르게 분배
  
느낌:
  "내 선택이 내 캐릭터를 만든다"
  (완전히 플레이어의 의도)
```

### Month 1 극한 플레이어

```
누적 특성 포인트: 150pt+

전략적 분배:
  무기 마스터리 12pt → +60%
  방어 강화 10pt → +50%
  악세서리 동조 8pt → +40%
  듀얼 웰드 10pt → +50%
  기본 스탯: 110pt → 모든 스탯 대폭 증가
  
최종 기본 공격:
  기본 25
  × (1 + 0.60) [무기]
  × (1 + 0.40) [악세서리]
  × (1 + 0.50) [듀얼]
  × (1 + 0.50) [장비 보너스]
  + 기본 스탯 보너스
  = 약 80+ 데미지
  
느낌:
  "완전히 내가 만든 캐릭터다"
  "모든 선택이 의미가 있다"
```

## 💰 재화 연계 (단순해짐)

### 골드

```
사용처:
  1. 장비 강화 (메인)
  2. 카드 강화 (부수)
  
특성과의 관계:
  강화 → 장비 스탯 증가
  + 플레이어가 별도로 포인트 투자
  = 명확한 분리
```

### 다이아

```
사용처:
  1. 가챭 (장비/카드 획득)
  2. 특성 포인트 구매 (선택적, 비추)
  
특성과의 관계:
  가챭 → 장비 획득
  + 플레이어가 포인트 투자로 강화
  = 자연스러운 유료화
```

## 📊 강화 vs 특성 포인트 (이제 명확)

### 강화 (Gear Enhancement)

```
비용: 골드
효과: 장비 스탯만 증가

예:
  무기 +0 → +5
  ATK 20 → 25
  → 강화도만 올라감
  
특징:
  자동 효과 없음
  장비를 더 찾아야 함 (가챭)
```

### 특성 포인트 (Trait Points)

```
비용: 특성 포인트
효과: 보너스 %

예:
  무기 마스터리에 5pt 투자
  → 무기 공격력 +25% 추가
  
특징:
  플레이어 선택
  다른 곳에 안 쓸 수 있음 (선택)
  명확한 트레이드오프
```

### 최종 공격력

```
최종 = 기본 스탯 
     + 강화 보너스 (장비 강화도에서)
     + 특성 포인트 보너스 (플레이어 투자에서)

모든 것이 명확하고 플레이어가 조작함
```

## ✅ 새 설계의 장점

```
명확성:
  ✅ 강화 = 장비 스탯 증가 (자동)
  ✅ 특성 = 플레이어 포인트 투자 (선택)
  ✅ 모두 분명히 구분됨

플레이어 선택:
  ✅ 매 레벨업마다 포인트 분배 선택
  ✅ 어떤 마스터리를 강화할지 결정
  ✅ 플레이 스타일 자유도 극대화

게임 밸런스:
  ✅ 강화로 인한 인플레 관리 쉬움
  ✅ 특성 포인트는 제한적 (총량 고정)
  ✅ 더 안정적인 경제

모바일 게임 친화적:
  ✅ 귀찮은 자동 관리 없음
  ✅ 명확한 선택만 있음
  ✅ UI 간단함
```

## 🎯 Game팀 구현

### 데이터 구조

```
Player.traits:
  - weapon_mastery_points: 10
  - defense_mastery_points: 8
  - accessory_resonance_points: 5
  - dual_wield_points: 6
  - equipment_bonus_points: 4

각 포인트 = 직접적인 보너스%
= 계산이 단순함
```

### UI

```
레벨업 시:
  특성 포인트 +5 획득
  
분배 화면:
  [무기 마스터리] [+] [-] 현재: 10pt
  [방어 강화] [+] [-] 현재: 8pt
  [악세서리 동조] [+] [-] 현재: 5pt
  [듀얼 웰드] [+] [-] 현재: 6pt
  [기본 스탯] [+] [-] 현재: 4pt
  
  분배 완료 버튼

표시:
  각 마스터리의 현재 효과 %
  포인트 투자로 얻을 수 있는 효과
```

### 계산

```
기본 공격 = 무기 ATK
          × (1 + weapon_mastery_points × 5%)
          × (1 + accessory_resonance_points × 5%)
          ... 등등
          
+ 기본 스탯 (특성 포인트로 직접 증가)

모두 명확하고 투명함
```
