# 🎮 Dream Collector — 게임 철학 분석 (StS + 방치형 + 로그라이크)

**작성:** Atlas (PM)  
**기준:** 기존 기획의 StS/방치형/로그라이크 특징 검증  
**버전:** v1.0  
**상태:** 📋 철학적 재검토 + 개선안

---

## 📌 개요

**목표:** Dream Collector가 3가지 장르 특징을 제대로 반영했는지 검증하고, 부족한 부분을 식별하여 개선안 제시

**3가지 핵심 장르:**
1. StS (Slay the Spire) - 로그라이크 덱빌딩
2. 방치형 게임 (Idle/Incremental Game) - 자동 진행
3. 로그라이크 (Roguelike) - 매 플레이 다른 경험

## 1️⃣ StS 특징 검증

### StS의 핵심 메커니즘

```
1. 덱 빌딩 (Deck Building)
   - 매 전투/아이템 선택 후 덱 구성 변함
   - 제한된 선택지로 최적의 조합 찾기
   - 높은 전술성

2. 로그라이크 진행 (Run-based)
   - 보스 선택: 3가지 경로 중 선택
   - 매 플레이 다른 시나리오
   - 완료 시 초기화 후 재시작

3. 리스크-리워드 (Risk/Reward)
   - 어려운 보스 선택 vs 쉬운 보스
   - 생명력/카드 트레이드오프

4. 높은 리플레이 (High Replayability)
   - 매 플레이가 다름
   - 100+ 시간 플레이 가능
```

### 현재 Dream Collector의 StS 반영도

```
✅ 덱 빌딩 (카드 시스템)
   - 200종 카드, 손패 5장
   - 매 전투마다 다른 카드 선택
   - 카드 조합의 전술성 높음
   - 평가: 95/100 ✅

⚠️ 보스 선택 및 경로 선택
   - 현재: 스토리 진행이 선형 (고정)
   - 문제: "3가지 경로 중 선택" 메커니즘 없음
   - 해결책 필요

✅ 리스크-리워드
   - 위험한 카드 (CURSE) vs 안전한 카드
   - 고비용 강력한 카드 vs 저비용 약한 카드
   - 평가: 90/100

✅ 리플레이 가능성
   - 200종 카드로 다양한 조합 가능
   - 평가: 85/100

전체 StS 반영: 91/100 (개선 가능)
```

### 부족한 부분 & 개선안

#### 개선안 1: 던전 선택 시스템 추가 (Critical)

```
현재:
  └─ 스토리 진행 (Lv 1-100 선형)
     └─ 막 1 → 막 2 → 막 3 → ... (고정)
     
문제:
  - 매 플레이가 항상 같은 경로
  - "다른 보스를 선택할 수 있다"는 자유도 없음
  - StS의 핵심인 "경로 선택"이 없음

개선안 1A: 로그라이크 모드 추가 (권장)
  ┌─ Roguelike Mode (특별 던전)
  │  └─ 매 시작 시 다른 경로:
  │     ├─ 녹색 보스 (쉬움, 보상 적음)
  │     ├─ 빨강 보스 (중간, 보상 중간)
  │     └─ 검정 보스 (어려움, 보상 많음)
  │  └─ 매 전투 후 3가지 경로 중 선택
  │  └─ 클리어 시 특별 칭호 & 보상
  │
  └─ Story Mode (기존 메인 스토리)
     └─ 진행도 명확하고 선형 (교육용)

효과:
  - StS 특징 100% 구현
  - 리플레이 가능성 대폭 증가
  - 극한 게이머 콘텐츠 추가
  
예상 시간: 2-3주 (구현)
우선순위: 높음 ⭐⭐⭐ (2.0 또는 1.1 패치)

개선안 1B: 스토리 내에서 부분 선택 추가 (선택)
  각 막마다 2-3가지 경로 선택:
    막 2: "숲" or "도시" or "탑" 순서 선택
    → 같은 장소지만 순서가 다르면 장비/카드 선택지도 달라짐
```

#### 개선안 2: 난이도 선택 시스템 (Important)

```
현재:
  난이도 고정 (각 레벨마다)
  
문제:
  극한 게이머: 콘텐츠 너무 쉬움 → 이탈
  캐주얼: 점점 어려워짐 → 이탈
  
개선안:
  각 던전마다 3가지 난이도 선택:
    Normal (기본): 표준 난이도
    Hard: 적 체력 +30%, 보상 +50%
    Nightmare: 적 체력 +50%, 보상 +100%
    
  또는 스토리별 난이도 (StS 스타일):
    Ascension 시스템:
      Ascension 1: 기본
      Ascension 2: 적 체력 +10%
      Ascension 3: 적 체력 +20%
      ...
      Ascension 20: 최고 난이도
```

## 2️⃣ 방치형 게임 특징 검증

### 방치형 게임의 핵심 메커니즘

```
1. 자동 진행 (Automation)
   - 최소 개입으로 계속 진행
   - 플레이어가 없어도 게임이 돈다

2. 성장 체감 (Growth/Progression)
   - 작은 변화를 누적하여 큰 성장으로
   - 지수적 증가 곡선

3. 시간 대비 보상 (Time Value)
   - 오래 플레이 = 더 많은 자원
   - 매분 자동으로 뭔가 증가

4. 중독성 (Addictive Loop)
   - 숫자 증가의 쾌감
   - 계속 플레이하고 싶은 느낌
```

### 현재 Dream Collector의 방치형 반영도

```
✅ 자동 진행 (기본 공격 ATB)
   - 플레이어가 카드 선택하면 자동 공격
   - 전투 중에도 자동 진행
   - 평가: 95/100 ✅

⚠️ 성장 체감
   - 레벨업: Lv 1-100 (비선형, 느림)
   - 강화: +0~+10 (선형)
   - 문제: 후반부 성장 체감 낮음
   - 평가: 70/100 ⚠️

⚠️ 시간 대비 보상
   - 일일 던전: 정해진 시간에만 가능
   - 보상: 시간 비례하지 않음
   - 문제: "오래 하면 더 많이 얻는다"는 느낌 없음
   - 평가: 60/100 🔴

⚠️ 중독성 (숫자 증가 쾌감)
   - 기본 공격 데미지는 천천히만 증가
   - 숫자 증가 체감 낮음
   - 평가: 65/100 ⚠️

전체 방치형 반영: 72/100 (개선 필요)
```

### 부족한 부분 & 개선안

#### 개선안 1: 자동 강화 시스템 (Critical)

```
현재:
  플레이어가 매번 강화 선택해야 함
  "방치형"인데 수동 개입 많음

개선안:
  "자동 강화" 토글 옵션:
    활성화 시:
      - 획득한 장비 자동으로 강화 시도
      - 실패하면 다른 장비 강화
      - 골드 조건에 따라 자동 배분
      
    플레이어는:
      - 세팅만 하고 방치 가능
      - 강화 우선순위 설정 (무기 > 방어 > 악세)
      - 강화 비용 제한 설정
      
  효과:
    - 진정한 "방치형" 경험
    - 오래 할수록 더 강해지는 느낌
    - 수동 플레이도 여전히 가능 (토글 끄기)
```

#### 개선안 2: 자동 카드 선택 (권장)

```
현재:
  매 전투마다 플레이어가 카드 선택
  
개선안:
  "자동 카드 선택" 모드:
    AI가 자동으로 최적 카드 선택
    
  난이도별:
    Easy AI: 무작위
    Normal AI: 데미지 우선
    Hard AI: 전술 고려
    
  적용:
    - 스토리 플레이할 때 "자동 모드" 선택 가능
    - 매 전투 자동으로 진행
    - 플레이어는 "보기만" 함
    
  효과:
    - 진정한 방치형 경험
    - 바쁜 플레이어도 콘텐츠 접근 가능
    - 리플레이할 때는 수동 플레이로 재미 증가
```

#### 개선안 3: 비활성 수입 시스템 (Important)

```
현재:
  플레이하지 않으면 아무것도 얻을 수 없음
  
개선안:
  오프라인 수입:
    플레이하지 않은 시간동안:
      - 골드 누적 (시간당 10-50 G)
      - 경험치 누적 (시간당 1-5 EXP)
      - 카드 드롭 (12시간마다 1장)
    
  상한선:
    - 최대 12시간치만 누적 가능
    - "플레이하지 말고 방치하기"는 막되
    - "플레이할 시간이 없으면 뭐라도 얻기" 가능
    
  효과:
    - 방치형 게임의 핵심: 오프라인 수입
    - 바쁜 플레이어도 진행 가능
    - 매일 접속 유도
```

#### 개선안 4: 성장 곡선 최적화 (Important)

```
현재:
  Lv 1-50: 빠름
  Lv 50-100: 느림
  
개선안:
  더 가파른 성장 곡선:
    레벨업 보상 증가:
      Lv 10: 특별 보상 (카드 5장)
      Lv 20: 특별 보상 (카드 10장 선택)
      Lv 30: 특별 보상
      Lv 40: 특별 보상
      ...
      Lv 100: 특별 보상 (최고의 칭호)
    
    강화 보상 증가:
      성공할 때마다 연쇄 보너스
      5연 성공: +10% 보상
      10연 성공: +20% 보상
      
  효과:
    - 성장 체감 증가
    - 숫자 증가 쾌감 증가
    - 계속 플레이하고 싶은 느낌
```

## 3️⃣ 로그라이크 특징 검증

### 로그라이크의 핵심 메커니즘

```
1. 매 플레이가 다름 (Variability)
   - 무작위 요소 (아이템, 적, 보스)
   - 매 플레이 완전히 다른 경험

2. 진행도 불확실 (Uncertainty)
   - 어디까지 갈 수 있을지 모름
   - 예측 불가능한 도전

3. Permanent Death (Permadeath) 또는 Progress Save
   - StS: 실패해도 경험/보상 유지
   - 전통 로그라이크: 모두 초기화

4. 높은 리플레이 (High Replayability)
   - 다양한 카드/아이템 조합
   - 100+ 시간 플레이 가능
```

### 현재 Dream Collector의 로그라이크 반영도

```
⚠️ 매 플레이가 다름 (Variability)
   - 스토리: 고정 (항상 같은 장소, 같은 순서)
   - 카드: 무작위 (다양함)
   - 장비: 무작위 (다양함)
   - 문제: 스토리 경로는 항상 같음
   - 평가: 65/100 ⚠️

⚠️ 진행도 불확실 (Uncertainty)
   - 스토리: 결정적 (언제 끝날지 알 수 있음)
   - 문제: "어디까지 갈 수 있을까"라는 의문 없음
   - 평가: 60/100 ⚠️

❌ Permadeath / Progress Save
   - 현재: 진행도 계속 유지 (방치형 스타일)
   - 로그라이크: 보통 매 런마다 초기화
   - 문제: 하이브리드 모델이 필요할 수도
   - 평가: 50/100 🔴

✅ 높은 리플레이 (Replayability)
   - 200종 카드 조합
   - 66종 장비 조합
   - 3가지 플레이 스타일
   - 평가: 90/100 ✅

전체 로그라이크 반영: 66/100 (개선 필요)
```

### 부족한 부분 & 개선안

#### 개선안 1: 런 기반 모드 추가 (StS 스타일)

```
현재:
  Story Mode만 있음 (Lv 1-100 장기 진행)

개선안:
  Roguelike Run Mode 추가:
    
    Run의 특징:
      1. 매 시작: 난이도 선택 (Normal/Hard/Nightmare)
      2. 첫 전투: 카드 3장 선택 (시작 덱)
      3. 각 전투 후: 3가지 경로 선택
         - 다음 보스 전투
         - 상점 (카드 구매)
         - 쉬기 (체력 회복)
      4. 5-10개 보스 격파 시 "Run 완료"
         → 특별 보상 (칭호, 카드 언락, 골드)
      5. Run 실패 시:
         → Story Mode 진행도는 유지 (방치형)
         → Run 기록만 초기화
    
    Run의 길이:
      - 한 번에 30-60분
      - 손쉽게 시작/종료 가능
      - Story는 길게, Run은 짧게
    
    효과:
      - StS 경험 100% 구현
      - 로그라이크 특징 100% 구현
      - Story와 Run 2가지 플레이 방식 제공
      - 극한 게이머와 캐주얼 게이머 모두 만족
```

#### 개선안 2: 무작위 이벤트 추가 (Important)

```
현재:
  전투와 보상만 있음
  
개선안:
  스토리 진행 중 무작위 이벤트:
    
    예시:
      "낯선 인물이 나타났다"
      - 선택지 1: 도움을 청한다 → 카드 획득
      - 선택지 2: 외면한다 → 시간 절약
      - 선택지 3: 싸운다 → 보스 전투 (위험)
      
      "갑자기 폭우가!"
      - 공격력 -20%, 방어력 +10%
      - 그 다음 전투에만 적용
      
      "신비한 상자를 발견했다"
      - 선택지:
        1. 열어본다 → 무작위 보상
        2. 무시한다 → 아무것도 안 함
    
    효과:
      - 매 플레이가 다른 경험
      - 진행도 불확실성 증가
      - 스토리 몰입감 증가
```

#### 개선안 3: 무작위 보스 및 적 생성 (Nice to have)

```
현재:
  매 지역마다 같은 보스
  
개선안:
  보스 풀에서 무작위 생성:
    
    예시:
      Forest of Whispers 지역:
        - Woodland Spirit (기본 보스)
        - Shadow Beast
        - Ancient Guardian
        - 위의 3가지 중 무작위 생성
      
      적의 조합도 무작위:
        - 항상 슬라임 3마리가 아님
        - 슬라임 2 + 박쥐 1 또는
        - 슬라임 1 + 고블린 2 또는
        - 박쥐 3
      
    효과:
      - 같은 지역이어도 매번 다른 경험
      - 전술 유연성 필요
      - 리플레이 가능성 증가
```

## 🎯 개선안 우선순위

### Critical (필수, 2.0 버전)

```
1. 로그라이크 런 모드 추가 (1-2주)
```
