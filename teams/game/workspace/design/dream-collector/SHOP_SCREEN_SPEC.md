## 💎 탭 1: 보석 구매 (IAP)

### **레이아웃**
- 3열 그리드
- 카드 크기: 110×160px
- 간격: 12px (가로/세로)
- 중앙 정렬

### **패키지 카드 구조**
```
┌────────────┐
│ 스타터 팩   │ ← 패키지 이름 (12px)
├────────────┤
│            │
│     💎     │ ← 아이콘 (32px)
│    100     │ ← 수량 (18px, Primary)
│            │
│ +0 보너스   │ ← 보너스 (8px, 있는 경우만)
├────────────┤
│  [$0.99]   │ ← 구매 버튼
└────────────┘
```

### **6개 패키지**
| 순서 | 이름 | 보석 | 보너스 | 가격 |
|------|------|------|--------|------|
| 1 | 스타터 팩 | 100 | - | $0.99 |
| 2 | 베스트 딜 | 500 | +50 | $4.99 |
| 3 | 메가 팩 | 1,000 | +200 | $9.99 |
| 4 | 슈퍼 팩 | 2,000 | +600 | $19.99 |
| 5 | 울트라 팩 | 5,000 | +500 | $49.99 |
| 6 | 메가 번들 | 10,000 | +2,000 | $99.99 |

### **상호작용**
1. **구매 버튼 클릭**
   - → IAP 결제 프로세스 (Google/Apple)
   - 성공 시: 모달 "구매 완료" + 보석 지급
   - 실패 시: 에러 메시지

2. **개발 버전**
   - IAP 스킵, 바로 보석 지급
   - 모달: "💎 X 보석을 받았습니다! (개발 버전: 무료 지급)"

## 🪙 탭 2: 재화 교환

### **레이아웃**
- 세로 리스트
- 아이템 크기: 358×70px
- 간격: 12px
- 중앙 정렬

### **교환 아이템 구조**
```
┌─────────────────────────────────┐
│ 🪙  골드 1,000    💎 10  [교환]  │
└─────────────────────────────────┘
```

**구성:**
- 아이콘 (32px) + 이름 (16px) - 왼쪽
- 필요 보석 (💎 X, 14px) - 중앙
- 교환 버튼 (80×40px) - 오른쪽

### **6개 교환 옵션**
| 순서 | 아이콘 | 이름 | 필요 보석 | 받는 재화 |
|------|--------|------|----------|-----------|
| 1 | 🪙 | 골드 1,000 | 💎 10 | 🪙 1,000 |
| 2 | 🪙 | 골드 5,000 | 💎 45 | 🪙 5,000 |
| 3 | 🪙 | 골드 10,000 | 💎 80 | 🪙 10,000 |
| 4 | ⚡ | 에너지 50 | 💎 5 | ⚡ 50 |
| 5 | ⚡ | 에너지 100 | 💎 9 | ⚡ 100 |
| 6 | ⚡ | 에너지 250 | 💎 20 | ⚡ 250 |

### **상호작용**
1. **교환 버튼 클릭**
   - 보석 충분 → 교환 성공
     - 모달: "교환 완료" + "🪙 골드 1,000를 받았습니다!"
     - 상단 카운터 업데이트
   
   - 보석 부족 → 실패
     - 모달: "보석 부족"
     - 메시지: "필요: 10, 보유: 5"
     - 버튼 2개: [💰 재충전] [닫기]

## 🪟 AlertModal (재화 부족)

### **구조**
```
┌─────────────────────┐
│   보석 부족          │ ← 타이틀 (파란색, 18px)
├─────────────────────┤
│ 보석이 부족합니다.   │
│                     │
│ 필요: 10            │
│ 보유: 5             │
├─────────────────────┤
│ [💰 재충전] [닫기]   │ ← 버튼 2개
└─────────────────────┘
```

### **버튼 동작**
1. **💰 재충전 버튼**
   - → 💎 보석 구매 탭으로 전환
   - 모달 닫기

2. **닫기 버튼**
   - 모달 닫기
   - 현재 화면 유지

3. **배경 클릭**
   - 모달 닫기

## 🎯 상호작용 흐름

### **시나리오 1: 보석 구매**
```
1. Shop 진입
2. "💎 보석 구매" 탭 (기본)
3. 패키지 선택 (예: 베스트 딜 $4.99)
4. IAP 결제 프로세스
5. 성공 시:
   - 모달: "구매 완료"
   - 💎 550 보석 지급
   - 상단 카운터 업데이트
```

### **시나리오 2: 재화 교환 (성공)**
```
1. "🪙 재화 교환" 탭 클릭
2. "골드 1,000" 교환 버튼 클릭
3. 보석 체크 (💎 10 필요, 💎 50 보유)
4. 교환 성공:
   - 💎 50 → 💎 40
   - 🪙 0 → 🪙 1,000
   - 모달: "교환 완료"
```

### **시나리오 3: 재화 교환 (실패 → 구매)**
```
1. "🪙 재화 교환" 탭
2. "골드 1,000" 교환 버튼 클릭
3. 보석 부족 (💎 5 보유, 💎 10 필요)
4. 모달: "보석 부족"
5. [💰 재충전] 버튼 클릭
6. → "💎 보석 구매" 탭으로 전환
7. 패키지 구매 → 교환 재시도
```
