# 🎲 Dream Collector — 가챠 시스템 통합 기획 (가챠 ↔ 강화 완전 연결)

**작성:** Atlas (PM) → Game팀  
**기준:** 가챠 시스템 + 강화 시스템 통합 설계  
**버전:** v1.0  
**상태:** 📋 완전 통합 시스템 설계 완료

---

## 📌 개요

**목표:** 가챠 시스템과 강화 시스템의 완전한 연결

**핵심 개념:**
```
가챠 (뽑기)
  ↓
장비/카드 획득 (희귀도별)
  ↓
낮은 희귀도 → 강화 재료로 변환
높은 희귀도 → 직접 사용 또는 강화
  ↓
강화
  ↓
더 강한 캐릭터
```

## 1️⃣ 가챠 시스템 (Gacha System)

### A. 장비 가챠 (Equipment Gacha)

**가챠 구성:**

```
가챠 비용: 50 다이아 / 1회
10회: 450 다이아 (10% 할인)
연속: 누적 횟수에 따라 특정 구간에서 보장

보장 시스템:
  50회 내: 최소 1회 Rare 이상
  100회 내: 최소 1회 Special 이상
  150회 이상: Special 또는 Legendary (하나는 보장)
```

**희귀도별 확률:**

```
COMMON (초급): 55%
  예: 철검, 천가죽갑, 동반지, 목재방패
  
RARE (중급): 30%
  예: 강철검, 비단로브, 은반지, 철방패
  
SPECIAL (상급): 13%
  예: 마검, 신비로운 로브, 황금반지, 마법방패
  
LEGENDARY (최고): 2%
  예: 전설의 검, 신의 로브, 영원의 반지, 신성한 방패
```

**가챠 획득 방법:**

```
무료 (매일):
  - 일일 1회 무료 뽑기 (50 다이아 값)
  - 매월 무료 10회 뽑기 (450 다이아 값)

유료:
  - 다이아 구매 (Real Money)
  
이벤트:
  - 이벤트 보상 (50-450 다이아)
  - 월간 로그인 (다이아 누적)
  - 마일스톤 달성 (다이아 보상)
```

### B. 카드 가챠 (Card Gacha)

**가챠 구성:**

```
가챠 비용: 50 다이아 / 1회
10회: 450 다이아 (10% 할인)
연속: 같은 보장 시스템
```

**희귀도별 확률:**

```
COMMON (초급): 45%
  예: 기본 공격, 약한 방어, 낮은 마법
  
RARE (중급): 35%
  예: 강력한 공격, 좋은 방어, 중간 마법
  
SPECIAL (상급): 15%
  예: 매우 강력한 공격, 강한 방어, 강력한 마법
  
LEGENDARY (최고): 5%
  예: 궁극의 공격, 절대 방어, 신의 마법
```

**카드 특성:**

```
각 희귀도별 3가지 기본 효과:

COMMON:
  ATTACK: 15 데미지
  SKILL (방어): 체력 10% 회복
  POWER (강화): ATK +10% (1턴)

RARE:
  ATTACK: 25 데미지
  SKILL (방어): 체력 20% 회복
  POWER (강화): ATK +20% (1턴)

SPECIAL:
  ATTACK: 40 데미지
  SKILL (방어): 체력 30% 회복
  POWER (강화): ATK +35% (1턴)

LEGENDARY:
  ATTACK: 60 데미지
  SKILL (방어): 체력 50% 회복
  POWER (강화): ATK +50% (1턴)
```

## 2️⃣ 강화 재료 시스템 (Enhancement Material System)

### 희귀도별 재료 가치

**재료 변환:**

```
COMMON 장비/카드:
  1개 = 강화 재료 1unit
  = "기본 강화석" 1개로 간주
  = 강화도 +1 가능

RARE 장비/카드:
  1개 = 강화 재료 3unit
  = "고급 강화석" 1개로 간주
  = 강화도 +3 가능

SPECIAL 장비/카드:
  1개 = 강화 재료 5unit
  = "특별 강화석" 1개로 간주
  = 강화도 +5 가능

LEGENDARY 장비/카드:
  1개 = 강화 재료 10unit
  = "전설의 강화석" 1개로 간주
  = 강화도 +10 가능
```

**강화 재료 활용:**

```
예 1: 장비 강화
  현재: 무기 +3 (필요: 2,000 G)
  방법:
    옵션 A: 골드 2,000 G 소비 → +4
    옵션 B: COMMON 카드 3개 + 골드 1,000 G → +4
    옵션 C: RARE 카드 1개 + 골드 500 G → +4

예 2: 카드 강화
  현재: 공격 카드 +2 (필요: 같은 카드 1개)
  방법:
    옵션 A: 같은 공격 카드 1개 → +3
    옵션 B: COMMON 다른 카드 3개 → +3
    옵션 C: RARE 다른 카드 1개 → +3
```

## 3️⃣ 전체 강화 시스템 (Unified Enhancement System)

### A. 장비 강화 상세

**강화 단계별 비용 (통합):**

```
+0→+1: 500 G
       또는 COMMON 장비 1개
       또는 COMMON 카드 1개

+1→+2: 750 G
       또는 COMMON 장비 1개 + 250 G
       또는 COMMON 카드 1개 + 250 G

+2→+3: 1,000 G
       또는 COMMON 장비 1개 + 500 G
       또는 RARE 장비 1개 (교체)
       또는 RARE 카드 1개

... (계속)

+9→+10: 10,000 G
        또는 LEGENDARY 장비 1개
        또는 LEGENDARY 카드 1개
        또는 SPECIAL 장비 2개 + 2,000 G
```

**선택 가능한 강화 경로:**

```
경로 1: 순수 골드 강화 (장기)
  비용: 총 36,250 G
  시간: 길게 (일이 더 필요)
  
경로 2: 장비 재료 활용 (균형)
  비용: 20,000 G + 재료 15개
  시간: 중간 (가챠 + 강화)
  
경로 3: 카드 재료 활용 (빠름)
  비용: 10,000 G + 재료 10개
  시간: 짧음 (카드 가챠 활용)
  
경로 4: 혼합 강화 (유연)
  비용: 18,000 G + 장비 5개 + 카드 10개
  시간: 유연함 (자신의 상황에 맞춰)
```

### B. 카드 강화 상세

**카드 강화 비용:**

```
기본→+1: 100 G + 같은 카드 1개
        또는 100 G + COMMON 카드 3개
        또는 100 G + COMMON 장비 3개

+1→+2: 150 G + 같은 카드 1개
       또는 150 G + COMMON 카드 3개
       
... (계속)

+9→+10: 500 G + 같은 카드 1개
        또는 500 G + RARE 카드 1개
        또는 500 G + SPECIAL 장비 1개
```

**카드 강화의 진행도:**

```
카드 획득 후 즉시 강화 가능:
  
  Day 1: 공격 카드 COMMON 획득
  → 즉시 기본→+1 강화 (100 G)
  
  Day 2: 공격 카드 COMMON 또 획득
  → 전 카드 +1→+2 강화 (150 G)
  
  ... 계속 가챠해서 얻는 중복 카드로
      계속 강화 진행
  
  결과: 자연스러운 강화 진행도
```

## 4️⃣ 가챠 ↔ 강화 완전 연결 시스템

### 전체 경제 루프

```
[매일 플레이]
  ↓
[보상]
  경험치 + 골드 + 다이아 (일부)
  ↓
[가챠 선택]
  
  옵션 A: 장비 가챠
    50 다이아 / 1회
    → COMMON 장비 (낮음)
    → RARE 카드 (중간)
    → SPECIAL 장비 (높음)
  
  옵션 B: 카드 가챠
    50 다이아 / 1회
    → COMMON 카드 (낮음)
    → RARE 카드 (중간)
    → SPECIAL 카드 (높음)
  
  옵션 C: 다이아 모으기
    → 더 큰 뽑기 준비
  ↓
[획득]
  장비 또는 카드 (희귀도별)
  ↓
[사용 선택]
  
  선택 1: 직접 장착/사용
    - SPECIAL 이상: 직접 사용
    - 강화도가 높아질수록 강해짐
  
  선택 2: 강화 재료로 변환
    - COMMON/RARE: 강화 재료로 사용
    - 더 좋은 장비/카드 강화에 사용
  ↓
[강화]
  재료 + 골드 → 강화도 +1
  ↓
[강화 완료]
  선택한 장비 또는 카드 더 강해짐
  ↓
[반복]
```

### 구체적인 시나리오

**Scenario 1: 신규 플레이어 (Week 1)**

```
Day 1:
  가챠 1회: COMMON 검 (방어력 약함)
  → 직접 사용 (선택지가 없음)

Day 2:
  가챠 1회: COMMON 카드 (공격)
  → 직접 사용

Day 3:
  가챠 1회: COMMON 갑옷
  → 직접 사용 또는 강화 재료로 판단

Day 4:
  가챠 1회: RARE 장비
  → 직접 사용 (RARE는 가치 있음)

Day 5-7:
  누적: COMMON 3개, RARE 1개
  선택:
    A) COMMON들을 강화 재료로 사용
       → RARE 장비를 +3까지 강화
    B) 계속 가챠 모으기
       → SPECIAL을 기다리기

결과: 자연스러운 강화 경험
```

**Scenario 2: 일반 플레이어 (Month 1)**

```
Weekly:
  가챠 총 21회 (일일 1회 × 7일)
  비용: 1,050 다이아 (무료 일부 포함)
  
결과:
  COMMON: 약 12개
  RARE: 약 7개
  SPECIAL: 약 2개
  LEGENDARY: 0개

강화 전략:
  1. SPECIAL 장비 2개 직접 사용 (+0)
  2. COMMON 12개 → 강화 재료 변환
     → SPECIAL 장비 2개를 +5까지 강화
  3. RARE 7개 → 강화 재료 변환 또는 직접 사용
     → 추가 강화 또는 예비 아이템
  
결과:
  SPECIAL 장비 2개가 +5 강화됨
  → 기본 공격 약 2.5배 증가
```

**Scenario 3: 극한 게이머 (Month 1)**

```
Daily:
  가챠 10회 (450 다이아)
  비용: 월 13,500 다이아 ($135)
  
결과:
  COMMON: 약 60개
  RARE: 약 90개
  SPECIAL: 약 39개
  LEGENDARY: 약 12개
  
강화 전략:
  1. LEGENDARY 12개 중 2-3개 직접 사용
  2. SPECIAL 39개를 완전 강화
     → 각각 +10까지
  3. RARE 90개를 강화 재료로 변환
     → SPECIAL들의 강화 가속화
  
강화 완료:
  모든 장비 거의 +10 근처
  캐릭터 매우 강함
  
결과:
  극한 게이머만 도달 가능한 성능
```

## 📊 경제 시스템 통합

### 월간 재료 흐름

```
무료 플레이어:
  월 가챠: 약 50회 (무료 다이아)
  → COMMON 27개, RARE 15개, SPECIAL 7개, LEGENDARY 1개
  → 강화 재료: 27 + (15×3) + (7×5) + (1×10) = 117 unit
  → 장비 +10 강화: 불가능 (35K G 필요)
  → 카드 +10 강화: 최대 3-4개 (제한적)

중간 플레이어:
  월 가챠: 약 200회 (부분 유료)
  → COMMON 110개, RARE 60개, SPECIAL 26개, LEGENDARY 4개
  → 강화 재료: 290 + 180 + 130 + 40 = 640 unit
  → 장비 +10 강화: 1-2개 (충분한 재료)
  → 카드 +10 강화: 10개 정도

극한 게이머:
  월 가챤: 약 450회 (높은 투자)
  → COMMON 247개, RARE 135개, SPECIAL 59개, LEGENDARY 9개
  → 강화 재료: 1,400+ unit
  → 장비 +10 강화: 모든 슬롯 가능
  → 카드 +10 강화: 모든 주요 카드 가능
```

### 골드 경제

```
강화 필요 골드:
  장비 +0→+10: 약 36,250 G
  4개 슬롯: 약 145,000 G
  
  카드 +0→+10: 약 2,750 G
  10개 카드: 약 27,500 G
  
  총: 약 180,000 G (초기 플레이)

월간 골드 수입:
  일일 던전: 약 1,000 G/일
  사이드 퀘스트: 약 500 G/회
  이벤트 보상: 약 5,000 G/월
  
  총: 약 40,000 G/월

결론:
  강화 = 장기 목표 (4-5개월)
  또는 가챠로 재료 수집 (빠름)
```

## 🎯 가챠 ↔ 강화 설계 요약

### 핵심 메커니즘

```
Step 1: 가챠 (뽑기)
  → 다양한 희귀도 장비/카드 획득

Step 2: 선택 (사용 vs 재료)
  높은 희귀도: 직접 사용
  낮은 희귀도: 강화 재료로 변환

Step 3: 강화 (2가지 경로)
  경로 A: 골드 투자 (느림, 무료)
  경로 B: 재료 투자 (빠름, 가챠 필요)

Step 4: 강화 완료
  스탯 증가 → 캐릭터 강해짐
  → 더 강한 콘텐츠 접근 가능

Step 5: 반복
  더 강하려면 → 더 가챠 해야 함
  → 다시 Step 1로
```

### 플레이어별 경험

```
신규 플레이어:
  - 낮은 희귀도로도 충분
  - 천천히 강화 진행
  - 자연스러운 학습 곡선

일반 플레이어:
  - 적절한 투자로 강화 진행
  - 일부 다이아 구매 (선택)
  - 2-3개월에 준 강화 완료

극한 게이머:
  - 높은 투자로 빠른 강화
  - 모든 시스템 활용
  - 최고 성능 도달 가능
```
