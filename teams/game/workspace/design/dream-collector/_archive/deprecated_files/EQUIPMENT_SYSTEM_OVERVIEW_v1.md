## 📊 장비 시스템 한눈에 보기

### 1️⃣ 장비 구성 (What)

```
4가지 장비 타입 × 6개 슬롯 착용

┌─────────────────┐
│ 무기 (1개)      │  → 주로 공격력 증가 + 회심율
├─────────────────┤
│ 방어구 (1개)    │  → 주로 방어력 증가 + HP
├─────────────────┤
│ 반지 (2개)      │  → 마나, 회피율, 특수 능력
├─────────────────┤
│ 목걸이 (2개)    │  → 꿈 수집 보너스, 경험치
└─────────────────┘

예시: 장비 1개 = "마법의 검" (무기, LV 5, 레어)
```

---

## 📈 장비가 제공하는 스탯 & 특성

### 기본 능력치 (Stats)

| 스탯 | 설명 | 효과 |
|------|------|------|
| **ATK (공격력)** | 데미지 증가 | 모든 공격이 X% 더 강함 |
| **DEF (방어력)** | 받는 피해 감소 | 적 공격 피해 X% 감소 |
| **MNA (마나)** | 최대 마나 증가 | 스킬 사용 가능 횟수 증가 |
| **EVA (회피율)** | 적 공격 회피 확률 | X% 확률로 적 공격 회피 |
| **CRI (회심율)** | 치명타 확률 | X% 확률로 데미지 1.5배 |
| **DRM (꿈 수집)** | 전투 보상 증가 | 전투 후 꿈 수집 X% 증가 |

### 능력치 예시 (캐릭터 기본값 + 장비 보너스)

```
기본: HP 500, ATK 100, DEF 50

장비 3개 장착:
├─ 무기 (마법의 검): +40 ATK, +5% 회심율
├─ 방어구 (방어의 갑옷): +30 DEF, +50 HP
└─ 반지 (지혜): +20 마나, +10% 회피율

결과: HP 550, ATK 140, DEF 80, 마나 20, 회피율 10%, 회심율 5%
```

## ⚡ 장비 스킬 (Equipment Skill)

### 개념: 각 장비의 **고유한 전투 능력**

```
【마법의 검】
스킬: "폭열 돌격" 
┌─────────────────────────────────────┐
│ 효과: 공격력 200% 데미지 (+50 꿈 수집)│
│ 소비: 마나 20                        │
│ 재사용: 2턴 대기 (턴마다 마나 +5 회복) │
│ 사용 가능 여부: ⭕ 아직 사용 가능     │
└─────────────────────────────────────┘

사용 타이밍:
├─ T1: 마나 20 소비 → 스킬 발동 ⚡
├─ T2: 마나 5 회복 (2턴 대기 중)
├─ T3: 마나 5 회복, 스킬 재사용 가능 ⭕
└─ ...
```

### 스킬 강화 (LV에 따른 강화)

```
LV 1: 공격력 120%, 마나 30 소비
LV 3: 공격력 150%, 마나 25 소비
LV 5: 공격력 200%, 마나 20 소비 ← 현재 레벨
LV 10: 공격력 300%, 마나 15 소비
```

## 🎛️ 장비 옵션 (선택 가능한 부가 능력)

### 개념: **플레이어가 선택**해서 장비의 특성을 커스터마이징

```
【마법의 검】선택 가능한 옵션:
┌────────────────────────────────┐
│ ✓ 옵션 A: 공격력 +15%          │ ← 선택됨
├────────────────────────────────┤
│ ☐ 옵션 B: 회피율 +10%          │ ← 변경 가능
├────────────────────────────────┤
│ ☐ 옵션 C: 꿈 수집 +20%         │ ← 변경 가능
└────────────────────────────────┘
```

### 옵션의 3가지 종류

| 종류 | 효과 | 예시 |
|------|------|------|
| **공격형** | 공격력 증가 | ATK +15%, 회심율 +8% |
| **방어형** | 방어력/HP 증가 | DEF +15%, HP +100 |
| **특수형** | 게임플레이 변화 | 꿈 수집 +20%, 마나 회복 +5% |

## ⚖️ 캐릭터 밸런스 & 등급

### 장비 등급 5단계

```
등급 1: Common (회색) ────────────────── 약함
등급 2: Uncommon (초록) ────────────────
등급 3: Rare (파란) ────────────────── 중간
등급 4: Epic (보라) ────────────────── 강함
등급 5: Legendary (주황) ───────────── 매우 강함
```

### 등급별 능력치 차이 (무기 LV 1 기준)

| 등급 | 공격력 | 스킬 효율 | 사용 횟수 |
|------|--------|---------|----------|
| Common | +15 | 100% | 0/3 |
| Uncommon | +25 | 110% | 0/5 |
| **Rare** | +40 | 120% | 0/8 |
| Epic | +60 | 130% | 0/10 |
| Legendary | +100 | 150% | 0/15 |

### 캐릭터 강화 경로

```
초반 (Level 1-10)
├─ Common/Uncommon 장비만 사용
├─ 기본 스탯만으로 진행 가능
└─ 난이도: 쉬움 ⭐

중반 (Level 11-30)
├─ Rare 장비 조합 시작
├─ 스킬 + 옵션 조합 중요해짐
└─ 난이도: 보통 ⭐⭐

후반 (Level 31+)
├─ Epic/Legendary 장비 추구
├─ 빌드 최적화 필수
└─ 난이도: 어려움 ⭐⭐⭐
```

## 🔗 전체 연결 구조

### A. 장비 선택 시나리오

```
1️⃣ 보유 장비 중 마법의 검 클릭
   ↓
2️⃣ 모달창 표시
   ├─ 기본 스탯: ATK +40, 회심율 +5%
   ├─ 스킬: "폭열 돌격" (공격력 200%, 마나 20)
   ├─ 옵션: ✓ "공격력 +15%" 선택됨
   └─ 사용 횟수: 3/8
   ↓
3️⃣ [장착] 버튼 클릭
   ↓
4️⃣ 캐릭터 스탯 즉시 업데이트
   └─ ATK 100 → 140 (장비 +40)
      회심율 0% → 5%
      옵션 있으면 +15% 추가
```

### B. 전투 중 스킬 사용

```
상황: 강한 적 등장 → 스킬로 빠르게 처치하려고 함

1️⃣ 전투 화면에서 [마법의 검 스킬] 버튼 보임
   ↓
2️⃣ 마나 20 이상인지 확인 ✓ (보유 마나 60)
   ↓
3️⃣ 스킬 발동
   ├─ 공격력 200% 데미지 → 적에게 500 데미지
   ├─ 꿈 수집 +50 추가
   ├─ 마나 20 소비 (60 → 40)
   └─ 2턴 재사용 대기 시작
   ↓
4️⃣ T2, T3: 마나 자동 회복 (매턴 +5)
   ↓
5️⃣ T3: 스킬 재사용 가능 ⭕
```

## 💡 핵심 밸런스 원칙

### 세 가지 빌드 선택지

```
1️⃣ 공격형 빌드 (DPS 극대화)
   ├─ 무기: "공격력 +15%" 옵션 선택
   ├─ 반지: "회심율 +10%" 옵션
   └─ 결과: 매턴 높은 데미지 (취약: 방어 약함)

2️⃣ 방어형 빌드 (생존 극대화)
   ├─ 방어구: "방어력 +15%" 옵션
   ├─ 목걸이: "최대 HP +50" 옵션
   └─ 결과: 많은 데미지 견딤 (취약: 공격 약함)

3️⃣ 균형형 빌드 (모든 스탯 중간)
   ├─ 무기: "공격력 +10%" 옵션
   ├─ 방어구: "방어력 +10%" 옵션
   ├─ 반지: "마나 +20%" 옵션
   └─ 결과: 안정적인 전투 (취약: 극단적 강점 없음)
```

### 마나 소비 밸런스

```
낮은 마나 소비 스킬 (마나 10-15)
├─ 자주 사용 가능 ⭕
├─ 효과도 적음 (공격력 100%)
└─ 초반에 좋음

높은 마나 소비 스킬 (마나 30-50)
├─ 마나 부족 → 자주 사용 불가
├─ 효과는 강함 (공격력 250%)
└─ 후반/보스전에서 활용
```
