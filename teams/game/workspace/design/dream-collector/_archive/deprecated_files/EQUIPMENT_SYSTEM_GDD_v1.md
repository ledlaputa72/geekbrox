## 시스템 개요

### 개념
- **기본 개념:** 캐릭터가 4종류의 장비(무기, 방어구, 반지, 목걸이)를 장착해 스탯을 강화하고 고유 스킬을 해제하는 시스템
- **게임 루프:** 카드 선택 → 전투 (장비 스킬 활용) → 꿈 수집 → 장비 강화
- **핵심 특징:**
  - 각 장비는 **고유한 장비 스킬** 제공
  - 장비 스킬은 **사용 횟수 제한** (재충전 가능)
  - 추가 옵션을 **플레이어가 선택** 커스터마이징 가능

### 참고 게임 분석
| 요소 | 참고 게임 | Dream Collector 적용 |
|------|---------|-----------------|
| 기본 속성 | ATK/DEF 수치 | ATK / DEF / 특수 (마나, 회피율 등) |
| 장비 스킬 | 사용 횟수 제한 (0/10, 0/2) | **Turn 재사용 또는 마나 소비** |
| 옵션 선택 | 여러 옵션 중 1개만 선택 | **최대 3개 옵션 중 선택 가능** |
| 등급 차별화 | 레어/에픽 색상 구분 | **5단계 등급 (Common~Legendary)** |

## 장비 타입

### 1. 무기 (Weapon)
**역할:** 공격력 강화, 공격 기반 스킬 제공  
**슬롯:** 1개  
**주요 속성:**
- 기본 공격력 +X%
- 회심율 증가
- 마나 회복율

**예시:**
```
【열음의 창】(Rare)
├─ 기본: ATK +40
├─ 스킬: "열음 돌격" LV.2 
│  └─ 아이콘 1개 소멸 → 공격력 165% 데미지 (사용 횟수 0/2)
└─ 옵션 선택 (1개):
   ├─ 전투 시작 시 공격력 +10%
   ├─ 방어 관통 데미지 +30%
   └─ 공격력 +15% (사용 횟수 증가: 0/3)
```

### 2. 방어구 (Armor)
**역할:** 방어력 강화, 생존 기반 스킬 제공  
**슬롯:** 1개  
**주요 속성:**
- 기본 방어력 +X%
- 최대 HP 증가
- 피해 감소

**예시:**
```
【방랑의 대검】(Epic)
├─ 기본: ATK +105
├─ 스킬: "방랑의 검세" LV.3
│  └─ 적군에게 공격력 180% 데미지 (사용 횟수 0/10)
└─ 옵션 선택 (1개):
   ├─ 전투 시작 시 공격력 +10%
   ├─ 공격 시 35% 확률로 다음 턴 추가 제공
   └─ 공격력 +15% (사용 횟수 증가: 0/5)
```

### 3. 반지 (Ring)
**역할:** 캐릭터/무기/방어구 스탯 강화 특화 (캐릭터 능력 증폭)  
**슬롯:** 2개  
**주요 속성:**
- 캐릭터 기본 스탯 강화 (ATK, DEF, HP 강화)
- 무기 스킬 효율 증가
- 방어구 피해 감소 효과 강화
- 회피율, 마나 회복, 상태 이상 저항

**설명:**
반지는 **캐릭터 자체를 강화하는 장비**입니다. 무기와 방어구의 효과를 더 극대화하고, 캐릭터의 기본 능력(회피, 마나 등)을 높입니다.

**예시:**
```
【힘의 반지】(Rare)
├─ 기본: ATK +20 (캐릭터 공격력 직접 강화)
├─ 스킬: "강화의 축복" LV.2
│  └─ 마나 15 소비 → 현재 턴 모든 스탯 +30% 증가 (사용 횟수 0/6)
└─ 옵션 선택 (1개):
   ├─ 무기 스킬 데미지 +20% (무기가 더 강해짐)
   ├─ 방어구 피해 감소 효과 +15%
   ├─ 캐릭터 최대 마나 +50 + 마나 회복률 +10%
   └─ 회피율 +12% + 상태 이상 저항 +20%

【지혜의 반지】(Epic)
├─ 기본: 최대 마나 +40, DEF +15
├─ 스킬: "마나 집중" LV.3
│  └─ 마나 25 소비 → 다음 턴 모든 공격 데미지 +50% (사용 횟수 0/8)
└─ 옵션 선택 (1개):
   ├─ 무기 스킬 재사용 대기 -1턴 (더 자주 사용 가능)
   ├─ 방어력 +25% + HP 회복 +2/턴
   └─ 캐릭터 공격력 +15% + 회심율 +10%
```

### 4. 목걸이 (Necklace)
**역할:** 카드 플레이 및 카드 기반 능력 강화 (카드 매커니즘 특화)  
**슬롯:** 2개  
**주요 속성:**
- 4가지 카드 종류 스탯/특성 영향 (ATTACK/SKILL/POWER/CURSE)
- 패링 효율 증가
- 카드 회피율 증가
- 꿈 수집 보너스

**설명:**
목걸이는 **카드 플레이를 강화하는 장비**입니다. 게임의 핵심인 카드 선택과 카드 활용을 더 효과적으로 만듭니다. 4가지 카드 종류(ATTACK/SKILL/POWER/CURSE)의 능력을 개별적으로 강화할 수 있습니다.

**예시:**
```
【공격의 목걸이】(Rare)
├─ 기본: ATTACK 카드 데미지 +25%, 꿈 수집 +15%
├─ 스킬: "공격 강화" LV.2
│  └─ 마나 20 소비 → 다음 ATTACK 카드 데미지 +100% (사용 횟수 0/5)
└─ 옵션 선택 (1개):
   ├─ ATTACK 카드 획득 시 체력 회복 +10
   ├─ SKILL 카드 재사용 대기 -1 (더 자주 사용)
   ├─ POWER 카드 확률 +20% (강력한 카드 더 자주)
   └─ CURSE 카드 피해 -30% (약화된 카드 영향 감소)

【수비의 목걸이】(Epic)
├─ 기본: SKILL 카드 회피율 +30%, 패링 효율 +25%
├─ 스킬: "방어 무술" LV.3
│  └─ 마나 18 소비 → 다음 공격 패링 성공 → 반격 데미지 +150% (사용 횟수 0/7)
└─ 옵션 선택 (1개):
   ├─ SKILL 카드 선택 시 다음 턴 DEF +20%
   ├─ POWER 카드 회피 불가 (필수 맞음) 대신 데미지 -50%
   ├─ CURSE 카드 상태 이상 저항 +50%
   └─ 모든 카드 패링 확률 +15%

【꿈의 목걸이】(Legendary)
├─ 기본: 모든 카드 타입 능력 +15%, 꿈 수집 +40%
├─ 스킬: "꿈 동기화" LV.4
│  └─ 마나 30 소비 → 모든 카드 다음 턴 능력 +50% (사용 횟수 0/10)
└─ 옵션 선택 (1개):
   ├─ ATTACK/SKILL/POWER 카드 능력 모두 +20%
   ├─ CURSE 카드도 긍정 효과로 변환 (피해 → 회복)
   └─ 카드 조합 보너스: 특정 카드 4개 연속 선택 시 +30% 추가 효과
```

## 기본 능력치

### 제공되는 스탯 종류

| 스탯 | 약자 | 설명 | 적용 장비 |
|------|------|------|---------|
| 공격력 | ATK | 기본 공격 데미지 증가 | 무기, 반지 |
| 방어력 | DEF | 받는 피해 감소 | 방어구, 목걸이 |
| 최대 마나 | MNA | 최대 마나 용량 증가 | 반지, 목걸이 |
| 회피율 | EVA | 적 공격 회피 확률 | 반지, 목걸이 |
| 회심율 | CRI | 회심 히트 확률 (데미지 1.5배) | 무기 |
| 꿈 수집 | DRM | 전투 후 꿈 수집량 증가 | 목걸이 |
| HP 회복 | HLG | 턴당 HP 자동 회복 | 방어구 |

### 스탯 계산식

```
최종 ATK = (캐릭터 기본 ATK) × (1 + 장비 ATK% 합계) + 장비 ATK 고정값 합계

최종 DEF = (캐릭터 기본 DEF) × (1 + 장비 DEF% 합계) + 장비 DEF 고정값 합계

최종 마나 = (캐릭터 기본 마나) + 장비 마나 보너스 합계
```

### 장비별 기본값 예시 (LV 1, Common)

| 장비 타입 | ATK | DEF | MNA | 기타 |
|---------|-----|-----|-----|------|
| 일반 무기 | +20 | - | - | 회심율 +5% |
| 일반 방어구 | - | +30 | - | HP 자동 회복 +1/턴 |
| 일반 반지 | +10 | +10 | +20 | 회피율 +3% |
| 일반 목걸이 | - | - | +15 | 꿈 수집 +10% |

## 장비 스킬

### 스킬 시스템

**정의:** 각 장비가 제공하는 **고유한 전투 스킬**, 사용 횟수 또는 마나 소비로 제한

### 스킬 레벨 진행

장비는 **LV 1 ~ LV 10**까지 강화되며, 레벨이 올라갈 때마다 스킬 효과가 증가한다.

```
【마법의 검】
├─ LV 1: "마법 돌격" - 공격력 120% 데미지
├─ LV 2: "마법 돌격" - 공격력 150% 데미지 (재사용 0/8)
├─ ...
└─ LV 10: "마법 돌격" - 공격력 250% 데미지 (재사용 0/15)
```

### 스킬 사용 메커니즘: 두 가지 옵션

#### **옵션 A: 사용 횟수 제한** (참고 게임 방식)

장비 스킬마다 **최대 사용 횟수 설정**. 전투 후 회복 또는 특정 아이템으로 충전.

```
【열음의 창】스킬: "열음 돌격"
├─ 사용 횟수: 0/2 (최대 2회)
├─ 전투 후 회복: +1 (매 전투마다 1회 회복)
├─ 복구 아이템: "스킬 충전 석" (1개 사용 시 +1회)
└─ 전투 중 사용 시:
   ├─ 아이콘 1개 소멸 (카드 버려짐)
   └─ 데미지: 공격력 165% (LV 2 기준)
```

**장점:**
- 전략적 운영 (언제 사용할지 결정)
- 명확한 자원 관리
- 참고 게임과 동일

**단점:**
- 복잡한 UI (사용 횟수 추적)
- 전투 중 충전 어려움

#### **옵션 B: 마나 소비 방식** (Dream Collector 추천)

장비 스킬은 **마나를 소비**해 사용. 마나는 턴마다 회복되거나 카드로 충전.

```
【꿈의 검】스킬: "꿈 베기"
├─ 마나 소비: 25
├─ 데미지: 공격력 180% + 추가 꿈 수집 +5
├─ 마나 회복: 턴마다 5 자동 회복 (또는 카드 사용)
└─ 전투 중 즉시 사용 가능 (마나 충분 시)
```

**장점:**
- Dream Collector 테마와 맞음 (마나 = 꿈 에너지)
- 자동 회복으로 더 부드러운 게임플레이
- UI 단순화

**단점:**
- 사용 횟수 제한 전략성 감소

## 추가 옵션 시스템

### 개념

각 장비마다 **고정 스킬 1개** + **선택 옵션 3~4개 중 1개 선택** 구조

### 옵션 분류 (장비 타입별)

#### 반지 옵션 (캐릭터 능력 강화)
```
【힘의 반지】옵션들
├─ "무기 강화": 무기 스킬 데미지 +20% (무기가 더 강해짐)
├─ "방어 극대화": 방어구 피해 감소 +15% + DEF +15%
├─ "마나 축적": 최대 마나 +50 + 회복률 +10%
└─ "민첩함": 회피율 +12% + 상태 이상 저항 +20%
   → 추천: 무기와 방어구 효과를 어느 쪽 극대화할지 선택
```

```
【지혜의 반지】옵션들
├─ "스킬 재사용": 무기 스킬 재사용 대기 -1턴 (더 자주 사용 가능)
├─ "탱크 강화": 최대 HP +100 + 자동 회복 +3/턴
├─ "공격력 증폭": 캐릭터 공격력 +15% + 회심율 +10%
└─ "마나 폭발": 마나 소비 시 추가 효과 +25% 발동
   → 추천: 캐릭터의 주요 역할(공격/방어/마나) 선택
```

#### 목걸이 옵션 (카드 타입별 강화)

```
【공격의 목걸이】옵션들
├─ "ATTACK 강화": ATTACK 카드 데미지 +30% (공격 카드만 강함)
├─ "SKILL 가속": SKILL 카드 재사용 대기 -1 (더 자주 사용)
├─ "POWER 확률": POWER 카드 출현 확률 +25%
└─ "CURSE 약화": CURSE 카드 피해 -40% (악화 영향 감소)
   → 추천: 어떤 카드 타입을 극대화할지 선택
```

```
【수비의 목걸이】옵션들
├─ "SKILL 회피": SKILL 카드 회피율 +40% (스킬 카드 회피 쉬움)
├─ "POWER 경화": POWER 카드 피해 -50% 대신 필수 맞음
├─ "CURSE 저항": CURSE 카드 상태 이상 저항 +60%
└─ "패링 극대": 모든 카드 패링 확률 +20%
   → 추천: 방어 중심 빌드 + 특정 카드 대응
```

```
【꿈의 목걸이】옵션들
├─ "전체 강화": ATTACK/SKILL/POWER 카드 모두 +20%
├─ "CURSE 전환": CURSE 카드를 긍정 효과로 변환
├─ "조합 보너스": 특정 카드 조합 시 추가 +30% 효과 발동
└─ "카드 드로우": 전투 중 추가 카드 1장 선택 기회 +1
   → 추천: 균형 잡힌 카드 조합 + 유연성
```

### 옵션 선택 메커니즘

**문제:** 참고 게임은 "옵션 1개만 선택" → Dream Collector는 복합 빌드 가능하게

**해결:** **최대 3개 옵션 중 선택 가능 (대신 각 선택마다 스탯 감소)**

```
【꿈의 검】선택 가능 옵션:
├─ 옵션 A: 공격력 +15%
├─ 옵션 B: 회피율 +10%
└─ 옵션 C: 꿈 수집 +20%

플레이어 선택 예시:
├─ A 선택: 공격력 +15% ✓
├─ B 선택: 공격력 +15% + 회피율 +10% (대신 회피율은 +8%만 적용) ✓
└─ A+B+C 선택: 모두 선택 (각각 70% 효율로 적용) ✓
```

## 사용 횟수 및 재사용

### 사용 횟수 메커니즘

**추천 방식: 마나 소비 + 턴 재사용**

```
【열정의 창】
├─ 장비 스킬: "폭열 돌격"
├─ 마나 소비: 20
├─ 재사용 대기: 2턴 (사용 후 2턴 대기)
├─ 데미지: 공격력 200% (턴당 마나 5 자동 회복)
└─ 전투 예시:
   ├─ T1: 마나 20 소비, 스킬 사용 ✓ → 2턴 재사용 대기 시작
   ├─ T2: 마나 5 회복 (총 5/60)
   ├─ T3: 마나 5 회복, 스킬 재사용 가능 ✓
   └─ T4: 마나 5 회복 (총 10/60)
```

### 마나 회복 방식

| 방식 | 설명 | 빈도 |
|------|------|------|
| 자동 회복 | 턴마다 고정 마나 회복 | 매 턴 |
| 카드 회복 | 특정 카드로 마나 +20 회복 | 카드 선택 시 |
| 장비 회복 | 특정 장비 옵션으로 회복 증가 | 옵션 선택 시 |
| 조건 회복 | 회심 히트 시 마나 +5 회복 | 조건 발동 시 |

## 등급 및 밸런스

### 장비 등급 5단계

| 등급 | 이름 | 색상 | 강도 | 획득 난이도 |
|------|------|------|------|----------|
| 1 | Common | 회색 | 기본 | 쉬움 (초반) |
| 2 | Uncommon | 초록색 | 낮음 | 보통 |
| 3 | Rare | 파란색 | 중간 | 어려움 |
| 4 | Epic | 보라색 | 높음 | 매우 어려움 |
| 5 | Legendary | 주황색 | 매우 높음 | 극도로 어려움 |

### 등급별 능력치 예시 (LV 1 기준)

**무기 (Weapon)**
| 등급 | ATK | 스킬 효율 | 옵션 수 | 사용 횟수 |
|------|-----|---------|--------|----------|
| Common | +15 | 100% | 1 | 0/3 |
| Uncommon | +25 | 110% | 2 | 0/5 |
| Rare | +40 | 120% | 2 | 0/8 |
| Epic | +60 | 130% | 3 | 0/10 |
| Legendary | +100 | 150% | 3 | 0/15 |

**방어구 (Armor)**
| 등급 | DEF | HP | 스킬 효율 | 사용 횟수 |
|------|-----|-----|---------|----------|
| Common | +15 | +50 | 100% | 0/3 |
| Uncommon | +25 | +100 | 110% | 0/5 |
| Rare | +40 | +150 | 120% | 0/8 |
| Epic | +65 | +250 | 130% | 0/10 |
| Legendary | +100 | +400 | 150% | 0/15 |

**반지 (Ring) - 캐릭터 스탯 강화**
| 등급 | ATK/DEF | MNA | 스킬 효율 | 옵션 수 |
|------|---------|-----|---------|--------|
| Common | +10/+8 | +20 | 100% | 1 |
| Uncommon | +15/+12 | +35 | 110% | 2 |
| Rare | +20/+18 | +50 | 120% | 2 |
| Epic | +35/+30 | +80 | 130% | 3 |
| Legendary | +50/+45 | +120 | 150% | 3 |

**목걸이 (Necklace) - 카드 능력 강화**
| 등급 | 카드 능력 | 꿈 수집 | 스킬 효율 | 옵션 수 |
|------|----------|--------|---------|--------|
| Common | +10% | +10% | 100% | 1 |
| Uncommon | +15% | +15% | 110% | 2 |
| Rare | +20% | +25% | 120% | 2 |
| Epic | +30% | +40% | 130% | 3 |
| Legendary | +50% | +60% | 150% | 3 |

### 사용 횟수 기준 (참고 게임 방식 사용 시)

```
사용 횟수 = (등급 * 2) + (LV / 2)

예시:
├─ Common LV 1: (1*2) + (1/2) = 2.5 → 0/2 또는 0/3
├─ Rare LV 5: (3*2) + (5/2) = 8.5 → 0/8 또는 0/9
└─ Legendary LV 10: (5*2) + (10/2) = 15 → 0/15
```

- **기본 능력치** (ATK, DEF, MNA 등)
- **장비 스킬** (스킬명 + 설명 + 효율)
- **추가 옵션** (선택된 옵션 표시 및 수정 버튼)
- **사용 횟수** (프로그래스 바 포함)

**예시 레이아웃:**

│ 장비 스킬 (열정):        │

│   (마나 20 소비, 2턴 대기) │

│ 선택 옵션:              │

├──────────────────────────┤

│ ████░░░░░░░░░░░        │
├──────────────────────────┤
