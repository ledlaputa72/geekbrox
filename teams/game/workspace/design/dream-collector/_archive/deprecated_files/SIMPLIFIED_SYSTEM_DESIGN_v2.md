### 2️⃣ 강화 시스템 (Enhancement System) 🔄 개선

#### A. 장비 강화 (Equipment Enhancement)

**새로운 설계:**

```
강화도: +0 ~ +10

비용 (간소화):
  +0→+1: 500 G
  +1→+2: 750 G
  +2→+3: 1,000 G
  +3→+4: 1,500 G
  +4→+5: 2,000 G
  +5→+6: 3,000 G
  +6→+7: 4,000 G
  +7→+8: 5,500 G
  +8→+9: 7,500 G
  +9→+10: 10,000 G
  ──────────────
  누적 (+0→+10): 약 36,250 G (1개 장비)
  4개 슬롯: 약 145,000 G

성공률:
  모든 단계: 100% (실패 없음)
  → "비용 지불 = 무조건 강화"

효과:
  각 +1 = 스탯 +5% (선형)
  
  예:
    무기 ATK 20
    +0: 20
    +1: 20 × 1.05 = 21
    +2: 20 × 1.10 = 22
    ...
    +10: 20 × 1.50 = 30 (50% 증가)

시각화:
  ▓▓▓▓░░░░░░ +4/10 (40%)
  비용: 1,500 G
  [강화하기] [취소]
  ↓
  ▓▓▓▓▓░░░░░ +5/10 (50%)
  (즉시 완료)
```

**강화와 특성의 상호작용:**

```
무기 강화 +5
  → 무기 마스터리 레벨 5
  → 보너스 = (+5 × 5%) = +25%
  
기본 공격 계산:
  무기 ATK: 20 × 1.50 (+10 강화) = 30
  × 무기 마스터리: 30 × 1.25 = 37.5
  × 다른 특성들...
  = 최종 데미지

간단한 공식:
  최종 = 기본 × ∏(1 + 특성%)
```

#### B. 카드 강화 (Card Enhancement) 🆕 새로 추가

**카드 강화 시스템:**

```
카드도 강화 가능 (장비처럼):

강화도: 기본 ~ +10

비용:
  기본 카드: 다이아 소비 없음 (무료)
  강화 비용: 카드 동일본 2개 또는 강화석 1개
  
  예시:
    기본 → +1: 같은 카드 1장 + 100 G
    +1 → +2: 같은 카드 1장 + 150 G
    ...
    +9 → +10: 같은 카드 1장 + 500 G

카드 강화 효과:
  각 +1 = 카드 능력 +10% (선형)
  
  예 (ATTACK 카드):
    기본: 25 데미지
    +5: 25 × 1.50 = 37.5 데미지
    +10: 25 × 2.00 = 50 데미지 (100% 증가!)

획득 방법:
  카드 강화 재료:
    - 같은 카드 (일반적)
    - 강화석 (특별)
    - 이벤트 보상

시각화:
  [기본 공격 카드] ▓▓░░░░░░░░ +2/10
  데미지: 25 → 27.5 (+2)
  
  강화 조건:
    - 같은 카드 소유
    - 비용 충분
    - [강화하기]
  
  [+3/10]
  데미지: 25 → 30 (+5)
```

**장비 강화 vs 카드 강화:**

```
장비 강화:
  목표: 기본 공격 강화
  비용: 골드 (경제 주요 소비)
  진행: 느림 (145K G 필요)
  체감: 지속적 (+5% 단위)

카드 강화:
  목표: 카드 능력 강화
  비용: 카드 중복 또는 강화석
  진행: 빠름 (자연스러운 획득)
  체감: 큼 (+10% 단위, 더 눈에 띔)

균형:
  둘 다 진행되어야 함
  - 장비: 기본 공격 (ATB)
  - 카드: 특별 공격 (전술)
```

### 4️⃣ 강화 석(Enhancement Material System) 🆕 새로 추가

**강화 재료 시스템:**

```
강화석 (Enhancement Stone):
  획득 방법:
    1. 레벨업 보상 (마일스톤)
    2. 일일 던전 드롭
    3. 이벤트 보상
    4. 상점 판매 (다이아)
  
  용도:
    1. 장비 강화 (선택적)
       - "카드 부족할 때" 대체 수단
    2. 카드 강화 대체 재료
       - "같은 카드 없을 때" 사용

희귀도별 강화석:
  일반 강화석: +1 강화 (누적 가능)
  고급 강화석: +3 강화 (1개 = 3단계)
  특별 강화석: +5 강화 (1개 = 5단계)

예시:
  카드 기본 +0 → +10
  
  방법 1: 같은 카드 10개 필요
  방법 2: 일반 강화석 10개 필요
  방법 3: 고급 강화석 4개 + 일반 2개
  방법 4: 특별 강화석 2개
  
  플레이어 선택:
    충분히 있으면 → 카드 사용 (무료)
    부족하면 → 강화석 사용 (획득한 만큼)
```

## 🎯 특성 시스템 최종 정의

### 5가지 마스터리 (개선)

```
1. 무기 마스터리 (Weapon Mastery)
   조건: 무기 장착
   레벨: 무기 강화도 (+0~+10)
   효과: ATK +5~+50%, 치명타 +1~10%
   
2. 방어 강화 (Defense Mastery)
   조건: 방어구 장착
   레벨: 방어구 강화도
   효과: DEF +5~+50%, HP +5~+50%
   
3. 악세서리 동조 (Accessory Resonance)
   조건: 악세서리 장착 (1~3개)
   레벨: 악세서리 개수 (1~3)
   효과: 악세서리 효과 100~150%
   
4. 듀얼 웰드 (Dual Wield)
   조건: 보조 무기 장착
   레벨: 보조 무기 강화도
   효과: 추가 데미지 +50~+200%
   
5. 장비 보너스 (Equipment Bonus) ← 세트 대체
   조건: 강화된 장비 착용 (각각 +1 이상)
   레벨: 착용한 강화된 장비 개수 (0~4)
   효과: 각 강화된 장비마다 모든 스탯 +5%
   
   예:
     무기 +5: 모든 스탯 +5%
     방어구 +3: 모든 스탯 +5%
     악세서리 +2: 모든 스탯 +5%
     보조무기 +1: 모든 스탯 +5%
     = 최대 +20% (4개 모두 강화)
```

### 기본 공격 계산 (간소화)

```
최종 데미지 = 기본 × ∏(1 + 각 특성%)

예:
  기본 데미지: 35
  
  × (1 + 무기 마스터리 +30%) = 45.5
  × (1 + 방어 강화 +15%) = 52.3
  × (1 + 악세서리 동조 +20%) = 62.8
  × (1 + 듀얼 웰드 +50%) = 94.2
  × (1 + 장비 보너스 +20%) = 113
  
  = 최종 113 데미지 (약 3.2배 증가)

간단함:
  - 곱셈만 사용 (세트 조합 X)
  - 강화도 1단계 = 5% 증가 (선형)
  - 모든 계산 투명함 (숨은 메커니즘 X)
```
