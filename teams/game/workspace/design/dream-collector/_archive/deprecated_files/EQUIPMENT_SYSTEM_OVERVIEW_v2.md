## 📊 장비 시스템 한눈에 보기

### 1️⃣ 장비 구성 (4종 × 6슬롯)

```
┌────────────────────────────────────┐
│ 무기 (1개)                         │
│ → 공격력 증가 + 공격 기반 스킬     │
├────────────────────────────────────┤
│ 방어구 (1개)                       │
│ → 방어력/HP 증가 + 생존 기반 스킬  │
├────────────────────────────────────┤
│ 반지 (2개) ⭐ NEW 역할 명확화      │
│ → 캐릭터/무기/방어구 능력 강화     │
│    (모든 능력 극대화, 마나 관리)   │
├────────────────────────────────────┤
│ 목걸이 (2개) ⭐ NEW 역할 명확화    │
│ → 카드 플레이 강화                 │
│    (4가지 카드 종류 특화)          │
└────────────────────────────────────┘
```

## 🎯 4가지 장비의 명확한 역할

### 무기 (Weapon) — 공격 담당
```
【폭열의 창】(Rare)
├─ 기본: ATK +40
├─ 스킬: "폭열 돌격" (공격력 200% 데미지)
└─ 옵션: 공격력 +15% / 방어 관통 +30%
   → 역할: 높은 데미지 딜링
```

### 방어구 (Armor) — 방어 담당
```
【용의 갑옷】(Rare)
├─ 기본: DEF +40, HP +150
├─ 스킬: "방어 무술" (다음 공격 패링 → 반격)
└─ 옵션: 피해 감소 +15% / HP 회복 +5/턴
   → 역할: 적 공격 견디기
```

### 반지 (Ring) — 캐릭터 능력 증폭 ⭐
```
【힘의 반지】(Rare)
├─ 기본: ATK +20, DEF +18
├─ 스킬: "강화의 축복" (현재 턴 모든 스탯 +30%)
└─ 옵션:
   ├─ 무기 스킬 데미지 +20% ← 무기 더 강해짐
   ├─ 방어구 피해 감소 +15% ← 방어구 더 강해짐
   ├─ 최대 마나 +50, 회복율 +10% ← 마나 관리 개선
   └─ 회피율 +12%, 상태이상 저항 +20%

   역할: 무기와 방어구의 효과를 극대화
        또는 보조 능력(마나, 회피) 강화
```

### 목걸이 (Necklace) — 카드 플레이 강화 ⭐
```
【공격의 목걸이】(Rare)
├─ 기본: ATTACK 카드 데미지 +25%
├─ 스킬: "공격 강화" (다음 ATTACK 카드 데미지 +100%)
└─ 옵션:
   ├─ ATTACK 카드 +30% ← 공격 카드만 강함
   ├─ SKILL 카드 재사용 -1 ← 스킬 카드 자주 사용
   ├─ POWER 카드 확률 +25% ← 강력한 카드 더 자주
   └─ CURSE 카드 피해 -40% ← 악화 카드 약화

   역할: 4가지 카드 종류 중 1개 또는 여럿 특화
        패링, 회피 등 카드 메커니즘 강화
```

---

## 📈 반지 vs 목걸이: 명확한 구분

| 항목 | 반지 | 목걸이 |
|------|------|--------|
| **영향 범위** | 캐릭터 기본 능력 | 카드 플레이 메커니즘 |
| **주요 역할** | 무기/방어구 강화 | 4가지 카드 종류 강화 |
| **스킬 효과** | 스탯 증가 | 카드 능력 증가 |
| **옵션 특징** | 무기/방어구 능력 극대화 | 특정 카드 종류 특화 |
| **게임플레이 영향** | 캐릭터 자체 강해짐 | 카드 선택의 의미 증가 |

## ⚖️ 캐릭터 밸런스 전략

### 세 가지 빌드 선택지

```
🔴 공격형 빌드 (DPS 극대화)
├─ 무기: "폭열의 창" (높은 데미지)
├─ 반지: "무기 스킬 데미지 +20%" 옵션
└─ 목걸이: "ATTACK 카드 +30%" 옵션
   → 결과: 엄청난 데미지! (취약: 방어 약함)

🔵 방어형 빌드 (생존 극대화)
├─ 방어구: "용의 갑옷" (높은 방어)
├─ 반지: "방어력 +15%" 옵션
└─ 목걸이: "SKILL 카드 회피 +40%" 옵션
   → 결과: 거의 안 죽음! (취약: 공격 약함)

🟢 균형형 빌드 (유연한 플레이)
├─ 무기 + 방어구 + 반지: 모두 중간값
├─ 목걸이: "전체 카드 +20%" 옵션
└─ 모든 카드 타입 고르게 강함
   → 결과: 안정적 (취약: 극단성 없음)

🟡 마나형 빌드 (스킬 회전)
├─ 반지: "최대 마나 +50, 회복율 +10%"
├─ 목걸이: "SKILL 카드 재사용 -1"
└─ 스킬을 매우 자주 사용
   → 결과: 스킬 중심 플레이 (취약: 기본 공격 약함)
```

등급 4: Epic (보라)
├─ 내구도: 높음
└─ 스탯: 높음

등급 5: Legendary (주황) ← 최고급
├─ 내구도: 매우 높음
└─ 스탯: 매우 높음

## 🎯 게임플레이 흐름

```
1️⃣ 전투 시작
   ↓
2️⃣ 카드 선택 (ATTACK/SKILL/POWER/CURSE 중 1~2개)
   └─ 목걸이가 각 카드의 효율에 영향
   ↓
3️⃣ 공격/방어/스킬 실행
   ├─ 공격력 = 무기 + 반지 강화
   ├─ 방어력 = 방어구 + 반지 강화
   ├─ 패링 성공 확률 = 목걸이 영향
   ↓
4️⃣ 마나 관리
   ├─ 스킬 사용 (반지가 마나량 결정)
   ├─ 마나 자동 회복
   ↓
5️⃣ 전투 후 보상
   └─ 꿈 수집량 = 목걸이 보너스
```

## 🔗 전체 연결 구조

### 반지의 연쇄 효과 예시

```
【힘의 반지 착용】
   ↓
반지 옵션: "무기 스킬 데미지 +20%"
   ↓
【마법의 검 스킬 발동】
   ├─ 원래 데미지: 공격력 200%
   ├─ 반지 효과: +20%
   └─ 최종 데미지: 공격력 240% ⭐ (20% 더 강함)

+ 반지 기본 스탯: ATK +20
   ↓
캐릭터 최종 공격력 = 기본 + 무기 40 + 반지 20 = 60 증가
```

### 목걸이의 연쇄 효과 예시

```
【공격의 목걸이 착용】
   ↓
목걸이 옵션: "ATTACK 카드 +30%"
   ↓
【ATTACK 카드 선택】
   ├─ 원래 데미지: 200
   ├─ 목걸이 효과: +30%
   └─ 최종 데미지: 260 ⭐ (30% 더 강함)

+ 목걸이 기본 스탯: ATTACK 카드 +25%
   ↓
총 ATTACK 카드 강화: +55% (매우 강해짐)
```
