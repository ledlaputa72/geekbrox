# 🛡️ Dream Collector — 캐릭터 장비 시스템 기획서

**작성:** Atlas (PM)  
**기준:** Dream Collector ATB 기본 공격 시스템  
**버전:** v1.0  
**상태:** 📋 기획 완료

---

## 📌 개요

**목적:** 카드 기반 전투에서 벗어나 기본 공격(ATB 오토 전투)을 강화하기 위한 장비 시스템

**역할:** 
- 캐릭터의 기본 공격 데미지/효과 결정
- 캐릭터 특성 강화
- 캐릭터 커스터마이징의 핵심 요소

**특징:**
- 카드와 다른 영역 (기본 공격 = 100% 자동)
- 장비를 통한 캐릭터 개성 표현
- 수집 및 강화 요소 제공

## 🔧 장비 시스템 구조

### 장비 슬롯 (4가지)

#### 1️⃣ Weapon (무기)
```
역할: 기본 공격 데미지 결정
특징: 
  - 가장 중요한 공격력 결정 요소
  - 공격 타입 결정 (물리/마법)
  - 기본 공격 애니메이션 변경
  
스탯:
  - 공격력 (ATK): 10~50
  - 치명타율 (CRI): 0~30%
  - 공격 속도 (SPD): -20% ~ +20%
```

**예시:**
```
剣の欠片 (검의 파편)
  ATK: 10, CRI: 0%, SPD: 0%
  
강철 검 (강철검)
  ATK: 20, CRI: 5%, SPD: +5%
  
마법 지팡이 (마법 지팡이)
  ATK: 15, CRI: 10%, SPD: +10%
```

---

#### 2️⃣ Armor (방어구)
```
역할: 기본 방어력 및 생명력 증가
특징:
  - 생존력 강화
  - 상태이상 저항 추가
  - 매턴 회복 가능
  
스탯:
  - 방어력 (DEF): 5~30
  - 생명력 (HP): 50~300
  - 저항력 (RES): 0~20% (독/화상/냉기 등)
```

**예시:**
```
청동 가슴판 (청동 가슴판)
  DEF: 5, HP: 50, RES: 0%
  
강철 갑옷 (강철 갑옷)
  DEF: 15, HP: 150, RES: 10%
  
신성한 로브 (신성한 로브)
  DEF: 10, HP: 200, RES: 20% (마법 저항)
```

---

#### 3️⃣ Accessory (악세서리)
```
역할: 특수 효과 부여
특징:
  - 가장 다양한 효과
  - 기본 공격에 추가 효과
  - 패시브 능력 부여
  
효과 종류:
  - 흡혈 (기본 공격 데미지의 X% 회복)
  - 반사 (받은 데미지의 X% 반격)
  - 버스트 (매 3턴마다 공격력 +30%)
  - 연쇄 (공격이 2회 연속 발동)
```

**예시:**
```
흡혈 반지 (흡혈 반지)
  효과: 기본 공격 데미지의 30% 회복
  
방어의 팔찌 (방어의 팔찌)
  효과: 받은 데미지의 20% 반격
  
속도의 신발 (속도의 신발)
  효과: 공격 속도 +25%
```

---

#### 4️⃣ Off-Hand (보조 무기)
```
역할: 보조 공격 또는 특수 능력
특징:
  - 기본 공격에 추가 타격
  - 특수 상태이상 추가
  - 도발 또는 회피 능력
  
스탯:
  - 추가 데미지 (5~25)
  - 상태이상 추가 (독/연소/냉기 등)
  - 도발율 / 회피율 (5~30%)
```

**예시:**
```
단검 (단검)
  추가 데미지: 5, 상태이상: 없음
  
독 칠한 단검 (독 칠한 단검)
  추가 데미지: 8, 상태이상: 독 (20% 확률)
  
방패 (방패)
  추가 데미지: 0, 회피율: 15%
```

## 👥 캐릭터 특성 강화

### 기존 특성 구조
```
- 주요 속성 (Nox = 기억/꿈 속성)
- 전투 스타일 (회피형/탱크형/공격형)
- 리액션 능력 (PARRY/DODGE/GUARD)
```

### 장비 연결 특성 추가

#### 특성 1: 무기 마스터리 (Weapon Mastery)
```
효과: 
  - 무기 공격력 +X% (특성 레벨에 따라)
  - 특정 무기 타입 전문화
  
예시:
  - 레벨 1: 무기 공격력 +5%
  - 레벨 3: 무기 공격력 +15%, 치명타 +5%
  - 레벨 5: 무기 공격력 +25%, 치명타 +15%
```

#### 특성 2: 방어 강화 (Defense Mastery)
```
효과:
  - 방어구 방어력 +X%
  - 생명력 +X
  
예시:
  - 레벨 1: 방어력 +5%, HP +20
  - 레벨 3: 방어력 +15%, HP +50
  - 레벨 5: 방어력 +25%, HP +100
```

#### 특성 3: 악세서리 동조 (Accessory Resonance)
```
효과:
  - 착용한 악세서리 효과 +X%
  - 여러 악세서리 착용 시 시너지
  
예시:
  - 1개: 효과 100%
  - 2개: 각각 120%, 시너지 +10% 데미지
  - 3개: 각각 150%, 시너지 +20% 데미지
```

#### 특성 4: 듀얼 웰드 (Dual Wield)
```
효과:
  - 보조 무기 공격 50% 강화
  - 보조 무기 추가 공격 확률 증가
  
예시:
  - 레벨 1: 보조 무기 +3% 추가 공격 확률
  - 레벨 3: 보조 무기 +10% 추가 공격 확률
  - 레벨 5: 보조 무기 +20% 추가 공격 확률, 데미지 +25%
```

#### 특성 5: 장비 연쇄 (Equipment Chain)
```
효과:
  - 같은 시리즈 장비 착용 시 추가 보너스
  - 세트 효과 발동
  
예시:
  - 검 세트 (검+방어구+반지): ATK +20%, DEF +15%
  - 마법사 세트 (지팡이+로브+신발): 마법 데미지 +30%, 속도 +15%
```

## 🔗 장비-캐릭터 특성 시너지

### 무기 마스터리 예시
```
Nox + 검
  기본 공격: 검의 파편 10 ATK
  무기 마스터리 레벨 3: +15% ATK = 11.5 공격력
  + 강철 검: 20 ATK
  + 마스터리: 20 × 1.15 = 23 ATK
```

### 악세서리 동조 예시
```
Nox + 흡혈 반지 + 속도의 반지
  기본 공격 데미지: 25
  흡혈 반지: 25 × 0.3 = 7.5 회복
  속도의 반지: 공격 속도 +20%
  악세서리 동조 레벨 3: 효과 150%, 시너지 +20% 데미지
  = 25 × 1.2 = 30 데미지, 9 회복
```

### 세트 효과 예시
```
Nox + 검 세트 (검 + 강철 갑옷 + 왕의 반지)
  - ATK +20%
  - DEF +15%
  - 기본 공격에 치명타 확률 +5%
```

## 📊 장비 획득 및 강화

### 획득 방법
```
1. 스토리 보상: 주요 악당 격파 시 고급 장비 획득
2. 일일 던전: 특정 던전에서 장비 드롭 (희귀도별)
3. 합성: 낮은 등급 장비 2개 → 높은 등급 장비 1개
4. 가챠: 프리미엄 뽑기로 희귀 장비 획득
5. 교환: 수집 아이템으로 NPC와 교환
```

### 강화 시스템
```
단계: +0 ~ +10
비용: 골드 (단계에 따라 증가)
효과: 단계마다 스탯 +5~10%
위험: +7부터 강화 실패 가능, 단계 하락
```

## 🎯 캐릭터 기본 공격 공식

```
기본 공격 데미지 = (무기 ATK + 보조 무기 추가 데미지) 
                 × (1 + 무기 마스터리%) 
                 × (1 + 캐릭터 ATK 스탯%)
                 × (1 + 악세서리 시너지%)
                 × (치명타 판정)

방어 계산 = 적 DEF - (우리 무기 관통력) 
         × (1 - 적 DEF 마스터리%)
         × (1 - 악세서리 저항력%)
```
