## 🐛 발견된 이슈

### 심각한 이슈 (High Priority)

```
1. 후반 게임 강화 비용 문제
   심각도: 높음
   영향: 15명 (30%) 이탈 고위험
   원인: 강화 비용 지수 증가
   해결책: 강화 비용 1.5배 → 1.4배 조정
   
2. 카드 게임 오버 (High Priority) 
   심각도: 높음
   영향: 카드에만 집중, 장비 무시
   원인: 카드 손패 시스템이 강함
   해결책: 장비 보너스 +10% 증가
   
3. 초반 튜토리얼 부족
   심각도: 중간
   영향: 새 플레이어 4명 이탈
   원인: 복잡한 UI, 설명 부족
   해결책: 튜토리얼 강화 (10분 추가)
```

### 중간 이슈 (Medium Priority)

```
4. 사이드 퀘스트 보상이 작음
   영향: 8명 불만
   해결책: 보상 +20%

5. NPC 친밀도 진행이 느림
   영향: 5명 불만
   해결책: 친밀도 +25%

6. 일일 던전 보상 불균형
   영향: 중간 (밸런스 이슈)
   해결책: 드롭률 조정
```

### 경미한 이슈 (Low Priority)

```
7. UI 버튼 너무 작음
   영향: 2명 (고령층)
   해결책: UI 스케일 옵션 추가

8. 음향 설정 부족
   영향: 3명
   해결책: 음량 조절, 음악 끄기 옵션

9. 저사양 핸드폰 성능
   영향: 1명
   해결책: 그래픽 최적화 (후속)
```

## 💰 경제 시스템 평가

### 금화 (골드) 균형

```
초기 (Lv 1-20): 균형 잡힘 ✅
중기 (Lv 30-60): 약간 부족 ⚠️
후기 (Lv 70-100): 심각히 부족 🔴

테스터 피드백:
  "처음에는 돈이 많은데 나중에 부족해진다" (25명)
  "강화 비용이 너무 비싸다" (15명)
  "보상이 적다" (10명)

권장 조정:
  1. 강화 비용 기본값: 1,500 G (1,000 G에서 50% 증가)
     → 1.4배 배수 적용 (1.5배에서 변경)
  2. 후기 보상 +30%
  3. 월간 보너스 +5,000 G
  4. 보스 드롭 골드 +50%

예상 효과:
  초기: 변화 없음
  중기: 약간 향상
  후기: 완전히 해결
```

### 다이아몬드 (유료) 균형

```
무료 플레이어: 월 500 D (자동 획득)
  "가챠 너무 비싸다" (35명)
  해결책: 매달 무료 가챠 2배 (10회 → 20회)

유료 플레이어: 월 $10 = 1,000 D
  "가치 있다" (평가 좋음)
  "더 저렴하면 좋겠다" (일부)

권장:
  - 무료 가챠 +100% (20회)
  - 월간 무료 다이아 +200 D
  - 신규 유저 선물 증가
```
