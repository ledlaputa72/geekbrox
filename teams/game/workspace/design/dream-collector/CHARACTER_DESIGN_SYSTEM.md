## 👤 메인 캐릭터: Nox (꿈수집가)

### 캐릭터 정보

```
이름: Nox (녹스)
한글: 녹스 / 꿈수집가
나이: 불명 (기억 상실)
성별: 중성 (선택 가능)
속성: 기억 / 꿈

시각적 특징:
- 3-4 head 동전(SD) 비율
- 앙증맞은 체형
- 신비로운 옷차림
- 꿈과 기억을 상징하는 액세서리
```

### 캐릭터 스토리

```
설정: Nox는 자신의 기억을 잃어버린 존재로,
      "The Mnemonic Weaver"의 세계를 떠돌며 
      잃어버린 기억(꿈)을 수집하려고 한다.

역할: 
- 플레이어 아바타
- 잃어버린 기억을 찾으려는 여정
- 다른 캐릭터들을 만나면서 세상을 구하는 영웅

성격:
- 호기심 많음
- 동정심 있음
- 신비로움
- 강한 의지
```

### 캐릭터 특성 (기본)

```
속성: 기억/꿈 (Memory/Dream)

기본 스탯 (Lv 1):
  HP: 100
  ATK: 15
  DEF: 10
  SPD: 12

성장률 (Lv 100까지):
  HP: +200
  ATK: +85
  DEF: +90
  SPD: +38

리액션 능력:
  - DODGE (회피): 본능적인 기억 회피
  - PARRY (격막): 기억을 이용한 방어
  - GUARD (경계): 꿈의 보호막
```

## 🎮 플레이 스타일 (3가지)

### 1️⃣ Architect (건축자) - 공격형

```
설정: 세상을 탐험하며 기억을 수집하는 여행자

특징:
  - ATK +20%
  - SPD +10%
  - DEF -10%

카드 보너스:
  - ATTACK 카드 효과 +15%
  - 기본 공격 +25%

권장 장비:
  - 무기: 검, 지팡이 (높은 공격력)
  - 특성: 무기 마스터리, 공격 중심

플레이 느낌:
  공격적이고 역동적인 전투
  빠른 진행 속도
  높은 리스크-리워드
```

### 2️⃣ Healer (치유자) - 방어형

```
설정: 잃어버린 영혼들을 돕는 치유자

특징:
  - ATK -10%
  - DEF +20%
  - HP +15%

카드 보너스:
  - SKILL 카드 효과 +15%
  - 매턴 회복 +3HP

권장 장비:
  - 방어구: 로브, 경갑 (높은 방어력)
  - 특성: 방어 강화, 악세서리 회복

플레이 느낌:
  방어적이고 안정적인 전투
  오래 살아남기
  낮은 리스크-높은 보상
```

### 3️⃣ Mystic (신비가) - 균형형

```
설정: 기억과 꿈 사이를 오가는 신비로운 존재

특징:
  - ATK: 0% (기본값)
  - DEF: 0% (기본값)
  - SPD +15%

카드 보너스:
  - POWER 카드 효과 +20%
  - CURSE 카드 사용 + 이득 +10%

권장 장비:
  - 무기+방어구 균형 (다양한 조합)
  - 특성: 악세서리 동조, 세트 효과

플레이 느낌:
  전략적이고 유연한 전투
  다양한 빌드 가능
  중간 난이도
```

## 📈 캐릭터 성장 시스템

### Lv 1 ~ Lv 100

```
경험치 획득:
  - 적 처치: 10-100 EXP
  - 퀘스트: 500-5,000 EXP
  - 이벤트: 500-10,000 EXP

레벨 기본 공식:
  필요 EXP = 100 × (Lv^1.5)
  
  Lv 1: 100 EXP
  Lv 10: 3,162 EXP
  Lv 50: 35,355 EXP
  Lv 100: 100,000 EXP
  
  누적: 약 1,000,000 EXP (Lv 100 도달)

성장 보상:
  5 Lv마다: 특별 스킬 포인트 +1
  10 Lv마다: 고급 장비 획득
  25 Lv마다: 새로운 카드 타입 언락
  50 Lv마다: 세계 확장 (새 지역)
```

### 특성 포인트 시스템

```
매 레벨업: 5 포인트 (Lv 1 = 5pt, Lv 100 = 500pt)

분배 가능 항목:
  - 공격력 (ATK): 1pt = +1 ATK
  - 방어력 (DEF): 1pt = +1 DEF
  - 생명력 (HP): 1pt = +5 HP
  - 속도 (SPD): 1pt = +0.5 SPD
  - 특수 스킬 언락 (10~50pt)

추천 빌드:
  Architect: ATK 60%, DEF 10%, HP 20%, SPD 10%
  Healer: ATK 10%, DEF 50%, HP 30%, SPD 10%
  Mystic: ATK 25%, DEF 25%, HP 25%, SPD 25%
```

## 🎬 캐릭터 애니메이션

### 기본 애니메이션

```
1. 대기 (Idle)
   - 부드러운 숨 쉬는 모션
   - 1.5초 루프
   - 신비로운 분위기

2. 기본 공격 (Attack)
   - 무기에 따라 다른 모션
   - 0.6초 (검), 0.8초 (지팡이)
   - 차원 효과 추가

3. 스킬 사용 (Skill)
   - 카드 선택 후 3초 시퀀스
   - 스킬별 독특한 이펙트
   - 파티클 효과 포함

4. 피해 (Damaged)
   - 0.3초 히트백
   - 빨강 색상 플래시
   - 사운드 효과

5. 회복 (Recovery)
   - 반짝이는 이펙트
   - 1초 동안 빛의 입자
   - 회복량 표시

6. 죽음 (Death)
   - 3초 페이드 아웃
   - 꿈이 사라지는 느낌
   - 부드러운 배경음악
```

### 플레이 스타일별 애니메이션

```
Architect:
  - 빠르고 역동적인 모션
  - 강한 공격 이펙트
  - 발광하는 검/지팡이

Healer:
  - 부드럽고 우아한 모션
  - 치유의 녹색 이펙트
  - 방어막 실드 이펙트

Mystic:
  - 신비로운 모션
  - 보라색/파란색 이펙트
  - 차원 넘나드는 느낌
```
