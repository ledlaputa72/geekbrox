# 🎲 Dream Collector — 가챠 + 강화 시스템 최종 설계 (간소화)

**작성:** Atlas (PM) → Game팀  
**기준:** 장비↔장비, 카드↔카드 강화 (명확한 구분)  
**버전:** v1.0 Final  
**상태:** 📋 최종 설계 완료

---

## 📌 핵심 설계 (간소화)

```
강화 = 같은 종류끼리만

장비 강화:
  장비 + 장비 (같은 종류) + 골드
  
카드 강화:
  카드 + 카드 (같은 종류) + 골드

크로스 강화 없음 (더 간단함!)
```

## 2️⃣ 강화 시스템 (분리된 구조)

### A. 장비 강화

**기본 구조:**

```
같은 장비 2개 + 골드 = 장비 강화도 +1

예:
  강철검 +3 강화 시키고 싶음
  필요: 강철검 (기본) × 1개 + 골드 1,000 G
  결과: 강철검 +4가 됨
  
기존 강철검 +3과 새 강철검이 합쳐져서
→ 강철검 +4 (1개)로 변환
```

**강화도별 비용:**

```
+0→+1: 500 G + 같은 장비 1개
+1→+2: 750 G + 같은 장비 1개
+2→+3: 1,000 G + 같은 장비 1개
+3→+4: 1,500 G + 같은 장비 1개
+4→+5: 2,000 G + 같은 장비 1개
+5→+6: 3,000 G + 같은 장비 1개
+6→+7: 4,000 G + 같은 장비 1개
+7→+8: 5,500 G + 같은 장비 1개
+8→+9: 7,500 G + 같은 장비 1개
+9→+10: 10,000 G + 같은 장비 1개

누적: 약 36,250 G + 장비 10개
```

**장비 강화의 보상:**

```
강화도 +1 = 스탯 +5%

예:
  무기 ATK 20
  +5 강화 → 20 × 1.25 = 25
  +10 강화 → 20 × 1.50 = 30 (50% 증가)

방어구, 악세서리도 동일
```

**성공률:**

```
모든 단계: 100% (실패 없음)
= 장비 준비 → 즉시 강화 완료
```

### B. 카드 강화

**기본 구조:**

```
같은 카드 2개 = 카드 강화도 +1

또는

같은 카드 1개 + 골드 = 카드 강화도 +1

예:
  공격 카드 +2 강화 시키고 싶음
  필요: 공격 카드 (기본) × 1개
  또는: 공격 카드 (기본) × 1개 + 100 G
  결과: 공격 카드 +3이 됨
```

**강화도별 비용:**

```
+0→+1: 100 G + 같은 카드 1개
       또는 같은 카드 2개 (골드 없음)

+1→+2: 150 G + 같은 카드 1개

+2→+3: 200 G + 같은 카드 1개

... (패턴: 50 G 증가)

+9→+10: 500 G + 같은 카드 1개

누적: 약 2,750 G + 카드 10개
```

**카드 강화의 보상:**

```
강화도 +1 = 카드 효과 +10%

예:
  공격 카드 기본 25 데미지
  +5 강화 → 25 × 1.50 = 37.5
  +10 강화 → 25 × 2.00 = 50 (100% 증가!)
  
같은 카드가 중복되면 자연스럽게 강화 진행
```

**성공률:**

```
모든 단계: 100% (실패 없음)
```

## 3️⃣ 가챠 → 강화 경제 루프

### 플레이어 진행 흐름

```
[가챤]
  ↓
[같은 종류 획득]
  
  예: 강철검 + 강철검 = 강화 재료!
  예: 공격 카드 + 공격 카드 = 강화 재료!
  
  ↓
[강화]
  
  장비:
    강철검 + 강철검 + 1,000 G
    → 강철검 한 단계 강화
  
  카드:
    공격 카드 + 공격 카드 + 100 G
    → 공격 카드 한 단계 강화
    
  또는 (카드 전용):
    공격 카드 + 공격 카드 (골드 없음)
    → 공격 카드 한 단계 강화
  
  ↓
[강화 완료]
  스탯 증가 (5~10%)
  
  ↓
[반복]
```

## 4️⃣ 희귀도별 활용법

### 장비의 경우

```
COMMON 장비:
  → 직접 사용 (초반)
  → 강화 재료로 사용 (나중)

RARE 장비:
  → 직접 사용 (중급 타겟)
  → 강화 재료로 사용 (고급 장비 강화)

SPECIAL 장비:
  → 직접 사용 및 강화 (중점)
  → 장기간 사용하는 메인 장비

LEGENDARY 장비:
  → 직접 사용 (최고급)
  → 장기간 강화 (최종 형태)
```

**예시:**

```
Week 1: COMMON 검 2개
  → 1개는 장착, 1개는 강화 재료로 사용
  → 검 +1 달성

Week 2: COMMON 검 2개 또 얻음
  → 검 +2 달성

Week 4: RARE 검 1개 획득
  → 직접 장착 (더 강함)
  → RARE 검을 강화할 때까지 사용

Month 1: SPECIAL 검 1개 획득
  → 직접 장착
  → 이걸 메인 장비로 강화 시작
  → +10까지 강화하면 메인 무기 완성
```

### 카드의 경우

```
COMMON 카드:
  → 직접 사용 (초반, 손패 구성)
  → 강화 재료로 사용 (자연스러움)

RARE 카드:
  → 직접 사용 (더 강함)
  → 강화 재료로 사용 (고급 카드 강화)

SPECIAL 카드:
  → 주로 직접 사용
  → 중복되면 강화 재료로

LEGENDARY 카드:
  → 거의 직접 사용
  → 매우 드문 중복만 강화 재료
```

**예시:**

```
Week 1: 공격 카드 COMMON 2개
  → 1개는 손패에, 1개는 강화
  → 공격 카드 +1

Week 2: 공격 카드 COMMON 1개, 방어 카드 RARE 1개
  → 공격 카드 +2
  → 방어 카드는 직접 사용

Week 3: 공격 카드 RARE 1개 획득
  → COMMON 공격 카드 강화 중단
  → RARE 공격 카드 사용 시작

Month 1: 공격 카드 RARE를 +5까지 강화
  → 처음 RARE 공격 카드는 +5
  → 더 나은 공격 카드 나올 때까지 사용
```

## 5️⃣ 강화 재료 획득 방법

### 자연스러운 강화 진행

**장비:**
- 매 가챠마다 일정 확률로 중복 장비 획득
- 중복된 같은 장비 = 강화 재료
- 처음에는 느리지만, 가챠 누적하면 자연스럽게 강화 재료 쌓임

**카드:**
- 매 가챠마다 자주 중복 카드 획득
- 중복된 같은 카드 = 강화 재료
- 자연스러운 강화 진행 (장비보다 빠름)

**강화석 (선택적):**
- 마일스톤 보상으로 제공
- 같은 아이템이 부족할 때만 사용

```
예: 장비 강화석
  1개 = 같은 장비 1개를 대체 가능
  (골드는 여전히 필요)
  
예: 카드 강화석
  1개 = 같은 카드 1개를 대체 가능
```

## 🎯 최종 강화 체계 (정리)

### 장비 강화

```
구조:
  같은 장비 2개 + 골드 → 1개 장비 +1

비용:
  총 36,250 G + 같은 장비 10개
  (1개 장비를 +0→+10)

속도:
  느림 (같은 장비 획득 대기)
  
성공률:
  100% (항상 성공)

효과:
  강화도 +1 = 스탯 +5%
```

### 카드 강화

```
구조:
  같은 카드 2개 → 1개 카드 +1
  또는
  같은 카드 1개 + 100~500 G → 1개 카드 +1

비용:
  총 2,750 G + 같은 카드 10개
  (1개 카드를 +0→+10)
  또는 골드 없이 중복만으로 진행 가능

속도:
  빠름 (카드가 자주 중복됨)
  
성공률:
  100% (항상 성공)

효과:
  강화도 +1 = 카드 효과 +10%
```

## ✅ 설계의 장점

```
간단함:
  ✅ 장비는 장비끼리, 카드는 카드끼리
  ✅ 크로스 강화 없음 (혼동 없음)

명확함:
  ✅ 필요한 것이 뭔지 명확함
  ✅ 같은 아이템 = 강화 재료 (직관적)

자연스러움:
  ✅ 가챠하다 보면 자연스럽게 중복 획득
  ✅ 중복 = 강화 (자연스러운 순환)

유연함:
  ✅ 카드: 골드 없이도 가능 (중복만으로)
  ✅ 장비: 골드 투자로 가속 가능
```
