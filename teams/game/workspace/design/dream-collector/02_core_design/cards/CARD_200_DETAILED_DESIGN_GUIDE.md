## 📌 개요

Dream Collector의 초기 론칭을 위한 **200개 카드**의 세부 데이터 생성 가이드입니다.

### 목표
- ✅ 4가지 메인 타입 (ATTACK/SKILL/POWER/CURSE)
- ✅ 10개 서브카테고리
- ✅ 200개 카드 완벽한 JSON/CSV 데이터
- ✅ CardDatabase.gd 연동 준비
- ✅ 게임 밸런스 검증

### 기본 구조
```
ATTACK (42)     → ATK-SGL(15) + ATK-MLT(12) + ATK-RCK(5) + ATK-CMP(10)
SKILL (60)      → SKL-GRD(20) + SKL-PAR(20) + SKL-DOD(20)
POWER (50)      → PWR-DRW(12) + PWR-ATK(15) + PWR-DEF(12) + PWR-SUS(11)
CURSE (48)      → CRS-STA(12) + CRS-SPD(12) + CRS-PEN(12) + CRS-RSK(12)
───────────────────────────────────────────────────────────────
합계: 200개
```

## 🎴 **타입 3: POWER (50장)**

### PWR-DRW (드로우) — 12장

**개념:** 카드 드로우 또는 에너지 회복 (즉시 효과).

**설계 원칙:**
```
0비용: 드로우+1, 에너지+1
1비용: 드로우+2, 에너지 회복 20%
2~3비용: 드로우+2~3, 복합 버프
```

**특징:**
- MAJOR_ARCANA 80% (10장)
- 희귀도: COMMON 20%, RARE 50%, SPECIAL 30%
- 지속 효과 아님 (즉시 효과)

---

### PWR-ATK (공격강화) — 15장

**개념:** 다음 공격들의 데미지 증가 (3~5턴 지속).

**설계 원칙:**
```
1비용: 힘+1 (3턴)
2비용: 힘+2 또는 민첩+1 (4턴)
3비용: 힘+3 (5턴)
```

**특징:**
- 스택 가능 (같은 버프 2회 = 효과 2배)
- MAJOR_ARCANA 50% (7~8장)
- 지속 턴수: 짧을수록 더 강함

---

### PWR-DEF (방어강화) — 12장

**개념:** HP 회복, 블록 자동생성, 저항 증가 (지속).

**설계 원칙:**
```
1비용: 회복 8~10
2비용: 회복 12~15 또는 매턴 블록+3
3비용: 회복 18~20 또는 매턴 블록+5 (3턴)
```

**특징:**
- 방어/회복 강화 중심
- MAJOR_ARCANA 40%
- 지속 효과 (턴 제한)

---

### PWR-SUS (지속) — 11장

**개념:** 복합 지속 버프 또는 조건부 효과. 가장 강력한 파워 카드.

**설계 원칙:**
```
예시:
- "매 공격 후 에너지+0.5 (전투 중)"
- "패링 성공 시 드로우+1"
- "약화 카드 사용 시 데미지+3"
- "매 턴 시작 시 일반 카드 1장 변환"
```

**특징:**
- 비용: 2~4 (높은 비용)
- MAJOR_ARCANA 60% (7장)
- 희귀도: RARE 50%, SPECIAL 35%, LEGENDARY 15%
- 전투 흐름을 완전히 바꾸는 카드

**개념:** 적의 방어 무시 또는 약화 상태 강화.

**특징:**
- 비용 높음, 강력함

### CRS-RSK (리스크) — 12장

**특징:**
- 저비용, 높은 효과

- 전략적 선택이 중요
- 희귀도: RARE 50%, SPECIAL 35%, LEGENDARY 15%
