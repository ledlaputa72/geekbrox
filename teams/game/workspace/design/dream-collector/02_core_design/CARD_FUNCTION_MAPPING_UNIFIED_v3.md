### 관계

```
기능 ID (1) → 카드 ID (1 이상)

예:
- ATK-01 (기본 공격) → SGL_001, SGL_002, SGL_003
- DEF-01 (기본 블록) → GRD_001, GRD_002, GRD_006
```
