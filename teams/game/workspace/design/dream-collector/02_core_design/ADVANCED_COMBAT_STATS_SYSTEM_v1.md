## 1. 전투 속성 완전 체계

### 1.1 속성 레이어 구조

카피바라 Go의 100+ 속성 체계를 참조하되, Dream Collector의 **카드·ATB 전투** 방식에 최적화하여 4개 레이어로 재설계한다.

```
┌───────────────────────────────────────────────────────────┐
│  Layer 0: 기본 속성 (Base)   — 플레이어가 항상 보는 수치    │
│  HP / 공격력 / 방어력 / 속도                               │
├───────────────────────────────────────────────────────────┤
│  Layer 1: 전투 속성 (Combat) — 요약 노출, 장비로 획득      │
│  치명타/회피/반격/흡혈/방어구파괴 등                       │
├───────────────────────────────────────────────────────────┤
│  Layer 2: 카드 타입 속성     — 장비·특성 연동, 백엔드      │
│  ATTACK/SKILL/POWER/CURSE별 치명타·데미지 증감             │
├───────────────────────────────────────────────────────────┤
│  Layer 3: 원소 속성          — 몬스터·던전 연동, 백엔드    │
│  꿈기억/불꽃/냉기/번개/독 계열 피해·저항                   │
└───────────────────────────────────────────────────────────┘
```

```

불꽃   → 냉기에 강함 (열이 얼음을 녹임)
냉기   → 번개에 강함 (전도율 차단)
```

## 4. 장비 속성 풀 확장

### 4.1 장비 옵션 시스템 개요

카피바라 Go의 세부 옵션 체계를 참조하여 Dream Collector 장비 옵션 풀을 확장한다.

**옵션 등급별 개수:**

| 장비 등급 | 기본 옵션 | 추가 옵션 | 잠금 해제 옵션 |
|----------|----------|----------|--------------|
| Common | 1 | 0 | 0 |
| Uncommon | 1 | 1 (선택) | 0 |
| Rare | 1 | 1 (선택) | 1 (잠금) |
| Epic | 2 | 1 (선택) | 1 (잠금) |
| Legendary | 2 | 2 (선택) | 2 (잠금) |

> **잠금 옵션** = 장비 강화 레벨 도달 시 해금 (Lv5, Lv10 등)

---

### 4.2 무기 (Weapon) 속성 풀

**기본 속성 (고정):**
- 공격력 +X (등급별 범위: Common +10~20 / Legendary +90~120)

### 5.3 몬스터 속성 상세 구조

각 몬스터는 플레이어와 동일한 속성 레이어를 보유:

```
몬스터 속성 구조
├─ 기본 속성: HP / ATK / DEF / SPD
├─ 원소: 고유 원소 타입 (피해 배율 적용)
├─ 면역: 특정 상태이상 면역 (예: 동결 면역)
├─ 취약: 특정 원소 데미지 ×1.5
├─ 특수 메커닉: 보스별 고유 시스템 (방어막, 격노, 페이즈 등)
└─ 드롭 테이블: 등급·원소별 드롭 아이템
```

| 11~30 | 300~800 | 25~60 | 2,000~5,000 | 50~100 |
| 31~60 | 1,000~3,000 | 80~150 | 10,000~30,000 | 150~250 |
| 61~100 | 5,000~15,000 | 200~400 | 50,000~200,000 | 350~600 |

### 5.4 보스 전투 메커닉 상세

Dream Collector 보스는 **턴베이스 전투 + 고유 페이즈**:

```
보스 전투 흐름:
Phase 1 (HP 100%~60%)
└─ 기본 패턴 + 원소 공격

Phase 2 (HP 60%~30%) ← 격노
└─ ATK ×1.5, 특수 스킬 추가
└─ 원소 효과 강화

Phase 3 (HP 30%~0%) ← 절망 모드
└─ 모든 공격에 상태이상 추가
└─ 1회용 필살기 발동

보스 고유 메커닉 예시:
- 방어막형: 매 3턴마다 방어막 재생 → CURSE/POWER 카드로 파괴
- 흡수형: 특정 원소 공격을 흡수하여 회복
- 반전형: 플레이어 카드 타입 효과 역전 (한 페이즈 동안)
- 카드 봉인형: 덱에서 특정 타입 카드 봉인
```

## 6. 데미지 계산 공식 및 연산 순서

### 6.1 전체 데미지 연산 순서

```
1. 기본 데미지
   = ATK × 카드 배율 (%)

2. 카드 타입 보너스
   × (1 + ATTACK/SKILL/POWER/CURSE 데미지 증가%)

3. 원소 배율
   × 원소 상성 배율 (0.7 / 1.0 / 1.5)
   × (1 + 원소 데미지 증가%)
   × (1 - 적 원소 경감%)

4. 치명타
   × 치명타 발동 여부 (확률 판정)
   × 치명타 데미지 배율 (기본 150% + 보너스)

5. 방어력 감소
   × (1 - DEF / (DEF + 정규화 상수))
   × (1 - 방어구 파괴 영향)

6. 최종 피해 증감
   × (1 + 최종 피해 증가%)
   × (1 - 최종 피해 감소%)

7. 흡혈 적용 (별도 처리)
   회복 HP = 최종 데미지 × 흡혈 효과%
```

### 6.2 방어력 공식

```
피해 감소율 = DEF / (DEF + K)
K = 100 (기준 상수, 조정 가능)

예시:
- DEF 63:  63 / (63+100) = 38.7% 감소
- DEF 150: 150 / (150+100) = 60.0% 감소
- DEF 300: 300 / (300+100) = 75.0% 감소 (최대 75% 제한)
```

### 6.3 연타 (Multi-hit) 공식

```
연타 발동: 공격 후 연타 확률 판정
추가 타격 수: 1~2회 (확률 가중치)
추가 타격 데미지 = 기본 공격 × 40%
연타 최대 적용: 1회 공격당 최대 2연타
```

### 6.4 반격 (Counter) 공식 (PARRY 연동)

```
반격 발동 조건: PARRY 성공 + 반격 확률 판정
반격 데미지 = ATK × 반격 데미지 배율 (기본 80%)
              × (1 + 반격 데미지 증가%)
반격 원소: 무기 원소 적용
```

## 7. 운영 밸런스 가이드라인

### 7.1 속성 캡 (Soft/Hard Cap)

| 속성 | Soft Cap | Hard Cap | 초과 시 |
|------|----------|----------|--------|
| 치명타 확률 | 40% | 60% | 초과분 50% 환산 |
| 회피 확률 | 30% | 50% | 초과 불가 |
| 최종 피해 감소 | 50% | 70% | 초과 불가 |
| 연타 확률 | 30% | 40% | 초과 불가 |
| 원소 경감 (단일) | 35% | 50% | 초과 불가 |

### 7.2 카드-장비-몬스터 삼각 연동

```
설계 원칙:
1. 몬스터 원소 약점 → 해당 원소 무기 유용성 보장
2. 보스 특수 메커닉 → 특정 카드 타입 요구 (다양한 덱 유도)
3. 정예 몬스터 → 전투 속성 카운터 관계 (예: 치명타 무시 몬스터 → 연타 전략으로 대응)
4. 드롭 테이블 → 해당 원소 장비 드롭 우선 (컨텐츠 연계)
```

### 7.3 플레이어 경험 설계

```
보이는 것 (플레이어 인지):
✅ 몬스터 원소 아이콘 (5가지 색상)
✅ "약점" / "강함" 텍스트 표시
✅ 보스 HP 페이즈 바
✅ 상태이상 아이콘 (화상/동결/감전/중독/망각)

숨겨진 것 (자동 계산):
✈ Layer 2~3 세부 속성 연산
✈ 연타/반격 확률 판정
✈ 원소 상성 배율 적용
✈ 치명타 세부 타입 분류
```
