## 시스템 개요

### 핵심 개념

**Dream Collector의 장비 시스템은 4가지 타입의 장비로 캐릭터의 전투 능력과 카드 플레이를 동시에 강화하는 시스템입니다.**

- **기본 구성:** 4가지 장비 타입 (무기/방어구/반지/목걸이) × 6개 슬롯
- **강화 방식:** 각 장비는 고유한 스킬(마나 소비)과 플레이어가 선택하는 옵션 제공
- **점진적 발전:** 등급(Common~Legendary)과 레벨(1~10)로 지속적 강화 가능

### 게임 루프에서의 역할

```
카드 선택 (목걸이 영향)
   ↓
전투 실행 (무기/방어구/반지 영향)
   ├─ 공격: 무기 + 반지 강화
   ├─ 방어: 방어구 + 반지 강화
   └─ 스킬: 장비 스킬 사용 (마나 소비)
   ↓
전투 후 보상 (목걸이 영향)
```

## 4가지 장비의 명확한 역할

### 1️⃣ 무기 (Weapon) — 공격 담당

**역할:** 기본 공격력 강화 및 공격 기반 스킬 제공

**슬롯:** 1개  
**주요 스탯:**
- ATK (공격력) 직접 증가
- 회심율 증가
- 공격 관련 보조 능력

**예시:**
```
【폭열의 창】(Rare, LV 5)
├─ 기본 스탯: ATK +40, 회심율 +5%
├─ 스킬: "폭열 돌격"
│  ├─ 효과: 공격력 200% 데미지 + 추가 꿈 수집 +50
│  ├─ 소비: 마나 20
│  ├─ 재사용: 2턴 대기
│  └─ 사용 시 마나 자동 회복: 턴마다 +5
└─ 옵션 선택 (1개):
   ├─ 전투 시작 시 공격력 +10%
   ├─ 방어 관통 데미지 +30%
   └─ 공격력 +15%
```

### 2️⃣ 방어구 (Armor) — 방어 담당

**역할:** 기본 방어력/HP 강화 및 방어 기반 스킬 제공

**슬롯:** 1개  
**주요 스탯:**
- DEF (방어력) 직접 증가
- HP (최대 체력) 증가
- 피해 감소 관련 능력

**예시:**
```
【용의 갑옷】(Rare, LV 5)
├─ 기본 스탯: DEF +40, HP +150, 자동 회복 +2/턴
├─ 스킬: "용의 방어"
│  ├─ 효과: 다음 공격 패링 → 반격 데미지 +150%
│  ├─ 소비: 마나 18
│  ├─ 재사용: 2턴 대기
│  └─ 성공 시 다음 턴 DEF +30%
└─ 옵션 선택 (1개):
   ├─ 받는 피해 15% 감소
   ├─ 최대 HP +100
   └─ 반격 데미지 +20%
```

### 3️⃣ 반지 (Ring) — 캐릭터 능력 강화 ⭐ KEY EQUIPMENT

**역할:** 캐릭터 자체 스탯을 강화하고, 무기/방어구의 효과를 극대화

**슬롯:** 2개  
**주요 스탯:**
- ATK/DEF 직접 증가 (캐릭터 능력 강화)
- 최대 마나, 마나 회복
- 회피율, 상태 이상 저항
- 무기/방어구 효과 강화 (간접 영향)

**설명:**  
반지는 **캐릭터 자체를 강화하는 장비**입니다. 무기와 방어구의 스킬 효율을 높이거나, 기본 스탯을 극대화합니다.

**예시:**
```
【힘의 반지】(Rare, LV 5)
├─ 기본 스탯: ATK +20, DEF +18, 최대 마나 +40
├─ 스킬: "강화의 축복"
│  ├─ 효과: 현재 턴 모든 스탯 +30% 증가
│  ├─ 소비: 마나 25
│  └─ 재사용: 1턴 대기
└─ 옵션 선택 (1개):
   ├─ "무기 극대화": 무기 스킬 데미지 +20% ⭐
   ├─ "방어 극대화": 방어구 피해 감소 +15%
   ├─ "마나 축적": 최대 마나 +50, 회복율 +10%
   └─ "민첩함": 회피율 +12%, 상태이상 저항 +20%

【지혜의 반지】(Epic, LV 8)
├─ 기본 스탯: ATK +35, DEF +30, 최대 마나 +70
├─ 스킬: "지혜의 강화"
│  ├─ 효과: 무기 스킬 재사용 대기 -1턴
│  ├─ 소비: 마나 30
│  └─ 지속: 전투 중 계속 유지
└─ 옵션 선택 (1개):
   ├─ "스킬 회전": 무기/방어구 스킬 재사용 -1턴 ⭐⭐
   ├─ "탱크": 최대 HP +100, 자동 회복 +3/턴
   ├─ "공격": 캐릭터 공격력 +15%, 회심율 +10%
   └─ "마나 폭발": 마나 소비 스킬 효율 +25%
```

### 4️⃣ 목걸이 (Necklace) — 카드 플레이 강화 ⭐ KEY EQUIPMENT

**역할:** 4가지 카드 종류(ATTACK/SKILL/POWER/CURSE) 능력 강화 및 카드 메커니즘 개선

**슬롯:** 2개  
**주요 스탯:**
- ATTACK/SKILL/POWER/CURSE 카드 능력 증가
- 패링 효율 증가
- 회피율 증가
- 꿈 수집 보너스

**설명:**  
목걸이는 **카드 플레이를 강화하는 장비**입니다. 게임의 핵심인 카드 선택과 카드 활용을 더 효과적으로 만듭니다.

**예시:**
```
【공격의 목걸이】(Rare, LV 5)
├─ 기본 스탯: ATTACK 카드 데미지 +25%, 꿈 수집 +15%
├─ 스킬: "공격 강화"
│  ├─ 효과: 다음 ATTACK 카드 데미지 +100%
│  ├─ 소비: 마나 20
│  └─ 재사용: 1턴 대기
└─ 옵션 선택 (1개):
   ├─ "ATTACK 특화": ATTACK 카드 +30% ⭐
   ├─ "SKILL 가속": SKILL 카드 재사용 대기 -1
   ├─ "POWER 강화": POWER 카드 출현 확률 +25%
   └─ "CURSE 약화": CURSE 카드 피해 -40%

【수비의 목걸이】(Epic, LV 8)
├─ 기본 스탯: SKILL 카드 회피율 +35%, 패링 효율 +25%
├─ 스킬: "방어 무술"
│  ├─ 효과: 다음 공격 패링 성공 → 반격 데미지 +150%
│  ├─ 소비: 마나 18
│  └─ 재사용: 2턴 대기
└─ 옵션 선택 (1개):
   ├─ "SKILL 특화": SKILL 카드 회피율 +40% ⭐⭐
   ├─ "POWER 견디기": POWER 카드 피해 -50% (필수 맞음)
   ├─ "CURSE 저항": CURSE 카드 상태이상 저항 +60%
   └─ "패링 극대": 모든 카드 패링 확률 +20%

【꿈의 목걸이】(Legendary, LV 10)
├─ 기본 스탯: 모든 카드 능력 +50%, 꿈 수집 +60%
├─ 스킬: "꿈 동기화"
│  ├─ 효과: 모든 카드 다음 턴 능력 +50%
│  ├─ 소비: 마나 30
│  └─ 재사용: 2턴 대기
└─ 옵션 선택 (1개):
   ├─ "전체 강화": ATTACK/SKILL/POWER 카드 모두 +20% ⭐⭐⭐
   ├─ "CURSE 전환": CURSE 카드를 긍정 효과로 변환
   ├─ "조합 보너스": 특정 카드 조합 시 추가 +30%
   └─ "카드 드로우": 전투 중 추가 카드 선택 기회 +1
```

## 기본 능력치 시스템

### 제공되는 스탯 종류

| 스탯 | 약자 | 설명 | 적용 장비 |
|------|------|------|---------|
| 공격력 | ATK | 기본 공격 데미지 증가 | 무기, 반지 |
| 방어력 | DEF | 받는 피해 감소 | 방어구, 반지 |
| 최대 체력 | HP | 최대 체력 증가 | 방어구, 반지(옵션) |
| 최대 마나 | MNA | 마나 용량 증가 | 반지, 무기(옵션) |
| 마나 회복 | MNR | 턴마다 자동 마나 회복 | 반지(옵션) |
| 회피율 | EVA | 적 공격 회피 확률 | 반지, 목걸이(옵션) |
| 회심율 | CRI | 치명타 확률 (데미지 1.5배) | 무기, 반지(옵션) |
| 패링 효율 | PAR | 패링 성공 확률 | 방어구(옵션), 목걸이 |
| 꿈 수집 | DRM | 전투 후 꿈 수집량 증가 | 목걸이 |
| 카드 능력 | CRD | 각 카드 타입 능력 증가 | 목걸이 |

### 스탯 계산식

```
최종 ATK = (캐릭터 기본 ATK) × (1 + 무기 ATK% + 반지 ATK%) + 고정값

최종 DEF = (캐릭터 기본 DEF) × (1 + 방어구 DEF% + 반지 DEF%) + 고정값

최종 마나 = (캐릭터 기본 마나) + 반지 마나 보너스 + 무기 마나(옵션)

카드 능력 = (카드 기본 능력) × (1 + 목걸이 카드% 보너스)

패링 확률 = (기본 10%) + 방어구 패링% + 목걸이 패링%
```

### 장비별 기본값 (LV 1, Common)

| 장비 타입 | ATK | DEF | HP | MNA | 특수 |
|---------|-----|-----|-----|-----|------|
| 무기 | +15 | - | - | - | 회심율 +5% |
| 방어구 | - | +15 | +50 | - | 회복 +1/턴 |
| 반지 | +8 | +8 | - | +20 | 회피율 +3% |
| 목걸이 | - | - | - | - | 카드 능력 +10% |

## 장비 스킬 (마나 기반)

### 스킬 시스템

**정의:** 각 장비가 제공하는 **고유한 전투 스킬**, 마나를 소비해 사용

**특징:**
- 마나 소비로 사용 (자동 회복 또는 카드로 충전)
- 재사용 대기 시간 존재 (턴 기반)
- 장비 레벨에 따라 효율이 증가

### 스킬 레벨 진행

```
【마법의 검】스킬: "마법 폭발"

LV 1: 공격력 120% 데미지, 마나 30 소비
LV 3: 공격력 150% 데미지, 마나 25 소비
LV 5: 공격력 200% 데미지, 마나 20 소비 ← 현재 레벨
LV 10: 공격력 300% 데미지, 마나 15 소비
```

### 마나 시스템

**마나 회복 방식:**

| 방식 | 설명 | 빈도 |
|------|------|------|
| 자동 회복 | 턴마다 고정 마나 회복 | 매 턴 (+5) |
| 카드 회복 | 특정 카드 선택 시 마나 충전 | 카드 선택 |
| 반지 강화 | 반지 옵션으로 회복률 증가 | 지속 |
| 조건 회복 | 회심 히트, 패링 성공 등 | 조건 발동 |

**마나 소비 가이드:**

```
낮은 소비 (10-15 마나)
├─ 자주 사용 가능 ⭕
├─ 효과도 적음 (공격력 100~120%)
└─ 초반/보조 역할

높은 소비 (25-40 마나)
├─ 마나 부족 → 전략적 사용
├─ 효과는 강함 (공격력 180~250%)
└─ 후반/핵심 공격
```

## 추가 옵션 시스템

### 개념

각 장비마다 **고정 스킬 1개** + **선택 옵션 2~3개 중 1개 선택** 구조

플레이어가 옵션을 선택함으로써 장비의 특성을 커스터마이징할 수 있습니다.

### 반지의 옵션 (캐릭터 능력 강화)

```
【힘의 반지】옵션 분류:

공격형 옵션
├─ "무기 극대화": 무기 스킬 데미지 +20%
├─ "회심 강화": 회심율 +10%, 회심 피해 +20%
└─ "공격 부스트": 캐릭터 공격력 +15%

방어형 옵션
├─ "방어 극대화": 방어구 피해 감소 +15%
├─ "탱크": 최대 HP +100, 자동 회복 +2/턴
└─ "방어력 부스트": 캐릭터 방어력 +15%

마나형 옵션
├─ "마나 축적": 최대 마나 +50, 회복율 +10%
├─ "마나 폭발": 마나 소비 스킬 효율 +25%
└─ "스킬 회전": 무기/방어구 스킬 재사용 -1턴 ⭐⭐

특수형 옵션
├─ "민첩함": 회피율 +12%, 상태이상 저항 +20%
├─ "균형": 모든 스탯 +8%
└─ "강화의 지속": 스킬 효과 지속 시간 +2턴
```

### 목걸이의 옵션 (카드 플레이 강화)

```
【공격의 목걸이】옵션 분류:

카드별 강화
├─ "ATTACK 특화": ATTACK 카드 데미지 +30%
├─ "SKILL 가속": SKILL 카드 재사용 대기 -1
├─ "POWER 강화": POWER 카드 출현 확률 +25%
└─ "CURSE 약화": CURSE 카드 피해 -40%

패링/회피 관련
├─ "패링 극대": 모든 카드 패링 확률 +20%
├─ "회피 극대": SKILL 카드 회피율 +40%
└─ "카운터": 패링 성공 시 반격 데미지 +30%

조합/시너지
├─ "카드 조합": 특정 카드 조합 시 보너스 +30%
├─ "꿈 극대": 모든 카드 꿈 수집 +20%
└─ "카드 드로우": 추가 카드 선택 기회 +1

균형형
├─ "전체 강화": ATTACK/SKILL/POWER 모두 +20%
├─ "CURSE 전환": CURSE 카드를 긍정 효과로 변환
└─ "카드 경험": 카드 사용 시 추가 경험치 +15%
```

## 등급 및 밸런스

### 장비 등급 5단계

| 등급 | 이름 | 색상 | 강도 | 획득 난이도 | 사용 시기 |
|------|------|------|------|----------|---------|
| 1 | Common | 회색 | 기본 | 쉬움 | 초반 (LV 1-5) |
| 2 | Uncommon | 녹색 | 낮음 | 보통 | 초반 (LV 5-10) |
| 3 | Rare | 파란색 | 중간 | 어려움 | 중반 (LV 10-25) |
| 4 | Epic | 보라색 | 높음 | 매우 어려움 | 후반 (LV 25+) |
| 5 | Legendary | 노란색 | 매우 높음 | 극도로 어려움 | 최고급 (LV 30+) |

### 등급별 능력치 비교 (LV 1 기준)

**무기 (Weapon)**

### 캐릭터 빌드 전략

```
🔴 공격형 빌드 (DPS 극대화)
├─ 무기: "폭열의 창" (높은 ATK + 회심율)
├─ 반지: "무기 극대화" 옵션 (무기 스킬 +20%)
├─ 목걸이: "ATTACK 특화" 옵션 (ATTACK 카드 +30%)
└─ 플레이: 높은 데미지 / 취약: 방어 약함

🔵 방어형 빌드 (생존 극대화)
├─ 방어구: "용의 갑옷" (높은 DEF + HP)
├─ 반지: "방어 극대화" 옵션 (피해 감소 +15%)
├─ 목걸이: "SKILL 회피" 옵션 (SKILL 카드 회피 +40%)
└─ 플레이: 거의 안 죽음 / 취약: 공격 약함

🟢 균형형 빌드 (유연한 플레이)
├─ 무기 + 방어구: 중간값 조합
├─ 반지: "균형" 옵션 (모든 스탯 +8%)
├─ 목걸이: "전체 강화" 옵션 (모든 카드 +20%)
└─ 플레이: 안정적 / 취약: 극단성 없음

🟡 마나형 빌드 (스킬 회전)
├─ 반지 2개: "마나 축적" + "스킬 회전" 옵션
├─ 목걸이: "SKILL 가속" 옵션
├─ 플레이: 스킬 매우 자주 사용 / 취약: 기본 공격 약함
└─ 추천: 초반 보조 역할 또는 특수 전략
```

**세부 요소:**

1. **상단 자원바**
   - 아이콘: 설정(⚙️), 다이아(💎), 골드(🪙), 마나(⚡)
   - 색상: 빨강 배경
   - 높이: 40px

2. **캐릭터 + 착용 장비 영역**
   - 중앙: 캐릭터 스프라이트 (큼, 140×180px)
   - 좌측: 무기, 반지1 (20×20 슬롯 × 2)
   - 우측: 방어구, 반지2 (20×20 슬롯 × 2)
   - 하단: 목걸이1, 목걸이2 (20×20 슬롯 × 2)

3. **스탯 표시**
   - 4개 스탯: HP, ATK, DEF, 기타(예: 마나/회피율)
   - 아이콘 + 숫자 형식
   - 폰트: 작음 (12px)

4. **보유 장비 그리드**
   - 구조: 5열 그리드
   - 각 셀: 티어 색상 배경 + 레벨 배지(좌상단) + 아이콘(중앙) + 착용 표시(우상단)
   - 스크롤: 최대 4행 보이고, 추가 장비는 스크롤로 보기

5. **하단 네비게이션**
   - 5개 탭: 홈, 카드, 업그레이드, **캐릭터(강조)**, 상점
   - 현재 탭 색상 구분 (노란색)

## 게임플레이 흐름

### 전투 중 장비 활용

```
T1 (전투 시작)
├─ 캐릭터 스탯 적용
│  └─ ATK 100 + 무기 40 + 반지 20 = 160
├─ 카드 선택 (목걸이 영향)
│  └─ 목걸이: ATTACK 카드 +25%
└─ 마나 충전: 초기 30/60

T2 (카드 플레이)
├─ ATTACK 카드 선택
│  ├─ 기본 데미지: 100
│  ├─ 목걸이 효과: +25% → 125
│  └─ 무기 스킬 사용 가능 ✓
├─ 무기 스킬 "폭열 돌격" 발동
│  ├─ 마나 20 소비 (30 → 10)
│  ├─ 데미지: 160(ATK) × 200% = 320
│  ├─ 반지 옵션 "무기 극대화" +20%
│  └─ 최종: 384 데미지
└─ 마나 자동 회복: +5 (10 → 15)

T3 (방어/회피)
├─ 방어구 스킬 "용의 방어" 발동 가능 ✓
│  ├─ 다음 공격 패링 → 반격
│  └─ 마나 18 소비
└─ 적 공격 (패링 시도)
   ├─ 목걸이: SKILL 카드 회피율 +40%
   └─ 성공 → 반격 데미지 증가

T4 (계속)
├─ 마나 회복: +5 (현재 마나 충전 중)
└─ 다음 카드 선택 (SKILL 또는 POWER)

전투 후
├─ 보상 계산
├─ 목걸이 보너스: 꿈 수집 +25% (기본 50 → 62.5)
└─ 장비 강화 경험치 획득
```

## 밸런스 기본 원칙

### 1. 역할의 명확한 구분

| 장비 | 역할 | 영향 범위 |
|------|------|---------|
| 무기 | 공격 | 공격 데미지, 회심율 |
| 방어구 | 방어 | 방어력, HP, 피해 감소 |
| 반지 | 캐릭터 강화 | 무기/방어구 효과 극대화 |
| 목걸이 | 카드 강화 | 4가지 카드 능력 향상 |

### 2. 등급별 진행도

```
Common (LV 1-3) ← 초반 주로 사용
   ↓
Uncommon (LV 3-8) ← 중초반 전환점
   ↓
Rare (LV 8-15) ← 중반 안정기
   ↓
Epic (LV 15-25) ← 후반 도전
   ↓
Legendary (LV 25+) ← 최고급 목표
```

### 3. 슬롯 구성의 전략성

```
의무 슬롯 (반드시 장착)
├─ 무기 1개 (공격 필수)
└─ 방어구 1개 (방어 필수)

선택 슬롯 (전략에 따라 선택)
├─ 반지 2개
│  ├─ 공격형: 무기 강화 옵션
│  ├─ 방어형: 방어구 강화 옵션
│  └─ 마나형: 마나/스킬 옵션
└─ 목걸이 2개
   ├─ 카드 특화형: 특정 카드 강화
   ├─ 패링형: 방어 관련 강화
   └─ 균형형: 모든 카드 강화
```
