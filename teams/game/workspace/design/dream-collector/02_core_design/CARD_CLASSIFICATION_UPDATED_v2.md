# v2.0 Final (2026-03 업데이트)

**상태**: ✅ v2.0 확정 (CARD_TYPE_SYSTEM_v2.md 기준 정렬 완료)

> ⚠️ v1 문서에서 변경된 주요 사항:
> - SKILL: 15장 → **18장** (DEF 8 + PAR 5 + DOD 5)

> - CURSE: 1장 → **6장** (신규 CUR_001~006)
> - M1 베이스라인: 30장 → **42장**

```
ATTACK:  "나는 적을 얼마나 빠르게 처리하는가"
SKILL:   "나는 얼마나 안전하게 살아남는가"
POWER:   "나는 전투 전체 방향을 바꾸는가"
CURSE:   "나는 적의 전략을 어떻게 무력화하는가"
```

### 시너지 설계 원칙

```
동일 타입 시너지:
├─ ATTACK × 3 → 콤보 보너스 (연속 공격)
├─ SKILL(GUARD) × 2 → 완벽한 방어 콤보
├─ POWER 누적 스택 → 전투 판도 완전 변경
└─ CURSE × 2 → 적 약화 심화 (취약 + 약화 = 데미지 극대화)

교차 타입 시너지:
├─ ATTACK + POWER(힘) → 데미지 2배 이상
├─ SKILL(DODGE) + POWER(꿈의 흐름) → 회피마다 드로우
├─ CURSE(취약) + ATTACK(고데미지) → 최대 1.5× 데미지
└─ PARRY + POWER(심판) → 패링 반격 2배
```

### 진입 장벽 (난이도 계층)

```
초급 (비용 0~1):
  ├─ 0비용 패링/회피 카드 (익숙해지면 강력)
  ├─ 1비용 공격 기본형 (검의 에이스, 번개)
  └─ 1비용 방어 기본형 (철벽, 정의, 절제)

중급 (비용 2):
  ├─ 2비용 중간 카드로 시너지 추구
  ├─ POWER 카드 활용 시작
  └─ CURSE 카드로 적 전략 교란

고급 (비용 3~4):
  ├─ 고비용 카드 + POWER 조합
  ├─ 복잡한 CURSE 리스크/리워드 관리
  └─ 메타 게임 (덱 구성 최적화)
```
