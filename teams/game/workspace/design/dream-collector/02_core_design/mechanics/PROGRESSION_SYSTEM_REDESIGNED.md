**작성:** Atlas (PM) → Game팀  
**기준:** 방치형 게임 특징 + 무한 인플레이션 + 지속적 보상  
**버전:** v1.0  
**상태:** 📋 최종 설계

## 📌 설계 철학

### 이전 설계 (문제)
```
Lv 1-100 (끝남)
특성 포인트 수동 분배 (복잡)
레벨 상한선 (진행도 정체)
```

```
Lv 1-1000+ (무한 진행)

방치형 게임 특성 극대화
```

## 1️⃣ 레벨 시스템 (무한 성장)

### 기본 구조

```
레벨 범위: Lv 1 ~ Lv 1000+

경험치 공식:
  필요 경험치 = 100 × (현재 레벨^1.2)
  
Lv 1: 100 EXP (가장 빠름)
Lv 50: 약 19,000 EXP
Lv 100: 약 100,000 EXP
Lv 500: 약 8,000,000 EXP (천만)
Lv 1000: 약 100,000,000 EXP (억)

특징:
  - 레벨이 높을수록 경험치 많이 필요
  - 하지만 계속 진행 가능 (상한선 없음)
  - 방치형 게임답게 무한정 성장
```

### 실제 진행 속도

```
초기 (Lv 1-50):
  빠름 (1-2주)
  일일 던전으로도 충분

중기 (Lv 50-100):
  보통 (2-4주)
  꾸준한 플레이 필요

후기 (Lv 100-500):
  느림 (수개월)
  자동화 필요 (방치형)

극후기 (Lv 500-1000+):
  매우 느림 (1년 이상)
  진정한 "끝없는 성장"
```

## 2️⃣ 자동 스탯 배분

### 레벨업 시 자동으로 증가

```
매 레벨업마다:
  ATK: +1.5
  DEF: +1
  HP: +5
  SPD: +0.3
  
예시:

Lv 1 캐릭터:
  ATK: 15, DEF: 10, HP: 100, SPD: 3.0

Lv 2 캐릭터:
  ATK: 16.5, DEF: 11, HP: 105, SPD: 3.3

Lv 50 캐릭터:
  ATK: 15 + (1.5 × 49) = 88.5
  DEF: 10 + (1 × 49) = 59
  HP: 100 + (5 × 49) = 345
  SPD: 3.0 + (0.3 × 49) = 17.7

Lv 100 캐릭터:
  ATK: 15 + (1.5 × 99) = 163.5
  DEF: 10 + (1 × 99) = 109
  HP: 100 + (5 × 99) = 595
  SPD: 3.0 + (0.3 × 99) = 32.7

Lv 500 캐릭터:
  ATK: 15 + (1.5 × 499) = 763.5 (!)
  DEF: 10 + (1 × 499) = 509
  HP: 100 + (5 × 499) = 2,595
  SPD: 3.0 + (0.3 × 499) = 152.7
```

### 플레이어 선택 없음

```
장점:
  ✅ 간단함 (자동)
  ✅ 귀찮은 관리 없음
  ✅ 모든 플레이어 동등 (불공정 없음)
  ✅ 방치형 게임에 최적

게임 경험:
  "레벨이 올라간다 = 자동으로 강해진다"
  (매우 직관적)
```

## 3️⃣ 레벨별 보상 시스템

### A. 매 레벨업 기본 보상

```
모든 레벨 (Lv 1~1000+)에서 레벨업 시:

1. 골드: 레벨 × 100 G
   Lv 1: 100 G
   Lv 50: 5,000 G
   Lv 100: 10,000 G
   Lv 500: 50,000 G
   
2. 다이아: 레벨 × 1 D (10레벨마다 1D)
   Lv 10: 1 D
   Lv 20: 2 D
   Lv 100: 10 D
   Lv 500: 50 D
   Lv 1000: 100 D
   
3. 카드 드롭: 확률 30%
   레어도: 레벨에 따라 증가
   Lv 1-10: COMMON만
   Lv 20+: RARE 포함
   Lv 50+: SPECIAL 가능
   Lv 100+: LEGENDARY 가능

4. 경험치 보너스: 다음 레벨 경험치 -10%
   (레벨링이 점점 빨라지는 착각 주기)
```

## 4️⃣ 컨텐츠 오픈 시스템

### 지역/던전 오픈

```
Lv 1-5:   초급 스토리 (1지역)
Lv 10-20: 중급 스토리 (2지역)
Lv 30-50: 고급 스토리 (3지역)
Lv 50-100: 전설의 지역 (4지역)

Lv 100+: 무한 던전 오픈
  - 무한 반복 가능
  - 레벨이 높을수록 더 많은 보상
  - 최상의 아이템만 드롭

특징:
  플레이어가 높은 레벨에 도달하면
  → 새로운 장소 자동 오픈
  → 도전할 새로운 것 생김
  → 계속 플레이 동기 부여
```

### 특별 이벤트/챌린지 오픈

```
Lv 25: 첫 보스 챌린지 오픈
  난이도: 매우 쉬움
  보상: 카드 3장

Lv 50: 월간 보스 오픈
  매월 새로운 보스 등장
  난이도: 중간
  보상: 장비 1개 + 500 G

Lv 100: 시즌 이벤트 오픈
  3개월마다 특별 이벤트
  보상: 특별 칭호 + 아이템

Lv 200+: 무한 챌린지
  클리어할 수 없는 난이도
  클리어 기록만 경쟁
  보상: 매일 갱신

특징:
  높은 레벨 = 더 많은 할 것들
  = 끝없는 목표 제시
```

## 💰 경제 시스템과의 연계

### 골드 흐름

```
Lv 1-50:
  수입: 월 150,000 G
  지출: 강화 50,000 G
  수지: +100,000 G (풍족)

Lv 51-100:
  수입: 월 500,000 G
  지출: 강화 200,000 G
  수지: +300,000 G (매우 풍족)

Lv 101-500:
  수입: 월 평균 500,000 G
  지출: 강화 300,000 G
  수지: +200,000 G (안정적)

결론:
  높은 레벨일수록 자원이 풍족함
  = 플레이어가 선택할 자유 많음
```

### 다이아 흐름

```
Lv 1-50:
  무료 획득: 약 25 D
  → 가챭 1회

Lv 51-100:
  무료 획득: 약 75 D
  → 가챭 3회

Lv 101-500:
  무료 획득: 약 2,500 D
  → 가챭 50회

결론:
  높은 레벨일수록 무료 가챭 많음
  = 무료 플레이도 강해짐
```

## 📈 인플레이션 관리

### 의도된 인플레이션

```
Lv 1: ATK 15, 데미지 15
Lv 100: ATK 163.5, 데미지 약 200+
Lv 500: ATK 763.5, 데미지 약 1000+

이 인플레이션은 의도됨:
  ✅ 방치형 게임의 특징
  ✅ 플레이어에게 성장의 쾌감
  ✅ 높은 레벨 = 보상
```

### 적절한 몬스터 난이도

```
Lv 1-50 몬스터: HP 100-500
Lv 51-100 몬스터: HP 1000-5000
Lv 101-500 몬스터: HP 10000-100000+

특징:
  플레이어 ATK 증가 = 몬스터 HP도 증가
  항상 2-3턴 정도의 적절한 전투 난이도 유지
  인플레이션이 게임 밸런스 깨지 않음
```
