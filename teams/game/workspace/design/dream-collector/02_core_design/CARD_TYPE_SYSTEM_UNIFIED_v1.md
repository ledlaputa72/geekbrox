## 2️⃣ SKILL (방어) — 18장

### **메인 태그: GRD / PAR / DOD**

SKILL 타입의 모든 카드는 아래 3가지 태그 중 하나를 반드시 보유합니다:

| 태그 | 의미 | 카드 수 | 리액션 윈도우 | 오토 모드 |
|------|------|--------|------------|---------|
| **GUARD** | 즉시 블록 | 8 | 없음 (항상 발동) | 가능 (50%) |
| **PARRY** | 패링 리액션 | 5 | 빨강(0.4초) | 불가능 |
| **DODGE** | 회피 리액션 | 5 | 노랑+빨강(1.4초) | 가능 (45~60%) |

### **추가 태그**

| 태그 | 의미 | 카드 |
|------|------|------|
| **MAJOR_ARCANA** | 타로 에너지 공급 | DEF_001, DEF_003, DEF_004, DEF_005, DEF_006, DEF_007, DEF_008 (8장) |

> **주의**: PARRY, DODGE는 MAJOR_ARCANA 태그가 없음!

### **서브 카테고리별 특성**

#### **SKL-GRD (GUARD 즉시 블록) — 8장**
- **태그**: GUARD, MAJOR_ARCANA (87.5%)
- **리액션**: 없음 (항상 손에서 발동 가능)
- **오토 모드**: 50% 확률로 자동 발동
- **특징**: 가장 안정적, 초보자 친화적
- **예시**: 철벽(DEF_002), 방패의 왕(DEF_001), 여황제(DEF_003)

#### **SKL-PAR (PARRY 패링 윈도우) — 5장**
- **태그**: PARRY (100%)
- **리액션**: 빨강 구간 (0.4초/Story, 더 짧음)
- **오토 모드**: 비활성(✕) — 수동 플레이 전용
- **특징**: 고난이도, 높은 보상, 플레이어 숙련도 중요
- **보상**: 에너지 +1~+3
- **예시**: 꿈의 쳐내기(PAR_001), 각성의 쳐내기(PAR_003), 달빛 반격(PAR_004)

#### **SKL-DOD (DODGE 회피 윈도우) — 5장**
- **태그**: DODGE (100%)
- **리액션**: 노랑+빨강 구간 (1.4초/Story, 더 김)
- **오토 모드**: 가능 (45~60% 성공률)
- **특징**: 중간 난이도, 균형잡힌 보상, 오토 모드 가능
- **보상**: 에너지 +1
- **예시**: 꿈의 스텝(DOD_001), 잔상(DOD_002), 황혼의 도약(DOD_003)

## 🔑 **태그 활용 규칙**

### **필수 태그**
- **ATTACK**: 타입이 ATTACK인 모든 카드
- **SKILL + (GRD|PAR|DOD)**: 타입이 SKILL인 모든 카드는 GRD/PAR/DOD 중 정확히 하나
- **POWER**: 타입이 POWER인 모든 카드
- **CURSE**: 타입이 CURSE인 모든 카드

### **선택 태그**
- **MAJOR_ARCANA**: 타로 에너지 공급원 (ATTACK, SKILL-GRD, POWER에 주로 붙음)
- **AOE**: 광역 공격 (ATTACK만)
- **PERSIST**: 지속 버프 (POWER에만 붙음, 향후)

## 🎯 **코드 구현 가이드**

### **Card 클래스 구조 (GDScript)**

```gdscript
class_name Card

# 기본 속성
var type: String  # "ATTACK", "SKILL", "POWER", "CURSE"
var tags: Array[String]  # ["MAJOR_ARCANA", "AOE", "GUARD" 등]

# SKILL 전용
var sub_category: String  # "GUARD", "PARRY", "DODGE"

# 검색 예시
func has_tag(tag_name: String) -> bool:
    return tag_name in tags

func is_major_arcana() -> bool:
    return has_tag("MAJOR_ARCANA")

func is_aoe() -> bool:
    return has_tag("AOE")

func is_guard() -> bool:
    return type == "SKILL" and has_tag("GUARD")
```

### **카드 필터링 쿼리**

```gdscript
# 타로 에너지 공급 가능한 카드
var tarot_cards = cards.filter(func(c): return c.is_major_arcana())

# GUARD 카드만
var guard_cards = cards.filter(func(c): return c.is_guard())

# 광역 공격
var aoe_cards = cards.filter(func(c): return c.type == "ATTACK" and c.is_aoe())
```
