## 6. 카드 창작 프레임워크 (Card Creation Matrix)

향후 신규 카드 제작 시 참조할 설계 기준.

### 6.1 타입별 역할 정의

```
공격 (ATTACK)  → "얼마나 빠르게 적을 처리하는가"
스킬 (SKILL)   → "얼마나 안전하게 살아남는가"
파워 (POWER)   → "전투 전체를 어떤 방향으로 유도하는가"
커스 (CURSE)   → "적의 전략을 어떻게 무력화하는가"
```

### 6.2 카드 설계 공식

| 요소 | 선택지 |
|------|--------|
| **비용** | 0 (반응형) / 1 (효율형) / 2 (밸런스형) / 3~4 (파워형) |
| **주 효과** | 데미지 / 블록 / 드로우 / 에너지 / 리액션 |
| **부 효과** | 상태이상 / 버프 / 드로우 / 에너지 회복 |
| **리스크** | 없음 / 자해 / 다음 턴 패널티 / 조건부 |
| **타이밍** | 즉시 사용 / 리액션 전용 / 조건 발동 |
| **희귀도** | COMMON(범용) / RARE(조합) / SPECIAL(특수메카닉) / LEGENDARY(게임체인저) |

### 6.3 비용 대비 기대값 기준표

| 비용 | 공격(데미지) | 스킬(블록) | 기타 부가효과 |
|------|------------|---------|------------|
| 0 | 해당없음 | 해당없음 (리액션만) | 에너지 +1~+2 |
| 1 | 6~8 | 5~7 | 드로우 0~1 |
| 2 | 9~15 | 8~12 | 드로우 1 or 상태이상 1 |
| 3 | 12~18 | 15~18 | 드로우 0~1 or 영구 버프 |
| 4 | 18~20+ | 없음 | 특수 효과 |

### 6.4 빈 구간 분석 (Gap Analysis)

| 빈 구간 | 현재 상태 | 권고 |
|--------|----------|------|
| 비용 0 공격 카드 | 없음 | 필요없음 (리액션 전용 영역) |
| 광역 스킬 카드 | 없음 | AOE 방어 카드 1~2장 권고 |
| 조건부 파워 카드 | 없음 | "X장 이하 HP시 발동" 등 권고 |
| 커스+PARRY 조합 | 없음 | 패링 성공 시 저주 발동형 권고 |
| 드로우 특화 공격 | ATK_009 1장 | 추가 필요 |
