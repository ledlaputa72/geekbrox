## 🎯 핵심 콘셉트 (빠른 요약)

### 1. 이중 시스템, 하나의 게임

```
같은 30장 카드로 두 가지 게임:

🔴 ATB (실시간)
   - 액션 게이머용
   - 빠른 반응 중요
   - 스트레스 높음
   - 속도감 있는 플레이

🔵 턴베이스 (전략)
   - 전략 게이머용
   - 계획 중요
   - 스트레스 낮음
   - 깊이 있는 게임
```

### 2. 에너지 = 자원 관리

```
방어 우수 플레이어:
  방어 성공 → 에너지 보너스 → 더 공격 가능
  = 반사신경 높은 플레이어가 실제로 강함!

결과: 방어를 잘할수록 더 강해지는 긍정적 피드백
```

### 3. 30장 카드 (엄선됨)

```
공격 10 + 방어 8 + 패링 5 + 회피 5 + 스킬 2

특징:
- PARRY/DODGE는 모두 0비용 (리소스 부담 없음)
- 비용 범위 0~4 (게임 초반부터 사용 가능)
- 각 카드는 명확한 역할 정의
```

### 4. 리액션 시스템

```
4가지 반응 타입:
1. PARRY (패링): 공격 무효화 + 에너지 +2
2. DODGE (회피): 공격 무효화 + 에너지 +1
3. GUARD (방어): 블록으로 부분 상쇄 + 에너지 +0.5
4. NONE (무반응): 풀 피해

특수: UNBLOCKABLE 🔱
      패링 불가능! 회피만 가능!
```

### 5. 콤보 시스템

```
4가지 콤보:
1. 연타 (ATK 3장): +75% 데미지
2. 패링 반격 (PARRY→ATK): +30% 데미지
3. 약점 폭로 (VULNERABLE→ATK): +50%
4. 완벽한 방어 (DEF 2장): 누적만

의도: 플레이어가 카드를 계획적으로 배열하게 함
```
