## 🎯 **밸런스 체크리스트**

### **신규 카드 설계 시 검증**

#### **1. 비용 밸런스**

```
□ 비용이 설정된 범위 내인가?
□ 비용 분포가 기존 카드와 균형인가?
□ 비용당 효과가 공정한가?
  - ATTACK: 비용×5~6 데미지
  - SKILL: 비용×5~7 블록
  - POWER: 비용 0~2, 즉시/지속 효과
  - CURSE: 비용 1~3, 약화 효과
```

#### **2. 효과 밸런스**

```
□ 2차 효과가 있는가?
□ 다른 카드와 시너지가 있는가?
□ 오버파워하지 않는가?
□ 게임플레이에 영향을 주는가?
```

#### **3. 희귀도 설정**

```
□ COMMON (50% 수준): 기본 효과
□ RARE (30% 수준): 강화 효과 또는 2차 효과
□ SPECIAL (13% 수준): 특수 메커닉 또는 극한 난이도
□ LEGENDARY (3% 수준): 게임 변화 카드
```

#### **4. 메인/서브 배치**

```
□ 적절한 메인 카테고리인가?
□ 적절한 서브 카테고리인가?
□ 기능 ID가 명확한가?
□ 기존 카드와 구분되는가?
```

#### **5. 태그 적용**

```
□ MAJOR_ARCANA 필요한가?
□ 기타 태그(AOE, GUARD, PARRY 등)는?
□ 태그 분포가 균형인가?
```

## 🔧 **데이터 생성 템플릿**

### **신규 카드 데이터 양식**

```gdscript
var card = Card.new()
card.id = "ATK_011"              # 카드 ID
card.name = "신규 공격"            # 카드명
card.cost = 2                     # 비용
card.type = "ATTACK"              # 메인 카테고리
card.damage = 12                  # 데미지
card.block = 0
card.draw = 0
card.tags = ["MAJOR_ARCANA"]      # 태그
card.description = "..."          # 설명
card.short_desc = "⚔️12"          # 짧은 설명
card.rarity = "COMMON"            # 희귀도
card.sub_category = "SGL"         # 서브 카테고리

# 2차 효과 (있으면)
card.status_effects.append({
    "target": "self",
    "type": "STRENGTH",
    "value": 1
})
```
