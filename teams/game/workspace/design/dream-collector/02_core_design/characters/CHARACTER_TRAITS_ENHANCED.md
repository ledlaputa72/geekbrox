## 🎯 캐릭터 특성 계층 (3단계)

### 1단계: 기본 정체성 특성 (고정)
```
예: Nox - 기억/꿈 속성
   - 각 캐릭터마다 고유
   - 게임 내 변경 불가
   - 스토리/설정과 연결
```

### 2단계: 전투 스타일 특성 (게임 진행 중 변경 가능)
```
예: 회피형 / 탱크형 / 공격형
   - 5가지 중 1가지 선택
   - 언제든지 변경 가능
   - 기본 스탯에 영향
```

### 3단계: 장비 기반 특성 (장비 장착으로 자동 활성화) ✨ NEW
```
예: 무기 마스터리 / 방어 강화 / 악세서리 동조 / 듀얼 웰드 / 장비 연쇄
   - 장비 장착 시 자동 활성화
   - 장비 해제 시 자동 비활성화
   - 특성 레벨 = 장비 강화 수준에 따라 증가
```

## 🛡️ 장비 기반 특성 시스템 (v2.0 신규)

### 특성 1: 무기 마스터리 (Weapon Mastery)

**활성 조건:** 무기 장착 (모든 무기)  
**특성 레벨:** 무기 강화 수준 (+0 ~ +10)

**효과:**
```
기본 (무기 장착): 무기 공격력 +5%

레벨 1 (+0 강화):  무기 ATK +5%, 치명타 +1%
레벨 2 (+2 강화):  무기 ATK +10%, 치명타 +2%
레벨 3 (+4 강화):  무기 ATK +15%, 치명타 +3%, 공격 속도 +5%
레벨 4 (+6 강화):  무기 ATK +20%, 치명타 +5%, 공격 속도 +10%
레벨 5 (+8 강화):  무기 ATK +25%, 치명타 +7%, 공격 속도 +15%
레벨 6 (+10 강화): 무기 ATK +30%, 치명타 +10%, 공격 속도 +20%, 추가 공격 15%

추가 효과 (특정 무기 타입):
- 검: 기본 공격 시 5% 확률로 2회 연속 공격
- 도끼: 기본 공격 시 방어력 무시 10%
- 지팡이: 기본 공격이 마법 데미지로 변환, 방어력 -30% 적용
```

**특성 보너스 (추가 선택사항):**
```
무기 전문화 (특정 무기 타입에만 +10% 추가 보너스):
- 검 전문가 (Swordmaster): 검 사용 시 +10% 모든 데미지
- 도끼 전문가 (Axemaster): 도끼 사용 시 +15% 치명타, 회피 -5% (무거움)
- 마법 전문가 (Mage): 지팡이 사용 시 +20% 공격 속도, ATK -10%
```

### 특성 2: 방어 강화 (Defense Mastery)

**활성 조건:** 방어구 장착 (갑옷/로브/경갑)  
**특성 레벨:** 방어구 강화 수준 (+0 ~ +10)

**효과:**
```
기본 (방어구 장착): 방어구 DEF +5%, HP +5%

레벨 1 (+0 강화):  방어구 DEF +5%, HP +5%, 상태이상 저항 +2%
레벨 2 (+2 강화):  방어구 DEF +10%, HP +10%, 상태이상 저항 +4%
레벨 3 (+4 강화):  방어구 DEF +15%, HP +15%, 상태이상 저항 +6%, 회복 +2% (지속)
레벨 4 (+6 강화):  방어구 DEF +20%, HP +20%, 상태이상 저항 +8%, 회복 +3%
레벨 5 (+8 강화):  방어구 DEF +25%, HP +25%, 상태이상 저항 +10%, 회복 +4%
레벨 6 (+10 강화): 방어구 DEF +30%, HP +30%, 상태이상 저항 +15%, 회복 +5%, 피해 감소 10%

추가 효과 (특정 방어구 타입):
- 갑옷: 피해를 받을 때마다 DEF +1% (최대 20% 중첩 가능, 1턴 유지)
- 로브: 마법 저항력 +20%, 받은 마법 데미지 -15%
- 경갑: 회피율 +15%, 받은 피해 20% 감소 (대신 방어력 -10%)
```

**특성 보너스:**
```
방어 전문화:
- 탱크 마스터 (Tankmancer): 갑옷 사용 시 피해 흡수 25%, 회복 -10%
- 신비 수호자 (Mystic Guardian): 로브 사용 시 마법 저항 +25%, 물리 저항 -5%
- 민첩 수호자 (Agile Guardian): 경갑 사용 시 회피율 +25%, DEF -10%
```

### 특성 3: 악세서리 동조 (Accessory Resonance) ✨ NEW

**활성 조건:** 악세서리 장착 (반지/팔찌/장신구)  
**특성 레벨:** 장착한 악세서리 개수 + 강화 수준

**효과:**
```
악세서리 1개 장착:
  - 효과 100% (기본값)
  - 특성 레벨 1

악세서리 2개 장착:
  - 각 악세서리 효과 120% (20% 증폭)
  - 시너지 보너스: 모든 데미지 +10%
  - 특성 레벨 2
  
악세서리 3개 장착:
  - 각 악세서리 효과 150% (50% 증폭)
  - 시너지 보너스: 모든 데미지 +20%, 모든 방어 +15%
  - 특성 레벨 3

추가 효과:
- 레벨 2: 악세서리 효과 발동 시 다음 공격력 +5%
- 레벨 3: 악세서리 효과 발동 시 다음 공격력 +10%, 다음 방어 +8%

특정 악세서리 조합:
- 3개 모두 반지: 스탯 +20% (최강의 공격)
- 3개 모두 팔찌: 방어력 +25% (최강의 방어)
- 혼합 (반지+팔찌+장신구): 전체 밸런스, +15% 모든 스탯
```

**특성 보너스:**
```
악세서리 대가:
- 반지 수집가: 반지 3개 착용 시 모든 스탯 +30% (팔찌/장신구 효과 -20%)
- 팔찌 수집가: 팔찌 3개 착용 시 방어력 +50% (공격력 -15%)
- 균형잡힌 수집가: 어떤 조합이든 +10% 모든 스탯
```

### 특성 4: 듀얼 웰드 (Dual Wield)

**활성 조건:** 보조 무기 (방패/단검/보조) 장착  
**특성 레벨:** 보조 무기 강화 수준 (+0 ~ +10)

**효과:**
```
기본 (보조 무기 장착): 보조 무기 효과 50% 활성화

레벨 1 (+0 강화):  추가 데미지 50%, 추가 공격 확률 +3%
레벨 2 (+2 강화):  추가 데미지 75%, 추가 공격 확률 +6%
레벨 3 (+4 강화):  추가 데미지 100%, 추가 공격 확률 +9%, 회피율 +5%
레벨 4 (+6 강화):  추가 데미지 125%, 추가 공격 확률 +12%, 회피율 +10%
레벨 5 (+8 강화):  추가 데미지 150%, 추가 공격 확률 +15%, 회피율 +15%
레벨 6 (+10 강화): 추가 데미지 200%, 추가 공격 확률 +20%, 회피율 +20%, 쌍무기 연계 (2회 연속 공격)

보조 무기별 특성:
- 방패 장착: 기본 공격 + 방어, 회피 +10%, 방어력 증가
- 단검 장착: 기본 공격 + 추가 타격, 치명타 +15%
- 보조 주술서 장착: 기본 공격 + 상태이상, 상태이상 확률 +20%
```

**특성 보너스:**
```
쌍무기 마스터:
- 검+방패: ATK +15%, DEF +20% (균형)
- 검+단검: ATK +30%, DEF -10% (공격)
- 지팡이+책: ATK +20%, 마법 저항 +15%
```

### 특성 5: 장비 연쇄 (Equipment Chain) ✨ NEW

**활성 조건:** 같은 시리즈 장비 2개 이상 장착  
**특성 레벨:** 장착한 같은 시리즈 장비 개수

**프리정의 세트 효과:**

#### 검 세트
```
구성: 검 + 강철 갑옷 + 왕의 반지 + 전투 방패
효과:
  - 2개 구성: ATK +10%, 전투력 +5%
  - 3개 구성: ATK +15%, DEF +10%, 전투력 +10%
  - 4개 구성: ATK +20%, DEF +15%, 전투력 +20%, 기본 공격 2회 연속
```

#### 마법 세트
```
구성: 지팡이 + 로브 + 신발 + 마력 주술서
효과:
  - 2개 구성: 마법 데미지 +15%, 공격 속도 +10%
  - 3개 구성: 마법 데미지 +25%, 공격 속도 +20%, 마법 저항 +15%
  - 4개 구성: 마법 데미지 +35%, 공격 속도 +30%, 마법 저항 +25%, 기본 공격 시 매턴 1회 무료 추가 공격
```

#### 방어 세트
```
구성: 도끼 + 청동 갑옷 + 방어의 반지 + 신성한 방패
효과:
  - 2개 구성: DEF +15%, HP +5%
  - 3개 구성: DEF +25%, HP +10%, 피해 감소 +10%
  - 4개 구성: DEF +35%, HP +15%, 피해 감소 +20%, 매턴 최대 HP 5% 회복
```

#### 회피 세트
```
구성: 지팡이 + 경갑 + 속도의 반지 + 신발
효과:
  - 2개 구성: 회피율 +15%, 공격 속도 +15%
  - 3개 구성: 회피율 +25%, 공격 속도 +25%, 받은 데미지 10% 감소
  - 4개 구성: 회피율 +35%, 공격 속도 +35%, 받은 데미지 20% 감소, 기본 공격 시 회피율 +50% (1턴 지속)
```

#### 사용자 정의 세트
```
자유로운 조합으로 추가 세트 효과 발동:
- 같은 타입 3개 이상: 기본 스탯 +10% × 개수
- 다양한 타입 4개 이상: 모든 스탯 +5% × 개수
```

## 📊 캐릭터 특성 시너지 예시

### Nox 캐릭터 풀 빌드 (최대 강화)

**장비 구성:**
```
무기: 빛의 검 (ATK 30, +10 강화)
방어구: 신성한 갑옷 (DEF 25, HP 250, +10 강화)
악세서리: 흡혈 반지 + 왕의 반지 + 차원의 보석 (+10 강화 모두)
보조 무기: 신성한 방패 (+10 강화)
```

**활성화 특성:**
```
1. 무기 마스터리 Lv 6 (검 +10 강화)
   - 무기 ATK +30%
   - 치명타 +10%
   - 공격 속도 +20%
   - 추가 공격 15%

2. 방어 강화 Lv 6 (갑옷 +10 강화)
   - 방어구 DEF +30%
   - HP +30%
   - 상태이상 저항 +15%
   - 회복 +5% (지속)
   - 피해 감소 10%

3. 악세서리 동조 Lv 3 (3개 악세서리)
   - 각 효과 150%
   - 모든 데미지 +20%
   - 모든 방어 +15%
   - 흡혈 반지: 30% 회복 (1.5배 증폭)
   - 왕의 반지: ATK +15, DEF +4.5, 속도 +15%
   - 차원의 보석: 추가 공격 확률 +15% (1.5배)

4. 듀얼 웰드 Lv 6 (보조 방패 +10 강화)
   - 추가 데미지 200%
   - 추가 공격 확률 +20%
   - 회피율 +20%
   - 쌍무기 연계 (2회 연속)

5. 장비 연쇄 Lv 4 (검 세트 4개 완성)
   - ATK +20%
   - DEF +15%
   - 전투력 +20%
   - 기본 공격 2회 연속
```

**최종 기본 공격 계산:**
```
기본 데미지: 30 (검) + 5 (방패) = 35
× (1 + 0.30) [무기 마스터리] = 45.5
× (1 + 0.20) [악세서리 동조] = 54.6
× (1 + 0.20) [장비 연쇄] = 65.52
× (1.2 치명타 판정) = 78.62 데미지

회복: 78.62 × 0.45 (흡혈 1.5배) = 35.38 HP 회복
추가 공격 확률: 35% (15% + 20%) → 약 1회 추가 공격
추가 회피: 20%
최종 방어력: 25 × 1.30 × 1.15 = 37.4 DEF
```
