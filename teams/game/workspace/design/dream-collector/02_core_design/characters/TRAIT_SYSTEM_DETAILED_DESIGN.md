# 🎯 Dream Collector — 특성 시스템 상세 설계 (Trait Mastery System)

**작성:** Atlas (PM) → Game팀  
**기준:** 5가지 마스터리의 완전한 정의 + 시스템 연계  
**버전:** v1.0  
**상태:** 📋 최종 상세 설계

---

## 📌 특성 시스템 개요

**정의:**
```
특성 = 장비 강화도에 기반한 자동 보너스 시스템

플레이어가 "특성을 활성화한다"는 개념 없음
→ 장비 착용 = 자동으로 특성 활성화
→ 장비 강화 = 자동으로 특성 보너스 증가
```

**핵심:**
```
단순함: 이해하기 쉬움 (플레이어 입장)
명확함: 수치가 투명함 (계산 공식 명시)
연계: 다른 모든 시스템과 연결됨
```

## 1️⃣ 마스터리 #1: 무기 마스터리 (Weapon Mastery)

### 정의

```
활성 조건:
  무기를 장착했을 때 자동 활성화
  
레벨:
  무기의 강화도 = 마스터리 레벨
  (무기 +3 강화 = Weapon Mastery Lv 3)
```

### 효과 공식

**기본 공격 데미지:**

```
최종 데미지 = 무기 ATK × (1 + Weapon Mastery Bonus)

Weapon Mastery Bonus = (강화도 + 1) × 5%

Lv 0 (+0): 5% (강화도 0)
Lv 1 (+1): 10% (강화도 1)
Lv 2 (+2): 15%
Lv 3 (+3): 20%
Lv 4 (+4): 25%
Lv 5 (+5): 30%
Lv 6 (+6): 35%
Lv 7 (+7): 40%
Lv 8 (+8): 45%
Lv 9 (+9): 50%
Lv 10 (+10): 55%
```

**추가 효과: 치명타**

```
기본 치명타율: 5%
추가: (강화도 × 1%)

Lv 0: 5% 치명타
Lv 5: 10% 치명타
Lv 10: 15% 치명타

치명타 데미지: 1.5배
```

### 구체적인 예시

```
시나리오: 강철검 (ATK 20)

Case 1: +0 강화 (Lv 0)
  기본 공격: 20 데미지
  × (1 + 5%) = 21 데미지
  추가 치명타: 5%

Case 2: +5 강화 (Lv 5)
  기본 공격: 20 데미지
  × (1 + 30%) = 26 데미지
  추가 치명타: 10%

Case 3: +10 강화 (Lv 10)
  기본 공격: 20 데미지
  × (1 + 55%) = 31 데미지
  추가 치명타: 15%

결론:
  강화도 +10 = 기본 공격 1.55배 증가
  + 치명타 확률 증가
```

### 다른 시스템과의 연계

**강화 시스템:**
```
무기 강화 +1 발생
  ↓
Weapon Mastery 자동 Lv 1 증가
  ↓
기본 공격 +5% 증가
  ↓
즉시 체감 (플레이어가 강화 효과를 느낌)
```

**경제 시스템:**
```
무기 강화 비용: 500 G (첫 강화)
보상: 기본 공격 +5%

투자 대비 효과 명확 = 플레이어 만족도 높음
```

**유료화:**
```
무기를 더 빨리 강화하고 싶음
  ↓
더 많이 가챭해야 함
  ↓
다이아 구매 동기 (자연스러운 유료화)
```

## 2️⃣ 마스터리 #2: 방어 강화 (Defense Mastery)

### 정의

```
활성 조건:
  방어구(갑옷)를 장착했을 때 자동 활성화
  
레벨:
  방어구의 강화도 = 마스터리 레벨
```

### 효과 공식

**방어력:**

```
최종 방어력 = 기본 방어력 × (1 + Defense Mastery Bonus)

Defense Mastery Bonus = (강화도 + 1) × 5%

Lv 0 (~0): 5%
Lv 5: 30%
Lv 10: 55%
```

**생명력:**

```
최종 HP = 기본 HP + (기본 HP × Defense Mastery HP Bonus)

HP Bonus = (강화도 + 1) × 5%

Lv 0: HP +5%
Lv 5: HP +30%
Lv 10: HP +55%
```

### 구체적인 예시

```
시나리오: 강철 갑옷 (DEF 15, HP 100)

Case 1: +0 강화 (Lv 0)
  기본 방어: 15 → 15.75 (5% 증가)
  기본 HP: 100 → 105 (5% 증가)

Case 2: +5 강화 (Lv 5)
  기본 방어: 15 → 19.5 (30% 증가)
  기본 HP: 100 → 130 (30% 증가)

Case 3: +10 강화 (Lv 10)
  기본 방어: 15 → 23.25 (55% 증가)
  기본 HP: 100 → 155 (55% 증가)

결론:
  방어 강화는 생존력 증가
  = 전술적 가치 (공격만 아니라 방어도 중요)
```

### 다른 시스템과의 연계

**플레이 스타일:**
```
Architect (공격형):
  방어구 강화 낮음 → 방어 약함 → 빠른 클리어 필요

Healer (방어형):
  방어구 강화 높음 → 방어 강함 → 안정적 진행

Mystic (균형형):
  방어구 강화 중간 → 균형잡힌 진행
```

**전투 난이도 조정:**
```
보스 전투에서 어려움
  ↓
방어 마스터리 강화 필요
  ↓
방어구 강화 동기 생성
```

**경제:**
```
방어구 강화 비용 = 무기 강화 비용과 동일
(공평한 경제)

플레이어 선택:
  공격 중심 강화 vs 방어 중심 강화
```

## 3️⃣ 마스터리 #3: 악세서리 동조 (Accessory Resonance)

### 정의

```
활성 조건:
  악세서리(반지, 팔찌, 목걸이 등)를 장착했을 때
  
레벨:
  착용한 악세서리 개수 = 마스터리 레벨
  (최대 3개, Lv 3)
  
구조:
  1개 착용 = Lv 1
  2개 착용 = Lv 2
  3개 착용 = Lv 3
```

### 효과 공식

**악세서리 효과 증폭:**

```
각 악세서리의 효과 = 기본 효과 × (1 + Resonance Bonus)

Resonance Bonus:
  Lv 0 (0개): 0%
  Lv 1 (1개): 10%
  Lv 2 (2개): 20%
  Lv 3 (3개): 30%
```

### 구체적인 예시

```
악세서리 예시:
  - 공격 반지: ATK +3
  - 방어 팔찌: DEF +2
  - 체력 목걸이: HP +10%

Case 1: 0개 착용
  아무 효과 없음

Case 2: 공격 반지 1개 착용 (Lv 1)
  공격 반지 효과: 3 × (1 + 10%) = 3.3 ATK
  동조 보너스: 10%

Case 3: 공격 반지 + 방어 팔찌 2개 (Lv 2)
  공격 반지: 3 × (1 + 20%) = 3.6 ATK
  방어 팔찌: 2 × (1 + 20%) = 2.4 DEF
  동조 보너스: 20%

Case 4: 3개 모두 착용 (Lv 3)
  공격 반지: 3 × (1 + 30%) = 3.9 ATK
  방어 팔찌: 2 × (1 + 30%) = 2.6 DEF
  체력 목걸이: 10% × (1 + 30%) = 13% HP
  동조 보너스: 30% (최고)

결론:
  3개 착용 = 모든 악세서리 효과 1.3배
```

### 다른 시스템과의 연계

**다양성:**
```
플레이어가 악세서리 3개를 모두 모으려고 함
  ↓
가챭 동기 증가
  ↓
자연스러운 다이아 구매 (유료화)
```

**전술성:**
```
어떤 악세서리를 선택할지 = 플레이 전략
  공격 중심 vs 방어 중심 vs 회복 중심
  
3개를 모두 갖추려면:
  → 가챭이 필요
  → 골드 투자도 필요 (강화)
  → 시간 투자 필요 (수집)
```

**경제:**
```
악세서리 강화: 선택적 (필수는 아님)
재료로 사용 가능: 다른 장비 강화에 활용
→ 경제 순환 구조
```

## 4️⃣ 마스터리 #4: 듀얼 웰드 (Dual Wield)

### 정의

```
활성 조건:
  보조 무기(방패, 단검, 그리모어 등)를 장착했을 때
  
레벨:
  보조 무기의 강화도 = 마스터리 레벨
```

### 효과 공식

**추가 데미지:**

```
보조 무기로부터의 추가 데미지:
  추가 데미지 = 보조 무기 ATK × (1 + Dual Wield Bonus)

Dual Wield Bonus = (강화도 + 1) × 10%

Lv 0: 10%
Lv 5: 60%
Lv 10: 110%
```

**추가 공격 확률:**

```
기본 공격 후 추가로 보조 무기로 공격할 확률:

기본: 3%
추가: (강화도 × 2%)

Lv 0: 3%
Lv 5: 13%
Lv 10: 23%
```

### 구체적인 예시

```
시나리오:
  주 무기: 강철검 ATK 20
  보조 무기: 단검 ATK 8

Case 1: 보조 무기 +0 (Lv 0)
  추가 데미지: 8 × (1 + 10%) = 8.8
  추가 공격 확률: 3%
  
Case 2: 보조 무기 +5 (Lv 5)
  추가 데미지: 8 × (1 + 60%) = 12.8
  추가 공격 확률: 13%
  
Case 3: 보조 무기 +10 (Lv 10)
  추가 데미지: 8 × (1 + 110%) = 16.8
  추가 공격 확률: 23%

최종 공격:
  기본: 강철검 데미지 (20)
  추가: 단검 데미지 (8-16.8, 확률 3-23%)
  = 기본 공격 1.5~2배 증가 가능
```

### 다른 시스템과의 연계

**플레이 스타일:**
```
Architect (공격형):
  보조 무기 강화 높음 → 높은 DPS

Healer (방어형):
  보조 무기 방패 사용 → 방어 증가 (별도 메커니즘)

Mystic (균형형):
  보조 무기 균형적 강화
```

**전술성:**
```
기본 공격의 변수성 증가
- 23% 확률로 추가 공격 = 게임 재미 증가
- 플레이할 때마다 다른 결과 = 중독성 증가
```

**경제:**
```
4번째 강화 경로 (무기, 방어구, 악세서리, 보조 무기)
플레이어의 강화 선택:
  우선순위 결정 필요
  → 경제 깊이 증가
```

## 5️⃣ 마스터리 #5: 장비 보너스 (Equipment Bonus)

### 정의

```
활성 조건:
  강화된 장비를 착용했을 때 (각 장비 +1 이상)
  
레벨:
  착용한 강화된 장비 개수 = 마스터리 레벨
  (최대 4개: 무기, 방어구, 악세서리, 보조 무기)
  
구조:
  장비 1개가 +1 이상 → 1 레벨
  장비 2개가 +1 이상 → 2 레벨
  장비 3개가 +1 이상 → 3 레벨
  장비 4개가 +1 이상 → 4 레벨 (최대)
```

### 효과 공식

**모든 스탯 증가:**

```
각 강화된 장비마다 모든 스탯 +5%

Equipment Bonus = 강화된 장비 개수 × 5%

1개 강화: 5%
2개 강화: 10%
3개 강화: 15%
4개 강화: 20% (최대)
```

### 구체적인 예시

```
시나리오:
  무기: 강철검 ATK 20 (강화도 +3)
  방어구: 강철 갑옷 DEF 15 (강화도 +2)
  악세서리: 공격 반지 ATK +3 (강화도 +1)
  보조 무기: 단검 ATK 8 (아직 강화 안 함)

Case 1: 무기만 강화 (1개)
  보너스: 5%
  
  최종 스탯:
    무기 ATK: 20 × 1.05 = 21
    방어구 DEF: 15 × 1.05 = 15.75
    악세서리 ATK: 3 × 1.05 = 3.15

Case 2: 무기 + 방어구 강화 (2개)
  보너스: 10%
  
  최종 스탯:
    무기 ATK: 20 × 1.10 = 22
    방어구 DEF: 15 × 1.10 = 16.5
    악세서리 ATK: 3 × 1.10 = 3.3
    보조 무기 ATK: 8 × 1.10 = 8.8

Case 3: 무기 + 방어구 + 악세서리 강화 (3개)
  보너스: 15%
  
Case 4: 모두 강화 (4개)
  보너스: 20%
  
  최종 스탯:
    무기 ATK: 20 × 1.20 = 24
    방어구 DEF: 15 × 1.20 = 18
    악세서리 ATK: 3 × 1.20 = 3.6
    보조 무기 ATK: 8 × 1.20 = 9.6

결론:
  모든 장비를 강화하면 모든 스탯 20% 증가
  = 플레이어에게 명확한 목표 제시
```

### 다른 시스템과의 연계

**장기 목표:**
```
"모든 장비를 +1 이상으로 강화하자"
= 게임의 장기 목표 설정
= 플레이어 몰입도 증가
```

**경제 효율:**
```
초기: 한두 개 장비만 강화 (자연스러운 진행)
중기: 3개 장비 강화 (본격적인 강화 단계)
후기: 모든 장비 강화 (최고 성능)

단계별 진행이 명확함 = 게임 진행도 체감
```

**유료화:**
```
모든 장비를 강화하려면:
  시간 부족 (바쁜 플레이어)
  
해결:
  더 많은 가챭 (다이아 구매)
  → 자연스러운 유료화 유도
```

## 📊 5가지 마스터리 통합 데미지 계산

### 기본 공격 최종 공식

```
최종 데미지 = 무기ATK × 마스터리 배수들의 곱

마스터리 배수:
  1. (1 + Weapon Mastery Bonus)
  2. (1 + Defense Mastery 효과도 고려) ← 방어력은 별도
  3. (1 + Accessory Resonance Bonus)
  4. (1 + Dual Wield Bonus) ← 보조 무기 추가 공격
  5. (1 + Equipment Bonus)

예: 모든 마스터리가 Lv 5일 때

기본 무기 ATK: 20

× (1 + 30%) [Weapon Mastery Lv 5]
× (1 + 20%) [Accessory Resonance Lv 2]
× (1 + 60%) [Dual Wield Lv 5]
× (1 + 15%) [Equipment Bonus 3개]
───────────────
= 20 × 1.30 × 1.20 × 1.60 × 1.15
= 20 × 2.864
= 약 57 데미지

vs 기본 20 데미지
→ 2.85배 증가
```

### UI 표시 (플레이어가 보는 화면)

```
[기본 공격 데미지 분석]

무기 ATK: 20
├─ 무기 마스터리: +30% = 26
├─ 악세서리 동조: +20% = 31.2
├─ 듀얼 웰드: +60% = 49.92
└─ 장비 보너스: +15% = 57.39

최종 데미지: 57 (기본 대비 2.85배)

---

[각 마스터리 상세]
✓ 무기 마스터리 Lv 5 (강화도 +5)
✓ 방어 강화 Lv 3 (강화도 +3)
✓ 악세서리 동조 Lv 2 (2개 착용)
✓ 듀얼 웰드 Lv 5 (보조 무기 +5)
✓ 장비 보너스 Lv 3 (3개 강화)
```

## 🎯 특성 시스템 & 다른 시스템 연계

### 강화 시스템과의 연계

```
[강화 발생]
  장비 +3 강화 완료
  
  ↓
  
[특성 자동 업데이트]
  무기 마스터리: Lv 3 (자동 계산)
  → 기본 공격 +20% (자동 적용)
  
[플레이어 체감]
  "기본 공격이 강해졌다!"
  (강화의 즉각적인 피드백)
```

### 성장 시스템과의 연계

```
[레벨업]
  Lv 30 → Lv 31 (특성 포인트 +5)
  
[플레이어 선택]
  특성 포인트를 ATK에 배분
  → ATK +5 증가
  
[최종 데미지]
  특성 보너스 + 특성 포인트 = 시너지
  
  예: 무기 마스터리 +30% × 스탯 포인트 ATK
     = 복합적인 강화 느낌
```

### 가챭 시스템과의 연계

```
[가챭: 새로운 악세서리 획득]
  공격 반지 (레어) 획득
  
  ↓
  
[특성 영향]
  악세서리 동조 Lv 2 → Lv 3
  → 모든 악세서리 효과 +30%
  → 즉시 강해진 느낌
  
[유료화 동기]
  "더 많은 악세서리가 필요하다"
  → 다이아 구매 (자연스러운 유료화)
```

### 경제 시스템과의 연계

```
[강화 비용]
  무기 강화: 1,000 G (강화도 +1)
  보상: 기본 공격 +5%
  
[가치 계산]
  투자 비용 명확 → 보상 명확
  = 플레이어가 이해하기 쉬움
  = 만족도 높음
```

  무기 마스터리 Lv 5 (+30%)

## 💰 유료화 연계

### 자연스러운 결정 포인트

```
1️⃣ "강화가 필요해"
   → 같은 장비 필요
   → 가챭 필요
   → 다이아 구매 (자동)

2️⃣ "모든 악세서리를 모으고 싶어"
   → 3개를 다 갖춰야 30% 효과
   → 가챭 반복
   → 자연스러운 유료화

3️⃣ "모든 장비를 강화하고 싶어"
   → 4개 장비 모두 강화
   → 긴 시간 투자 필요
   → 시간이 없으면 다이아 (유료)

결론:
  강제하지 않음
  자연스러운 선택
  = 플레이어 만족도 높음
```
