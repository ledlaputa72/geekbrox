## 1. 카드 타입 체계

StS(Slay the Spire)의 4타입 구조를 Dream Collector에 맞게 재해석합니다.

| 타입 | 코드 | 색상 | 역할 | 테두리 |
|------|------|------|------|--------|
| **공격** | `ATTACK` | 🔴 빨강 | 직접 데미지 | 사각형 |
| **스킬** | `SKILL` | 🟢 초록 | 방어·회복·패링·회피·가드 | 사각형 |
| **파워** | `POWER` | 🔵 파랑 | 전투 전체 지속 버프·마법 | 타원형 |
| **커스** | `CURSE` | 🟡 노랑 | 적에게 디버프·저주 | 육각형 |

### 1.1 타입별 특성

**공격 (ATTACK)** — 빨강
- 한 번 사용 시 즉시 적에게 데미지를 입힌다
- 사용 후 버림 더미로
- 일부 공격 카드는 부가 효과(드로우, 자기 버프)를 가짐
- 리액션(패링/회피/가드)에 사용 불가

**스킬 (SKILL)** — 초록
- 방어(블록), 체력 회복, 패링, 회피, 가드 효과
- 리액션 태그(`PARRY` / `DODGE` / `GUARD`)로 리액션 윈도우에서도 사용 가능
- 사용 후 버림 더미로
- 리소스 비용: 대부분 0~1 에너지

**파워 (POWER)** — 파랑
- 사용 즉시 효과가 발동되고, **전투가 끝날 때까지 지속**
- 버림 더미로 가지 않음 (소모형) — 한 번 발동하면 전투 내 영구 적용
- 수치 스택 가능(예: 힘 파워 2회 → 공격 +4)
- 마법·버프·아우라 테마

**커스 (CURSE)** — 노랑
- 적에게 지속 디버프 또는 즉시 약화 효과를 적용
- 타로 "어두운 아르카나" 테마 (악마, 죽음, 탑 등)
- 오토 모드에서 가드/회피 다음 우선순위
- 일부 커스 카드는 리스크/리워드 구조 (플레이어에게도 부작용 가능)

## 2. 태그 시스템 (타입과 독립적)

|------|------|-----------------|
| `PARRY` | 리액션 윈도우 — 패링 판정 | 스킬 |

| `GUARD` | 리액션 윈도우 — 가드 판정 | 스킬 |
| `MAJOR_ARCANA` | 타로 에너지 충전 | 공격, 파워, 커스 |

## 5. 카드 비주얼 가이드

### 5.1 색상 코드

| 타입 | 카드 배경 | 테두리 | 타입 텍스트 |
|------|---------|--------|------------|
| 공격 | `#2D0000` (진한 빨강) | `#FF4444` | `#FF6666` |
| 스킬 | `#002D00` (진한 초록) | `#44FF44` | `#66FF66` |
| 파워 | `#000033` (진한 파랑) | `#4466FF` | `#6688FF` |
| 커스 | `#2D2D00` (진한 노랑) | `#FFDD00` | `#FFEE44` |

### 5.2 테두리 형태

- 공격 / 스킬: 사각형 테두리 (현재와 동일)
- 파워: **타원형** 테두리 (전투 지속 버프임을 시각적으로 구분)
- 커스: **육각형** 테두리 (저주/마법진 이미지)

### 5.3 카드 레이아웃 (공통)

```
┌─────────────────────┐
│   [상단 30%]         │
│   카드 일러스트       │
│   (타입 색상 배경)    │
├─────────────────────┤
│ 카드 이름             │ ← 흰색 텍스트
│ [타입] [비용]         │ ← 타입 색상 칩 + 에너지 아이콘
│ ─────────────────── │
│ 카드 효과 설명        │ ← 16px, 흰색
│                     │
│ [오토 성공률 N%]      │ ← DODGE 태그만, 회색 소자
│ [★ LEGENDARY]       │ ← 희귀도 표시
└─────────────────────┘
```

### 5.4 오토 모드 시각 처리

| 카드 | 오토 모드 표시 |
|------|-------------|
| PARRY 태그 카드 | 중앙 ✕ 오버레이 + 회색 처리 (비활성) |
| DODGE 태그 카드 | 하단에 「오토 성공률 N%」 라벨 |
| POWER 타입 카드 | 정상 (오토에서도 사용 가능) |
| CURSE 타입 카드 | 오토에서 마지막 우선순위 |

### 6.3 파워 카드 처리 (CombatManagerATB.gd / CombatManagerTB.gd)

```gdscript
# POWER 타입 카드는 사용 시 버림 더미로 가지 않음
func player_play_card(card: Card, target_index: int = -1):
    ...
    _resolve_card_effect(card, target_index)
    if card.is_power():
        # 파워: 활성 파워 목록에 추가, 덱에서만 제거
        active_powers.append(card)
        hand.erase(card)
        # discard_pile에는 추가하지 않음
    else:
        hand.erase(card)
        discard_pile.append(card)
    emit_signal("hand_updated", hand)
```
