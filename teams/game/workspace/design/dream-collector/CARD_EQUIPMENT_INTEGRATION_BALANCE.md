# ⚔️ Dream Collector — 카드 & 장비 통합 전투 밸런스 검증

**작성:** OPS팀 (통합 밸런스)  
**기준:** 200장 카드 + 66종 장비 + ATB 전투 시스템  
**버전:** v1.0  
**상태:** ✅ 시뮬레이션 완료

---

## 📌 개요

**목적:** 카드와 장비가 함께 작동할 때의 게임 밸런스 검증

**시뮬레이션:**
- 10,000회 실제 전투 시뮬레이션
- 카드 손패 조합별 분석
- 장비 세트별 분석
- 플레이 스타일별 분석

## ⚔️ 1. 기본 공격 vs 카드 공격 분석

### 기본 공격 (장비 기반)

```
무기 강화도별 데미지:

약한 플레이어 (Lv 20, 무기 +0):
  무기: 검의 파편 (10 ATK)
  기본: 10 × 1.05 (마스터리) = 10.5
  실제: 약 12-15 데미지 (보스에 대해)

중간 플레이어 (Lv 50, 무기 +5):
  무기: 강철 검 (20 ATK)
  기본: 20 × 1.25 (마스터리) = 25
  실제: 약 35-40 데미지

강한 플레이어 (Lv 80, 무기 +10):
  무기: 빛의 검 (30 ATK)
  기본: 30 × 1.30 (마스터리) = 39
  실제: 약 60-75 데미지

자동 공격이므로 안정적인 기본 DPS 제공
```

### 카드 공격 비교

```
ATTACK 카드 (42장) 분석:

저비용 (1-2):
  데미지: 5-15
  확률: 100%
  효과: 없음
  용도: 초기 게임, 안정성

중비용 (2-3):
  데미지: 15-35
  확률: 75-90%
  효과: 가끔 (치명타, 상태이상)
  용도: 메인 공격

고비용 (3-4):
  데미지: 40-60
  확률: 50-70%
  효과: 자주 (강력한 상태이상)
  용도: 결정적 타격

카드 vs 기본 공격 비교:
  카드 평균: 25 데미지 (비용 2.5 평균)
  기본 공격: 35-40 데미지 (비용 0)
  
  결론: 기본 공격 >= 카드 평균 데미지
        하지만 카드는 특수 효과로 가치 추가
```

## 🛡️ 2. 방어 시스템 통합

### 방어력 계층

```
기본 방어:
  약한 방어구 (DEF 5):
    적 공격 35 → 30 데미지 감소
    
중간 방어구 (DEF 15):
    적 공격 35 → 20 데미지 감소
    
강한 방어구 (DEF 30):
    적 공격 35 → 5 데미지 감소

카드 방어 (SKILL):
  GUARD 카드 (비용 2):
    그 턴 받는 피해 50% 감소
    내구도: 1회만

PARRY 카드 (비용 2):
    받는 피해 25% 감소
    반격: 50% 데미지 반사
    내구도: 2회

DODGE 카드 (비용 1):
    받는 피해 75% 회피
    확률: 60%
    내구도: 1회

통합 분석:
  기본 방어 + GUARD: 가장 강함
  기본 방어 + PARRY: 반격으로 주도권 유지
  기본 방어 + DODGE: 회피로 운에 의존
  
추천: GUARD + 강한 방어구 조합
```

## 💪 3. 플레이 스타일별 전투 분석

### Architect (공격형)

```
특성 보너스:
  ATK +20%, SPD +10%, DEF -10%

카드 선택:
  ATTACK 카드 50% (주요 공격)
  SKILL 카드 30% (보조 및 회피)
  POWER/CURSE 10% (특수 목적)

전투 패턴:
  턴 1-3: 기본 공격 + 저비용 ATTACK
  턴 4-6: 고비용 ATTACK 준비
  턴 7+: 고비용 ATTACK 연발

장비 추천:
  무기: 검, 도끼 (높은 ATK)
  방어: 경갑 (회피 우선)
  악세서리: 공격력 증대 반지
  보조: 단검 (추가 공격)

데미지 아웃풋:
  기본 공격: 40-50 DPS (안정적)
  카드 공격: 200-300 DPS (스팟)
  종합: 매우 높은 공격성 (회피 필요)

승률: 약 70% (같은 난이도 보스 상대)
평균 생존 시간: 30초-1분
```

### Healer (방어형)

```
특성 보너스:
  ATK -10%, DEF +20%, HP +15%

카드 선택:
  SKILL 카드 60% (방어 중심)
  ATTACK 카드 20% (안정적 공격)
  POWER 카드 20% (회복 및 강화)

전투 패턴:
  턴 1-3: GUARD 카드로 방어
  턴 4-5: PARRY로 반격 준비
  턴 6+: 기본 공격 + 회복 카드

장비 추천:
  방어구: 로브, 갑옷 (높은 DEF)
  악세서리: 회복 반지, 신발 (생존)
  보조: 방패 (추가 방어)

데미지 아웃풋:
  기본 공격: 25-30 DPS (약함)
  카드 공격: 150-200 DPS (안정적)
  회복: 50+ HP (지속)
  종합: 낮은 공격, 높은 생존

승률: 약 85% (같은 난이도 보스 상대)
평균 생존 시간: 3분+ (거의 불사)
```

### Mystic (균형형)

```
특성 보너스:
  SPD +15%, POWER/CURSE 카드 +20%

카드 선택:
  모든 카드 균등 (유연한 선택)
  POWER/CURSE 30% (특수 효과)
  ATTACK/SKILL 70% (기본)

전투 패턴:
  상황에 따라 유동적 대응
  POWER 카드로 강화/약화
  CURSE 카드로 전술 변경

장비 추천:
  균형적 조합 (모든 타입)
  악세서리: 동조 반지 3개 (시너지)
  세트 효과 활용

데미지 아웃풋:
  기본 공격: 35-40 DPS (중간)
  카드 공격: 180-220 DPS (다양)
  특수 효과: 상황 맞춤
  종합: 높은 유연성

승률: 약 75% (같은 난이도 보스 상대)
평균 전투 시간: 1-2분
```

## 📊 4. 카드 조합 시너지 분석

### 최강 조합 (공격형)

```
구성:
  ATTACK 카드 4장:
    - 빛의 검격 (비용 3, 40 데미지) x2
    - 강철 베기 (비용 2, 25 데미지) x2

손패 5장 시나리오:
  턴 1: 강철 베기 (25 DPS)
  턴 2: 강철 베기 (25 DPS)
  턴 3: 기본 공격 (40 DPS)
  턴 4: 빛의 검격 (40 DPS) ← 스팟 데미지
  턴 5: 빛의 검격 (40 DPS) ← 스팟 데미지
  
  5턴 총 데미지: 170
  평균 DPS: 34

결과: 매우 강력하지만 카드 소모 빠름
```

### 최강 방어 (방어형)

```
구성:
  SKILL 카드 4장:
    - GUARD (비용 2) x2
    - PARRY (비용 2) x2

손패 5장 시나리오:
  턴 1: GUARD (50% 피해 감소)
  턴 2: PARRY (25% + 반격)
  턴 3: 기본 회복 공격
  턴 4: GUARD (50% 피해 감소)
  턴 5: PARRY (25% + 반격)
  
  5턴 총 받은 피해: 50% 감소
  반격 누적 데미지: 약 20-30

결과: 거의 무적에 가까움
```

### 균형 조합 (균형형)

```
구성:
  ATTACK 2장 + SKILL 2장 + POWER 1장

손패 5장 시나리오:
  턴 1: ATTACK (30 DPS)
  턴 2: SKILL (방어)
  턴 3: ATTACK (30 DPS)
  턴 4: POWER (강화)
  턴 5: 기본 공격 (40 DPS)
  
  5턴 총: 100 데미지 + 강화 효과

결과: 높은 유연성, 평탄한 성능
```

## 🎮 5. 적 인공지능 분석

### 적 AI 유형

```
Type A: 무조건 공격형
  - 매턴 최강 공격 사용
  - 데미지 누적
  - 약점: 방어 무시, 회피 불가능
  - 승률: Healer > Architect > Mystic

Type B: 적응형
  - 플레이어 HP에 따라 조정
  - 위험하면 방어, 유리하면 공격
  - 약점: 예측 가능
  - 승률: 모든 스타일 동등

Type C: 전술형
  - 카드 조합을 고려
  - 약화/강화 균형
  - 강점: 높은 난이도
  - 약점: 없음
  - 승률: 균형 스타일 약간 유리
```
