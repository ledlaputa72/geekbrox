## 📖 메인 스토리: 5막 구성

### 막 1: 깨어남 (Act 1: Awakening)

**배경:** 꿈과 현실의 경계 지역

**스토리:**
```
Nox는 기억을 잃은 채 "Memory Void"라는 
어두운 공간에서 깨어난다.
주변은 깨진 꿈의 조각들로 가득하다.

첫 번째 NPC: Lyra (빛의 정령)
- 목소리로만 나타남
- Nox를 깨우고 탈출 방법을 알려줌
- "기억을 찾아야 한다"는 미션 제시

주요 이벤트:
1. 깨어나기
2. 처음 전투 (약한 적 3마리)
3. Lyra 만나기
4. 세상으로의 탈출

기간: 약 30분
필요 레벨: Lv 1-5
보상: 첫 카드 5장, 기본 장비, 1,000 EXP
```

**주요 대사:**
```
Lyra: "넌 누구인가? 기억이 있는가?"
Nox: "...모른다. 어디지?"
Lyra: "넌 기억을 잃었다. 하지만 여기 조각들이 있다.
      수집해야 한다. 모두 모으면 진실을 알 수 있다."
```

### 막 2: 깨진 세상 (Act 2: Fragmented World)

**배경:** 꿈과 현실이 뒤섞인 세상, 여러 지역

**스토리:**
```
Nox는 Memory Void에서 탈출하여 
깨진 세상의 여러 지역을 탐험한다.

각 지역마다 "꿈의 조각"을 가진 NPC들을 만난다:
1. Forest of Whispers (속삭임의 숲) - Finn (숲의 정령)
2. City of Echoes (메아리 도시) - Aria (음악가)
3. Tower of Memories (기억의 탑) - Sage (현인)

각 NPC는 자신의 꿈/기억을 잃어버렸고,
Nox와 협력하여 되찾으려고 한다.

주요 이벤트:
1. 3개 지역 탐험
2. 각 지역에서 "꿈의 보스" 격파
3. 보스 격파 후 NPC와 감정 교감
4. 각 NPC로부터 중요한 기억 조각 습득

기간: 약 2시간
필요 레벨: Lv 5-20
보상: 중형 카드 15장, 중급 장비, 10,000 EXP × 3
```

**각 지역의 NPC:**

#### Finn (숲의 정령)
```
특징: 파란색 머리, 자연 속성
꿈: 깨진 숲을 다시 살리는 것
기억: 과거에 사람들을 지키는 수호자였음
역할: 방어형 NPC (Healer 유형)
카드: SKILL 카드 기증
```

#### Aria (음악가)
```
특징: 은색 옷, 음악을 연주함
꿈: 아름다운 음악을 만드는 것
기억: 과거에 유명한 음악가였음
역할: 공격형 NPC (Architect 유형)
카드: ATTACK 카드 기증
```

#### Sage (현인)
```
특징: 고대 옷, 책을 많이 소유
꿈: 모든 지식을 기억하는 것
기억: 과거에 세상의 역사를 기록함
역할: 균형형 NPC (Mystic 유형)
카드: POWER 카드 기증
```

### 막 3: 진실의 문 (Act 3: The Door of Truth)

**배경:** 꿈과 현실의 경계에 있는 대궁전

**스토리:**
```
Nox가 3개의 꿈의 조각을 모으면,
"Mnemonic Weaver"의 궁전으로 가는 길이 열린다.

여기서 비로소 Nox는 자신이 
"기억을 짜는 자"의 일부였음을 알게 된다.

그리고 중요한 진실을 마주한다:
"세상이 사라지고 있다. 
 기억을 되찾지 못하면 모든 것이 사라질 것이다."

주요 이벤트:
1. 궁전 입장
2. Mnemonic Weaver와 만나기 (처음)
3. 기억 깊숙이 내려가기
4. 중간 보스 격파 (자신의 과거 버전)
5. Mnemonic Weaver와 협상

기간: 약 1시간
필요 레벨: Lv 20-30
보상: 고급 카드 10장, 고급 장비, 20,000 EXP
```

**중간 보스: The Forgotten Self (잊혀진 자신)**
```
특징: Nox의 과거 버전, 어두운 옷
공격 패턴: 어두운 공격들, CURSE 카드 사용
약점: LIGHT/MEMORY 속성 공격
보상: 기억 조각 3개, 특별 카드 "The Past"
```

### 막 4: 선택의 순간 (Act 4: The Choice)

**배경:** 현실과 꿈의 경계가 무너지고 있는 세상

**스토리:**
```
Nox는 Mnemonic Weaver로부터 충격적인 사실을 듣는다:

"넌 선택해야 한다.
 1. 모든 기억을 되찾고 진정한 자신이 되기
 2. 현재의 기억을 잃은 자신으로 살며 모두를 구하기"

이것이 게임의 가장 중요한 선택의 순간이다.
플레이어의 선택에 따라 엔딩이 달라진다.

주요 이벤트:
1. 모든 기억 조각 모으기
2. 최종 선택: 기억 되찾기 vs 현재 모습 유지
3. 선택에 따른 다른 엔딩으로 진행

기간: 약 1시간 30분
필요 레벨: Lv 30-40
보상: 선택에 따른 다른 보상
```

### 막 5: 새로운 시작 (Act 5: New Beginning)

**배경:** 선택에 따라 다른 세상

**엔딩 1: 기억을 되찾은 Nox**
```
Nox가 진정한 이름과 정체성을 되찾는다.
하지만 그와 동시에 과거의 모든 고통도 기억한다.

결국 Nox는:
- 자신의 과거를 받아들이고
- 새로운 정체성으로 세상을 구한다
- Lyra와 다른 NPC들과 함께 새로운 시대 시작

엔딩 영상: 밝음 (행복하지만 복잡한 감정)
음악: 희망차지만 약간의 슬픔
```

**엔딩 2: 기억을 잃은 채 사는 Nox**
```
Nox가 자신의 정체성을 포기하고
"잃어버린 존재"로서 세상을 구한다.

이 선택은:
- 더 큰 힘을 얻게 함 (모든 기억이 에너지가 됨)
- 하지만 더 이상 진정한 Nox가 아님
- 모두를 구하지만 자신은 사라짐

엔딩 영상: 어둠 (희생과 용기)
음악: 장엄하지만 애련함
```

**엔딩 3: 숨겨진 엔딩 (게임 100% 완료 필요)**
```
특별한 조건을 만족하면 진정한 엔딩이 열린다:
- 모든 사이드 퀘스트 완료
- 모든 숨겨진 기억 수집
- 모든 NPC와 완전히 신뢰 형성

진정한 엔딩:
- Nox와 Mnemonic Weaver가 협력
- 세상 자체를 다시 짜는 (Weave) 이야기
- 모든 잃어버린 것들이 복구됨
- 새로운 시대의 진정한 시작

엔딩 영상: 밝음 (완전한 희망과 재생)
음악: 장엄하고 아름다운 오케스트라
```

**기간:** 약 2시간
**필요 레벨:** Lv 40+
**보상:** 최종 보상, 특별 칭호 "The True Weaver"

```

2. NPC 친밀도 대사
   - 친밀도 수준에 따라 변함
   - 심화된 이야기 전개

3. 선택지 대사
   - 플레이어가 선택하는 대사
   - 결과에 영향

4. 숨겨진 대사
   - 특정 조건 만족 시에만 등장
   - 추가 정보 제공

5. 반복 대사
   - 같은 NPC와 만날 때마다 변함
   - 친밀도에 따른 차이
```
