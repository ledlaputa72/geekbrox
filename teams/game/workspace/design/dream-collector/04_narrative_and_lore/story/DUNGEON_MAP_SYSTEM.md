## 🌍 메인 지역 5개

### 지역 1: Memory Void (기억의 공백) - 막 1

```
위치: 게임 시작점
크기: 소형 (약 10분 길이)
난이도: Lv 1-5 (튜토리얼)

테마:
- 어두운 파란색과 검은색
- 깨진 기억의 조각들
- 몽환적인 분위기

주요 요소:
- 시작점: 깨어나는 공간
- 길 찾기: 3개의 경로 선택
- 첫 전투: 약한 적 (슬라임 3마리)
- 보스: Memory Guardian (약한 보스)

수집 요소:
- 기억 조각 3개 (필수)
- 숨겨진 카드 1개 (선택)

NPC: Lyra (음성만)

진행도: 약 30분
보상: 첫 카드 5장, 기본 장비, 1,000 EXP
```

---

### 지역 2: Forest of Whispers (속삭임의 숲) - 막 2-1

```
위치: Memory Void 탈출 후
크기: 중형 (약 20분 길이)
난이도: Lv 5-15

테마:
- 초록색과 파란색
- 울창한 숲, 신비로운 분위기
- 나뭇잎 소리와 새 울음

주요 요소:
- 입구: 거대한 나무
- 탐험: 3개 구역 (숲 깊숙이)
- 전투: 숲의 정령 (약 5-6마리)
- 보스: Woodland Spirit (Finn의 과거 버전)

구조:
```
┌────────────────────────────┐
│  Forest of Whispers        │
│                            │
│  [입구]                    │
│    ↓                       │
│  [숲 1구역] [숲 2구역]     │
│    ↓         ↓            │
│  [깊숙한곳]   [보스 방]    │
│    ↓         ↓            │
│  [숨겨진곳] [출구]        │
│                            │
└────────────────────────────┘
```

수집 요소:
- 기억 조각 2개
- 특별 카드 2개 (숨겨진)
- 강화 재료 3개

NPC: Finn (만남 & 협력)

진행도: 약 45분
보상: 중급 카드 5장, 특급 장비, 5,000 EXP
```

---

### 지역 3: City of Echoes (메아리 도시) - 막 2-2

```
위치: 지도 동쪽
크기: 중형 (약 25분 길이)
난이도: Lv 10-20

테마:
- 은색과 보라색
- 도시 건축물, 음악 소리
- 신비로운 야경

주요 요소:
- 중앙 광장: 연주회 무대
- 골목길: 3개 경로
- 타워: 높이 올라가기
- 보스: Echo Knight (Aria의 정신 버전)

지역 구성:
```
광장 (중앙)
├── 북쪽 골목 → 숨겨진 음악실
├── 동쪽 타워 → 전망대
├── 서쪽 거리 → 과거의 가옥들
└── 남쪽 문 → 보스 전장
```

수집 요소:
- 기억 조각 3개
- 악기 조각 5개
- 특별 카드 1개

NPC: Aria (만남 & 협력)

진행도: 약 1시간
보상: 고급 카드 5장, 전설 장비 1개, 7,500 EXP
```

### 지역 4: Tower of Memories (기억의 탑) - 막 2-3

```
위치: 지도 중앙
크기: 대형 (약 40분 길이)
난이도: Lv 15-30

테마:
- 갈색과 금색
- 고대 건축, 책과 지식
- 엄숙한 분위기

주요 요소:
- 입구: 거대한 문
- 1층-5층: 각 층마다 테마 다름
  * 1층: 기억의 도서관
  * 2층: 시간의 정원
  * 3층: 몽환의 방
  * 4층: 진실의 방
  * 5층: 보스 전장
- 전투: 각 층마다 4-5명의 적
- 보스: Keeper of Memories (Sage의 진정한 모습)

층별 구조:
```
5층 (보스 전장)
  ↑
4층 (진실의 방)
  ↑
3층 (몽환의 방)
  ↑
2층 (시간의 정원)
  ↑
1층 (도서관) → [입구]
```

수집 요소:
- 기억 조각 4개
- 책 조각 10개
- 특별 카드 2개

NPC: Sage (만남 & 협력)

진행도: 약 1시간 30분
보상: 전설 카드 5장, 최고급 장비, 12,500 EXP
```

---

### 지역 5: Mnemonic Palace (기억의 궁전) - 막 3-4

```
위치: 지도 중앙 (숨겨진 영역)
크기: 매우 큼 (약 1시간 길이)
난이도: Lv 30-45

테마:
- 모든 색상 혼재
- 초현실적 건축
- 시간과 공간이 뒤섞임

주요 요소:
- 대청: 출발점
- 4개 날개: 각각 다른 테마
  * 북쪽 날개: 과거 (Memory)
  * 동쪽 날개: 현재 (Present)
  * 남쪽 날개: 미래 (Future)
  * 서쪽 날개: 꿈 (Dream)
- 중앙 홀: 최종 보스 전장
- 최종 보스: Mnemonic Weaver

지도:
```
        [북쪽 날개]
             ↑
[서쪽] ← [중앙 홀] → [동쪽]
  날개    (보스)     날개
             ↓
        [남쪽 날개]
```

각 날개의 특징:
- 북쪽: 어두운 기억들 (CURSE 카드)
- 동쪽: 현재의 도전 (ATTACK 카드)
- 남쪽: 미래의 희망 (POWER 카드)
- 서쪽: 꿈의 영역 (SKILL 카드)

수집 요소:
- 기억 조각 8개
- 특별 카드 5개
- 궁극의 보물 1개

NPC: Mnemonic Weaver (최종 만남)

진행도: 약 2시간
보상: 최고급 모든 것, 50,000 EXP, 특별 칭호
```

## 🎮 일일 던전 5개 (반복 가능)

### 던전 1: Novice's Hall (초보자 연습장)

```
난이도: Lv 1-20
시간: 약 5분
보상: 약한 카드 2-3개, 100-500 EXP, 50-200 골드

특징:
- 매일 3회 무료 진입 가능
- 1회에 추가 금 50 다이아 소비로 진입 가능
- 약한 적들만 등장
- 초보 플레이어 연습용

구성:
- 5개 방 (각 방 1-2마리 적)
- 보스: 없음 (마지막 방에 조금 강한 적)
```

### 던전 2: Forest Trial (숲의 시험)

```
난이도: Lv 10-25
시간: 약 10분
보상: 중급 카드 2-4개, 500-1,500 EXP, 200-500 골드

특징:
- 매일 2회 무료 진입
- 강화 재료 드롭 시작
- 숲 테마, 자연 속성 적

구성:
- 3개 구역 (각 구역 3-4마리 적)
- 보스: Forest Guardian (중간 난이도)
- 보상: 특급 카드 가능
```

### 던전 3: City Challenge (도시의 도전)

```
난이도: Lv 20-35
시간: 약 15분
보상: 고급 카드 2-5개, 1,500-3,000 EXP, 500-1,000 골드

특징:
- 매일 1회 무료 진입 (또는 150 다이아)
- 높은 확률로 RARE/SPECIAL 카드 드롭
- 도시 테마, 다양한 적

구성:
- 4개 지역 (각 지역 4-5마리 적)
- 보스: City Guardian (높은 난이도)
- 보상: SPECIAL 카드 자주 드롭
```

### 던전 4: Tower Ascent (탑 등반)

```
난이도: Lv 30-45
시간: 약 20분
보상: 전설 카드 1-3개, 3,000-6,000 EXP, 1,000-2,000 골드

특징:
- 매일 1회 무료 진입 (또는 200 다이아)
- 매우 높은 확률로 SPECIAL/LEGENDARY 카드
- 탑 테마, 매우 어려운 적들

구성:
- 5층 (각 층 5마리 적)
- 보스: Tower Keeper (매우 높은 난이도)
- 보상: LEGENDARY 카드 가능성 높음
```

### 던전 5: Nightmare Abyss (악몽의 심연)

```
난이도: Lv 40-50+ (최고 난이도)
시간: 약 30분
보상: 최고급 카드 1-2개, 5,000-10,000 EXP, 2,000-5,000 골드

특징:
- 매일 1회만 진입 가능 (300 다이아 소비)
- 가장 높은 보상
- 극도로 어려운 적들
- 진정한 고난도 플레이어 대상

구성:
- 무한 루프 방식 (클리어할 때까지 계속)
- 깊이 증가마다 난이도 상향
- 최종 보스: Abyss Keeper (불가능에 가까운 난이도)
- 보상: LEGENDARY 카드 높은 확률, 특별 장비
```

## 🎉 이벤트 던전 (기간 한정)

### 이벤트 던전 구조

```
월별 1개씩 열림:
- 특정 기간만 진입 가능 (보통 2주)
- 특별한 테마와 보상
- 기한을 놓치면 다음 해에만 가능
- 컬렉션 목표

예시 (월별):
1월: "새해의 꿈" - 새해 테마, 신규 카드 5개
2월: "사랑의 기억" - 로맨스 테마, 특별 카드
3월: "봄의 재생" - 자연 테마, 강화 재료 2배
...
12월: "겨울의 평온" - 축제 테마, 최고 보상
```
