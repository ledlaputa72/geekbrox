# 📚 꿈(Dream) 레퍼런스 리포트 - Dream Collector 기획용

**문서 목적**: 게임 'Dream Collector'의 세계관, 스토리, 카드 기획의 깊이를 더하기 위한 심리학적, 역사/문화적, 문학/신화적 레퍼런스 정리.

---

## 1. 심리학적 측면: 무의식의 지도

프로이트와 융으로 대표되는 심리학은 꿈을 '무의식으로 향하는 왕도'로 보았습니다. 이는 우리 게임의 핵심적인 세계관이 될 수 있습니다.

| 이론/현상 | 설명 | 게임 기획 적용 아이디어 |
| :--- | :--- | :--- |
| **프로이트적 꿈**<br>(Freudian Dream) | 억압된 욕망의 성취. 현실에서 금기시된 감정이나 소원이 상징적으로 나타남. | **[카드]** "억압된 분노": 내 HP를 5 소모하고 적에게 20의 피해를 준다. <br>**[스토리]** 특정 꿈(스테이지)은 특정 욕망(보상/패널티)과 연결됨. |
| **융의 원형**<br>(Jungian Archetypes) | 모든 인류가 공유하는 '집단 무의식' 속 보편적인 상징. 페르소나, 그림자, 아니마/아니무스 등. | **[카드/캐릭터 원형]** "그림자 자아" 카드: 강력하지만, 덱에 '불안' 카드를 섞어 넣는다.<br>**[NPC]** "현명한 노인" NPC가 조언을 해줌. "페르소나"는 방어도와 관련. |
| **위협 시뮬레이션**<br>(Threat Simulation) | 꿈은 현실의 위협적인 상황에 대처하는 능력을 훈련하는 과정이라는 현대 이론. | **[핵심 컨셉]** 게임 플레이 자체가 '악몽'이라는 위협에 맞서 싸우는 훈련.<br>**[카드]** 특정 위협(보스)에 강한 효과를 발휘하는 '예지몽' 카드. |
| **자각몽**<br>(Lucid Dreaming) | 꿈 속에서 스스로 꿈임을 인지하고 통제하는 상태. | **[카드]** "자각몽": 이번 턴에 카드를 3장 더 뽑는다. 또는, 버린 카드 더미에서 원하는 카드를 1장 가져온다. **[파워 카드]** '꿈 조종자'. |
| **악몽 / 야경증**<br>(Nightmare) | 극심한 공포와 불안을 동반하는 꿈. 강력한 부정적 감정의 표출. | **[몬스터]** '악몽' 역할군 몬스터. 플레이어에게 '공포' 디버프 부여.<br>**[카드]** 강력하지만 '악몽' 디버프를 받는 고위험 고수익 카드. |

## 2. 역사/문화적 측면: 시대의 꿈

역사 속에서 꿈은 신의 계시, 예언, 또는 질병의 징후 등 다양한 의미로 해석되었습니다.

| 문화/시대 | 꿈의 의미 | 게임 기획 적용 아이디어 |
| :--- | :--- | :--- |
| **고대 이집트** | 신과의 소통, 미래에 대한 예언. 꿈에서 본 상징을 해몽하여 미래를 읽음. | **[카드]** "상형문자의 계시": 적의 다음 행동(의도)을 3턴 동안 엿본다.<br>**[이벤트]** '오래된 파피루스' 이벤트에서 꿈 해몽 선택지로 보상/패널티. |
| **고대 그리스** | 아스클레피오스 신전에서의 치유몽(Incubation). 꿈을 통해 병을 진단하고 치료법을 얻음. | **[휴식 노드]** '치유의 신전': HP 회복량이 더 많거나, 덱에서 '상처' 카드를 제거.<br>**[카드]** "아스클레피오스의 손길": HP 10 회복, 모든 디버프 제거. |
| **아메리카 원주민** | 영적 탐구(Vision Quest), 정체성 발견. 동물을 통해 메시지를 받음. 드림캐쳐로 악몽을 걸러냄. | **[파워 카드]** "드림캐쳐": 매 턴 시작 시, 상태이상 카드 1장을 소멸시킨다.<br>**[카드]** '정령 동물' 시리즈: 곰(방어), 늑대(협공), 독수리(드로우) 등. |
| **성경 (기독교/유대교)**| 신의 경고나 예언. 요셉의 꿈 해몽, 다니엘의 환상 등. | **[카드]** "일곱 마리 암소": 다음 7턴 동안 에너지가 1 감소하는 대신, 그 후 7턴 동안 에너지가 1 증가.<br>**[스토리]** 거대한 '예언의 꿈'을 해석하는 것이 메인 스토리. |

## 3. 문학/신화적 측면: 꿈의 인격화

문학과 신화는 꿈을 의인화하거나, 꿈과 현실의 경계를 허무는 환상적인 이야기를 제공합니다.

| 출전 | 꿈의 묘사 | 게임 기획 적용 아이디어 |
| :--- | :--- | :--- |
| **그리스 신화** | **모르페우스(Morpheus)**: 꿈의 신, 인간의 형상을 흉내 냄.<br>**포베토르(Phobetor)**: 악몽의 신, 동물의 모습.<br>**판타소스(Phantasos)**: 비현실적 꿈, 사물의 모습. | **[보스]** 3명의 보스: 모르페우스(플레이어 복제), 포베토르(강력한 몬스터 소환), 판타소스(함정 카드 생성).<br>**[카드]** "모르페우스의 형상": 이번 전투에서 적 하나를 복제하여 아군으로 사용. |
| **일본 신화** | **바쿠(獏)**: 사람의 악몽을 먹고 사는 상상의 동물. | **[카드]** "바쿠의 포식": 내 손패와 덱에 있는 모든 '악몽', '불안', '상처' 카드를 소멸시킨다. <br>**[NPC/상점]** '바쿠'에게 돈을 지불하고 저주 카드를 제거. |
| **닐 게이먼, 『샌드맨』** | '꿈' 그 자체가 세상의 법칙을 관장하는 일곱 영원 중 하나. 강력하고 창조적인 힘. | **[세계관]** 플레이어는 '꿈'의 영역을 여행하는 '꿈 수집가'.<br>**[희귀 카드]** "꿈의 투구", "모래 주머니", "루비" 등 강력한 유물 카드 시리즈. |
| **H.P. 러브크래프트**| 악몽은 이해할 수 없는 우주적 공포(코즈믹 호러)와 연결되는 통로. 광기를 유발. | **[디버프]** `광기(Madness)`: 덱에 쓸모없고 해로운 '광기' 카드를 섞어 넣음. <br>**[카드]** "금단의 지식": 강력한 효과를 발휘하지만, '광기' 카드를 3장 얻는다. |

## 4. 게임 기획 적용 방안

### 🃏 **타로 카드 시스템과의 연계**

게임의 핵심인 '3단계 꿈 선택'을 메이저 아르카나 타로 카드와 연결하여 깊이를 더합니다.

| 단계 | 타로 카드 (예시) | 의미 | 게임 내 효과 |
| :--- | :--- | :--- | :--- |
| **시작 (Start)** | **0. The Fool (광대)** | 순수한 시작, 미지의 여정 | 쉬운 난이도, 다양한 가능성(노드) |
| | **I. The Magician (마법사)** | 창조, 잠재력 | 시작 시 유용한 파워 카드 1장을 가지고 시작 |
| **여정 (Journey)**| **VII. The Chariot (전차)** | 돌진, 승리 | 전투 노드의 비중이 높고, 보상이 큼 |
| | **IX. The Hermit (은둔자)** | 탐색, 성찰 | 휴식/이벤트 노드의 비중이 높음 |
| **종료 (End)** | **XIII. Death (죽음)** | 끝과 새로운 시작 | 어려운 보스, 하지만 클리어 시 매우 희귀한 보상 |
| | **XXI. The World (세계)** | 완성, 통합 | 모든 종류의 노드가 균형 있게 등장, 최종 보상 강화 |

### 🎴 **카드 디자인 예시**

| 카드 이름 | 원형 | 등급 | 비용 | 효과 | 레퍼런스 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **억압된 분노** | 꿈의 분노 | 고급 | 1 | 내 HP를 5 잃는다. 적에게 20의 피해를 준다. | 프로이트 |
| **그림자 강림** | 뒤틀린 악몽 | 희귀 | 2 | `힘`을 5 얻는다. 덱에 `죄책감`(상태이상) 카드 2장을 섞어 넣는다. | 융의 '그림자' |
| **자각몽** | 샘솟는 영감 | 고급 | 0 | 버린 카드 더미에서 카드 1장을 골라 손으로 가져온다. 소멸된다. | 자각몽 |
| **바쿠의 포식** | 고요한 방벽 | 희귀 | 2 | 내 덱, 손패, 버림패의 모든 상태이상 카드를 소멸시킨다. | 바쿠(일본 신화) |
| **예지몽** | (공용) | 고급 | 1 | 다음 2턴 동안 모든 적의 행동을 미리 본다. | 예언(이집트/성경) |
