# 💰 Dream Collector — 경제 시스템 & 비용 분석

**작성:** OPS팀 (경제 분석)  
**기준:** 200장 카드 + 66종 장비의 획득 비용  
**버전:** v1.0  
**상태:** ✅ 분석 완료

---

## 📌 개요

**목적:** 각 카드와 장비의 획득 비용을 산출하여 경제 밸런스 검증

**범위:**
- 카드 획득 비용 (가챠 + 퀘스트 + 드롭)
- 장비 획득 비용 (가챠 + 던전 + 합성)
- 강화 비용 (골드 소모)
- 월간 경제 흐름

## 🎴 1. 카드 획득 비용

### 희귀도별 가챠 비용

```
COMMON (40%):
  가챠 가격: 50 다이아 / 1회
  확률: 40%
  예상 소모: 125 다이아 (평균 1장 획득)
  = 약 $1.25 / 장

RARE (35%):
  확률: 35%
  예상 소모: 143 다이아 (평균 1장 획득)
  = 약 $1.43 / 장

SPECIAL (15%):
  확률: 15%
  예상 소모: 333 다이아 (평균 1장 획득)
  = 약 $3.33 / 장

LEGENDARY (10%):
  확률: 10%
  예상 소모: 500 다이아 (평균 1장 획득)
  = 약 $5.00 / 장

200장 전부 구매:
  COMMON (80장): 100 다이아 × 80 = 8,000 D
  RARE (70장): 143 다이아 × 70 = 10,010 D
  SPECIAL (30장): 333 다이아 × 30 = 9,990 D
  LEGENDARY (20장): 500 다이아 × 20 = 10,000 D
  ─────────────────────────────────
  총 비용: 38,000 다이아 = $380
  
  현실적 시나리오 (무료 플레이):
  - 월 무료 가챠: 30회 (30 카드)
  - 월 유료 가챠: 5회 (5 카드)
  - 이벤트 보상: 5회 (5 카드)
  ─────────────────
  월간: 40 카드
  
  200장 완성: 약 5개월
```

### 스토리/퀘스트 보상

```
메인 스토리:
  - 막 1: 5장 (필수 학습)
  - 막 2: 15장 (3개 지역 × 5장)
  - 막 3: 10장 (중간 보스)
  - 막 4: 8장 (선택)
  - 막 5: 7장 (엔딩)
  ─────────
  총 45장 (무료, 게임 진행에 포함)

사이드 퀘스트:
  - 초급 (15개): 30장
  - 중급 (15개): 40장
  - 상급 (15개): 50장
  ─────────
  총 120장 (선택, 플레이 시간 투자)

일일 던전:
  - 매일 3-5장
  - 월 90-150장
  - 누적: 1년에 1,080-1,800장

결론:
  200장 완성 = 약 2-3개월 (무료 플레이)
  비용: $0 (시간만 투자)
```

## 💎 3. 강화 비용 (골드 소모)

### 기본 공식

```
강화 비용 = 1,000 골드 × 1.5^강화도

+0: 1,000 G
+1: 1,500 G
+2: 2,250 G
+3: 3,375 G
+4: 5,062 G
+5: 7,594 G
+6: 11,391 G
+7: 17,086 G
+8: 25,630 G
+9: 38,445 G
+10: 57,667 G

누적 비용 (+0→+10):
  단계별 합: 170,500 G / 1개 장비
  
  4개 슬롯 풀 강화:
  170,500 × 4 = 682,000 G
```

### 현실적 강화 시나리오

```
초기 (Lv 1-10):
  목표: 무기 +2, 방어구 +1
  비용: (2,250 + 3,375) × 1 = 5,625 G
  획득: 일일 100-200 G
  기간: 약 2주

중기 (Lv 20-50):
  목표: 무기 +5, 방어구 +4, 나머지 +2
  비용: (7,594 + 5,062 + 2,250 + 2,250) × 1 = 17,156 G / 월
  획득: 일일 300-500 G = 월 9,000-15,000 G
  상황: 약간의 자금 부족

후기 (Lv 50-80):
  목표: 모든 슬롯 +6~+7
  비용: 약 50,000-100,000 G / 월
  획득: 일일 500-800 G = 월 15,000-24,000 G
  상황: 자금 부족 (추가 보상 필요)

극후기 (Lv 80-100):
  목표: 모든 슬롯 +8~+10
  비용: 약 150,000-200,000 G / 월
  획득: 일일 800-1,200 G = 월 24,000-36,000 G
  상황: 심각한 자금 부족 (보스/이벤트 보상 필수)

결론:
  모든 슬롯 풀 강화(+10): 약 12-18개월 필요
  비용: 682,000 G
```

## 💳 5. 유료 및 무료 플레이 분석

### 무료 플레이어 (F2P)

```
첫 달:
  카드 40장, 장비 30개, 강화 +3 평균
  시간: 20-30시간
  만족도: 높음 (지속 성장)

3개월:
  카드 120장, 장비 66개, 강화 +5 평균
  시간: 60-90시간
  만족도: 중상 (강화 속도 느려짐)

6개월:
  카드 200장, 장비 100개+, 강화 +6-7 평균
  시간: 120-180시간
  만족도: 중간 (고스탑, 추가 콘텐츠 필요)

1년:
  완전 강화 (거의 불가능)
  시간: 240+시간
  만족도: 낮음 (진행도 정체)

결론:
  - 초중기: 무료도 매우 즐거움
  - 후기: 유료 플레이 유인 필요
  - 전체: 약 70% 유지율 (일반적 게임 수준)
```

```

  카드 80장, 장비 50개, 강화 +4 평균

3개월:
  카드 200장, 장비 100개+, 강화 +7-8 평균

6개월:
  완전 강화 가능, 풀 극강화 진행 중
  만족도: 높음 (모든 콘텐츠 주도권)

1년:
  강화 +9~+10 완료, 프리플레이어 대비 20배 강함
```

## 📈 6. 수정 권장사항

### 초기 (Lv 1-20) - 문제 없음 ✅

### 중기 (Lv 30-60) - 약간의 조정

```
현재: +3,000 G 흑자 (경계)
개선:
  1. 사이드 퀘스트 보상 +1,000 G
  2. 일일 던전 보상 +500 G/회
  
개선 후: +5,500 G (안정적)
```

### 후기 (Lv 70-100) - 적극 조정 필요

```
현재: -18,000 G 적자 (심각)
개선:
  1. 월간 특별 이벤트 +20,000 G
  2. 보스 드롭 +5,000 G
  3. 강화 비용 10% 감소
  4. 대체 강화 경로 추가 (궁극의 장비)

개선 후: +5,000 G (안정적)
```
