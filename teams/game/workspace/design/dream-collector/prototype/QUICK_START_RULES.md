# Dream Collector - 빠른 시작 룰 (5분 이해)

## 🎯 게임 목표
- 꿈 던전을 탐험하며 적을 물리치고 **Reverie(꿈 조각)**를 모으기
- HP가 0이 되면 게임 오버

---

## 🎴 턴 구조 (4단계)

### 1️⃣ DRAW (드로우)
- 카드 5장을 뽑는다
- 덱이 비면 버린 더미를 섞어서 새 덱으로

### 2️⃣ REVEAL (공개)
- 적의 다음 행동을 확인
- "공격 8" 또는 "방어 5" 등

### 3️⃣ ACTION (행동)
- **Energy 3**으로 시작
- 카드를 사용해서:
  - 적 공격
  - 방어막 쌓기
  - Reverie 수집
- 사용한 카드는 버린 더미로

### 4️⃣ END (종료)
- 남은 Block(방어막)은 사라짐
- 적의 행동 실행
- 다음 턴 시작

---

## 🎴 카드 읽는 법

```
[카드 이름]
코스트: 1 Energy
효과: Deal 5 damage
```

- **코스트**: 카드 사용에 필요한 Energy
- **효과**: 즉시 발동

---

## ⚔️ 전투 예시

```
[플레이어 턴]
HP: 50, Energy: 3, Hand: 5장

1. Basic Strike (1코스트) → 적에게 5 피해
2. Dream Shield (1코스트) → Block 5 획득
3. Memory Shard (1코스트) → Reverie 2 획득
4. 턴 종료

[적 턴]
적의 공격 8 → Block 5 소모 → 남은 3 피해 받음
HP: 50 → 47
```

---

## 🎲 준비물

### 카드
- 시작 덱 12장
- 획득 풀 20장
- 적 카드 6장

### 토큰
- HP: 동전 10개
- Energy: 클립 4개  
- Reverie: 구슬 20개
- Block: 작은 돌 10개

---

## 🚀 첫 게임 시작 (10분)

### 1. 셋업
- 시작 덱 12장 섞기
- HP 50으로 시작
- 첫 적 선택: Shadow Whisper (HP 15)

### 2. 첫 턴
1. 카드 5장 뽑기
2. Energy 3 확인
3. 적 행동 확인 ("공격 5")
4. 카드 사용 (총 3 Energy까지)
5. 턴 종료 → 적 공격 받기

### 3. 승리
- 적 HP 0 → 카드 3개 중 1개 선택
- 다음 적 등장

---

## 💡 첫 플레이 팁

- **첫 3턴**: 공격 위주 (적 빨리 죽이기)
- **HP 낮으면**: 방어 카드 사용
- **Reverie**: 나중에 강화에 사용 (지금은 무시해도 OK)
- **Energy 관리**: 0~1 남기는 게 이상적

---

## ❓ 자주 하는 실수

1. **Block은 턴마다 사라짐** - 쌓아두기 불가
2. **Energy는 매 턴 3** - 이월 안 됨
3. **덱 비면 섞기** - 게임 오버 아님
4. **적 행동 먼저 보기** - 행동 후 카드 사용
