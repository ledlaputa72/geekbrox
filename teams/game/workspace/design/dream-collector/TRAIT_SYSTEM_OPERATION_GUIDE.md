# 🎮 Dream Collector — 특성 시스템 운영 가이드 (플레이어 경험)

**작성:** Atlas (PM) → Game팀  
**기준:** 플레이어가 실제로 조작하는 방식, UI/UX, 재화 연계  
**버전:** v1.0  
**상태:** 📋 완전한 운영 가이드

---

## 📌 개요

**목표:** 플레이어가 특성 시스템을 어떻게 사용하는가

```
플레이어는 특성을 직접 조작하지 않음
→ 장비 선택 & 강화가 특성을 자동으로 조정
→ 비자동(강제) 메커니즘은 "특성 포인트 분배"뿐
```

## 1️⃣ 플레이어가 실제로 하는 행동 3가지

### 행동 1: 장비 장착 (자동 특성 변경)

**시나리오:**

```
[장비 화면]

현재 장착:
  무기: 강철검 +0
  방어구: 천 옷 +0
  악세서리: (없음)
  보조무기: (없음)

가방:
  강철검 +3 ← 더 강한 버전 발견!
  [강철검 +3 장착하기] 버튼
```

**플레이어 선택:**
```
[강철검 +3 장착하기] 클릭
```

**즉시 일어나는 일:**

```
[화면 전환]

기존 무기 마스터리: Lv 0 (+0)
신규 무기 마스터리: Lv 3 (+3)

[기본 공격 데미지 업데이트]

기존: 20 데미지
신규: 26 데미지 (30% 증가)

UI 표시:
  ✨ 기본 공격 +6 증가!
  강철검 +3 장착됨

플레이어 체감:
  "아! 강한 무기를 장착하니 공격력이 올랐다!"
  (자동이지만 직관적으로 느낌)
```

### 행동 2: 장비 강화 (특성 자동 레벨업)

**시나리오:**

```
[강화 화면]

현재 무기: 강철검 +3
필요: 같은 강철검 1개 + 1,000 G

[강화하기] 버튼
```

**플레이어 선택:**
```
[강화하기] 클릭
```

**즉시 일어나는 일:**

```
[강화 진행]

강철검 +3 → 강철검 +4로 강화됨

[특성 자동 변경]

무기 마스터리: Lv 3 → Lv 4
보너스: 20% → 25%

[기본 공격 데미지 재계산]

기존: 26 데미지 (무기 마스터리 Lv 3: +20%)
신규: 27 데미지 (무기 마스터리 Lv 4: +25%)

UI 표시:
  ✨ 강화 완료!
  강철검 +4 획득
  
  기본 공격:
  ❌ 26 → ✅ 27 (+1 증가)
  
  무기 마스터리 Lv 4 (25%)

플레이어 체감:
  "강화할 때마다 자동으로 강해진다!"
  (강화의 보상 명확함)
```

### 행동 3: 특성 포인트 분배 (유일한 수동 조작)

**시나리오:**

```
[레벨업]

Nox가 Lv 30에서 Lv 31로 업그레이드
특성 포인트: +5 획득

총 특성 포인트: 147 → 152

[특성 포인트 분배 화면 자동 열림]
```

**플레이어가 보는 화면:**

```
╔════════════════════════════════╗
║   특성 포인트 분배 화면         ║
╚════════════════════════════════╝

사용 가능: 5pt

[현재 스탯]
  공격력 (ATK): 58
  방어력 (DEF): 35
  생명력 (HP): 280
  속도 (SPD): 12

[플레이 스타일: Architect (공격형)]
추천: ATK에 집중

┌─────────────────┐
│ 공격력 (ATK)   │
│  현재: 58      │
│ [+1] [+2] [+5] │  ← 한 번에 1/2/5pt 추가
└─────────────────┘

┌─────────────────┐
│ 방어력 (DEF)   │
│  현재: 35      │
│ [+1] [+2] [+5] │
└─────────────────┘

┌─────────────────┐
│ 생명력 (HP)    │
│  현재: 280     │
│ [+1] [+2] [+5] │
└─────────────────┘

┌─────────────────┐
│ 속도 (SPD)     │
│  현재: 12      │
│ [+1] [+2] [+5] │
└─────────────────┘

[분배 완료] [취소]
```

**플레이어 선택:**
```
공격형이므로 ATK [+5] 클릭
```

**즉시 일어나는 일:**

```
[분배 완료 애니메이션]

특성 포인트: 5pt → 0pt (소비됨)

[스탯 업데이트]

공격력 (ATK): 58 → 63 (+5 증가)

[기본 공격 데미지 재계산]

기존: 27 데미지 (강화도로부터)
신규: 32 데미지 (강화도 + 특성 포인트)

UI 표시:
  ✨ 특성 포인트 분배 완료!
  공격력: 58 → 63 (+5)
  기본 공격: 27 → 32 (+5)

플레이어 체감:
  "공격력을 선택했더니 공격이 강해졌다!"
  (명확한 선택과 결과)
```

## 3️⃣ 재화 활용 (골드 & 다이아)

### 골드 (무료 재화)

**사용처:**

```
1. 장비 강화 (메인)
   무기 +1 강화: 500 G
   무기 +2 강화: 750 G
   ...
   무기 +10 강화: 10,000 G
   
   총 36,250 G (1개 무기)
   4개 무기: 145,000 G

2. 카드 강화 (부수)
   카드 강화: 100~500 G
   
3. 특성과의 연계
   강화 비용 ← 특성 포인트 효과 없음
   (골드 사용은 플레이어 자유)
```

**획득 방법:**

```
1. 전투 보상
   - 일일 던전: 1,000 G/회
   - 이벤트: 5,000 G/회

2. 퀘스트
   - 메인 스토리: 10,000 G
   - 사이드 퀘스트: 1,000 G/회

3. 상점 판매
   - 불필요한 장비 판매
   - 중복 카드 판매

월간 수입: 약 40,000~50,000 G
월간 지출: 강화에 따라 다름
```

**특성과의 관계:**

```
플레이어 생각:
  "기본 공격을 강하게 하려면 무기를 강화해야 한다"
  → 강화 비용 = 골드 소모
  → 골드를 얻기 위해 던전 반복
  → 게임 플레이 동기 생성
  
특성의 역할:
  강화 → 특성 레벨 증가 → 스탯 증가
  (골드와 특성의 자연스러운 연결)
```

### 다이아 (유료 재화)

**사용처:**

```
1. 가챭 (메인)
   - 장비 가챭: 50 D / 1회
   - 카드 가챭: 50 D / 1회
   - 10회 뽑기: 450 D (10% 할인)

2. 특성과의 연계
   가챭 → 장비 획득 → 강화 가능 → 특성 강화
   가챭 → 카드 획득 → 강화 가능
   
   자연스러운 연결!
```

**유료화 포인트:**

```
① 무기 마스터리를 Lv 10으로 키우고 싶음
   → 같은 무기가 많이 필요
   → 가챭 반복 (다이아 필요)

② 모든 악세서리를 3개 다 갖고 싶음
   → 가챭 반복 (다이아 필요)

③ 모든 장비를 +10으로 강화하고 싶음
   → 같은 장비가 많이 필요
   → 가챭 반복 (다이아 필요)

결론:
  특성 마스터리 목표
  → 가챭 필요
  → 자연스러운 다이아 구매
```

## 5️⃣ 특성 시스템 조작 정리

### 플레이어가 할 수 있는 행동

```
1️⃣ 자동 (플레이어가 모르는 사이에 발생)
   ✅ 장비 강화 → 특성 자동 레벨업
   ✅ 장비 장착 → 특성 자동 활성화
   ✅ 장비 해제 → 특성 자동 비활성화
   ✅ 기본 공격 → 자동 데미지 계산

2️⃣ 반자동 (선택하지만 간단함)
   ✅ 장비 선택/장착
   ✅ 가챭에서 장비 획득
   ✅ 강화 실행

3️⃣ 수동 (플레이어가 명시적으로 선택)
   ✅ 특성 포인트 분배 (ATK/DEF/HP/SPD)
   ✅ 목표 설정 (무기 +10까지 강화하기 등)
```

### 플레이어가 할 수 없는 행동

```
❌ 특성 직접 활성화 (자동)
❌ 마스터리 레벨 수동 조정 (자동)
❌ 특성 보너스 직접 변경 (자동)

→ 단순함 = 게임 이해도 높음
```

## 💡 특성 시스템 운영의 핵심

### "자동이지만 투명함"

```
플레이어는 이해합니다:
  ✅ "강화도 +1 = 특성 Lv +1"
  ✅ "특성 Lv +1 = 보너스 +5%"
  ✅ "보너스 +5% = 공격력 향상"

모든 과정이 명확하고 투명함
→ 플레이어 신뢰 구축
```

### "선택과 결과의 명확한 연결"

```
플레이어 선택:
  1. 어떤 장비를 강화할까?
     (무기 vs 방어구 vs 악세서리)
  
  2. 특성 포인트를 어디에?
     (ATK vs DEF vs HP vs SPD)
  
  3. 가챭은 언제?
     (무료 범위 내 vs 다이아 구매)

즉시 피드백:
  강화 → 특성 Lv 증가 → 스탯 증가 → 데미지 증가
  (매 단계마다 체감)
```

### "자연스러운 유료화"

```
강제하지 않음:
  ✅ 무료로도 진행 가능
  ✅ 시간 투자로 진행 가능

자연스러운 유료화:
  "더 빨리 강해지고 싶다"
  → 가챭 반복 (다이아 필요)
  → 자연스러운 구매 동기
```

### UI 화면 구현

```
[ ] 특성 정보 탭
    - 5가지 마스터리 표시
    - 각 마스터리 레벨 표시
    - 각 마스터리 효과 표시

[ ] 기본 공격 데미지 분석 화면
    - 세부 계산 과정 표시
    - 각 마스터리별 기여도 표시
    - 최종 데미지 표시

[ ] 다음 목표 추천 화면
    - 플레이 스타일별 추천
    - 필요한 강화 목표 표시
    - 예상 비용 표시

[ ] 특성 포인트 분배 화면
    - 4가지 스탯 선택
    - 한 번에 1/2/5pt 추가
    - 실시간 스탯 변화 표시
```

### 로직 구현

```
[ ] 자동 특성 레벨 계산
    - 장비 강화도 → 특성 레벨
    - 악세서리 개수 → 특성 레벨
    - 강화된 장비 개수 → 특성 레벨

[ ] 자동 데미지 계산
    - 기본 공격 = 무기 ATK
    - × 각 마스터리 보너스
    - × 특성 포인트 보너스
    - = 최종 데미지

[ ] 실시간 업데이트
    - 장비 변경 → 즉시 특성 재계산
    - 강화 완료 → 즉시 특성 재계산
    - 포인트 분배 → 즉시 스탯 재계산

[ ] 저장/로드
    - 모든 특성 데이터 직렬화
    - 장비 정보와 동기화
    - 특성 포인트 분배 기록
```
