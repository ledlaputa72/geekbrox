# 🎴 카드 시스템 요약 (Cowork용)

**목적:** 200개 카드 시스템 이해 및 빠른 학습  
**대상:** Cowork, 신규 기획자  
**학습 시간:** 30분  
**마지막 업데이트:** 2026-03-06

---

## 📊 카드 시스템 개요

### 전체 구성

```
200개 카드 (4가지 타입)
├─ ATTACK  (50개) - 빠른 공격
├─ SKILL   (50개) - 보조/방어
├─ POWER   (50개) - 강력한 공격
└─ CURSE   (50개) - 디버프/약화
```

---

## 🎴 1️⃣ 카드 타입별 분석

### **ATTACK (공격) 🔥** - 50개

**특징:**
- 비용: 1-3 Mana
- 쿨다운: 0-1턴
- 효과: 직접 피해 50-200%

**전략:**
- 빠른 속도 + 낮은 비용 = 초반 주력
- 연계 공격 가능
- 높은 회심율

**예시 카드:**
- 기본 공격: 1 Mana, 50% 피해
- 화염 공격: 2 Mana, 100% 피해 + 불 상태
- 연쇄 공격: 2 Mana, 75% × 3회

**대상:** 공격형 빌드, ATTACK 특화 플레이어

---

### **SKILL (스킬) ✨** - 50개

**특징:**
- 비용: 2-4 Mana
- 쿨다운: 1-2턴
- 효과: 버프/디버프/회복

**전략:**
- 전투 지속력 확보
- 아군 강화 / 적 약화
- 생존 극대화

**예시 카드:**
- 방어: 2 Mana, DEF +30% (3턴)
- 회복: 3 Mana, HP 회복 200
- 약화: 2 Mana, 적 ATK -20% (2턴)
- 버프: 3 Mana, 모든 스탯 +15% (2턴)

**대상:** 방어형 빌드, 지능형 빌드

---

### **POWER (파워) ☄️** - 50개

**특징:**
- 비용: 4-6 Mana
- 쿨다운: 3-4턴
- 효과: 강력한 피해 200-400%

**전략:**
- 높은 비용 = 높은 피해
- 쿨다운 관리 중요
- 결정적 순간에 사용

**예시 카드:**
- 강펀치: 4 Mana, 200% 피해
- 메테오: 5 Mana, 300% 광역 피해
- 필살기: 6 Mana, 400% 피해 (회심율 2배)

**대상:** 공격형 빌드, 보스 전투

---

### **CURSE (저주) ☠️** - 50개

**특징:**
- 비용: 2-4 Mana
- 쿨다운: 1-3턴
- 효과: 디버프/중독/약화

**전략:**
- 적의 능력 제한
- 지속 피해
- 적 무력화

**예시 카드:**
- 독: 2 Mana, 중독 (3턴, 매턴 10% 피해)
- 약화: 3 Mana, 적 DEF -30% (2턴)
- 봉인: 4 Mana, 적 특성 1개 봉인 (3턴)

**대상:** 지능형 빌드, 보스 전투

## 🔄 2️⃣ 덱 빌드 패턴 (3가지)

### **패턴 1: 공격형 (Attacker)**

```
카드 구성:
- ATTACK: 60% (30장)
- SKILL: 20% (10장) - 응급 방어
- POWER: 15% (7장) - 강타
- CURSE: 5% (3장) - 기본

특징:
- 초반부터 끝까지 공격
- 높은 DPS
- 낮은 생존력

권장 스탯: STR 높음, DEX 높음
```

### **패턴 2: 방어형 (Tank)**

```
카드 구성:
- SKILL: 50% (25장) - 방어/회복
- ATTACK: 30% (15장) - 반격
- POWER: 10% (5장) - 강력한 반격
- CURSE: 10% (5장) - 약화

특징:
- 오래 버티기
- 천천히 누적 피해
- 높은 생존력

권장 스탯: VIT 높음, DEF 높음
```

### **패턴 3: 균형형 (Balanced)**

```
카드 구성:
- ATTACK: 35% (17장)
- SKILL: 35% (17장)
- POWER: 20% (10장)
- CURSE: 10% (6장)

특징:
- 상황별 대응
- 안정적인 플레이
- 초보자 추천

권장 스탯: 모든 스탯 균형
```

## 🎯 3️⃣ 카드 콤보 시스템

### 기본 콤보 매커니즘

**같은 타입 3장 연속 사용 → 보너스**
```
예시:
1턴: ATTACK 1 사용 (50% 피해)
2턴: ATTACK 2 사용 (50% 피해)
3턴: ATTACK 3 사용 (50% 피해 + 50% 추가) ⭐
합계: 200% 피해 (150% 기본 + 50% 콤보)
```

### 고급 콤보 (크로스 타입)

**ATTACK + SKILL + POWER 순서 → 특수 효과**
```
예시:
- ATTACK: 100% 피해
- SKILL: 적 DEF -30%
- POWER: 200% 피해 (×1.3 배수)
= 260% 피해 (100% + 200%×1.3)
```
