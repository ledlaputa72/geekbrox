# 🎨 Dream Collector — UI 메뉴 재설계 & 장비 탭 추가

**작성:** Atlas (PM)  
**대상:** Game팀 (Cursor IDE)  
**날짜:** 2026-03-04  
**변경 사항:** Progress 탭 → 캐릭터(장비) 탭, 장비 탭 추가

---

## 📌 변경 요약

### 현재 메인 메뉴 (기존)
```
[홈] [던전] [카드] [Progress] [설정]
```

### 새로운 메인 메뉴
```
[홈] [던전] [카드] [장비] [캐릭터] [설정]
```

**변경 사항:**
- ❌ Progress 탭 제거
- ✅ 캐릭터(장비) 탭으로 변경
- ✅ 장비 탭 신규 추가

---

## 🎯 메인 메뉴 구조

### 5가지 메인 탭

#### 1️⃣ 홈 (Home)
- 현재 레벨, 경험치
- 최근 보상
- 빠른 액션 버튼

#### 2️⃣ 던전 (Dungeon)
- 스토리 던전
- 일일 던전
- 무한 던전
- 월간 이벤트

#### 3️⃣ 카드 (Cards)
- 보유 카드 목록
- 카드 정보 (type, rarity, stats)
- 카드 강화
- 카드 정렬/필터

#### 4️⃣ 장비 (Equipment) ⭐ NEW
- 보유 장비 목록
- 장비 강화
- 장비 구분 (4슬롯)
- 강화 재료 관리

#### 5️⃣ 캐릭터 (Character) ✏️ RENAMED
- 현재 레벨/경험치 (기존 Progress)
- 착용 장비 현황
- 스탯 정보
- 마일스톤 진행도

#### 6️⃣ 설정 (Settings)
- 음량
- 그래픽 설정
- 약관

## 🎨 메뉴 탭 디자인

### 메뉴 바 위치
```
화면 하단 고정 메뉴바
- 5개 탭 (또는 스크롤 가능)
- 각 탭 아이콘 + 텍스트
- 현재 활성 탭 강조 표시
```

### 탭 아이콘 가이드
```
🏠 홈         (집 모양)
⚔️ 던전       (검 모양 또는 지도)
🃏 카드       (카드 모양)
🛡️ 장비       (방패 또는 갑옷) ⭐ NEW
👤 캐릭터    (사람 모양, Progress에서 변경)
⚙️ 설정      (톱니바퀴)
```


### 1️⃣ 상단: 착용 중인 장비 (Quick View)

```
착용 중인 장비
┌─────────────────────┐
│ 무기: 강철검 +3     │ [강화]
│ 방어구: 강철갑옷 +2 │ [강화]
│ 악세서리: 공격반지  │ [강화]
│ 보조무기: 단검 +1   │ [강화]
└─────────────────────┘

특징:
- 각 슬롯별로 현재 착용 장비 표시
- 각 장비에 [강화] 버튼
- 클릭하면 그 장비의 강화 화면으로 이동
- 장비 아이콘 표시 (시각적 구분)
```

### 2️⃣ 중단: 보유 장비 목록 (Inventory)

```
필터/정렬:
[전체] [무기] [방어구] [악세서리] [보조무기]
[강화도순] [희귀도순] [이름순]

보유 장비 그리드 또는 리스트:
┌──────────────────────────────────┐
│ 강철검 +3 (착용중)   ATK 23       │
│ 가격: 1,000 G                     │
│ [정보] [강화]                    │
├──────────────────────────────────┤
│ 강철검 +2           ATK 22       │
│ 강화 재료로 사용 가능            │
│ [정보] [강화]                    │
├──────────────────────────────────┤
│ 강철검 +1           ATK 21       │
│ [정보] [강화]                    │
├──────────────────────────────────┤
│ 강철갑옷 +2 (착용중) DEF 17      │
│ [정보] [강화]                    │
└──────────────────────────────────┘

정보 표시:
- 이름, 강화도, 기본 스탯
- 착용 상태 표시
- 강화 재료로 사용 가능 표시
- 실행 버튼: [정보], [강화]
```

### 3️⃣ 장비 상세 정보 화면 (팝업)

```
┌─────────────────────────┐
│  강철검 +3              │
```

## 💡 Cursor 프롬프트 (UI 구현용)

### 프롬프트 1: 메인 메뉴 변경

```
UI_EQUIPMENT_TAB_DESIGN.md를 참조해서 메인 메뉴를 다음과 같이 변경해줘:

현재:
[홈] [던전] [카드] [Progress] [설정]

변경:
[홈] [던전] [카드] [장비] [캐릭터] [설정]

요구사항:
1. Progress 탭 제거
2. 캐릭터 탭 추가 (5번째)
3. 장비 탭 추가 (4번째)
4. 탭 아이콘: 🛡️ 장비, 👤 캐릭터
5. 탭 전환 애니메이션 추가
6. 현재 활성 탭 강조 표시

기술:
- Godot UI (Control nodes)
- 탭 패널 구조
- 신호(Signal) 기반 탭 전환
```

### 프롬프트 2: 장비 탭 UI

```
UI_EQUIPMENT_TAB_DESIGN.md의 "장비 탭" 섹션을 참조해서
장비 탭 UI를 만들어줘:
```
