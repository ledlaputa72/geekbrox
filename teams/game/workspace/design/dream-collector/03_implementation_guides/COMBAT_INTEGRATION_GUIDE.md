**목적:** 캐릭터/카드/아이템이 전투에서 어떻게 연동되는지 완전 이해  
**대상:** Cowork, 신규 기획자

## 🎮 전투 시스템 개요

### ATB (Active Time Battle) 기본 메커닉

```
┌─────────────────────────────────────────┐
│        전투 시작 (ATB 시스템)           │
│                                         │
│  플레이어 │░░░░░░░░░░░│ (ATB 게이지 증가)
│  적1     │████████░░░│ (먼저 행동)
│  적2     │██░░░░░░░░│
│                                         │
│  → 가장 먼저 채워진 캐릭터 행동       │
│  → 카드 선택                           │
│  → 효과 적용                           │
│  → 게이지 초기화                       │
│  → 반복                                │
└─────────────────────────────────────────┘
```

### 전투 흐름

```
1️⃣ 캐릭터 선택
   └─ ATB 게이지 100% 도달 시 차례 획득

2️⃣ 카드 선택
   └─ 덱에서 가능한 카드 선택 (마나 충분 시)

3️⃣ 카드 효과 적용
   └─ 스탯 + 아이템 + 특성 계산

4️⃣ 결과 처리
   └─ 피해/버프/디버프 적용 → 게이지 리셋

5️⃣ 적 차례
   └─ AI가 카드 선택 → 효과 적용

6️⃣ 반복
   └─ 승리 또는 패배까지 반복
```

## 🔗 시스템 연동 상세

### **1단계: 캐릭터 스탯 초기 설정**

```
기본 스탯 (Lv 기반)
├─ STR: 75 (무기 요구 충족)
├─ DEF: 60
├─ VIT: 85
├─ DEX: 68
├─ INT: 70
└─ LUK: 60

활성 특성:
├─ 맹렬함: +10% 공격
├─ 정확함: 회심율 +10%, 회심 데미지 +50%
└─ 무한가능성: 모든 스탯 +2 추가

결과:
├─ STR: 77 (기본 75 + 특성 +2)
├─ DEF: 62
├─ VIT: 87
└─ DEX: 70
```

### **2단계: 아이템 장착**

```
장착 아이템:
├─ 무기: 성검
│  └─ ATK +100, 모든 효과 +30%
├─ 방어구: 용의 갑옷
│  └─ DEF +60, HP +220
├─ 반지 1: 무한의 반지
│  └─ 모든 스탯 +50% (= STR +38.5 → 77+38 = 115)
├─ 반지 2: 회심 반지
│  └─ 회심율 +10%
├─ 목걸이 1: ATTACK 목걸이
│  └─ ATTACK 카드 +35%
└─ 목걸이 2: SKILL 목걸이
   └─ SKILL 카드 +35%

최종 능력치:
├─ STR: 115 (기본 77 + 반지 보너스 +38)
├─ DEF: 92 (기본 62 + 반지 +31 + 방어구 +60)
├─ VIT: 130 (기본 87 + 반지 +43)
├─ 최대 HP: 520 (기본 435 + 방어구 +220 - 2)
├─ ATK: 100 (무기)
└─ 회심율: 20% (특성 10% + 반지 10%)
```

### **3단계: 전투 중 카드 사용**

```
플레이어 차례 도착:
- 현재 마나: 200 (INT 70 → 최대 190 + 특성/아이템 +10)
- 덱에 사용 가능한 카드:
  ├─ ATTACK 카드 (비용 1-3)
  ├─ SKILL 카드 (비용 2-4)
  ├─ POWER 카드 (비용 4-6)
  └─ CURSE 카드 (비용 2-4)

카드 선택: "화염 공격" (ATTACK)
├─ 기본 비용: 2 Mana
├─ 기본 데미지: 100% (= 115 × 100% = 115)
├─ 아이템 보너스: +30% (무기)
├─ 카드 타입 보너스: +35% (목걸이)
└─ 특성 보너스: +10% (맹렬함)

최종 데미지 계산:
└─ 115 × (100% + 30% + 35% + 10%) = 115 × 175% = 201 피해

실제 적용:
├─ 적의 DEF: 40
├─ DEF 감소: 201 × (1 - 40/(40+100)) = 201 × 0.71 = 143 피해
└─ 최종 피해: 143
```

### **4단계: 적의 반격**

```
적의 능력치 (적도 같은 방식으로 계산):
├─ ATK: 50
├─ 기본 공격 (물리):
│  └─ 50 × 100% = 50 피해
└─ 플레이어 DEF: 92
   ├─ 감소율: 92/(92+100) = 0.48 (48% 감소)
   └─ 최종 피해: 50 × 0.52 = 26 피해

플레이어 현재 HP: 520 - 26 = 494
```

### **5단계: 특성/버프 발동**

```
플레이어 특성 발동:
├─ "정확함" (회심율 +10%)
│  └─ 회심 확률: 20% (10% + 10%)
│  └─ 20% 확률로 143 × 1.5 = 214 피해 (회심!)
│
└─ "맹렬함" (공격 +10%)
   └─ 이미 계산에 반영됨

적의 특성:
├─ 없음 (일반 몬스터)
└─ → 특성 보너스 없음

버프/디버프:
├─ 활성 버프: 없음
├─ 활성 디버프: 없음
└─ 다음 턴 유지
```

## 🎯 실제 전투 시나리오

### 시나리오: 일반 몬스터 vs 플레이어

```
┌─────────────────────────────────────────────────┐
│ Turn 1: 플레이어 차례                          │
├─────────────────────────────────────────────────┤
│                                                  │
│ 플레이어 선택: "화염 공격" (ATTACK)             │
│ ├─ 데미지: 143 (또는 214 회심 시)              │
│ └─ 적 HP: 200 → 57 (또는 -14 패배)            │
│                                                  │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│ Turn 2: 적 차례 (생존)                         │
├─────────────────────────────────────────────────┤
│                                                  │
│ 적 선택: "기본 공격"                            │
│ ├─ 데미지: 26                                  │
│ └─ 플레이어 HP: 494 → 468                     │
│                                                  │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│ Turn 3: 플레이어 차례                          │
├─────────────────────────────────────────────────┤
│                                                  │
│ 플레이어 선택: "강펀치" (POWER)                 │
│ ├─ 비용: 4 Mana                               │
│ ├─ 데미지: 115 × 200% × 1.75 × 0.71 = 286   │
│ └─ 적 HP: 57 → -229 (플레이어 승리!)          │
│                                                  │
└─────────────────────────────────────────────────┘

✅ 승리 조건:
- 적 모든 HP 제거
- 보상 획득:
  ├─ 골드: 100~200
  ├─ 경험치: 500~1,000
  └─ 아이템 드롭: 5% 확률 (LUK 영향)
```

## 📊 시스템 간 영향 관계도

### 캐릭터 스탯이 전투에 주는 영향

```
STR (힘)
├─ 기본 ATK 증가 → 카드 데미지 증가
└─ 특정 카드 (POWER) 효율 증가

DEF (방어)
├─ 받는 피해 감소 (방어력)
└─ SKILL 카드 (방어) 효율 증가

VIT (체력)
├─ 최대 HP 증가
└─ 총 생존 시간 증가

DEX (민첩)
├─ ATB 속도 증가 (더 자주 차례)
├─ 회피율 증가
└─ 회심율 증가

INT (지능)
├─ 최대 마나 증가 (더 많은 카드 사용)
└─ 마법 카드 (SKILL/CURSE) 효율 증가

LUK (운)
├─ 아이템 드롭 확률 증가
└─ 특별 이벤트 발동률 증가
```

### 아이템이 전투에 주는 영향

```
무기
├─ ATK 직접 증가
└─ 카드 효과 글로벌 증가 (특정 무기)

방어구
├─ DEF/HP 직접 증가
└─ 상태이상 저항

반지
├─ 스탯 글로벌 증가 (모든 스탯 +50%)
└─ 특수 효과 (회심율, 마나 등)

목걸이
├─ 카드 타입별 효율 증가
└─ 카드 콤보 보너스
```

### 특성이 전투에 주는 영향

```
공격 특성 (맹렬함, 정확함, 분노)
└─ 데미지 증가 → 전투 시간 단축

방어 특성 (강건함, 재생)
└─ 생존율 증가 → 전투 지속력 강화

특수 특성 (마나효율, 기술숙련)
└─ 전략적 깊이 증가
```

## 🎯 전투 밸런싱 원칙

### 1. 파워 커브

```
낮은 레벨 (Lv 1-20)
├─ 약한 적 (HP 100)
├─ 플레이어 공격: 50 피해
└─ 예상 전투 시간: 2 시간

중간 레벨 (Lv 50)
├─ 중간 강도 (HP 500)
├─ 플레이어 공격: 200 피해
└─ 예상 전투 시간: 2.5 시간

높은 레벨 (Lv 100)
├─ 강한 적 (HP 1,000+)
├─ 플레이어 공격: 500+ 피해
└─ 예상 전투 시간: 3 시간
```

### 2. 전략적 깊이

```
초급 플레이어
└─ ATTACK만 반복 → 해결 가능 (효율 50%)

중급 플레이어
└─ ATTACK + SKILL 조합 (효율 75%)

고급 플레이어
└─ 카드 콤보 + 아이템 시너지 (효율 150%)
```

## 📞 Q&A

**Q: 한 턴에 여러 카드를 사용할 수 있나?**  
A: 아니오. 한 캐릭터 차례 = 한 카드만 선택.

**Q: 회피 100%는 불가능한가?**  
A: 네. 회피율 최대 50%, 나머지 50%는 항상 맞음.

**Q: 특성을 전투 중 변경할 수 있나?**  
A: 아니오. 전투 전 설정한 특성만 사용.
