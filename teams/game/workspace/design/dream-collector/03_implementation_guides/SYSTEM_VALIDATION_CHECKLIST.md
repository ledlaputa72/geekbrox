## 🎭 1️⃣ 캐릭터 시스템 검증

### 기본 스탯 검증

- [ ] **6가지 스탯의 역할이 명확한가?**
  - 현재: STR, DEF, VIT, DEX, INT, LUK
  - 평가: ⭐⭐⭐⭐⭐ (명확함)
  - 개선안: _________________

- [ ] **레벨별 성장 곡선이 적절한가?**
  - 현재: Lv 1 (55총점) ~ Lv 100 (700총점)
  - 평가: ⭐⭐⭐⭐ (합리적)
  - 부족: 게임 초중반 (Lv 10-30) 성장 속도가 빠른가?
  - 개선안: _________________

- [ ] **스탯 균형이 맞는가?**
  - 현재: STR 우위 (전투 유리)
  - 평가: ⭐⭐⭐ (약간 불균형)
  - 부족: DEX 과소평가되는가? (회피/회심이 너무 약한가?)
  - 개선안: _________________

### 특성 시스템 검증

- [ ] **20가지 특성이 충분한가?**
  - 현재: 공격 5 + 방어 5 + 특수 5 + 고급 5
  - 평가: ⭐⭐⭐⭐ (충분함)
  - 부족: 조합 다양성은? (같은 특성만 사용하는가?)
  - 개선안: _________________

- [ ] **특성 획득이 명확한가?**
  - 현재: 조건만 설명 (STR ≥ 40 등)
  - 평가: ⭐⭐⭐ (부분적)
  - 부족: 어디서 획득하는가? (레벨업/아이템/이벤트?)
  - 개선안: _________________

- [ ] **특성 시너지가 게임을 흥미롭게 하는가?**
  - 현재: 개별 특성 효과만 설명
  - 평가: ⭐⭐ (약함)
  - 부족: 특성 조합 권장안이 없다
  - 개선안: 
    ```
    예: 맹렬함 + 무한가능성 = 초반 압도
    예: 강건함 + 재생 = 탱커 극대화
    ```

- [ ] **캐릭터 커스터마이제이션이 충분한가?**
  - 현재: 스탯 + 특성 + 아이템 조합
  - 평가: ⭐⭐⭐ (기본적)
  - 부족: 직접 스탯 배분이 없는가? (자동 배분만?)
  - 개선안: _________________

## 🎴 2️⃣ 카드 시스템 검증

### 카드 타입 검증

- [ ] **4가지 타입의 역할이 명확한가?**
  - ATTACK (50개): 빠름, 낮은 비용 ⭐⭐⭐⭐⭐
  - SKILL (50개): 보조, 중간 비용 ⭐⭐⭐⭐
  - POWER (50개): 강함, 높은 비용 ⭐⭐⭐⭐
  - CURSE (50개): 약화, 중간 비용 ⭐⭐⭐
  - 평가: ⭐⭐⭐⭐ (명확함)
  - 부족: CURSE는 실제로 사용되는가? (약할 수도)
  - 개선안: _________________

### 카드 밸런싱 검증

- [ ] **모든 카드가 사용되는가?**
  - 현재: 200개 카드 모두 설계됨
  - 평가: ⭐⭐⭐ (미확정)
  - 부족: 실제 게임에서는 일부 카드만 사용될 가능성
  - 개선안: 
    ```
    1. 약한 카드 버프
    2. 강한 카드 너프
    3. 상황별 카드 활용도 분석
    ```

- [ ] **덱 빌딩 자유도가 충분한가?**
  - 현재: 3가지 패턴 제시
  - 평가: ⭐⭐⭐ (기본적)
  - 부족: 더 다양한 빌드가 가능한가?
  - 개선안: 
    ```
    추가 패턴:
    - 콤보 중심 (같은 타입 3개 연계)
    - 버프 중심 (팀 강화)
    - 디버프 중심 (적 약화)
    ```

- [ ] **카드 콤보 시스템이 충분히 깊이 있는가?**
  - 현재: 같은 타입 3개 + 크로스 타입
  - 평가: ⭐⭐⭐ (기본적)
  - 부족: 더 복잡한 콤보가 필요한가?
  - 개선안: 
    ```
    3연 콤보 보너스 강화
    특정 카드 조합 시너지 추가
    (예: ATTACK + SKILL + POWER = 특수 효과)
    ```
