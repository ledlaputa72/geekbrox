## 📊 1️⃣ 기본 스탯 시스템 (6가지)

### 전체 스탯 구성도

```
┌──────────────────────────────────────────────────┐
│         캐릭터 스탯 (Lv 1~100)                   │
├──────────────────────────────────────────────────┤
│ 공격력 관련        방어력 관련       보조 스탯    │
│ ├─ STR (힘)       ├─ DEF (방어)    ├─ DEX (민첩) │
│ └─ ATK (공격력)   ├─ VIT (체력)    ├─ INT (지능) │
│                  └─ HP (생명력)     └─ LUK (운)   │
└──────────────────────────────────────────────────┘
```

### **1. STR (Strength / 힘) ⚔️**

**역할:** 물리 공격 기본 능력  
**영향도:** 
- 무기 데미지 +5~10% 증가
- 장비 요구 STR 충족
- 특정 카드 효과 상승 (POWER 카드 +15%)

**레벨별 성장:**
```
Lv 1:  10 STR
Lv 10: 35 STR
Lv 50: 75 STR
Lv 100: 120 STR
```

**최적화 대상:**
- 공격 중심 빌드
- 무거운 무기 사용자
- POWER 카드 주력 플레이어

---

### **2. DEF (Defense / 방어) 🛡️**

**역할:** 피해 감소 기본 능력  
**영향도:**
- 받는 피해 -5~15% 감소
- 방어구 효율 +10~20%
- 특정 카드 효과 상승 (SKILL 카드 방어 계산)

**레벨별 성장:**
```
Lv 1:  8 DEF
Lv 10: 28 DEF
Lv 50: 60 DEF
Lv 100: 100 DEF
```

**최적화 대상:**
- 방어 중심 빌드
- 탱커 역할 플레이어
- SKILL 카드 (방어) 주력 플레이어

---

### **3. VIT (Vitality / 체력) ❤️**

**역할:** 체력 기본값 결정  
**영향도:**
- 최대 HP = 기본값 + (VIT × 5)
- 독성 피해 저항 +3% per VIT
- 상태이상 저항력 증가

**레벨별 성장:**
```
Lv 1:  12 VIT  → 60 HP
Lv 10: 40 VIT  → 200 HP
Lv 50: 85 VIT  → 425 HP
Lv 100: 150 VIT → 750 HP
```

**최적화 대상:**
- 오래 버틸 수 있는 빌드
- 장시간 전투 플레이어
- HP 관련 카드 사용자

---

### **4. DEX (Dexterity / 민첩) 🏃**

**역할:** 회피, 회심, 행동 속도  
**영향도:**
- 회피율 +2% per DEX (최대 50%)
- 회심율 +1% per DEX (최대 25%)
- ATB 속도 +0.5% per DEX
- 카드 선택 시간 +0.1초 단축 per 10 DEX

**레벨별 성장:**
```
Lv 1:  9 DEX
Lv 10: 32 DEX
Lv 50: 68 DEX
Lv 100: 110 DEX
```

**최적화 대상:**
- 빠른 플레이 중심 빌드
- ATTACK 카드 주력 플레이어
- 회피 위주 플레이어

---

### **5. INT (Intelligence / 지능) 🧠**

**역할:** 마나(카드 비용) 관리  
**영향도:**
- 최대 Mana = 50 + (INT × 2)
- Mana 회복 속도 +1% per INT
- 카드 효과 강화 +2% per INT (특히 SKILL/CURSE)
- 디버프 저항 +1% per INT

**레벨별 성장:**
```
Lv 1:  8 INT  → 66 Mana
Lv 10: 30 INT → 110 Mana
Lv 50: 70 INT → 190 Mana
Lv 100: 120 INT → 290 Mana
```

**최적화 대상:**
- 마나 소비 많은 카드 플레이 (POWER/CURSE)
- 버프/디버프 중심 플레이어
- SKILL 카드 주력 플레이어

---

### **6. LUK (Luck / 운) 🍀**

**역할:** 아이템 드롭, 골드, 특별 이벤트  
**영향도:**
- 아이템 드롭율 +1% per LUK
- 레어 아이템 드롭율 +0.5% per LUK
- 골드 보상 +1% per LUK
- 특별 이벤트 발동률 증가 (+1% per 10 LUK)

**레벨별 성장:**
```
Lv 1:  8 LUK
Lv 10: 28 LUK
Lv 50: 60 LUK
Lv 100: 100 LUK
```

**최적화 대상:**
- 아이템 수집 중심 플레이어
- 경제 효율 중심 플레이어
- 롱런 게임 플레이어

## 📋 2️⃣ 특성(Traits) 시스템 (20가지)

### 특성 개요

**특성이란:** 캐릭터의 고유한 능력으로 스탯을 보완/강화  
**획득 방법:** 레벨 업, 아이템 장착, 특별 이벤트  
**효과:** 각 특성마다 다른 보너스 제공

### 공격 특성 (5가지)

#### 1. **맹렬함 (Fervor)** 🔥
- 효과: 모든 공격 +10% 데미지
- 조건: STR ≥ 40
- 시너지: POWER 카드, 무거운 무기

#### 2. **정확함 (Precision)** 🎯
- 효과: 회심율 +10%, 회심 데미지 +50%
- 조건: DEX ≥ 50
- 시너지: ATTACK 카드, 빠른 카드

#### 3. **분노 (Wrath)** 💢
- 효과: HP 감소 시 다음 공격 +20% 데미지
- 조건: HP < 50% + STR ≥ 30
- 시너지: 위험 플레이 스타일

#### 4. **회복력 (Resilience)** 🛡️
- 효과: 받는 피해 -8%, 회복 +15%
- 조건: DEF ≥ 45, VIT ≥ 35
- 시너지: 방어 카드, 탱커 빌드

#### 5. **운명의 인도 (Destiny)** ✨
- 효과: 전투 중 특별 이벤트 발동률 +5%
- 조건: LUK ≥ 60
- 시너지: 운에 의존하는 전술

### 방어 특성 (5가지)

#### 6. **강건함 (Fortitude)** 💪
- 효과: 최대 HP +20%, 스탠드 (상태이상 1회 무시)
- 조건: VIT ≥ 50
- 시너지: 탱커 빌드, 장전투

#### 7. **재빠름 (Swift)** ⚡
- 효과: ATB 속도 +15%, 회피율 +10%
- 조건: DEX ≥ 55
- 시너지: ATTACK 빠른 카드 연계

#### 8. **보호막 (Shield)** 🔐
- 효과: 매 턴 시작 시 DEF 10 증가 (최대 5턴)
- 조건: DEF ≥ 50
- 시너지: 방어 카드 중첩

#### 9. **회피 전문 (Evasion)** 🏃‍♂️
- 효과: 회피율 +20% (50% 초과 불가)
- 조건: DEX ≥ 60, VIT ≥ 30
- 시너지: 회피 위주 플레이

#### 10. **재생 (Regeneration)** 💚
- 효과: 매 턴 최대 HP 5% 회복
- 조건: INT ≥ 40, VIT ≥ 45
- 시너지: 장전투, SKILL 카드

### 특수 특성 (5가지)

#### 11. **마나 효율 (Mana Efficiency)** 🔵
- 효과: 모든 카드 Mana 비용 -15%
- 조건: INT ≥ 55
- 시너지: Mana 소비 많은 카드

#### 12. **마법 증폭 (Spell Amplify)** 📈
- 효과: SKILL/CURSE 카드 효과 +20%
- 조건: INT ≥ 50
- 시너지: 마법 카드 위주 플레이

#### 13. **기술 숙련 (Technique Master)** 🎓
- 효과: 카드 콤보 시 추가 효과 +30%
- 조건: DEX ≥ 45, INT ≥ 40
- 시너지: 카드 콤보 플레이

#### 14. **약점 공략 (Weak Point)** 🎯
- 효과: 특정 적에 대해 데미지 +25%
- 조건: STR ≥ 35, DEX ≥ 40
- 시너지: 보스 전투

#### 15. **동료의 신뢰 (Companion's Trust)** 🤝
- 효과: 보조 캐릭터 능력 +15%
- 조건: INT ≥ 30 (멀티플레이 시)
- 시너지: 팀 플레이

### 고급 특성 (5가지)

#### 16. **무한 가능성 (Infinite Potential)** ✨
- 효과: 매 레벨 업마다 스탯 +2 추가 증가
- 조건: Lv 50+ 달성
- 시너지: 장기 플레이, 모든 빌드

#### 17. **결정력 (Determination)** 🔥
- 효과: HP 25% 이하 시 모든 스탯 +15%
- 조건: STR ≥ 50, VIT ≥ 50
- 시너지: 역전 플레이

#### 18. **지혜의 경험 (Wisdom)** 🧙
- 효과: 경험치 획득 +20%, 스탯 습득 +10%
- 조건: INT ≥ 60, LUK ≥ 40
- 시너지: 레벨링 최적화

#### 19. **완벽한 조화 (Perfect Harmony)** ⚖️
- 효과: 모든 스탯 균형 유지 시 각 스탯 +8%
- 조건: 모든 스탯 ± 5 범위 내
- 시너지: 밸런스 빌드

#### 20. **꿈 수집가의 운명 (Dreamer's Fate)** 🌙
- 효과: 꿈 획득량 +30%, 특별 이벤트 발동률 +10%
- 조건: Lv 75+ 달성 + 모든 특성 5개 이상 해금
- 시너지: 게임 최종 목표 달성

### 예시: 강한 빌드

**"무고한 전사" 빌드:**
```
기본 스탯:
- STR 80   (공격 위주)
- DEF 60   (어느정도 방어)
- VIT 70   (체력 확보)
- DEX 40   (약간의 회피)

- 맹렬함 (공격 +10%)
- 강건함 (HP +20%)
- 약점공략 (특정 적 +25%)

결과:
- 높은 물리 피해
- 탱킹 가능
- 특정 적에 강함
```

## 🎯 5️⃣ 캐릭터 빌드 예시

### 빌드 1: 공격형 (Attacker)

```
특징: 높은 피해, 빠른 속도
추천 스탯 (Lv 100):
- STR: 120  (최대)
- DEX: 110  (2위)
- INT: 60   (마나 확보)
- VIT: 80   (최소 생존력)
- DEF: 40   (무시)
- LUK: 50   (기본)

특성: 맹렬함, 정확함, 분노, 무한가능성, 결정력

카드: ATTACK 중심 (빠른 카드)

장비: 
- 무기: 높은 ATK
- 반지: DEX, STR 강화
- 목걸이: ATTACK 카드 +30%
```

### 빌드 2: 방어형 (Tank)

```
특징: 높은 방어, 오래 생존
추천 스탯 (Lv 100):
- VIT: 150  (최대)
- DEF: 100  (최대)
- STR: 70   (반격용)
- INT: 80   (마나)
- DEX: 50   (회피)
- LUK: 50   (기본)

특성: 강건함, 보호막, 재생, 완벽한조화, 무한가능성

카드: SKILL 중심 (방어 카드)

장비:
- 방어구: 높은 DEF
- 반지: VIT, DEF 강화
- 목걸이: SKILL 카드 +30%
```

### 빌드 3: 지능형 (Mage)

```
특징: 마나 효율, 특수 효과
추천 스탯 (Lv 100):
- INT: 120  (최대)
- DEX: 100  (빠른 시전)
- VIT: 90   (최소 생존)
- STR: 50   (기본)

카드: SKILL + CURSE (마법 카드)

장비:
- 무기: INT 보너스
- 반지: INT, DEX 강화
- 목걸이: SKILL + CURSE +30%
```

## 📞 Q&A

**Q: 특성을 여러 개 동시에 가질 수 있나?**  
A: 네. 특성은 누적되며, 최대 10개까지 동시 보유 가능 (Lv 100)

**Q: 특성을 제거하거나 변경할 수 있나?**  
A: 아니오. 특성은 영구적입니다. 대신 새로운 캐릭터를 만들 수 있음.

**Q: 스탯을 수동으로 배분할 수 있나?**  
A: 기본값은 자동 성장. 아이템/특성으로만 추가 가능.

**Q: 특성 조건을 충족하지 못하면?**  
A: 조건을 충족할 때까지 특성이 활성화되지 않음. 자동으로 활성화됨.
