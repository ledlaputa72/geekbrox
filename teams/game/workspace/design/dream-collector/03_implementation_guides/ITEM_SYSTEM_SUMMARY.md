# ⚔️ 아이템 시스템 요약 (Cowork용)

**목적:** 90개 아이템 시스템 이해 및 빠른 학습  
**대상:** Cowork, 신규 기획자  
**학습 시간:** 25분  
**마지막 업데이트:** 2026-03-06

---

## 📊 아이템 시스템 개요

### 전체 구성

```
90개 아이템 (4가지 타입, 5가지 등급)
├─ 무기 (20개)     - ATK 강화
├─ 방어구 (20개)   - DEF/HP 강화
├─ 반지 (25개)     - 스탯 강화
└─ 목걸이 (25개)   - 카드 효율 증가

등급:
├─ Common (33%)
├─ Uncommon (25%)
├─ Rare (22%)
├─ Epic (11%)
└─ Legendary (6%)
```

---

## ⚔️ 1️⃣ 아이템 타입별 분석

### **무기 (Weapon) 🔫** - 20개

**역할:**
- ATK 증가 (기본값 +15~100)
- 특수 스킬 부여 (데미지 유형)

**등급별 예시:**
```
Common (Lv 1-2):
- 나무 검: ATK +15
- 돌 도끼: ATK +17

Uncommon (Lv 3-4):
- 철 검: ATK +25, 회심 +3%
- 화염 검: ATK +28, 불 피해 +5%

Rare (Lv 5-6):
- 용의 도끼: ATK +42, 광역 공격 +10%

Epic (Lv 7-8):
- 검은 검: ATK +60, 적 약화 10%

Legendary (Lv 9-10):
- 성검: ATK +100, 모든 효과 +30%
```

**특징:**
- 가장 높은 ATK 증가량
- 장착 시 바로 효과 발생
- 캐릭터 빌드의 핵심

**대상:** 공격형 빌드

---

### **방어구 (Armor) 🛡️** - 20개

**역할:**
- DEF 증가 (기본값 +13~80)
- HP 증가 (기본값 +45~280)
- 상태이상 저항

**등급별 예시:**
```
Common (Lv 1-2):
- 천 옷: DEF +13, HP +45

Uncommon (Lv 3-4):
- 가죽 갑옷: DEF +24, HP +90

Rare (Lv 5-6):
- 철 갑옷: DEF +38, HP +150

Epic (Lv 7-8):
- 용의 갑옷: DEF +60, HP +220, 불 저항 +10%

Legendary (Lv 9-10):
- 신성한 갑옷: DEF +80, HP +280, 모든 저항 +20%
```

**특징:**
- 가장 높은 HP 증가량
- DEF 증가로 피해 감소
- 탱킹 능력 결정

**대상:** 방어형 빌드

---

### **반지 (Ring) 💍** - 25개

**역할:**
- 캐릭터 스탯 강화 (+20~50%)
- 특수 능력 부여
- 2개 동시 장착 가능

**등급별 예시:**
```
Common (8개):
- 힘의 반지: STR +15%
- 방어 반지: DEF +15%

Uncommon (7개):
- 강한 힘의 반지: STR +30%
- 민첩함의 반지: DEX +30%

Rare (5개):
- 회심 반지: 회심율 +10%
- 마나 반지: INT +30%

Epic (3개):
- 힘과 방어: STR +40%, DEF +40%
- 완벽한 균형: 모든 스탯 +25%

Legendary (2개):
- 무한의 반지: 모든 스탯 +50%
- 영원한 반지: 모든 스탯 +50%, 회복 +30%
```

**특징:**
- 스탯 강화 전문
- 2개 동시 장착으로 시너지
- 빌드 커스터마이제이션의 핵심

**대상:** 모든 빌드 (필수 아이템)

---

### **목걸이 (Necklace) 📿** - 25개

**역할:**
- 카드 효율 증가 (+20~50%)
- 카드 타입별 특화
- 2개 동시 장착 가능

**등급별 예시:**
```
Common (8개):
- ATTACK 목걸이: ATTACK 카드 +20%
- SKILL 목걸이: SKILL 카드 +20%

Uncommon (6개):
- 강화 ATTACK: ATTACK +35%
- 강화 CURSE: CURSE +35%

Rare (7개):
- 완벽한 공격: ATTACK +40%, 회심 +10%
- 마법의 향상: SKILL/CURSE +40%

Epic (3개):
- 카드 극대화: 모든 카드 +50%
- 위대한 강화: 모든 카드 +45%

Legendary (1개):
- 꿈의 목걸이: 모든 카드 +50%, 콤보 보너스 +30%
```

**특징:**
- 카드 효율 증가 전문
- 덱 빌드에 맞춘 선택
- 2개 동시 장착으로 극대화

**대상:** 모든 빌드 (필수 아이템)

## 📊 2️⃣ 아이템 등급별 분석

### 등급 분배

| 등급 | 개수 | 비율 | 획득 난이도 | 가격대 |
|------|------|------|----------|--------|
| Common | 30개 | 33% | 쉬움 | 800-1,100 G |
| Uncommon | 22개 | 25% | 중간 | 3,000-8,000 G |
| Rare | 20개 | 22% | 높음 | 15,000-30,000 G |
| Epic | 10개 | 11% | 매우 높음 | 50,000-150,000 G |
| Legendary | 6개 | 6% | 극한 | 100,000+ G |

### 각 등급의 특징

**Common (초입)**
- 모든 플레이어 접근 가능
- 기본 성능
- 초반 게임 진행 필수

**Uncommon (초중반)**
- 약간의 특수 효과
- 목표 달성용 장비
- 게임의 대부분을 감당

**Rare (중반)**
- 눈에 띄는 능력치
- 특정 빌드 최적화
- 장기 플레이 목표

**Epic (후반)**
- 강력한 능력 + 특수 효과
- 엔드게임 컨텐츠
- 보스 전투 권장

**Legendary (최종)**
- 극한의 성능
- 게임의 최고 목표
- 수집 가치 + 기능성
