# 🔮 타로 카드 시스템 기획 가이드 v2.0

## 1. 핵심 컨셉: "타로 카드로 직조하는 나만의 꿈(런)"

플레이어는 매 런(Run) 시작 시, **3개의 막(Act)에 해당하는 타로 카드를 각각 뽑아** 이번 탐험의 전체적인 구조와 테마를 결정합니다. 이는 매번 새로운 경험과 전략적인 선택을 제공하는 핵심 로그라이크 요소가 됩니다.

- **런 구조**: `1막 (초반)` → `2막 (중반)` → `3막 (후반)`
- **타로 선택**: 각 막이 시작될 때마다, 제시된 2~3장의 타로 카드 중 하나를 선택하여 해당 막의 규칙을 정합니다.

## 2. 타로 조합 매트릭스: 스테이지 생성의 법칙

각 타로 카드는 해당 막의 **'노드 구성'** 과 **'특수 규칙'** 이라는 두 가지 핵심 요소를 변경합니다.

| 아이콘 | 노드 이름 | 설명 |
| :--- | :--- | :--- |
| ⚔️ | **일반 전투 (Combat)** | 일반 몬스터와 전투. 카드와 소량의 재화 획득. |
| 👑 | **엘리트 전투 (Elite)** | 강력한 몬스터와 전투. 희귀 카드 및 유물 획득. |
| ❓ | **미지 (Unknown)** | 무작위 이벤트 발생. 긍정/부정/중립적 결과. |
| 💰 | **상점 (Merchant)** | 재화를 사용하여 카드, 유물, 소모품 구매 및 카드 제거. |
| 🔥 | **휴식 (Rest)** | HP 회복 또는 카드 강화 중 하나를 선택. |
| ボス | **보스 (Boss)** | 막의 마지막에 등장하는 강력한 적. 클리어 시 막대한 보상. |

## 3. 레벨 디자인 가이드: 난이도 곡선 설계

플레이어가 선택한 3장(1~3막)의 타로 카드 조합이 해당 런의 난이도와 경험을 결정합니다.

### ① 막(Act)별 타로 카드 풀 설계

- **1막 카드 풀 (초반)**: 성장의 기반을 다지는 카드 위주
  - 예: `광대`, `마법사`, `여황제`(최대 HP 증가), `황제`(재화 획득 증가)
  - **디자인 의도**: 플레이어가 덱의 방향성을 잡고 안전하게 성장할 수 있도록 지원.
- **2막 카드 풀 (중반)**: 본격적인 도전과 컨셉 심화
  - 예: `전차`, `힘`, `정의`(밸런스), `악마`
  - **디자인 의도**: 플레이어가 구축한 덱의 강점을 시험하고, 위험을 감수하여 더 큰 보상을 얻을 기회 제공.
- **3막 카드 풀 (후반)**: 런의 컨셉을 극대화하는 파격적인 규칙
  - 예: `탑`, `별`(강화 효율 증가), `달`(미지 이벤트 강화), `심판`(버린 카드 활용)
  - **디자인 의도**: 최종 보스를 향한 마지막 관문. 런의 재미와 도전을 극한으로 끌어올림.

### ② 난이도 조절 메커니즘

- **저위험 조합 (Easy)**: `은둔자(1막)` + `절제(2막)` + `별(3막)`
  - **경험**: 휴식과 강화 위주로 안전하게 플레이. 전투 비중이 낮아 덱 압축과 강화에 집중.
- **고위험 조합 (Hard)**: `전차(1막)` + `악마(2막)` + `탑(3막)`
  - **경험**: 엘리트 전투와 저주가 난무하는 극한의 전투 중심 플레이. 휴식과 상점이 없어 매 선택이 치명적.
- **예측불가 조합 (Random)**: `광대(1막)` + `운명의 수레바퀴(2막)` + `달(3막)`
  - **경험**: 미지 이벤트와 무작위 효과가 가득한 혼돈의 런. 운과 적응력이 중요.

### ③ 스테이지 생성 알고리즘 (개발팀 가이드)

1.  **막(Act) 시작**: 해당 막의 '타로 카드 풀'에서 무작위 카드 3장을 플레이어에게 제시.
2.  **카드 선택**: 플레이어가 1장을 선택.
3.  **노드 생성**:
    - 기본 노드 배치 규칙(예: 15층, 분기 2~3개)을 먼저 생성.
    - 선택된 타로 카드의 **'노드 비중 변화'** 효과를 적용하여 특정 노드를 증감/변형. (예: `전차` 선택 시, 일반 전투 노드 3개를 엘리트 전투로 변경)
4.  **규칙 적용**:
    - 선택된 타로 카드의 **'특수 규칙'** 을 이번 막에 적용되는 전역 버프/디버프로 추가.
