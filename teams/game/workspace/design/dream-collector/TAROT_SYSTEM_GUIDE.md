# 🔮 타로 카드 시스템 기획 가이드 v2.1

## 1. 핵심 철학: "3개의 층(Layer)으로 직조하는 예측 불가능한 꿈"

타로 카드를 3개의 층위로 나누어, 각각 런(Run)의 다른 측면에 영향을 주도록 설계합니다. 이 시스템은 78종의 타로 카드 덱 전체를 활용하여 거의 무한한 리플레이 가치를 제공합니다.

1.  **제1층: 메이저 아르카나 (Major Arcana - 22종)**: 막(Act)의 **'테마'와 '큰 규칙'** 을 결정합니다.
2.  **제2층: 코트 카드 (Court Cards - 16종)**: 플레이어에게 해당 막 동안 지속되는 **'역할'과 '지속 능력'** 을 부여합니다.
3.  **제3층: 핍 카드 (Pip Cards - 40종)**: 개별 노드, 특히 `미지(❓)` 노드에 **'작은 변수'와 '소소한 이벤트'** 를 추가합니다.

---

## 2. 플레이어 경험의 흐름

1.  **막(Act) 시작 시**:
    -   제시된 **메이저 아르카나** 3장 중 1장을 선택 → 이번 막의 **테마** 결정 (예: `VII. 전차` 선택, 엘리트 전투 비중 증가)
    -   제시된 **코트 카드** 2장 중 1장을 선택 → 이번 막의 **지속 능력** 획득 (예: `검의 왕` 선택, 모든 전투 시작 시 힘+1)
2.  **맵 생성**:
    -   선택된 메이저 아르카나의 '노드 비중' 규칙에 따라 맵의 전체 구조가 생성됩니다.
3.  **맵 탐험 중**:
    -   `❓ 미지` 노드에 들어갈 때, **핍 카드** 덱에서 1장을 뽑아 해당 노드의 이벤트를 결정합니다. (예: `컵 에이스`를 뽑아 체력을 대량 회복)

## 3. 확장된 타로 조합 매트릭스 (78종)

### 👑 제1층: 메이저 아르카나 (22종) - 막(Act)의 테마 결정

막의 큰 흐름과 구조를 결정합니다. 각 카드는 노드 구성과 막 전체에 영향을 미치는 특수 규칙을 정의합니다.

| 타로 카드 (예시) | **노드 비중 변화** | **특수 규칙** |
| :--- | :--- | :--- |
| **VII. 전차** | `👑 엘리트` +50%, `🔥 휴식` -30% | 엘리트 승리 시 최대 HP +3 |
| **XV. 악마** | `👑 엘리트`가 '저주받은 엘리트'로 변경 | 엘리트 보상 2배, 승리 시 저주 카드 획득 |
| **...(20종 추가)** | ... | ... |

### 🎭 제2층: 코트 카드 (16종) - 막(Act)의 지속 능력 부여

4개의 슈트(Suit)가 능력의 종류를, 4개의 계급이 능력의 방향성을 결정합니다.

| 슈트 | 테마 (게임 적용) |
| :--- | :--- |
| **지팡이 (Wands)** | **공격과 행동**: 힘, 에너지, 공격 카드 강화 등 |
| **컵 (Cups)** | **회복과 방어**: HP 회복, 방어도, 상태이상 정화 등 |
| **검 (Swords)** | **도전과 시련**: 고위험 고수익, 엘리트/보스 강화 및 보상 증가 |
| **펜타클 (Pentacles)** | **재화와 성장**: 골드, 상점, 카드 강화 등 |

| 계급 | 역할 | **지팡이(Wands)** 예시 | **펜타클(Pentacles)** 예시 |
| :--- | :--- | :--- | :--- |
| **왕 (King)** | **지배 (강력한 지속 효과)** | 모든 전투 시작 시 `힘` +1 | 모든 전투 승리 시 골드 +10 |
| **여왕 (Queen)**| **풍요 (자원 생성)** | 3턴마다 무작위 공격 카드 1장 생성 | 상점에서 모든 아이템 가격 20% 할인 |
| **기사 (Knight)**| **추진 (조건부 보상)** | `엘리트`를 처치하면 무작위 공격 카드 2장 발견 | 골드를 200 이상 모으면 최대 에너지 +1 |
| **시종 (Page)** | **시작 (초기 보너스)** | 이 막을 `에너지` +1 상태로 시작 | 이 막을 골드 +100 상태로 시작 |

### 🎴 제3층: 핍 카드 (40종) - `❓ 미지` 노드의 이벤트 결정

슈트 테마는 코트 카드와 동일하며, 숫자는 이벤트의 강도와 내용을 결정합니다.

| 숫자 | 의미 | **컵(Cups)** 예시 | **검(Swords)** 예시 |
| :--- | :--- | :--- | :--- |
| **Ace (1)** | **정수 (가장 강력한 순수 효과)** | 최대 HP의 50% 회복 | 즉시 `엘리트`와 전투 (승리 시 유물 획득) |
| **2 ~ 5** | **소소한 긍정/부정** | HP 약간 회복 / 카드 1장 강화 | 소량의 HP 잃음 / 덱에 '상처' 카드 추가 |
| **6 ~ 8** | **선택과 교환** | "HP 20% 잃고 유물 얻기" vs "그냥 지나가기"| "모든 골드 잃고 희귀 카드 얻기" vs "그냥 지나가기"|
| **9 ~ 10**| **극적인 시련/결과** | "덱의 카드 절반 소멸, 최대 HP +20" | "최대 HP 절반 감소, 유물 3개 획득" |

## 4. 레벨 디자인: 무한한 조합과 난이도 곡선

이 3중 레이어 시스템은 거의 무한에 가까운 조합을 만들어냅니다.
- **조합의 수**: `(메이저 22종) x (코트 16종) x (핍 40종 이벤트)` → **수천 가지 이상의 런 경험**

### 레벨 디자인 예시
- **"광전사의 길" 조합**:
  - **메이저**: `VII. 전차` (엘리트 증가)
  - **코트**: `지팡이의 왕` (전투 시작 시 힘+1)
  - **결과**: 전투가 매우 잦고 강력한 적이 등장하지만, 플레이어 역시 초반부터 강력한 힘 버프로 맞설 수 있는 하이리스크 하이리턴 런.

- **"부유한 상인" 조합**:
  - **메이저**: `VI. 연인` (상점 2개 등장)
  - **코트**: `펜타클의 여왕` (상점 20% 할인)
  - **결과**: 상점을 자주 들러 덱을 최적화하는 운영 중심의 런. 전투보다는 골드 수급과 전략적 구매가 중요.

## 5. 다음 단계

1.  **콘텐츠 전체 설계**: 78종의 모든 타로 카드에 대한 구체적인 `노드 비중`, `특수 규칙`, `지속 능력`, `이벤트 내용`을 담은 **마스터 스프레드시트**를 작성합니다.
2.  **시스템 구현**: 개발팀은 '메이저/코트/핍' 3중 구조에 따라 맵과 이벤트를 생성하는 로직을 구현합니다.
3.  **밸런싱**: 각 카드의 효과와 조합 결과를 지속적으로 테스트하며 난이도 곡선을 조정합니다.
