## 1.1 전투 시스템

### 1.1.1 턴 기반 전투 플로우

**전투 구조:**
```
┌──────────────────────────────────────┐
│ 전투 초기화                          │
│ - 데이터베이스에서 적 스탯 로드      │
│ - 런 상태에서 플레이어 HP/Energy 설정│
│ - 4장 드로우 (시작 핸드)             │
└──────────────────────────────────────┘
           ↓
┌──────────────────────────────────────┐
│ 플레이어 턴                          │
│ 1. 드로우 단계: 카드 1장 뽑기        │
│ 2. 행동 단계: 카드 플레이            │
│    - Energy 소모하여 카드 플레이     │
│    - 카드 효과 해결                  │
│    - HP/Energy 상태 업데이트         │
│ 3. 종료 단계: 핸드 제한 확인 (5장)   │
└──────────────────────────────────────┘
           ↓
┌──────────────────────────────────────┐
│ 적 턴                                │
│ 1. AI 패턴 실행                      │
│ 2. 공격 데미지 적용                  │
│ 3. 특수 능력 확인                    │
│ 4. 상태 효과 발동                    │
└──────────────────────────────────────┘
           ↓
┌──────────────────────────────────────┐
│ 승리/패배 조건 확인                  │
│ - 적 HP ≤ 0? → 승리                  │
│ - 플레이어 HP ≤ 0? → 패배            │
│ - 그 외: 플레이어 턴으로 루프        │
└──────────────────────────────────────┘
```

### 1.1.2 데미지 계산

**공격 공식:**
```
기본 데미지 = 카드 공격력
보정치 = 시너지 보너스 + 상태 효과
최종 데미지 = 기본 데미지 × (1 + 보정치)

예시:
- 카드: [Lucid Strike] = 10 데미지
- 시너지: [Fear Essence]로 +20%
- 상태: 플레이어 버프 +10%
- 최종 = 10 × (1 + 0.2 + 0.1) = 13 데미지
```

**방어 공식:**
```
방어된 데미지 = MIN(방어력, 받는 데미지)
실제 피해 = MAX(0, 받는 데미지 - 방어력)

예시:
- 적 공격: 8 데미지
- 플레이어 방어: 5 ([Dream Shield])
- 실제 피해 = MAX(0, 8 - 5) = 3 HP 손실
```

**크리티컬 히트 (선택 사항 - 고급 기능):**
```
크리티컬 확률 = 5% 기본 + 카드 보정치
크리티컬 배율 = 1.5×

예시:
- 공격력: 10 데미지
- 5% 확률 → 15 데미지 (1.5× 크리티컬)
```

### 1.1.4 상태 효과 시스템

**상태 효과 종류:**

| 효과 | 지속 | 메커닉 |
|------|------|--------|
| **취약 (Vulnerable)** | 1-3턴 | 받는 데미지 +50% |
| **힘 (Strength)** | 1-3턴 | 주는 데미지 +3 |
| **약화 (Weak)** | 1-3턴 | 주는 데미지 -3 |
| **재생 (Regeneration)** | 2-4턴 | 턴당 +2 HP |

| **기절 (Stunned)** | 1턴 | 턴 생략 (행동 불가) |

**상태 구현:**
```python
class StatusEffect:
    def __init__(self, type, duration, potency):
        self.type = type
        self.duration = duration
        self.potency = potency
    
    def apply(self, entity):
        if self.type == "vulnerable":
            entity.damage_taken_multiplier += 0.5
        elif self.type == "strength":
            entity.damage_dealt_bonus += 3
        # ... 등등
    
    def tick(self):
        self.duration -= 1
        if self.duration <= 0:
            self.remove()
```

**덱 제약조건:**
```yaml
최소 카드: 8장
최대 카드: 12장
중복 제한: 카드당 최대 3장
시작 덱: 8장 사전 선택 (튜토리얼)
```

**덱 구성 규칙:**
```python
class Deck:
    def __init__(self):
        self.cards = []
        self.max_size = 12
        self.min_size = 8
        self.duplicate_limit = 3
    
    def can_add_card(self, card_id):
        if len(self.cards) >= self.max_size:
            return False, "덱이 가득 찼습니다"
        
        count = self.cards.count(card_id)
        if count >= self.duplicate_limit:
            return False, f"최대 {self.duplicate_limit}장까지 가능"
        
        return True, "OK"
    
    def add_card(self, card_id):
        success, msg = self.can_add_card(card_id)
        if success:
            self.cards.append(card_id)
        return success, msg
```

```python
def calculate_synergy_bonus(deck, active_cards):
    bonus_multiplier = 1.0
    
    # Nightmare 시너지 확인
    nightmare_count = sum(1 for c in active_cards if c.has_tag("nightmare"))
    if nightmare_count >= 2:
        bonus_multiplier += 0.2  # 2장 이상: +20%
    if nightmare_count >= 3:
        bonus_multiplier += 0.3  # 3장 이상: +30%
    
    # 활성화 카드 확인 (예: Fear Essence)
    if "fear_essence" in active_cards:
        nightmare_energy_discount = 1  # 모든 Nightmare 카드 비용 -1
    
    return bonus_multiplier, nightmare_energy_discount
```

### 1.3.1 런 구조

**단일 런 플로우:**
```
1. 드리머 선택 (난이도: 쉬움/보통/어려움)

3. 런 시작 (10개 노드)
4. 노드 탐색:
   - Memory 노드 (60%): Reveries 수집
   - Event 노드 (20%): 선택 기반 보상
   - Combat 노드 (15%): 적과 전투
   - Boss 노드 (5%): 최종 보스전
5. 런 완료:
   - 승리: 전체 보상
   - 패배: 50% 보상
6. 로비 복귀: Reveries로 업그레이드 구매
```

**노드 생성 알고리즘:**
```python
def generate_run_path(length=10):
    nodes = []
    
    # 첫 노드는 항상 Memory (안전한 시작)
    nodes.append("memory")
    
    # 중간 노드 생성 (노드 2-9)
    for i in range(1, length - 1):
        roll = random.uniform(0, 100)
        
        if roll < 60:
            nodes.append("memory")
        elif roll < 80:
            nodes.append("event")
        else:
            nodes.append("combat")
    
    # 마지막 노드는 항상 Boss
    nodes.append("boss")
    
    # 보스 전 최소 2개 전투 보장
    combat_count = nodes.count("combat")
    if combat_count < 2:
        for i in range(2 - combat_count):
            idx = random.randint(5, 8)
            nodes[idx] = "combat"
    
    return nodes
```

```python
def calculate_offline_rewards(time_away_seconds):
    # 8시간(28800초) 제한
    capped_time = min(time_away_seconds, 28800)
    
    # 기본 속도: 플레이어 덱 아이들 파워
    idle_power = calculate_deck_idle_power(player.deck)
    
    # 프레스티지 보너스
    idle_multiplier = 1.0 + (player.prestige_idle_bonus / 100)
    
    # Reveries 계산
    hours_away = capped_time / 3600
    reveries_earned = idle_power * 10 * hours_away * idle_multiplier
    
    return {
        "reveries": int(reveries_earned),
        "time_capped": time_away_seconds > 28800,
        "hours_calculated": min(hours_away, 8)
    }
```

**덱 아이들 파워 계산:**
```python
def calculate_deck_idle_power(deck):
    power = 0
    
    for card in deck:
        if card.has_tag("collection"):
            power += 5  # Collection 카드 +5 파워
        if card.has_tag("synergy"):
            power += 3  # Synergy 카드 +3 파워
        
        # 업그레이드된 카드는 더 강력
        power += card.level * 0.5
    
    return power
```

**아이들 계산 예시:**
```

- 4개 Collection 카드 (4 × 5 = 20 파워)
- 2개 Synergy 카드 (2 × 3 = 6 파워)
- 평균 카드 레벨: 3 (+1.5 파워 × 12장 = 18 파워)

총 아이들 파워: 20 + 6 + 18 = 44

6시간 오프라인:
Reveries = 44 × 10 × 6 × 1.5 = 3,960 Reveries
```

### 1.5.2 자동 전투 시스템

**자동 전투 AI:**
```python
class AutoBattleAI:
    def __init__(self, deck, player_stats):
        self.deck = deck
        self.player = player_stats
        self.strategy = self.determine_strategy()
    
    def determine_strategy(self):
        attack_cards = [c for c in self.deck if c.type == "attack"]
        defense_cards = [c for c in self.deck if c.type == "defense"]
        
        if len(attack_cards) > 6:
            return "aggressive"
        elif len(defense_cards) > 4:
            return "defensive"
        else:
            return "balanced"
    
    def play_turn(self, hand, enemy):
        if self.strategy == "aggressive":
            # 가장 높은 데미지 카드 우선 플레이
            playable = [c for c in hand if c.cost <= self.player.energy]
            playable.sort(key=lambda c: c.damage, reverse=True)
            return playable[:2]  # 상위 2개 데미지 카드 플레이
        
        elif self.strategy == "defensive":
            # 적 공격력 > 5이면 방어 플레이
            if enemy.attack > 5:
                defense = [c for c in hand if c.type == "defense"]
                return defense[:1]
            else:
                attack = [c for c in hand if c.type == "attack"]
                return attack[:1]
        
        # 밸런스: 공격/방어 교대
        return self.balanced_choice(hand)
```

```python
def generate_dungeon_path():
    nodes = []

    nodes.append(Node("memory", reveries=10))
    
    # 노드 2-9: 절차적 생성
    for i in range(2, 10):
        roll = random.uniform(0, 100)
        
        if roll < 60:
            nodes.append(Node("memory", reveries=random.randint(10, 15)))
        elif roll < 80:
            nodes.append(generate_event_node())
        elif roll < 95:
            nodes.append(generate_combat_node(difficulty=i))
        else:
            nodes.append(generate_treasure_node())

    nodes.append(generate_boss_node())
    
    # 품질 검사: 보스 전 최소 2개 전투 보장
```

```python
import hashlib
import json

def save_with_checksum(data):
    """저장 손상 방지"""
    json_str = json.dumps(data, sort_keys=True)
    checksum = hashlib.sha256(json_str.encode()).hexdigest()
    
    save_file = {
        "data": data,
        "checksum": checksum,
        "timestamp": time.time()
    }
    
    with open("save.json", "w") as f:
        json.dump(save_file, f, indent=2)

def load_with_verification():
    """로드 시 저장 무결성 검증"""
    with open("save.json", "r") as f:
        save_file = json.load(f)
    
    data = save_file["data"]
    expected_checksum = save_file["checksum"]
    
    json_str = json.dumps(data, sort_keys=True)
    actual_checksum = hashlib.sha256(json_str.encode()).hexdigest()
    
    if actual_checksum != expected_checksum:
        raise SaveCorruptedError("저장 파일 손상됨, 백업 로드 중...")
    
    return data
```

```python
        return (time.time() - self.last_sync) > self.sync_interval
    
    def sync_to_cloud(self, save_data):
        """클라우드에 저장 업로드"""
        if not self.should_sync():
            return
        
        # 저장 데이터 압축
        compressed = gzip.compress(json.dumps(save_data).encode())
        
        # 백엔드에 업로드
        if self.backend == "firebase":
            firebase.upload(user_id, compressed)
        
        self.last_sync = time.time()
    
    def sync_from_cloud(self):
        """클라우드에서 저장 다운로드"""
        if self.backend == "firebase":
            compressed = firebase.download(user_id)
        
        # 압축 해제
        json_str = gzip.decompress(compressed).decode()
        return json.loads(json_str)
    
    def resolve_conflict(self, local_save, cloud_save):
        """저장 충돌 처리"""
        # 최신 저장 사용
        if local_save["last_played"] > cloud_save["last_played"]:
            return local_save
        else:
            return cloud_save
```

**충돌 해결:**
```yaml
시나리오 1: 클라우드가 로컬보다 최신
  → 클라우드 저장 사용, 로컬 덮어쓰기

시나리오 2: 로컬이 클라우드보다 최신
  → 로컬 저장 사용, 클라우드에 업로드

시나리오 3: 둘 다 수정됨 (충돌)
  → 타임스탬프 비교
  → 최신 저장 사용
  → 수동 병합 제공 (고급)
```
