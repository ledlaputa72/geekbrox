# 꿈 수집가 (Dream Collector)
## 게임 기획 문서 (Game Design Document) v2.0 - 2D 모바일 방치형 에디션

**문서 작성일:** 2026년 2월 20일  
**문서 버전:** 2.0 (주요 개정)  
**장르:** 방치형 / 증분형 + 로그라이크 + 덱빌딩  
**플랫폼:** 모바일 (iOS/Android) 주력, PC 보조  
**타겟 유저층:** 12세 이상, 방치형 게임 및 분위기 있는 경험을 선호하는 유저  
**세션 길이:** 1-5분 (능동 플레이), 연속 (방치형 진행)

## 하이 컨셉

**한 줄 요약:** *잠자는 동안에도 꿈 수집 제국을 건설하세요—말 그대로.*

꿈 수집가는 잠자는 영혼들로부터 꿈을 자동으로 수집하는 **2D 모바일 방치형 게임**입니다. 당신이 게임을 하지 않는 동안에도 꿈 수집가는 끊임없이 레버리(꿈 조각)를 모으며, 이는 새로운 능력, 카드, 그리고 더 깊은 꿈의 세계를 해금합니다.

**핵심 기둥:**
- **방치형 자동화:** 시간이 지남에 따라 꿈이 스스로 수집됩니다
- **전략적 덱빌딩:** 꿈 카드 덱을 완벽하게 구성하여 수집을 최적화하세요
- **로그라이크 런:** 각 몽상가는 무작위 이벤트를 포함하는 독특한 런입니다
- **프레스티지 루프:** 승천하여 강력한 메타 진행을 잠금 해제하세요
- **몽환적인 분위기:** 편안하고 명상적인 꿈의 미학

**장르 융합:**
- **방치형/증분형** (Adventure Capitalist, Egg Inc.)
- **덱빌딩** (Slay the Spire, Balatro)
- **로그라이크** (Hades, Dead Cells)
- **수집** (Pokémon, Genshin Impact)

## 핵심 게임플레이 루프

### 1차 루프 (능동 플레이: 1-5분)

```
1. 앱 실행 → 축적된 오프라인 진행 상황 확인
   ↓
2. 보상 수집 (레버리, 화폐, 카드)
   ↓
3. 다음 몽상가 선택 (로그라이크 런)
   ↓
4. 덱 구성/조정 (8-12개의 꿈 카드 선택)
   ↓
5. 꿈 런 시작 (자동화)
   ↓
6. 전략적 선택 (카드 플레이, 분기 경로)
   ↓
7. 런 완료 → 보상 획득
   ↓
8. 카드/능력 업그레이드
   ↓
9. 앱 종료 → 방치형 수집 지속
```

### 방치형 루프 (수동: 앱 종료 중)

```
꿈 수집가 자동 탐험 → 조우 자동 진행 →
레버리 축적 → 오프라인 보상 8시간 상한 →
플레이어 복귀 시 수집
```

### 메타 루프 (장기: 수일/수주)

```
레버리 수집 → 꿈 세트 완성 → 새로운 몽상가 해금 →
프레스티지 (승천) → 영구 보너스 획득 → 더 깊은 콘텐츠 →
스토리 챕터 발견
```

## 방치형 메카닉

### 1. 오프라인 진행

**작동 방식:**
- **기본 속도:** 덱 파워에 따라 시간당 X개의 레버리 수집
- **상한:** 8시간 동안의 오프라인 보상 (무한 축적 방지)
- **알림:** 상한 도달 시 푸시 알림
- **수집:** "수집"을 탭하여 축적된 모든 보상 획득

**방치형 속도 수정자:**
- **카드 시너지:** 특정 카드 조합은 방치형 속도를 증가시킵니다
- **프레스티지 보너스:** 승천을 통해 얻는 영구적인 배율
- **능동 보너스:** 광고 시청 또는 IAP를 통한 일시적 부스트

### 2. 자동화 시스템

**자동 수집:**
- 꿈 수집가는 꿈의 세계를 자동으로 탐색합니다
- 방치 시간 동안 플레이어의 입력은 필요하지 않습니다
- 인벤토리가 가득 차면 기록소로 돌아옵니다

**자동 전투 (선택 사항):**
- 악몽 조우는 자동으로 해결됩니다
- 플레이어가 설정한 덱의 AI 전략을 사용합니다
- 수동 플레이를 위해 토글 오프 가능

**자동 업그레이드:**
- 화폐를 자동으로 사용하여 업그레이드하는 토글
- 플레이어가 설정한 우선순위 (카드 > 능력 > 몽상가)를 따릅니다

### 3. 방치형 최적화

플레이어는 다음을 통해 방치형 수익을 최적화합니다:
- **덱 구성:** "방치형 부스트" 태그가 있는 카드
- **몽상가 선택:** 일부 몽상가는 더 높은 방치형 속도를 가집니다
- **업그레이드:** 수집 속도에 대한 영구 업그레이드
- **프레스티지:** 각 승천은 기본 방치형 속도를 증가시킵니다

## 카드 시스템 (덱빌딩)

### 카드 기반 게임플레이

직접적인 캐릭터 능력 대신, 플레이어는 수집 전략을 정의하는 **꿈 카드 덱**을 구성합니다.

### 카드 종류

#### 1. **수집 카드** (수동 소득)
레버리 수집 속도를 증가시킵니다.

**예시:**
```
[기억 파편]
타입: 수집
희귀도: 일반
효과: 레버리 +5/시간
업그레이드: 레벨당 +2
```

#### 2. **액션 카드** (능동 이벤트)
조우/선택 중에 사용됩니다.

**예시:**
```
[자각몽]
타입: 액션
비용: 에너지 2
효과: 즉시 레버리 50 획득
재사용 대기시간: 5분
```

#### 3. **시너지 카드** (콤보)
특정 카드와 함께 플레이될 때 보너스 효과.

**예시:**
```
[악몽 방어막]
타입: 시너지
효과: [공포 꿈]과 조합 시 수집 +20%
보너스: 덱에 악몽 카드 3개 이상 시 효과 두 배
```

#### 4. **이벤트 카드** (로그라이크 무작위성)
런 중에 뽑히며, 선택지를 제공합니다.

**예시:**
```
[갈림길]
타입: 이벤트
선택 A: 레버리 100 획득 (안전)
선택 B: 50% 확률로 레버리 500 획득, 50% 확률로 50 손실 (위험)
```

### 덱 구성 규칙

- **덱 크기:** 8-12장 (업그레이드를 통해 확장)
- **카드 제한:** 카드당 최대 3장
- **희귀도 균형:** 높은 희귀도 카드는 덱 비용이 있습니다 (밸런싱)
- **시너지 집중:** 테마별 덱 구성 장려 (모든 악몽, 모든 기억 등)

### 카드 획득

- **드롭:** 런 완료를 통해 카드 획득
- **상점:** 화폐로 특정 카드 구매
- **제작:** 중복 카드 조합으로 업그레이드
- **가챠 (선택 사항):** IAP를 위한 카드 팩 (필수 아님)

## 로그라이크 요소

### 런 구조

각 몽상가는 **로그라이크 런**입니다:

1. **몽상가 선택** (난이도/테마)
2. **덱 구성** (8-12장)
3. **꿈의 세계 진입** (자동 탐험)
4. **이벤트 조우** (무작위)
5. **런 완료** (성공 = 보상, 실패 = 부분 보상)

### 절차적 이벤트

각 런 동안 무작위 이벤트를 조우합니다:

| 이벤트 타입 | 빈도 | 효과 |
|------------|-----------|--------|
| 기억 노드 | 60% | 레버리 수집 (안전) |
| 선택 이벤트 | 20% | 2-3가지 옵션 중 선택 (위험/보상) |
| 악몽 전투 | 15% | 액션 카드를 사용하여 전투 |
| 보물 | 5% | 보너스 희귀 카드/화폐 |

### 부분적인 영구 사망 (Permadeath Lite)

- **런 실패:** 런별 보상 손실, 레버리의 50% 유지
- **영구 진행:** 카드 업그레이드 및 프레스티지 보너스는 유지
- **페널티 없음:** 실패는 학습 및 실험의 일부입니다

## 진행 시스템

### 1. 카드 업그레이드

- **레벨업:** 레버리를 사용하여 카드 레벨업 (효과 증가)
- **진화:** 희귀 카드는 최대 레벨에서 에픽으로 진화 가능
- **각성:** 새로운 효과를 추가하는 특별 업그레이드

### 2. 프레스티지 시스템 (승천)

**프레스티지 시기:**
- 10,000 레버리 수집 후 해금
- 대부분의 진행 상황이 리셋되지만 영구 보너스 부여

**프레스티지 보너스:**
- **드림 샤드:** 영구 화폐, 다음 용도로 사용:
  - 방치형 속도 +10% (중첩 가능)
  - 새로운 몽상가 해금
  - 카드 슬롯 해금
  - 자동 기능 해금

**프레스티지 단계:**
- 1단계: 방치형 +10%, 몽상가 2명 해금
- 2단계: 방치형 +25%, 희귀 카드 해금
- 3단계: 방치형 +50%, 악몽 모드 해금
- ...
- 10단계: 방치형 +500%, 비밀 엔딩 해금

### 3. 꿈의 기록 (수집)

- **레버리 갤러리:** 발견한 모든 레버리의 시각적 컬렉션
- **로어 항목:** 읽으면 스토리 조각 해금
- **완료 보상:** 테마별 세트 완료 시 보너스

### 4. 몽상가 진행

- **몽상가 해금:** 프레스티지 및 스토리 마일스톤을 통해
- **몽상가 레벨:** 동일 몽상가로 반복 런 = 더 나은 보상
- **몽상가 보너스:** 각 몽상가는 고유한 패시브 능력을 가집니다

## 수익화 (F2P 옵션)

**사업 모델:** 선택적 IAP가 있는 부분 유료화 (Free-to-play)

### 무료 플레이어 경험

- **풀 게임:** 모든 콘텐츠를 무료로 이용 가능
- **진행:** 느리지만 공정한 진행 속도
- **광고:** 임시 부스트를 위한 선택적 광고 시청

### 수익화 핵심 기둥

#### 1. **시간 가속**
- **방치형 속도 두 배 (1시간):** $0.99
- **즉시 수집:** $0.99 (방치형 보상을 즉시 수집)

#### 2. **카드 팩**
- **스탠다드 팩 (5장):** $1.99
- **프리미엄 팩 (10장, 희귀 1장 보장):** $4.99
- **페이 투 윈 아님:** 모든 카드는 무료로 획득 가능

#### 3. **꾸미기**
- **꿈 테마:** 비주얼 스타일 변경 ($2.99)
- **수집가 스킨:** 아바타 커스터마이징 ($1.99)

#### 4. **배틀 패스**
- **시즌 패스 ($9.99):** 
  - 독점 카드
  - 보너스 화폐
  - 꾸미기 보상
- **무료 트랙:** 모두가 일부 보상 획득

#### 5. **광고 제거**
- **영구:** $4.99 (일회성)

### 수익화 철학

- **가챠 도박 없음:** 카드 팩은 내용물 공개
- **에너지 시스템 없음:** 원하는 만큼 플레이
- **페이월 없음:** 모든 콘텐츠는 무료로 해금 가능
- **관대한 F2P:** 일일 로그인 보상, 이벤트

## 아트 스타일 가이드 (2D)

### 시각적 미학

**스타일:** **손으로 그린 2D 일러스트 + 파티클 효과**

**참고 게임:**
- Gris (색상 팔레트, 미니멀리즘)
- Monument Valley (불가능한 기하학, 색상)
- Florence (감성적인 스토리텔링)
- Hades (스타일리시한 캐릭터, VFX)

### 색상 팔레트

**꿈 테마:**
각 몽상가는 고유한 색상 팔레트를 가집니다:

| 테마 | 주 색상 | 보조 색상 | 강조 색상 |
|-------|---------|-----------|--------|
| 평온 | 부드러운 파랑 | 라벤더 | 흰색 |
| 불안 | 진한 보라 | 회색 | 강렬한 빨강 |
| 향수 | 따뜻한 주황 | 세피아 | 금색 |
| 공포 | 깊은 검정 | 피 같은 빨강 | 옅은 노랑 |
| 기쁨 | 밝은 노랑 | 분홍 | 하늘색 |

### 캐릭터 디자인 (2D)

**꿈 수집가:**
- 미니멀리스트 실루엣 (성별 중립)
- 파티클 궤적을 남기는 흐르는 망토
- 빛나는 눈 (유일한 얼굴 특징)
- 떠다니거나 공중 부양 (걷기 애니메이션 불필요)

**몽상가 (NPC):**
- 스타일화된 초상화로 등장 (전신 아님)
- 왜곡되거나 추상적인 얼굴 특징 (꿈의 논리)
- 배경은 테마를 반영

### 환경 (2D 레이어)

**시차 스크롤링 (Parallax Scrolling):**
- 배경 레이어 (느림)
- 중간 레이어 (중간)
- 전경 레이어 (빠름)
- 파티클 오버레이 (떠다니는 꿈의 정수)

**꿈의 세계 요소:**
- 떠다니는 섬/플랫폼
- 불가능한 건축물 (에셔에서 영감)
- 영묘한 식물
- 기억 조각 (빛나는 구체)

### UI/UX 미학

- **글래스모피즘 (Glassmorphism):** 흐림 효과가 있는 반투명 패널
- **부드러운 그림자:** 강렬하지 않은 깊이
- **발광 효과:** 상호작용 가능한 요소는 빛남
- **부드러운 애니메이션:** 모든 전환은 완화된 움직임

## 이 피벗의 이유

### 2D 모바일 방치형의 장점:

1. **낮은 개발 비용:**
   - 3D 모델링보다 2D 아트가 저렴
   - 더 작은 팀 (1-3명)
   - 빠른 반복 작업

2. **더 큰 시장:**
   - 모바일 게임 시장 > PC 게임 시장 (수익)
   - 방치형 장르는 성공 검증됨 (Egg Inc., AdVenture Capitalist)
   - 캐주얼 유저에게 접근성 높음

3. **잔존율 메카닉:**
   - 일일 로그인 보상
   - 프레스티지 루프 (영원히 플레이)
   - 방치형 진행 (FOMO 없음)

4. **하이브리드 장르 매력:**
   - 방치형 팬 (수동적 게임플레이)
   - 덱빌딩 팬 (Slay the Spire 유저층)
   - 로그라이크 팬 (높은 리플레이성)

### 완화된 위험 요소:

- **퍼즐 피로:** 방치형은 막히는 퍼즐의 좌절감 제거
- **세션 길이:** 모바일 플레이어는 짧은 세션을 선호
- **수익화:** F2P 모델이 프리미엄보다 더 검증됨
