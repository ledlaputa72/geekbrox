## 하이 컨셉

**한 줄 요약:** *당신이 괴물입니다—빙의하고, 변이하고, 포식하며 던전을 헤쳐나가세요.*

**던전 기생충**은 던전 몬스터를 탈취하는 기생 생명체로 플레이하는 다크 2D 모바일 로그라이크 덱빌더입니다. 적과 싸우는 대신, **적이 되세요**—그들의 몸을 조종하고, 능력을 훔치고, 기괴한 변이를 통해 진화하세요.

**핵심 기둥:**
- **기생 빙의:** 전투 중 몸을 바꿔 전술적 우위 확보
- **감염 및 변이:** 적을 감염된 아군으로 전환
- **적응형 덱빌딩:** 어떤 숙주에 빙의하느냐에 따라 카드 변경
- **다크 판타지 미학:** 기괴하고 러브크래프트적 바디 호러

**독특한 판매 포인트:**
1. **숙주 교체 메카닉:** 전투 중 몸 교체 (장르 내 독특)
2. **이중 덱 시스템:** 기생체 코어 + 숙주별 카드
3. **포식 시스템:** 영구 파워를 위해 숙주 소비 (위험/보상)
4. **모바일 최적화:** 5-15분 런, 세로 모드, 한 손 플레이

**장르 융합:**
- **로그라이크 덱빌더** (Slay the Spire, Monster Train)
- **바디 호러 전략** (Carrion, Prototype)
- **다크 판타지** (Darkest Dungeon, Dead Cells)

## 핵심 게임플레이 루프

### 1차 루프 (단일 런: 5-15분)

```
1. 기생체 계통 선택 (시작 로드아웃)
   ↓
2. 시작 숙주 빙의 (약한 몬스터)
   ↓
3. 던전 탐색 (8층, 층당 3가지 선택지)
   ↓
4. 조우 (전투 70%, 이벤트 20%, 상점 10%)
   ↓
5. 전투: 카드 플레이, 적 감염, 숙주 교체
   ↓
6. 승리: 보상 선택 (새 카드, 변이, 골드)
   ↓
7. 결정: 숙주 소비 (즉시 파워) vs 유지 (나중 유연성)
   ↓
8. 보스 전투까지 반복 (8층)
   ↓
9. 보스 전투: 궁극의 빙의 도전
   ↓
10. 런 완료: 메타 진행 잠금 해제
```

### 전투 루프 (조우당: 1-3분)

```
당신 턴:
  5장 드로우 → 카드 플레이 (에너지 소비) → 
  공격/방어/감염 → 
  숙주 교체 (선택, 에너지 1) →
  턴 종료

적 턴:
  적이 활성 숙주 공격 → 
  숙주 무결성 확인 (HP = 0이면 강제 분리) →
  당신 턴

승리:
  패배한 적 소비 또는 보존 →
  보상 선택 →
  계속
```

### 메타 진행 루프

```
런 완료 → 기생체 계통 잠금 해제 → 
변이 트리 잠금 해제 → DNA 포인트 획득 → 
시작 숙주 잠금 해제 → 어센션 레벨
```

## 빙의 및 전투 메카닉

### 1. 빙의 시스템

**핵심 컨셉:**
당신은 기생 생명체입니다. 생존하려면 숙주 몸을 반드시 점령해야 합니다. 모든 숙주 상실 = 게임 오버.

**숙주 슬롯:**
- 시작: 숙주 슬롯 1개
- 잠금 해제: 최대 3개 숙주 슬롯 (메타 업그레이드 통해)
- 활성 숙주: 피해 받음, 카드 제공
- 비활성 숙주: 안전, 행동 불가

**숙주 등급:**

| 등급 | 예시 | HP | 덱 크기 | 감염 확률 % |
|------|----------|-----|-----------|-------------|
| **포더** | 고블린, 쥐, 슬라임 | 30 | 3 카드 | 80% |
| **표준** | 스켈레톤, 오크, 거미 | 60 | 5 카드 | 60% |
| **엘리트** | 미노타우로스, 가고일, 망령 | 100 | 7 카드 | 40% |
| **보스** | 드래곤, 리치, 악마 군주 | 150 | 10 카드 | 20% |

**빙의 메카닉:**

**감염 굴림:**
- 기본 성공 확률은 등급에 따라 다름
- 대상 HP 50% 미만 시 턴당 +15%
- 대상 기절/디버프 시 +10%
- 보스 등급이면 -20%
- 실패: 에너지 1 낭비, 다음 턴 재시도 가능

**숙주 교체:**
- 비용: 에너지 1
- 즉시: 행동 지연 없음
- 제한: 턴당 1회
- 전술적 용도: 피해 회피, 숙주별 능력 발동

### 2. 전투 시스템

**턴 구조:**

**1단계: 드로우 단계**
- 5장 드로우 (기생체 덱 + 활성 숙주 덱에서)
- 3 에너지 획득 (기본) + 숙주 보너스
- 상태 효과 확인 (독, 버프 등)

**2단계: 액션 단계 (당신 턴)**
- 카드 플레이 (무제한, 에너지 소진까지)
- 공격, 방어, 감염, 변이
- 숙주 교체 (에너지 1, 턴당 1회)
- 수동 턴 종료 OR 에너지 없을 때 자동 종료

**3단계: 적 단계**
- 적이 선제권 순서대로 행동 (의도 아이콘 표시)
- 공격은 활성 숙주 대상
- 특수 적 능력 발동

**4단계: 종료 단계**
- 상태 효과 틱 (독, 재생 등)
- 숙주 무결성 확인 (HP ≤ 0이면 강제 분리)
- 새 턴 시작

**분리 상태 (비상):**
- 활성 숙주가 죽으면 기생체 자동 분리
- 기생체 형태: HP 15, 기생체 코어 카드만 플레이 가능
- 2턴 내 새 숙주 감염 필수, 아니면 게임 오버
- 취약: 피해 2배

### 3. 감염 메카닉

**감염 방법:**
1. "감염" 카드 플레이 (대상 HP 감소)
2. 대상 HP 낮을 때 빙의 시도
3. 등급 기반 굴림 (포더 80%, 보스 20%)
4. 성공: 적이 숙주가 되고 HP 완전 회복
5. 실패: 에너지 낭비, 대상 격노 (+25% 피해)

**감염된 적:**
- 영구 아군이 됨 (이 전투에서)
- 당신 턴에 행동
- 덱을 카드 풀에 제공
- 전투 후 영구 파워를 위해 소비 가능

## 덱 시스템

### 이중 덱 메카닉

**1. 기생체 코어 덱 (15장)**
당신의 진정한 정체성. 숙주와 무관하게 항상 접근 가능.

**카테고리:**
- **감염 카드:** 피해 + 감염 확률
- **변이 카드:** 기생체에 영구 버프
- **에너지 카드:** 추가 에너지 생성
- **유틸리티 카드:** 드로우, 힐, 디버프

**예시 카드:**
```
[촉수 타격]
비용: 1 | 8 피해. 킬 시 100% 감염.

[신경 탈취]
비용: 2 | 1턴 동안 적 조종.

[대사 흡수]
비용: 1 | 숙주에서 HP 15 흡수, 에너지 2 획득.
```

**2. 숙주 덱 (숙주당 3-10장)**
빙의한 각 몸은 고유 카드 추가. 해당 숙주 활성 시에만 사용 가능.

**예시: 스켈레톤 숙주**
```
[뼈 창]
비용: 1 | 10 피해.

[죽음의 딸랑이]
비용: 0 | 패시브: 이 숙주 죽을 때 광역 20 피해.

[재조립]
비용: 2 | 이 숙주 HP 30 회복.
```

**예시: 오크 숙주**
```
[잔인한 베기]
비용: 2 | 18 피해. 카드 1장 드로우.

[분노]
비용: 1 | 이번 턴 모든 공격 +5 피해. HP 5 손실.

[전쟁 함성]
비용: 1 | 모든 감염된 아군 이번 턴 +3 피해.
```

### 카드 획득

- **전투 후 보상:** 3장 중 1장 선택
- **상점:** 특정 카드 구매 (골드)
- **이벤트:** 무작위 카드 보상
- **숙주 소비:** 시그니처 카드 1장 영구 획득

### 카드 업그레이드 시스템

- **캠프파이어:** 카드 1장 업그레이드 (3층마다)
- **엘리트 승리:** 희귀 카드 보장
- **숙주 소비:** 모든 숙주 카드 자동 업그레이드

## 진행 시스템

### 1. 런 내 진행

**덱 빌딩:**
- 시작: 기생체 카드 15장 + 숙주 1개 (5장) = 20장
- 끝: 25-30장 (이상적)
- 카드 제거: 상점에서 가능 (골드 비용)

**변이 (임시 버프):**
- 이벤트 및 엘리트 승리에서 획득
- 예시:
  - *강화 외골격:* 모든 숙주 최대 HP +20
  - *빠른 재생:* 턴당 HP 5 회복
  - *다중 감염:* 턴당 감염 시도 2회 가능

**골드 및 상점:**
- 승리로 골드 획득
- 4-5층마다 상점
- 구매: 카드, 물약, 숙주 업그레이드

### 2. 메타 진행 (런 간)

**기생체 계통 (시작 로드아웃):**
8가지 고유 계통 잠금 해제, 각각 다른 시작 덱.

| 계통 | 시작 덱 | 플레이 스타일 |
|--------|---------------|-----------|
| **집단 의식** | 다중 감염 집중 | 약한 숙주 다수 무리 |
| **정점 포식자** | 고피해, 위험 | 강한 숙주 1개 단독 |
| **변신체** | 숙주 교체 보너스 | 유동적, 적응형 |
| **역병 보균자** | 독 및 지속 피해 | 느림, 소모전 |

**DNA 포인트 (화폐):**
- 획득: 런당 50-200 (성능 기반)
- 사용처:
  - 기생체 계통 잠금 해제 (500 DNA)
  - 시작 숙주 잠금 해제 (300 DNA)
  - 영구 업그레이드 (+1 카드 드로우, +1 에너지 등)

**어센션 레벨 (난이도 단계):**
- 20 어센션 레벨
- 각각 수정자 추가 (적 강화, 당신 약화)
- 보상: 더 많은 DNA, 독점 잠금 해제

### 3. 업적 및 도전

- **일일 런:** 고정 시드, 리더보드
- **주간 도전:** 특수 수정자
- **업적:** 50+ 업적 (꾸미기 잠금 해제)

## 모바일 특화 기능

### 1. 터치 제스처

**한 손 플레이 최적화:**
- **탭:** 카드 선택, 카드 플레이
- **드래그:** 카드를 대상 적에게 드래그 (공격)
- **길게 누르기:** 카드 상세 정보 보기
- **스와이프:** 숙주 간 전환 (초상화를 좌/우 스와이프)
- **핀치:** 전장 확대 (선택)

### 2. 세로 모드 UI

**화면 레이아웃:**
```
┌─────────────────┐
│ [에너지] [HP]   │ ← 상단: 자원
├─────────────────┤
│                 │
│  [적 영역]      │ ← 적 (3개 이상이면 가로 스크롤)
│  🦴 👹 🐉        │
│                 │
├─────────────────┤
│  [활성 숙주]    │ ← 현재 몸 (크게)
│      🧟         │
├─────────────────┤
│ [비활성 숙주]   │ ← 스와이프로 전환 (잠금 해제 시)
│   🐀 ⚔️         │
├─────────────────┤
│ [패: 5장 카드]  │ ← 가로 스크롤
│ 🃏🃏🃏🃏🃏       │
│ [턴 종료]       │ ← 큰 버튼
└─────────────────┘
```

### 3. 세션 유연성

**빠른 런 모드:**
- 8층 → 5층 (6-10분 런)
- 카드 풀 축소 (쉬운 결정)
- 통근 게임용

**표준 런 모드:**
- 8층 (10-15분 런)
- 완전한 경험

**무한 모드 (5승 후 잠금 해제):**
- 무한 층, 난이도 증가
- 리더보드

### 4. 클라우드 저장 및 크로스플레이

- Google Play / iCloud 통해 동기화
- 휴대폰에서 플레이, 태블릿/PC에서 계속
- 진행 손실 없음

## 아트 스타일 가이드 (2D)

### 시각적 미학

**스타일:** **다크 판타지 손그림 2D + 픽셀 아트 하이브리드**

**참고 게임:**
- Darkest Dungeon (고딕, 회화적)
- Dead Cells (유동적 애니메이션, 픽셀 아트)
- Slay the Spire (깔끔한 UI, 읽기 쉬운 카드)
- Inscryption (으스스한, 손그림 카드)

### 색상 팔레트

**핵심 팔레트:**
- **주 색상:** 진한 진홍색 (피), 뼈 흰색, 그림자 검정
- **보조 색상:** 부패한 초록 (감염), 멍든 보라, 녹슨 갈색
- **강조 색상:** 빛나는 청록 (기생체 에너지), 병든 노랑 (약점)

**분위기:** 억압적, 폐쇄공포증, 내장적

### 캐릭터 디자인 (2D)

**기생체 (플레이어):**
- 무정형 덩어리 + 촉수
- 빛나는 핵 (청록/청록)
- 맥동하는 유기적 애니메이션
- 얼굴 없음, 순수 바디 호러

**숙주 (몬스터):**
- 스타일화, 과장된 비율
- 빙의 시 감염 정맥 (빛나는 청록) 보임
- 각 등급마다 뚜렷한 실루엣
- 포더: 작고 불쌍함
- 엘리트: 위압적, 상세함
- 보스: 화면 가득, 다단계

**감염 상태:**
- 빛나는 눈 (청록)
- 피부 아래 보이는 정맥/촉수
- 경련하는 부자연스러운 움직임
- 입에서 기생체 잔류물 흘림

### 환경 (2D 레이어)

**던전 테마:**
1. **지하묘지:** 뼈 더미, 거미줄, 횃불
2. **하수도:** 슬라임, 떨어지는 물, 쥐
3. **지옥:** 불, 유황, 쇠사슬

**시차 레이어:**
- 배경: 석조 벽, 주변 불길
- 중간: 플랫폼, 장애물
- 전경: 안개, 파티클 효과

### 카드 아트

- **일러스트 스타일:** 손그림, 기괴한 디테일
- **프레임:** 유기적, 뼈 같은 테두리
- **에너지 비용:** 빛나는 숫자 (좌상단)
- **카드 희귀도:** 
  - 일반: 회색 테두리
  - 언커먼: 초록 테두리
  - 레어: 보라 테두리
  - 전설: 금색 테두리

## UI/UX 디자인 (모바일)

### 디자인 원칙

1. **가독성 우선:** 큰 텍스트, 높은 대비
2. **한 손 플레이:** 엄지 도달 범위 내 모든 버튼
3. **빠른 결정:** 드래그 대상 자동 확대
4. **최소 혼란:** 탭할 때까지 필수 아닌 정보 숨김

### 핵심 화면

#### 1. 메인 메뉴
```
┌─────────────────┐
│  DUNGEON        │ ← 제목
│  PARASITE       │
├─────────────────┤
│ [런 계속]       │ ← 진행 중 런이 있으면
│ [새 런]         │
│ [일일 런]       │
│ [컬렉션]        │ ← 잠금 해제 카드/숙주 보기
│ [업그레이드]    │ ← DNA 포인트 사용
│ [설정]          │
└─────────────────┘
```

#### 2. 런 시작 전 설정
```
┌─────────────────┐
│ 계통 선택       │
├─────────────────┤
│ [집단 의식] 🦠  │ ← 스와이프로 탐색
│ 다중 감염       │
│ 집중            │
├─────────────────┤
│ [런 시작]       │
│ [어센션: 3]     │ ← 난이도 토글
└─────────────────┘
```

#### 3. 전투 (주 화면)
위 "세로 모드 UI" 섹션 참조.

**전투 피드백:**
- **피해 숫자:** 대상에서 위로 날아감
- **힐:** 초록 +HP 텍스트
- **에너지 비용:** 카드 플레이 시 간략히 빛남
- **숙주 교체:** 화면 깜박임 (빠른 전환)

#### 4. 보상 화면
```
┌─────────────────┐
│ 승리!           │
├─────────────────┤
│ 보상 1개 선택:  │
│                 │
│ [카드 1]        │ ← 탭하여 상세 보기
│ [카드 2]        │
│ [카드 3]        │
├─────────────────┤
│ 패배한 숙주:    │
│ 🦴 스켈레톤     │
│ [소비] [유지]   │ ← 중요한 결정
└─────────────────┘
```

**소비 vs 유지:**
- **소비:** 시그니처 카드 1장 영구 획득, 숙주 영원히 잃음
- **유지:** 숙주가 풀에 남음 (향후 전투에 사용 가능)

### 접근성 기능

- **색맹 모드:** 대체 색 구성
- **텍스트 크기:** 작게 / 중간 / 크게
- **스크린 리더 지원:** 모든 UI 요소 레이블
- **햅틱 피드백:** 켜기/끄기 토글
- **배터리 절약 모드:** 애니메이션 감소

   - 2D 아트가 3D보다 저렴

   - 빠른 반복

- **화면 크기:** 세로 모드 + 큰 UI 요소
