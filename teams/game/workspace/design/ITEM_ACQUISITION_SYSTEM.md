## 개요

### 목적

Dream Collector에서 플레이어가 아이템을 획득하는 모든 경로를 체계적으로 관리하여:
- 게임 진행도에 따른 자연스러운 파워 상승
- 플레이어 성취감과 동기부여
- 운영상 수익성 확보 (프리미엄 아이템 판매)

### 획득 경로 다이어그램

```
플레이어
  ├─ 게임 플레이
  │  ├─ 전투 보상
  │  ├─ 스테이지 클리어 보상
  │  ├─ 일일/주간 퀘스트
  │  └─ 이벤트 보상
  │
  ├─ 골드 획득
  │  └─ 상인 → 아이템 구매
  │
  ├─ 보석 획득
  │  └─ 프리미엄 뽑기 → 아이템
  │
  ├─ 운영 활동
  │  ├─ 로그인 보상
  │  ├─ 신입 환영 패키지
  │  ├─ 시즌 패스
  │  └─ 콜라보 이벤트
  │
  └─ 기타
     ├─ 우편함 (선물)
     ├─ 길드 (길드 숍)
     └─ 달성 목표 (마일스톤)
```

## 획득 방법 분류

### 분류 체계

| 획득 방법 | 주요 화폐 | 획득 빈도 | 등급 범위 | 동기부여 |
|---------|---------|--------|---------|--------|
| **게임 보상** | 골드 | 자주 | Common~Rare | 즉시 |
| **상인** | 골드 | 선택적 | Common~Epic | 전략적 |
| **골드 뽑기** | 골드 | 자주 | Common~Uncommon | 저가 |
| **보석 뽑기** | 보석 | 드물게 | Uncommon~Legendary | 고가 |
| **운영 보상** | 보석/골드 | 주기적 | Common~Legendary | 장기 |
| **기타** | 다양 | 특수 | 전범위 | 특수 |

## 게임 보상

### 1️⃣ 전투 보상

**획득 방법:** 몬스터 처치 시 자동 지급

#### 보상 설계

```
기본 보상:
├─ 골드: 몬스터 레벨 × 10 ~ 50 (변동)
├─ 경험치: 몬스터 레벨 × 5
├─ 아이템: 15% ~ 60% 확률

아이템 드롭 확률:
├─ Common: 60% (높음)
├─ Uncommon: 25%
├─ Rare: 12%
├─ Epic: 2.5%
└─ Legendary: 0.5%

레벨별 드롭:
├─ 1-5레벨 몬스터: Common만 드롭
├─ 6-10레벨 몬스터: Common + Uncommon
├─ 11-20레벨 몬스터: Common + Uncommon + Rare
├─ 21-35레벨 몬스터: 모든 등급 (Rare 증가)
└─ 36+레벨 몬스터: 모든 등급 (Epic 증가)
```

#### 보상 테이블 예시

```json
{
  "monsters": [
    {
      "monsterId": "slime_lv1",
      "monsterName": "스라임",
      "level": 1,
      "baseGold": 10,
      "goldVariation": [5, 20],
      "baseEXP": 5,
      "itemDropTable": [
        { "itemId": "weapon_crystal_sword_common_lv1", "probability": 15 },
        { "itemId": "armor_leather_common_lv1", "probability": 15 },
        { "itemId": "ring_stat_common_lv1", "probability": 20 },
        { "itemId": "necklace_attack_common_lv1", "probability": 15 }
      ]
    }
  ]
}
```

### 2️⃣ 스테이지 클리어 보상

**획득 방법:** 스테이지 완료 시 보상

#### 설계

```
스테이지별 보상:
├─ 1-5스테이지: 보상 1개 (Common)
├─ 6-15스테이지: 보상 2개 (Common + Uncommon)
├─ 16-30스테이지: 보상 2-3개 (Uncommon + Rare)
├─ 31-50스테이지: 보상 3개 (Rare + Epic)
└─ 51+스테이지: 보상 3-4개 (Epic + Legendary)

클리어 난이도 보너스:
├─ Normal (기본): 1배 보상
├─ Hard: 1.5배 보상
└─ Nightmare: 2배 보상

첫 클리어 보너스:
├─ 첫 완료: 추가 골드 100%
├─ 별 완료: 추가 아이템 1개
```

### 3️⃣ 일일/주간 퀘스트

**획득 방법:** 퀘스트 완료 시 지급

#### 설계

```
일일 퀘스트 (5개):
├─ "몬스터 10마리 처치" → Common 아이템 1 + 골드 50
├─ "스테이지 2개 클리어" → Uncommon 아이템 1 + 골드 100
├─ "전투 3회" → 아이템 선택 상자 1개
├─ "꿈 수집 1000" → 아이템 보상
└─ "상점 방문" → 골드 50

주간 퀘스트 (3개):
├─ "총 50마리 처치" → Rare 아이템 1 + 골드 300
├─ "15개 스테이지 클리어" → 아이템 선택 상자 2개
└─ "일일 퀘스트 5회 완료" → Epic 아이템 1개

완료 보너스:
├─ 일일 퀘스트 5개 완료: 보석 10개
└─ 주간 퀘스트 3개 완료: 보석 50개 + Epic 아이템
```

### 4️⃣ 이벤트 보상

**획득 방법:** 한정 이벤트 참여 시 지급

#### 설계

```
시즌 이벤트 (월 1회, 2주간):
├─ 입장: 무료
├─ 보상: Common~Rare 아이템 + 골드
├─ 최종 보상: Rare 아이템 1개 보장

콜라보 이벤트 (분기 1회):
├─ 제한 시간: 2주
├─ 보상: 콜라보 한정 아이템 (Uncommon~Legendary)
├─ 진입 가능 재화: 무료 또는 보석

생일 이벤트:
├─ 플레이어 생일: 특별 패키지 (아이템 3개 선택)
└─ 게임 생일: 전체 플레이어 (아이템 1개 + 보석 20개)
```

## 게임내 상인

### 1️⃣ 상인 시스템

**목적:** 플레이어의 의도적 선택을 통한 아이템 획득

#### 상인 타입

```
A. 일반 상인 (마을)
├─ 판매 가능 아이템: Common~Uncommon
├─ 재고: 5개 (일일 갱신)
├─ 가격: 정가 (변동 없음)
├─ 특징: 항상 접근 가능

B. 희귀 상인 (던전 보스 처치 후 등장)
├─ 판매 가능 아이템: Rare~Epic
├─ 재고: 2-3개 (주간 갱신)
├─ 가격: 정가 + 20%
├─ 특징: 제한적 접근

C. 흑시장 상인 (숨겨진 지역)
├─ 판매 가능 아이템: Epic~Legendary
├─ 재고: 1개 (월간 갱신)
├─ 가격: 정가 + 50%
├─ 특징: 매우 제한적 (찾기 어려움)

D. 길드 상인 (길드 소속 시)
├─ 판매 가능 아이템: Common~Rare
├─ 재고: 10개 (주간 갱신)
├─ 가격: 정가 - 10% (길드 할인)
├─ 특징: 길드 길드 포인트 필요
```

#### 가격 정책

```
아이템 가격 = (기본 가격) × (등급 계수) × (수요도 계수)

기본 가격 = 아이템 능력치 × 500 골드
등급 계수:
├─ Common: 1.0
├─ Uncommon: 1.5
├─ Rare: 2.5
├─ Epic: 4.0
└─ Legendary: 8.0

수요도 계수 (게임 시즌에 따라 조정):
├─ 저수요: 0.8
├─ 중수요: 1.0
└─ 고수요: 1.2

예시:
"불꽃 검" (Uncommon)
= (기본 능력치 24 × 500) × 1.5 × 1.0
= 12,000 × 1.5
= 18,000 골드
```

## 뽑기 시스템

### 1️⃣ 골드 뽑기 (가우차)

**특징:** 저비용, 자주 사용, 기본 아이템 중심

#### 설계

```
뽑기 가격: 500 골드 (1회)
할인 패키지:
├─ 10회: 4,500 골드 (10% 할인)
├─ 50회: 20,000 골드 (20% 할인)
└─ 100회: 35,000 골드 (30% 할인)

아이템 확률 (1회):
├─ Common: 60%
├─ Uncommon: 35%
├─ Rare: 4.5%
└─ Epic: 0.5%

보증 시스템:
├─ 50회 뽑기 없이 Rare 미획득 → 51회차 Rare 보증
├─ Epic: 보증 없음 (매우 낮은 확률)

타입별 균형:
├─ 무기: 25%
├─ 방어구: 25%
├─ 반지: 25%
└─ 목걸이: 25%
```

### 2️⃣ 보석 뽑기 (프리미엄)

**특징:** 고비용, 드물게 사용, 강력한 아이템 중심

#### 설계

```
뽑기 가격: 50 보석 (1회) [약 2,000원 환율]
할인 패키지:
├─ 10회: 450 보석 (10% 할인)
├─ 50회: 2,000 보석 (20% 할인)
└─ 100회: 3,500 보석 (30% 할인)

아이템 확률 (1회):
├─ Uncommon: 30%
├─ Rare: 55%
├─ Epic: 13%
└─ Legendary: 2%

천장 시스템 (Pity):
├─ Epic: 90회 뽑기 없이 미획득 → 91회차 Epic 보증
├─ Legendary: 180회 뽑기 없이 미획득 → 181회차 Legendary 보증

기간 한정 픽업:
├─ 신규 아이템 2주간 픽업 (확률 2배)
├─ 콜라보 아이템 픽업 (한정)
```

#### 매월 로테이션

```
1주차: 신규 아이템 픽업 (Epic/Legendary 확률 UP)
2주차: 시즌 아이템 픽업 (Rare 확률 UP)
3주차: 추천 아이템 (무작위 조합)
4주차: 풀 아이템 (모든 아이템 등확률)
```

## 기타 지급 방법

### 1️⃣ 로그인 보상

**획득 방법:** 매일 게임 접속 시 자동 지급

```
일일 로그인:
├─ 1일: 골드 100
├─ 3일: Common 아이템 1
├─ 7일: Uncommon 아이템 1 + 보석 10
├─ 14일: Rare 아이템 1 + 보석 30
└─ 30일: Epic 아이템 1 + 보석 100

월간 누적 (31일):
├─ 보상 내용: 아이템 5개 (Common~Rare) + 보석 200
├─ 리셋: 매월 1일
```

### 2️⃣ 신입 환영 패키지 (온보딩)

**획득 방법:** 게임 시작 후 특정 단계에서 지급

```
1단계 (3레벨): "시작 선물"
├─ Common 무기 1 (고정)
├─ Common 방어구 1 (고정)
└─ 골드 500

2단계 (10레벨): "초급 선물"
├─ Uncommon 아이템 2개 선택 (무기/방어구)
├─ 골드 2,000
└─ 보석 50

3단계 (20레벨): "중급 선물"
├─ Rare 아이템 1개 선택 (4가지)
├─ 골드 5,000
└─ 보석 100
```

### 3️⃣ 시즌 패스

**획득 방법:** 유료 구독 (월간 5,000원 추천)

```
프리 시즌 패스:
├─ 50단계, 무료 진행
├─ 보상: 골드 1,000 + 공통 아이템 10개

프리미엄 시즌 패스:
├─ 50단계 추가 보상 해금
├─ 보상: 보석 500 + 한정 아이템 10개 (Epic 3개 포함)
├─ 월간 비용: 4,900 보석 (약 5,000원)

시즌 기간:
├─ 1시즌: 60일
├─ 매시즌 신규 아이템 기획
```

### 4️⃣ 우편함 (선물)

**획득 방법:** 운영팀에서 직접 지급 (이벤트, 사과 등)

```
우편함 내용물 (정책에 따라):
├─ 보상 회수 실패: 사과 아이템 + 보석 50
├─ 업데이트 이벤트: 골드 1,000 + 아이템 1
├─ 플레이어 감사: 보석 100 + 한정 아이템
├─ 버그 수정: 보상 재지급 + 추가 보석
```

### 5️⃣ 길드 숍

**획득 방법:** 길드 포인트 사용 구매 (길드 레벨과 무관)

```
길드 포인트 획득:
├─ 일일 로그인: 10포인트
├─ 길드 퀘스트 완료: 20포인트
├─ 길드 기여: 50포인트 (한계 없음)

길드 숍 라인업:
├─ Uncommon 아이템: 300포인트
├─ Rare 아이템: 1,000포인트
├─ 특수 아이템: 2,000포인트 (한정)
```

### 6️⃣ 달성 목표 (마일스톤)

**획득 방법:** 게임 진행도 달성 시 수여

```
마일스톤 보상:
├─ "10레벨 달성": Uncommon 아이템 1
├─ "50레벨 달성": Rare 아이템 1 + 보석 50
├─ "100레벨 달성": Epic 아이템 1 + 보석 200
├─ "스테이지 50 클리어": Rare 아이템 2 선택
├─ "총 150시간 플레이": Legendary 아이템 1개 선택 (3개 중)
```

## 등급별 획득 확률

### 전체 획득 경로별 확률

| 획득 방법 | Common | Uncommon | Rare | Epic | Legendary |
|---------|--------|----------|------|------|-----------|
| 몬스터 드롭 | 60% | 25% | 12% | 2.5% | 0.5% |
| 골드 뽑기 | 60% | 35% | 4.5% | 0.5% | - |
| 보석 뽑기 | - | 30% | 55% | 13% | 2% |
| 로그인 (7일) | 50% | 50% | - | - | - |
| 시즌 패스 | 40% | 30% | 20% | 10% | - |
| 길드 숍 | - | 60% | 30% | 10% | - |
| 마일스톤 | - | 25% | 50% | 20% | 5% |

### 월간 예상 획득량

**일반 플레이어 (일일 1시간):**
```
Common:    50-80개
Uncommon:  20-40개
Rare:      5-15개
Epic:      0-2개
Legendary: 0개 (보석 뽑기 필요)
```

**적극적 플레이어 (일일 3시간 + 보석 구매):**
```
Common:    100-150개
Uncommon:  60-100개
Rare:      30-50개
Epic:      10-20개
Legendary: 2-5개 (보석 기반)
```

## 보상 밸런싱

### 게임 밸런스 원칙

```
1. 자연스러운 파워 상승
   └─ 초반: Common 중심
      중반: Uncommon + Rare
      후반: Rare + Epic
      극후반: Epic + Legendary

2. 성취감 제공
   └─ 즉시 보상 (몬스터 드롭)
      단기 보상 (일일 퀘스트)
      중기 보상 (시즌 패스)
      장기 보상 (마일스톤, 길드)

3. 수익화 전략
   └─ 프리 플레이어: 월 Rare 10~15개
      가벼운 과금: 월 Rare 30~40개 + Epic 5개
      헤비 과금: 월 Epic 30개 + Legendary 10개
```

### 월별 보상 예산

```
"좋은 운영 = 플레이어 만족도 + 운영 수익"

목표:
├─ 월간 신규 플레이어: 1,000명
```
