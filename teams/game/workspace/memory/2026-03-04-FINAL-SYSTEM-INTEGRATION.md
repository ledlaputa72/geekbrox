```

희귀도:
  COMMON (55% 또는 45%)
  RARE (30% 또는 35%)
  SPECIAL (13% 또는 15%)
  LEGENDARY (2% 또는 5%)

보장:
  50회: Rare 이상 1개
  100회: Special 이상 1개

```

```
RARE = 3 unit (강화 +3)
SPECIAL = 5 unit (강화 +5)
LEGENDARY = 10 unit (강화 +10)

낮은 희귀도 아이템 = 강화 재료로 활용!
```
