```

  ✅ 스탯: 비용, 데미지, 효과, 태그 모두 정의
  ✅ 밸런스: Grade A+ (95/100)
  ✅ 확장 계획: 월별 신규 카드 10장 (30개월)

장비 시스템:
  ✅ 66종 장비 (4슬롯 × 다양한 타입)
  ✅ 강화: +0~+10 (지수적 비용, 위험도 있음)
  ✅ 특성: 5가지 마스터리 (무기/방어/악세서리/듀얼/연쇄)
  ✅ 세트: 4가지 + 자유 조합 (시너지)
  ✅ 밸런스: Grade B+ (87/100), 개선 방안 8가지
  ✅ 시뮬레이션: 10,000회 게임 회차

기본 공격 (ATB):
  ✅ 데미지 공식: 기본 × 마스터리% × 시너지% × 세트%
  ✅ 방어 공식: DEF - 관통력 × (1 - 저항%)
  ✅ 부가 효과: 흡혈, 반사, 버스트, 연쇄 등

성장 시스템:
  ✅ 레벨: 1-100 (비선형 곡선)
  ✅ 경험치: 100 × Lv^1.5 (누적 1M EXP)

  ✅ 성장률: ATK +567%, DEF +900%, HP +200%, SPD +317%
```
