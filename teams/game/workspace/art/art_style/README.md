# 🎨 Dream Weave - 아트 스타일 가이드 및 레퍼런스 보드 v1.0

## 1. 핵심 방향성: "정교한 동화 (Sophisticated Fairy Tale)"
- **캐릭터**: 3~4등신 SD, 귀여움, 신비로움
- **배경**: 몽환적, 서정적, 수채화/파스텔 톤
- **UI & FX**: 미니멀, 직관적, 부드러운 애니메이션
- **무드**: 따뜻함, 평온함, 치유(Healing)
