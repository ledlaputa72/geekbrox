# 아트 스타일 가이드 (STYLE_GUIDE)

> 서브에이전트는 이 파일을 읽고 모든 아트 작업의 일관성을 유지한다.
> 스타일 확정 후 이 파일을 업데이트하면, 이후 모든 아트 서브에이전트가 자동으로 일관된 스타일로 작업한다.

## 상태
**⏳ 컨셉 확정 후 아트 방향 결정 예정**

---

## 방향성 후보 (컨셉 확정 후 선택)

### 픽셀 아트 스타일
- 툴: Aseprite
- 해상도: 16x16 ~ 32x32 타일
- 팔레트: 제한 팔레트 (64색 이내)
- 장점: 소규모 팀 적합, 복고풍 감성

### 2D 일러스트 스타일
- 툴: Procreate / Clip Studio Paint
- 특징: 손그림 느낌
- 장점: 감성적 차별화

### 미니멀 벡터 스타일
- 툴: Figma / Illustrator
- 특징: 깔끔한 아이콘 중심
- 장점: 개발 속도 빠름, 모바일 최적화
