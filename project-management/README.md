## 📊 관리 방법론

### 스프린트 사이클
- **주간 스프린트** (월요일 시작)
- 매주 금요일: 스프린트 리뷰 + 다음 주 계획
- 매월 말: 월간 회고 및 로드맵 조정

### 우선순위 시스템
- 🔴 **P0 (Critical):** 즉시 처리 필요
- 🟠 **P1 (High):** 이번 주 완료
- 🟡 **P2 (Medium):** 이번 달 완료
- 🟢 **P3 (Low):** 백로그

### 상태 라벨
- ⏳ **TODO:** 시작 전
- 🚧 **IN PROGRESS:** 진행 중
- ⏸️ **BLOCKED:** 차단됨
- ✅ **DONE:** 완료
- ❌ **CANCELLED:** 취소됨
