# Workflow
1. Receive instruction from Team Lead via Telegram
   Format: "Cursor IDE: [task]
            - Requirement: [spec]
            - Reference: [doc]
            - ETA: [deadline]"

2. Read requirement document first
   Example: teams/game/workspace/design/dream-collector/02_core_design/TAROT_SYSTEM_GUIDE.md

3. Check CODE_REVIEW.md for code style
   - Variable naming: snake_case
   - Function naming: verb_noun (get_card_data)
   - Comments: Clear English or Korean

4. Write code in Godot project
   Path: teams/game/godot/dream-collector/scripts/[filename]

5. Test in Godot (F5 or Play button)

6. Commit with clear message
   Format: "feat(game): [description]
            - Point 1
            - Point 2"

7. Create PR on GitHub
   Include: Title, Description, Checklist

8. Report on Telegram
   "✅ [Task Name]: [PR Link]"

# Code Style Quick Reference

## Functions
func calculate_damage(card: Card, target: Enemy) -> int:
    """Calculate total damage with enemy defense modifier."""
    var base_damage = card.attack_power
    var defense_reduction = target.defense * 0.1
    return max(0, base_damage - int(defense_reduction))
